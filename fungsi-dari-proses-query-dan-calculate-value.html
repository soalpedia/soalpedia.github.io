23. A finance company offers compound interest to all its investors. Calculate : Investment value = RM 52 500, Matured value =RM 84 551.78, Time = 5 years, interest is compounded every year. Calculate the interest rate.​


Jawaban:

10%

Penjelasan dengan langkah-langkah:

the general formula:

V = P* (1+i) ^T

84,551.78 = 52,500 (1+i) ^5

(1+i) ^5 = 84,551.78 / 52,500

= 1.6105101

1+i = (1.6105101)^(1/5)

= 1.10

i = 1.10 - 1 = 0.10

so, the interest rate per year = 0.10 = 0.10 × 100%

= 10%

Jawaban:

A = P (1 + i)ⁿ

A = final amount = 84,551.78

P = intial principal balance = 52,500

i = interest rate = ?

n = number of time period = 5 years

84,551.78 = 52,500 (1 + i)⁵

(1 + i)⁵ = 84,551.78 / 52.500

(1 + i)⁵ = 1.6105100952

1 + i = 1.10

i = 0.10

i = 10%

so, the interest rate = 10%

27. Jika dipanggil fungsi calculate(2903), apa outputnya?


Suatu fungsi bernama calculate(2903) akan berfungsi sebagaimana method yang diberikan kepadanya ketika mendefinisikan fungsi tersebut. Jika fungsi tersebut isinya adalah method operasi matematika penjumlahan misalnya maka argument yang ada dalam kurung dalam fungsi di atas akan diproses dengan operasi matematika.

Pembahasan

Misalnya dalam Python versi 3 kita membuat sebuah fungsi seperti ini:

def calculate(the_number):

   return (the_number*10) + 200

Dalam fungsi yang dibuat di atas dengan perintah def kita umpankan argument yang ada dalam kurung ke dalam sebuah proses yang kita buat dalam fungsi, yaitu (the_number*10) + 200. Bila kita berikan the_number adalah 2903 maka akan memberikan hasil 29230. Hal ini karena angka yang diberikan tersebut akan dikalikan dengan 10 lalu ditambah 200 sehingga menjadi 29230.

37. Help me to do this. Calculate the value of h.Thank you​


Answer:

Total height = 27 cmPyramid height (h) = 9 cm; prism height = 18 cm

Explanation with steps:

Cross section area = luas alas = 325 cm²

Volume = 6,825 cm³

Height of pyramid = h; height of prism with triangular cross section = 2h

Therefore the equation to find

pyramid volume = (cross section area x height)/3= 325h/3

Prism with triangle cross sec. = cross sec. area x height = 325 x 2h = 650h

Pyramid vol + prism vol = Total vol.

(325h/3) + 650h = 6,825

2,275h/3 = 6,825

2,275 h = 6,825 x 3

2,275 h = 20,475

h = 20,475 / 2,275

h = 9 cm

(height of pyramid = 9 cm)

2h = 9 x 2 = 18 cm (height of prism = 18 cm)

Total height = 27 cm
