1. domain dan range fungsi dari fungsi rasional (pecahan)​


Penjelasan dengan langkah-langkah:

Dalam matematika, fungsi rasional adalah fungsi yang dinyatakan sebagai pecahan polinomial, di mana pembilang dan penyebutnya adalah polinomial. Untuk fungsi rasional f(x), domain dan range-nya dapat dijelaskan sebagai berikut:

1. Domain (daerah asal):

Domain dari fungsi rasional adalah himpunan semua nilai x yang dapat digunakan sebagai input dalam fungsi tersebut tanpa menyebabkan pembagian dengan nol dalam penyebut. Jadi, domain f(x) adalah semua nilai x yang membuat penyebutnya bukan nol.

2. Range (daerah hasil):

Range dari fungsi rasional tergantung pada bentuk fungsi dan batasan-batasan yang mungkin ada. Range adalah himpunan semua nilai y yang dapat dihasilkan oleh fungsi tersebut untuk nilai x dalam domainnya. Range bisa beragam tergantung pada karakteristik fungsi rasional tersebut.

Sebagai contoh, jika kita memiliki fungsi rasional f(x) = (3x + 2) / (x - 1), maka domain f(x) adalah semua nilai x kecuali x = 1 (karena itu akan membuat penyebutnya nol). Range f(x) akan bergantung pada cara grafik fungsi ini berperilaku dan apakah ada batasan lain pada variabel x. Range biasanya dapat ditentukan dengan mengamati grafik fungsi atau dengan menggunakan analisis matematis lebih lanjut.

10. domain dari fungsi rasional f(x)=x-3/2x-8​


Domain atau daerah asal dari f(x) = [tex]\frac{x - 3}{2x - 8}[/tex] adalah Df = {x | x ≠ 4, x ∈ R}.

Untuk menentukan daerah asal suatu fungsi, bergantung dari bentuknya, yaitu

Jika y = [tex]\frac{f(x)}{g(x)}[/tex] maka domainnya adalah g(x) ≠ 0.Jika y = [tex]\sqrt{f(x)}[/tex] maka domainnya adalah f(x) ≥ 0.   Jika y = [tex]\frac{f(x)}{\sqrt{g(x)}}[/tex] maka domainnya adalah g(x) > 0.Jika y = [tex]^{f(x)} log \: g(x)[/tex] maka domainnya adalah f(x) ≠ 1, f(x) > 0 dan g(x) > 0.

Penjelasan dengan langkah-langkah

Diketahui

f(x) = [tex]\frac{x - 3}{2x - 8}[/tex]

Ditanyakan

Tentukan domain dari fungsi tersebut!

Jawab

Langkah 1

Fungsi rasional (bentuk pecahan) akan terdefinisi jika penyebutnya tidak sama dengan nol.

Langkah 2

f(x) = [tex]\frac{x - 3}{2x - 8}[/tex]

pembilang = x – 3penyebut = 2x – 8

f(x) akan terdefinisi jika (2x – 8) ≠ 0

Langkah 3

Jadi berdasarkan langkah 2, domain dari f(x) adalah

2x – 8 ≠ 0

     2x ≠ 8

        x ≠ [tex]\frac{8}{2}[/tex]

        x ≠ 4

Daerah asal atau domain dari f(x) = [tex]\frac{x - 3}{2x - 8}[/tex] adalah

Df = {x | x ≠ 4, x ∈ R}

19. tentukan domain untuk fungsi fungsi berikut​


Jika diketahui fungsi  [tex]f(x) = \frac{x^2}{x - 1}[/tex] , maka domain fungsi tersebut adalah x ≠ 1.

Pembahasan

Suatu fungsi dapat dinyatakan dalam bentuk: f : x ---> f(x). Nilai fungsi untuk setiap nilai x yang diberikan dapat dihitung dengan cara mensubstitusikan nilai x ke dalam rumus fungsi f(x).

Domain (daerah asal) fungsi adalah himpunan nilai-nilai agar suatu fungsi terdefinisi.

Daerah hasil fungsi (range) adalah himpunan nilai-nilai fungsi dari setiap anggota daerah asal (domain).

Penyelesaian

diket:

[tex]f(x) = \frac{x^2}{x - 1}[/tex]

ditanya:

domain....?

jawab:

karena fungsi [tex]f(x) = \frac{x^2}{x - 1}[/tex]  berbentuk pecahan a/b, maka fungsi f(x) agar terdefinisi adalah b ≠ 0, sehingga

x - 1 ≠ 0

x ≠ 1

Kesimpulan

Jadi, domain untuk fungsi [tex]f(x) = \frac{x^2}{x - 1}[/tex] adalah x ≠ 1.

36. contoh soal fungsi rasional beserta domain dan rangenya


Contoh soal fungsi rasional:

f(x) = 1/x

Tentukanlah daerah asal (domain) dan daerah hasil (range) fungsi di atas!

Penyelesaian:

Karena penyebut tidak terdefinisi pada saat x = 0, maka domain fungsi tersebut adalah x ≠ 0, atau biasa ditulis sebagai berikut.

Df = { x | x ≠ 0, x ∈ bilangan riil}

f(x) = 1/x

y = 1/x

x = 1/y

Karena penyebut tidak terdefinisi pada saat y = 0, maka range fungsi tersebut adalah y ≠ 0, atau biasa ditulis sebagai berikut.

Rf = { y | y ≠ 0, y ∈ bilangan riil}
