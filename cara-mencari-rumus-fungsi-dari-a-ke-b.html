38. Untuk menentukan persamaan garis dalam bentuk ax + by = c dari grafik garis, kita perlu menentukan nilai a, b, dan c terlebih dahulu. Nilai a adalah koefisien x, b adalah koefisien y, dan c adalah nilai konstanta. Untuk menentukan nilai a dan b, dapat digunakan dua titik pada garis tersebut. Setelah nilai a dan b ditemukan, nilai c dapat dicari dengan mengganti nilai x dan y dengan koordinat salah satu titik yang sudah diketahui. Sebagai contoh, jika diperoleh nilai a = 2 dan b = 3 dari dua titik pada garis, dan salah satu titik yang diketahui koordinatnya adalah (4, 5), maka nilai c dapat dicari dengan mengganti nilai x dan y dengan 4 dan 5 pada persamaan ax + by = c, sehingga menjadi:

2(4) + 3(5) = c

8 + 15 = c

c = 23

Sehingga persamaan garis tersebut adalah 2x + 3y = 23.
