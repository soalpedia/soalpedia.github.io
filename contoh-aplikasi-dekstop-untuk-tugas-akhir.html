1. PROGAM UNTUK MEMBUAT APLIKASI DEKSTOP


1. TSPlus

￼TSPlus

Aplikasi TSPlus adalah sebuah aplikasi dimana kamu bisa menggunakannya untuk membangun sebuah sistem Client Server dimana aplikasi yang kamu buat ada di Server, sebetulnya TSPlus punya cara kerja yang sama seperti Remote Desktop nya windows, namun dengan TSPlus kamu dapat mendifinisikan lebih mendetail mengenai User, Pemilihan Printer, kamu juga bisa membuat executable filenya dengan menggunakan TSPlus App Generator. Oh iya dengan TSPlus ini juga kamu bisa memilih Resolusi Aplikasi yang digunakan jika Bandwidth yang tersedia kecil pada client. Untuk mencoba software ini, kamu dikenakan biaya ya sob, alias software ini Berbayar, tapi kamu bisa coba kok versi trial nya disini.

2. 2X Application

￼2X Application Server

Tidak seperti TSPlus, 2X Application menggunakan port tersendiri untuk komunikasi antar client dan servernya,dengan menggunakan 2X Application kamu dapat meng hosting lebih dari 2 aplikasi pada sisi server, sehingga kamu bisa lebih leluasa dalam mempublish kan Aplikasi yang kamu buat. Berbeda dengan TSPlus, 2X Application memerlukan 2X Application client untuk dapat mengakses Aplikasi yang ada pada server. Untuk mencoba aplikasi ini kamu bisa dapatkan versi trial nya disini.

3. Winflector

￼Winflector

Software WinFlector adalah software yang boleh dibilang paling ringan diantara kedua software diatas, WinFlector bekerja seperti 2X Application yang membutuhkan Software yang terinstall pada sisi server, dan juga pada sisi Client untuk dapat mengakses aplikasi yang berada pada sisi server. Oh iya guys, Winflector sangat ringan digunakan dikoneksi yang boleh dibilang agak kurang cepat, dengan fitur TCP/IP Compress nya, membuat aplikasi dekstop kamu seperti halnya aplikasi yang dapat diakses dimanapun. kamu bisa coba versi gratis dari winflector disini, oh iya guys, versi gratis daripada winflector ini dibatasi 2 session saja. tapi so far, sangat berguna bagi mereka yang ingin membuat online sebuah program desktop menjadi online.

Beberapa pilihan yang bisa diatur dalam menu Appearance antara lain sebagai berikut. a. Windows and buttons, digunakan untuk memilih style dari Windows. b. Color scheme, digunakan untuk memilih tampilan warna baik di desktop ataupun pada program aplikasi lainnya. Pilih dan lihat hasilnya dengan cara klik kanan pada salah satu pilihan dan lihat layar (dialog box) akan ada perubahan warna. c. Font size, digunakan untuk memilih ukuran huruf dalam Window. d. Effects, digunakan untuk menapilkan kotak dialog Effects. e. Advanced, digunakan untuk mengatur Appearence lebih lanjut. .
