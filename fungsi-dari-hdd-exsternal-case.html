3. fungsi dari SSd dan HDD RAM ROM ​


Jawaban:

RAM bertugas sebagai cache yang sangat cepat untuk CPU kamu dan sebagai penyimpanan data yang sangat sedikit dibandingkan HDD atau SSD. Ia hanya menyimpan data sementara dari kinerja sistem operasi dan data dari aplikasi yang sedang kamu gunakan

1.ROM (Read Only Memory) adalah perangkat atau peralatan proses yang terdapat di

dalam CPU. ROM berisikan suatu program yang telah diterapakan oleh pembuat

perangkat komputer dan keberadaan program ini tidak dapat diubah, ditambah,

maupun dikurangi oleh pemakai Komputer. ROM pada komputer diguanakan untuk

sebagai penyimpan BIOS (Basic Input Output System).

2.SSD (Solid State Drive) adalah sebuah perangjat penyimpanan data yang

bersifat solid state yang menggunakan rangkaian integrated circuit (IC)

sebagai memory untuk menyimpan data secara terus mennnerus. SSD adalah

penyimpanan data terbaru. Teknologi ini memiliki fungsi yang sama dengan

HDD, namun dengan kinerja yang lebih baik, efisien, dengan konsumsi daya

lebih rendah. Perngakat SSD menambahkan chip memory berbasis silikon

sebagai media penyimpanan untuk menulis dan membaca data persisten.

Penjelasan paling mudah untuk SSD adalah di dalamnya, hanya terdapat chip

memori yang biasa kita temukan di flash disk. Sehingga, SSD tidak memiliki

bagian yang bergerak dan tidak seperti HDD yang di dalamnya bergerak.

3.HDD ( Hard disk drive ) adalah perangkat penyimpanan data yang digunakan untuk menyimpan dan mengambil informasi digital menggunakan disk (cakram) yang dilapisi dengan bahan magnetik.

4.RAM (Random Acces Memory) adalah sebuah perangkat keras yang berfungsi menyimpan data dan instruksi program yang akan dan sudah dieksekusi oleh prosesor. Penyimpanan RAM bersifat sementara, ini artinya setelah komputer dimatikan, RAM akan dikosongkan lalu akan diisi data baru yang diperlukan saat komputer dinyalakan dan dipergunakan.

Penjelasan:

RAM, random access memory. Berfungsi sebagai cache program, semua prpgram termasuk sistem operasi. Jadi, saat mnejalankan program (misalkan, Microsoft Word), data inti program dimasukkan terlebih dahulu ke RAM dan kemudian dijalankan prosesor. 25. memori dan hdd berfungsi untuk menyimpan data pada komputer.apa perbedaan keduanya?jelsakan​


Memori tidak sama dengan hardisk, tapi memori & hardisk mempunyai kesamaan untuk meyimpan informasi.

Memori biasanya berbentuk modul elektronik, menyimpan informasi sementara sedangkan hardisk menyimpan informasi secara permanen.

Jawaban:

Perbedaan terletak pada fungsi dan kegunaan keduanya. Memory (RAM) digunakan untuk menyimpan data secara sementara, sedangkan HDD digunakan untuk menyimpan data secara permanen.

Penjelasan:

Memory atau yang biasa disebut dengan RAM, merupakan singkatan dari Random Access Memory. Dari kata 'random' saja kita sudah bisa menyimpulkan bahwasanya memory ini bersifat acak, tidak teratur. Secara detail, Memory memiliki fungsi untuk menyimpan suatu data secara sementara, tidak tetap, tidak permanen. Contoh kegunaannya dapat kita lihat dalam kehidupan sehari-hari, misalnya pada komputer, jika kita membuka suatu aplikasi, sebagai contoh kita ambil aplikasi game, maka Memory yang ada pada komputer kita akan meyimpan data terkait game tersebut secara sementara, misalnya data mengenai variabel-variabel yang ada pada game tersebut, atau kalkulasi yang terjadi dalam game, ingat ya, memory ini menyimpan data, bukan memproses data. Tujuannya tentu saja agar komputer kita dapat mengakses data tersebut dengan sangat cepat, rata-rata diatas 1 Gbps.

Sedangkan untuk HDD, secara detail memiliki fungsi untuk menyimpan suatu data secara permanen. Contohnya sangat banyak, kita ambil yang paling simple: file. File yang kita buat, dapatkan, dan miliki pada komputer kita, disimpan pada HDD dan tidak akan bisa hilang terkecuali jika dihapus sendiri oleh kita atau terjadi kesalahan pada sistem. Karena sifatnya yang menyimpan data secara permanen, HDD memiliki kecepatan yang jauh lebih lambat dibandingkan ram, rata-rata sekitar 10 - 50 Mbps atau dalam satuan HDD sering disebut dengan RPM.

32. HDD(Hard Disk Drive) berfungsi sebagai


Untuk menyimpan dataHardisk adalah media penyimpanan data permanen, jadi data tidak hilang meskipun listrik sudah dimatikan. Hardisk berisi cakram magnetik yang mampu menyimpan data. Ukuran harddisk dinyatakan dalam Byte (B), contoh: 160GB (160 milyar byte). Hardisk ditemukan pertama kali oleh Reynold Johnson di tahun 1956. Harddisk pertama berukuran 4.4 MB. FUNGSI HARDISK Adapun fungsi Hardisk merupakan ruang simpan utama dalam sebuah komputer. Di situlah seluruh sistem operasi dan mekanisme kerja kantor dijalankan, setiap data dan informasi disimpan. Dalam sebongkah hardisk, terdapat berbagai macam ruangruang kecil (direktori, folder, subdirektori, subfolder), yang masing-masing dikelompokkan berdasarkan fungsi dan kegunaannya. Di situlah data-data diletakkan. Ruang kecil dalam hardisk bekerja dalam logika saling tergantung (interdependent). Data/informasi dalam satu ruang kadangkala diperlukan untuk menggerakkan data/ informasi yang berada di ruang lain. Ada ruang di mana data di dalamnya tidak boleh diutak-atik atau dipindahkan ke tempat lain, ada ruang di mana kita bisa membuang dan menaruh data secara bergantian sesuai kebutuhan. Hardisk terdiri atas beberapa komponen penting. Komponen utamanya adalah pelat (platter) yang berfungsi sebagai penyimpan data. Pelat ini adalah suatu cakram padat yang berbentuk bulat datar, kedua sisi permukaannya dilapisi dengan material khusus sehingga memiliki pola-pola magnetis. Pelat ini ditempatkan dalam suatu poros yang disebut spindle.
