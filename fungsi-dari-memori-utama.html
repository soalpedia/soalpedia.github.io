2. jelaskan fungsi dari memori utama


Fungsi Memori / RAM dan bagian - bagiannya yaitu :
1. Dynamic Random Access Memory (DRAM)
DRAM memiliki bentuk yang beragam. Umumnya berbentuk persegi / petak dengan warna hitam dan terletak di bagian PCB memori. DRAM berfungsi untuk menampung data dalam waktu yang singkat, dan juga akan selalu direfresh secara periodik / berkala.

2. Contact Point
Contact point merupakan bagian memori yang berfungsi sebagai penghubung ke motherboard / papan induk. Yang terdiri dari 1 atau 2 buah lekukan (NOTCH) dan beberapa titik. NOTCH bertujuan untuk mencegah kesalahan pasang memori pada motherboard.

3. Printed Circuit Board (PCB)
PCB merupakan sebuah board/papan berfungsi sebagai tempat komponen-komponen yang terdapat jalur koneksi antara satu komponen dengan yang lainnya. PCB dibuat dari fiber / pertinak yang dilapisi perak atau tembaga. PCB biasanya memiliki warna hijau dan terdiri dari beberapa layer / lapis tembaga.

3. apa fungsi memori utama pada komputer


Saya akan mencoba menjawab dengan dua jawaban:

Kelas: X
Mata pelajaran: Teknologi Informasi
Materi: Memori utama

Jawaban pendek:

Fungsi dari memori utama (main memory) adalah untuk menyimpan data yang akan diolah dan program yang akan dijalankan oleh komputer.

Jawaban panjang:

Memori utama, atau disebut juga main memory dan Random Access Memory (RAM) adalah bagian dari komputer yang merupakan tempat tersimpannya data dan program yang akan dijalankan oleh CPU saat komputer bekerja, termasuk sistem operasi.

Komputer hanya dapat mengakses secara langsung data dan program yang tersimpan dalam memori utama. Sehingga data dan program yang tersimpan dalam penyimpanan sekunder seperti di hard disk atau CD harus dibawa dulu ke memori utama untuk bisa diproses atau dijalankan.

Memori utama dihubungkan dengan CPU komputer dengan penghubung yang disebut memory bus.

Besarnya memori berfungsi menentukan kompleksitas operasi yang bisa dilakukan oleh suatu komputer. Tanpa memory utama yang besar, maka komputer hanya bisa melakukan perhitungan sederhana karena besar input yang bisa di proses kecil. Demikian juga output harus dikeluarkan langsung dan tidak bisa disimpan untuk perhitungan berikutnya yang lebih rumit.

Memori utama bersifat volatil, artinya hanya bisa berisi data ketika komputer menyala. Ketika komputer mati maka data di memori utama akan hilang. Ini berbeda dengan memori sekunder sperti hard disk yang bersifat non volatil, dimana data akan tersimpan meski tidak ada arus listrik.

Namun memori utama memiliki keunggulan yaitu kecepatan aksesnya oleh CPU yang lebih cepat dari pada kecepatan akses data memori sekunder.

4. Apa dua item utama yang disimpan dalam memori utama? Bagaimana Anda bisa membedakannya?


Penjelasan:

1 . S RAM

Static Random Access Memory atau S RAM adalah jenis RAM yang mampu menyimpan data tanpa harus ada penyegaran atau refresh dari CPU. S RAM memiliki proses kerja yang lebih cepat dibandingkan dengan DDRAM atau SDRAM.

2. HDD

Drive hard disk terbuat dari cakram bermagnet—dikenal sebagai piring—yang berputar cepat biasanya antara 5.400 dan 15.000 putaran per menit (RPM). Semakin cepat cakram magnetis berputar, semakin cepat komputer dapat mengakses informasi darinya.

Semua data digital berasal dari bentuk kode biner—serangkaian angka satu dan nol yang dapat mewakili tiap potongan informasi. Kepala baca/tulis dari hard drive biasanya dipakai untuk memasukkan angka satu dan nol ini dengan memagnetkan bagian dari piring. Setiap porsi kecil dari piring menampung satu bit, yang sama dengan 1 atau 0. Kepala dapat mendeteksi magnetisme dari setiap porsi, sehingga "membaca" informasi darinya. Kepala yang sama yang dapat "membaca" data dapat juga "menulis" data, dengan mengubah magnetisasi dari bit pada piring.

Kapan saja perubahan dibuat—seperti file baru disimpan atau file dihapus—kepala hard drive akan menyesuaikan magnetisme dari piring. Anda dapat membayangkannya seperti pemutar rekaman, dengan cakram piringan hitam sebagai piring yang berisi informasi, dan lengan sebagai kepala yang memindai informasi ini.

Karena data disimpan secara magnetis, HDD adalah perangkat non-volatile, yang berarti bahwa perangkat ini menyimpan data meski komputer dimatikan.

Sekarang ini, HDD internal dapat mencapai kapasitas maksimum 20 TB. Sejak kemunculan SSD, hard disk drive jarang dipakai sebagai penyimpanan sekunder komputer, namun masih merupakan opsi penyimpanan eksternal.

Metode penggunaan memori cache ini meningkatkan kinerja sistem sehingga prosesor mengakses memori cache dengan akses yang lebih cepat daripada mengakses memori utama secara langsung.

26. pada VIrtual memori Bagaimana mengetahui page mana yang masih di swap space dan yang ada di memori utama ?


Jawaban:

Untuk mengetahui page mana yang masih ada di swap space dan yang ada di memori utama, Anda dapat menggunakan perintah swapon -s pada terminal atau command prompt. Perintah ini akan menampilkan informasi tentang swap space yang digunakan oleh sistem operasi, termasuk page yang masih ada di swap space dan yang ada di memori utama.

Perlu diingat bahwa perintah ini hanya dapat digunakan pada sistem operasi Linux atau Unix. Jika Anda menggunakan sistem operasi lain, Anda perlu menggunakan perintah yang sesuai dengan sistem operasi tersebut untuk mengetahui informasi tentang swap space.

Penjelasan:

Swap space adalah bagian dari hard disk yang digunakan sebagai tempat sementara untuk menyimpan data yang tidak digunakan saat ini oleh sistem operasi. Jika sistem operasi membutuhkan lebih banyak memori daripada yang tersedia di memori utama, maka sistem operasi akan memindahkan data yang tidak digunakan ke dalam swap space untuk membebaskan ruang di memori utama.

Penjelasan:

Virtual memory adalah fitur yang memungkinkan komputer untuk menggunakan ruang penyimpanan sekunder, seperti hard drive, sebagai tempat penyimpanan tambahan untuk data yang melebihi kapasitas memori utama. Hal ini memungkinkan komputer untuk melanjutkan operasi meskipun memori utama sudah penuh.

Untuk mengetahui page mana yang masih di swap space (ruang penyimpanan sekunder) dan yang ada di memori utama, Anda bisa menggunakan beberapa cara berikut:

Menggunakan perintah ps. Perintah ps (process status) adalah perintah yang bisa digunakan untuk menampilkan informasi tentang proses yang sedang berjalan di sistem operasi. Anda bisa menggunakan perintah ps dengan opsi -l (long format) untuk menampilkan informasi yang lebih detail tentang setiap proses, termasuk informasi tentang virtual memory yang sedang digunakan.Menggunakan perintah top. Perintah top adalah perintah yang bisa digunakan untuk menampilkan informasi tentang proses yang sedang berjalan di sistem operasi secara real-time. Anda bisa menggunakan perintah top untuk menampilkan informasi tentang virtual memory yang sedang digunakan oleh setiap proses, seperti jumlah page yang ada di memori utama dan yang ada di swap space.Menggunakan fitur Resource Monitor.Untuk menggunakan Resource Monitor, Anda dapat membukanya melalui Task Manager. Klik kanan pada taskbar, kemudian pilih "Task Manager". Pada jendela Task Manager, klik pada tab "Performance", lalu klik pada tombol "Open Resource Monitor".

31. Apa beda fungsi memori utama dan tambahan


Jawaban:

Secondary dan main memory adalah dua jenis perangkat memori yang terdapat dalam sebuah sistem komputer dan berfungsi menyimpan data.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan perbedaan dari kedua memori tersebut. Berikut kakak akan mencoba menjawabnya.

MAIN MEMORI atau memori utama adalah perangkat penyimpanan data pada komputer dan umumnya berbentuk RAM.

Memori utama berfungsi sebagai media penyimpanan data sementara yang kemudian akan diolah oleh unit processor untuk kemudian disampaikan kepada pengunakan melalui media output.

Sementara itu, SECONDARY MEMORY atau memori tambahan adalah perangkat penyimmpanan data jangka panjang dan biasanya berbentuk ROM.

Dalam praktiknya, memori tambahan juga kerap difungsikan untuk menyediakan ruang penyimpanan data tambahan ketika tidak tersedia ruang cukup di memori utama.

35. memori yang berfungsi agar pemroses tidak langsung mengacu pada memori utama agar kinerja dapat ditinggalkan adalah​


Memori yang mana berfungsi agar pemroses tidak langsung tertuju ke memori utama agar kinerja dapat ditinggalkan, memori tersebut adalah cache memory.

Pembahasan

Cache memory adalah sebuah sistem memori yang dibuat untuk tambahan umum menyimpan data yang sering digunakan di sistem, hal tersebut dilakukan agar pemrosesan pada CPU menjadi lebih cepat tanpa mengakses memori utama pun. Berikut beberapa fungsi cache memory:

Mempercepat proses akses data. Akses data di komputer menjadi lebih cepat karena adanya cache memory, cache tersebut menjadi data penting pengingat terhadap data tersebut. Hal tersebut membuat dorongan menjadi terbiasa untuk mengakses data tersebut, maka dari hal tersebutlah pentingnya suatu cache memory agar meningkatkan efisien waktu dalam mengakses suatu data di komputer.Meringankan beban kerja processor perangkat. Processor sering mengangkat beban besar di komputer, hal tersebut dapat terjadi karena biasanya processor dibagi ke dalam beberapa core agar bisa mengerjakan tugas-tugas di komputer. Cache lah mendorong agar data-data tersimpan banyak dan mudah diakses, maka dari hal tersebutlah mendorong mudahnya kerja proccesor melalui cache memory.Mempercepat kinerja serta peforma memori perangkat. Cache memory memang memakan banyak data di komputer, hal tersebut terjadi karena cache memory membuat data sering dijumpai atau pintasan penting digunakan dalam banyak. Dari hal tersebutlah membuat dorongan kinerja memori perangkat secara umum semakin cepat meskipun memakan kapasitas data yang relatif banyak.
