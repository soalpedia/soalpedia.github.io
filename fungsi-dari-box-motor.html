30. seorang pedagang buah buahan akan menyewa sebuah motor box dan mobil box untuk mengakut 10.000.kg buah.sekali jalan motor box dapat mengakut 200 kg dan mobil box dapat mengakut 500 kg untuk mengakut buah buahan tersebut diperlukan jumlah motor box dan mobil box paling sedikit 35 unit dengan biaya angkut motor box RP 100.000.00 dan mobil box RP 150.000.00 biaya minimum untuk mengakut buah buahan tersebut beserta cara penyelesaian nyaA.RP 3.000.000.00B.RP 3.500.000.00C.RP 4.000.000.00D.RP 5.000.000.00E.RP 5.250.000.00​


pertama anggap saja semua motor box

35 x 200kg = 7.000kg

10.000 - 7.000 = 3.000kg (yg belum diangkut)

maka skg selisih antara motor dan mobil, 500kg - 200kg = 300kg

3.000kg : 300kg = 10 Mobil

maka motor = 35 - 10 = 25motor

oke, pembuktiannya dulu

25 x 200kg = 5.000kg (motor)

10 x 500kg = 5.000kg (mobil)

maka pas 10.000kg

skg hitung biayanya

25 x 100.000 = 2.500.000

10 x 150.000 = 1.500.000

total biaya = 2.500.000 + 1.500.000 = Rp. 4.000.000

Jawab:

C. Rp 4.000.000,00

Penjelasan dengan langkah-langkah:

Dik :

Daya angkut Maksimum = 10000 kg

Daya angkut Motor box = 200 kg

Daya Angkut Mobil box = 500 kg

Jumlah minimum kendaraan  = 35 unit

Biaya angkut motor per box = 100.000

Biaya angkut mobil per box = 150.000

Dit :

Biaya minimum pengangkutan buah

Jwb :

Daya angkut sementara

= 35 x 200kg = 7.000 kg

Sisa daya angkut yang belum terangkut

= 10.000 - 7.000 = 3.000 kg

Selisih Daya angkut Motor box dan Mobil Box

= 500kg - 200kg = 300 kg

Banyak Mobil box untuk mengangkut buah

= 3.000kg/300kg = 10 Mobil

Banyak Motor box untuk mengambil buah

= 35 - 10 = 25 Motor

Biaya pengangkutan kedua kendaraan :

Motor box

= 25 x 100.000 = 2.500.000

Mobil box

= 10 x 150.000 = 1.500.000

Total biaya minimum pengangkutan

= 2.500.000 + 1.500.000 = Rp 4.000.000

Jawabannya adalah C. Rp 4.000.000,00

32. bagaimana prospek usaha box motor delivery


Kelas ; X
Pelajaran : Kewirausahaan
Kategori : Transportasi
Kata kunci : prospek, box motor delivery

Pembahasan:
Prospek usaha box motor delivery sangat potesial ke depannya. Beberapa hal yang mendukung usaha ini adalah
1. Semakin padatnya jalan saat ini.
Dengan layanan pesan antar menggunakan motor, dapat melalui jalan-jalan yang tidak dapat dilalui oleh mobil. Sehingga waktu tempuhnya lebih cepat daripada menggunakan mobil.
2. Meningkatnya jumlah pelanggan layanan pesan antar.
Kesibukan para pelanggan dalam membutuhkan sesuatu atau mengantarkan sesuatu dapat dijangkau dengan cepat dan murah dengan menggunakan motor.

Kendala yang ditemukan pada layanan pesan antar dengan motor adalah
1. Dibutuhkan wadah yang aman dari panas dan hujan.
2. Wadah yang digunakan untuk layanan terbatas kapasitasnya.
