13. Konsep pre-defined class dengan pemograman java


[jawaban]

Mata Pelajaran: Teknologi Informasi
Kata kunci: Java, Pemrogaman

• Jawaban pendek:

Konsep pre-defined class dengan pemograman java

Java class library (Java API) memiliki banyak sekali kelas (Java predefined classes) yang diletakkan di paket (package) untuk dapat dipergunakan kembali di program. Paket tersebut digunakan untuk mengelompokkan kelas-kelas yang mempunyai kemiripan fungsi (related class). Kelas-kelas Java yang akan digunakan di dalam program, terlebih dahulu harus diimpor beserta dengan nama paket di mana kelas tersebut berada, kecuali untuk kelas Java yang berada di paket java.lang seperti kelas JOptionPane yang berada di paket javax.swing dan kelas DecimalFormat yang berada di paket java.text.

20. As a group, mangroves cannot _____ too closely. There are some 70 species from two dozen familiesâ€”among them palm, hibiscus, holly, plumbago, acanthus, legumes, and myrtle. fill the blank? (A) define (D) defining (B) defined (E) be defining (C) be defined


As a group, mangroves cannot be defined too closely. There are some 70 species from two dozen familiesâ€”among them palm, hibiscus, holly, plumbago, acanthus, legumes, and myrtle. (C)

Pembahasan

Untuk melengkapi kalimat di atas, yang dibutuhkan adalah kata kerja atau verb dalam bentuk pasif.

Kata 'define' berarti menetapkan atau mendefinisikan.

Pada kalimat di atas, bila diartikan "sebagai sebuah kelompok, mangrove tidak dapat didefinisikan terlalu dekat". Kata 'didefinisikan' di sini berbentuk pasif, karena bila dilanjutkan muncul pertanyaan "oleh siapa?".

25. . In Australia, dangerous goods _________ by the Australian Dangerous Goods Code (ADG). a. is defined b. is defining c. are defining d. are defined


Jawaban:

d. are defined

Penjelasan:

Dari kalimatnya kita menemukan dangerous goods. Berarti dia kata benda plural (jamak, lebih dari satu). Kerena subjeknya berbentuk jamak, maka to be nya bukan is, bukan am, tapi are

Jadi jawaban yang benar adalah d. are defined.

Kenapa bukan c. are definining?

Karena ini adalah kalimat pasif (passive voice), di mana verb nya harus berbentuk v3, jadi defined
