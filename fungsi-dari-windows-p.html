1. Apakah fungsi Windows XP,Windows Me,dan Windows Vista????


Windows XP, Windows Me, dan Windows Vista adalah suatu OS (Operating System) yg terdapat di dalam sebuah komputer. OS berguna sebagai suatu kesatuan sistem yg terdiri dari berbagai proses dimana terdapat input-proses-output dalam sebuah komputer. Sebuah komputer tidak akan dapat difungsikan jika tidak terdapat OS terinstal didalamnya.

16. fungsi windows server 2012 dan keunggulan dibandingkan dengan windows lainnya


A. Didasari komputasi melalui jaringan dan fitur cloud.

Windows server 2012 memiliki kelebihan dalam fitur yang telah disebutkan tadi. Karena jarang sistem operasi lainnya yang mensupport teknologi ini. Seperti fitur cloud dari Microsoft Azure yang dapat dihubungkan ke Windows Server 2012.

A. Banyak Versi yang dapat disesuaikan kebutuhan

Dengan bertumbuhnya perusahaan dan bisinis, diperlukan juga akan komputasi server yang sesuai dengan budget dan kebutuhan suatu bisnis. Dengan banyaknya versi membuat bisnis dapat memilih versi Windows Server 2012 yang tepat bagi mereka, tanpa perlu mengkonfigurasi suatu sistem operasi untuk sesuai dengan kebutuhan mereka ( bisnis ).

F. Kekurangan Windows Server 2012

1. Harga yang relatif mahal.
2. Fitur baru yang masih jarang digunakan, sehingga meningkatkan kemungkinan pemakaian yang lebih sulit.

29. 1.jelaskan fungsi search pada windows Explorer 2. jelaskan fungsi view pada windows explorer


1. Search
Digunakan untuk membuka kotak dialog file All files. Jadi pilihan ini sama dengan pilihan find dari Start Microsoft Windows XP. Begitu juga kegunaannya sama yaitu untuk mencari file yang dikehendaki.

2. View
Pilihan ini digunakan untuk mengkostumisasikan jendela1. Fungsi search:
untuk mencari file yang akan kamu gunakan. baik itu foto, dokumen, folder, dll. Kamu juga dapat mencari file yang bersifat executable seperti gimpsetup.exe. jika kamu menekan file tersebut maka akan muncul kotak instalansi.
2. View:
berguna untuk melihat file yang dipilih tanpa harus membukanya. cara ini hanya memerlukan klik 1 kali tanpa membuka program yang mendukung file tersebut seperti halo.docx akan terlihat di mode view jika Microsoft Word terinstal tetapi tidak perlu membuka Ms. Word untuk melihat filenya. Tidak dapat membaca file berekstensi .zip, .exe, dll.

semoga membantu
