9. Pada VCS Git, terdapat empat jenis workflow. Sebutkan dan jelaskan disertai gambar workflownya dari masing-masing jenis workflow tersebut.


1. Working Directory

Working directory merupakan tempat dimana kita bisa bebas untuk mengedit file

2. Staging Index / Staging Area

Staging Index / Staging Area tempat dimana kita sudah selesai mengedit file dan siap disimpan di local repository

3. Local Repository

Local Repository tempat dimana seluruh file yang sudah kita kita kerjakan, posisi nya masih ada di laptop atau pc masing-masing user

4. Remote Repository

Remote Repository tempat dimana kita menyimpan file yang sudah kita kerjakan, bedanya dengan local repository adalah remote sendiri posisinya ada di server

14. Salah satu contoh pemrograman visualyang berorientasi objek berbasis Windowsadalah ....a. Visual Basicb. GambasPascald. Git HubC.​


Salah satu pemrograman visual yang berorientasi objek dan berbasis Windows adalah a. Visual Basic (VB). Visual Basic adalah bahasa pemrograman yang menggunakan antarmuka grafis untuk membuat aplikasi Windows. Bahasa ini menggabungkan konsep pemrograman objek dengan kemudahan penggunaan untuk mengembangkan aplikasi desktop yang interaktif. Visual Basic juga mendukung pengembangan aplikasi web dengan menggunakan teknologi .NET.

Pembahasan

Pemrograman visual yang berorientasi objek berbasis Windows merujuk pada pendekatan pengembangan perangkat lunak yang menggunakan bahasa pemrograman visual dan berfokus pada konsep pemrograman berorientasi objek. Dalam konteks Windows, ini berarti menggunakan bahasa pemrograman dan lingkungan pengembangan yang mendukung pembuatan aplikasi yang kompatibel dengan sistem operasi Windows.

Berikut adalah beberapa bahasa pemrograman visual yang berorientasi objek dan berbasis Windows:

Visual Basic .NET (VB.NET) adalah bahasa pemrograman yang dikembangkan oleh Microsoft. C#: C# (C sharp) adalah bahasa pemrograman modern yang dikembangkan oleh Microsoft. Visual C++ adalah versi yang ditingkatkan dari bahasa pemrograman C++. Delphi adalah lingkungan pengembangan perangkat lunak yang menggunakan bahasa pemrograman Object Pascal. PowerBuilder adalah lingkungan pengembangan aplikasi yang dirancang untuk membuat aplikasi bisnis berbasis Windows. Lazarus adalah lingkungan pengembangan perangkat lunak bebas dan sumber terbuka yang memungkinkan pengembangan aplikasi dengan menggunakan bahasa pemrograman Object Pascal.

29. Arteri pada GIT berasal dari truncus?


GIT singkatan dari Traktus Gastrointestinal artinya saluran pencernaan makanan. Jadi arteri pada GIT berasal dari truncus coeliacus.  Arteri coeliacus berbentuk pendek dan tunggal serta bercabang.

Pembahasan

Arteri coeliacus besarnya sebesar ibu jari dengan panjang 2,5 cm kemudian bercabang mejadi:

Arteria gastrica sinistra

Arteri ini berjalan ke atas dan ke kiri untuk mencapai esofagus dan kemudian berjalan ke bawah di sepanjang lambung curvatura minor. Sisi kiri jantung mendarahi sepertiga bagian bawah esofagus dan bagian kanan atas perut.

Arteria gastrica dextra

Arteri ini berasal dari arteri hepatika pada tepi atas pilorus dan berjalan di sepanjang lengkungan usus kecil. Arteria mendarahi bagian kanan bawah gaster.

Arteriae gastricae breves

Arteri ini berasal dari arteria lienalis pada hilum lienale dan mengalir maju ke dalam ligamentum gastrosplenicum untuk mendarahi fundus.

Arteria gastroomentalis sinistra

Arteri ini berasal dari splenica arteria pada hilus lienale dan berjalan ke depan di dalam ligamentum gastrolienale untuk mendarahi lambung di sepanjang bagian atas kelengkungan lambung besar.

Arteria gastroomentalis dextra

Arteri yang memasok darah ke perut berasal dari cabang yang lebih besar arteri yang memasok darah ke hati. Arteri ini berjalan ke kiri dan mendarahi  lambung di sepanjang bagian bawah dari lengkungan besar.

A. GIMP

 

GIMP adalah program (softwareatau perangkat lunak) editor gambar, yang dapat digunakan untuk memanipulasi gambar dengan format poplar seperti PNG (prtable network graphics), JPEG dan Bitmap.

 

GIMP adalah cross-platform yang tersedia untuk sistem operasi seperti GNU / Linux (dan variannya seperti Ubuntu dan Debian), OS X (sistem operasi yang digunakan untuk komputer Apple), Microsoft Windows dan lebih banyak lagi.

 

GIMP adalah perangkat lunak bebas (open source), sehingga kita menggunakan dan dapat mengubah kode sumber serta mendistribusikan perubahan secara bebas.

 

B. GOM

 

GOM Player adalah media player (software untuk memainkan media seperti video dan musik). GOM Player adalah software untuk sistem operasi Microsoft Windows, dan dikembangkan oleh GOM & Company of Korea Selatan. Fitur utamanya meliputi kemampuan memutar beberapa file media yang rusak dan menemukan codec yang hilang menggunakan layanan finder codec.

 

C. GNOME

 

GNOME adalah lingkungan desktop (tampilan antar muka di desktop) yang terdiri dari perangkat lunak bebas dan open-source yang berjalan di sistem operasi Linux.

 

D. GIT

 

Git adalah sistem kontrol versi terdistribusi bebas dan open source. GIT umumnya digunakan untu proyek dari yang kecil hingga sangat besar dengan cepat dan efisiensi. Git mudah dipelajari dan memiliki beban dan jejak program kecil sehingga memiliki kinerja yang cepat.

 

E.GNU

 

GNU adalah kumpulan sistem operasi dan perangkat lunak komputer yang termasuk sistem operasi LINUX dan program-program yang berjalan di Linux. GNU seluruhnya terdiri dari perangkat lunak bebas (free and open source).
