11. Daerah yang diarsir pada grafik berikut adalah himpunan penyelesaian sistem pertidaksamaan linier. Nilai maksimum dari fungsi objektif f(x,y) = 6x + 10y adalah ….​


Jawaban:

Karena koefisien x dan y pada fungsi objektif mempunyai nilai yang hampir sama besar, nilai maksimum fungsi objektif tersebut terletak pada titik potong kedua garis.

Garis yang melalui titik (0, 5) dan (5, 0).

5x + 5y = 5 × 5

x + y = 5 … (1)

Garis yang melalui titik (0, 2) dan (−1, 0).

2x − y = 2×(−1)

2x − y = −2 … (2)

Titik potong kedua garis dapat dicari dengan melakukan eliminasi persamaan (1) dan (2).

x + y = 5

2x − y = −2

⎯⎯⎯⎯⎯⎯⎯ +

3x = 3

x = 1

Eliminasi x = 1 ke persamaan (1) diperoleh:

1 + y = 5

y = 4

Dengan demikian, nilai maksimum tercapai saat x = 1 dan y = 4.

f(x, y) = 6x + 10y

f(1, 4) = 6×1+10×4

= 6 + 40

= 46

Nilai maksimum fungsi objektif z = 3x + 4y dari system pertidaksamaan x + y ≤ 500; 2x + y ≤ 800; x ≥ 0; y ≥ 0 adalah

Penyelesaian:

X + y = 500

Jika x = 0 maka y = 500 ---------> (0, 500)

Jika y = 0 maka x = 500 ---------> (500, 0)

2x + y = 800

Jika x = 0 maka y = 800 ---------> (0, 800)

Jika y = 0 maka x = 400 ---------> (400, 0)

Eliminasi pertidaksamaan:

X + y = 500

2x + y = 800  -

-----------------  

-x = -300

X = 300

Y = 200 -----------------------------> (300, 200)

Fungsi objektif z = 3x + 4y

Titik (0, 500)

Z = 0 + 2000 = 2000

Titik (500, 0)

Z = 1500 + 0 = 1500

Titik (0, 800)

Z = 0 + 3200 = 3200

Titik (400, 0)

Z = 1200 + 0 = 1200

Titik (300, 200)

Z = 900 + 800 = 1700

 

Jadi, nilai maksimum fungsi objektif tersebut adalah 3.200

Langkah 1

x ≥ 0 dan y ≥ 0 berarti daerah penyelesaiannya berada di kuadran I (sebelah kanan sumbu y dan di atas sumbu x).

Langkah 2

Menggambar grafik x + 2y ≤ 6.

x + 2y = 6

Jika x = 0, maka y = 3 ⇒ (0, 3)Jika y = 0, maka x = 6 ⇒ (6, 0)

Menggambar grafik 2x + y ≤ 8.

2x + y = 8

Jika x = 0, maka y = 8 ⇒ (0, 8)Jika y = 0, maka x = 4 ⇒ (4, 0)

Hubungkan titik (0, 8) dan (4, 0) sehingga membentuk garis lurus dan perpanjang, kemudian arsir ke bawah, maka diperoleh grafik 2x + y ≤ 8.

Langkah 4

Titik potong kedua garis.

x + 2y = 6 |×2| 2x + 4y = 12

2x + y = 8 |×1| 2x + y = 8

                       ------------------  –

                              3y = 4

                                y = [tex]\frac{4}{3}[/tex]

x + 2y = 6

        x = 6 – 2y

        x = 6 – 2([tex]\frac{4}{3}[/tex])

        x = [tex]\frac{18}{3} \:-\: \frac{8}{3}[/tex]

        x = [tex]\frac{10}{3}[/tex]

Jadi titik potong kedua garis tersebut adalah

[tex]\left(\frac{10}{3}, \: \frac{4}{3}\right)[/tex]

(0, 0)(4, 0)(0, 3)[tex]\left(\frac{10}{3}, \: \frac{4}{3}\right)[/tex]

Langkah 6

Substitusi keempat titik sudut tersebut ke fungsi objektif z = 3x + 6y

Untuk titik (0, 0), nilai z adalah:

z = 3(0) + 6(0) = 0

Untuk titik (4, 0), nilai z adalah:

z = 3(4) + 6(0) = 12

Untuk titik (0, 3), nilai z adalah:

z = 3(0) + 6(3) = 18

Untuk titik [tex]\left(\frac{10}{3}, \: \frac{4}{3}\right)[/tex], nilai z adalah:

z = 3([tex]\frac{10}{3}[/tex]) + 6([tex]\frac{4}{3}[/tex]) = 10 + 8 = 18

Berdasarkan langkah 6, nilai maksimum dan minimum dari z = 3x + 6y adalah:

Nilai maksimum = 18.Nilai minimum = 0.

4x + 2y ≤ 4 . 2

 2x + 3y = 6    | × 2 ---> 4x + 6y = 12

 4x + 2y = 8    | × 1 ---> 4x + 2y = 8
