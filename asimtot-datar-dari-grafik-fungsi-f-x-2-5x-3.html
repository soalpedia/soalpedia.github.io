38. asimtot datar pada grafik fungsi [tex]f(x) = 3 {}^{x - 2} - 2[/tex]​


Jawaban Dan Penjelasan :

Untuk menemukan asimtot datar pada grafik fungsi f(x) = 3^x - 2, kita perlu memeriksa dua jenis asimtot: asimtot horizontal dan asimtot vertikal.

Asimtot Horizontal

Untuk mencari asimtot horizontal, kita perlu memeriksa perilaku fungsi saat x mendekati positif tak terhingga dan negatif tak terhingga. Dalam hal ini, saat x mendekati positif tak terhingga (x → ∞), fungsi akan mendekati nilai positif tak terhingga. Jadi, tidak ada asimtot horizontal di atas.

Saat x mendekati negatif tak terhingga (x → -∞), fungsi juga akan mendekati nilai positif tak terhingga. Jadi, tidak ada asimtot horizontal di bawah.

2. Asimtot Vertikal

Untuk mencari asimtot vertikal, kita perlu memeriksa apakah ada nilai x yang membuat penyebut dari fungsi menjadi nol. Jika ada, maka ada asimtot vertikal pada nilai tersebut.

Dalam fungsi f(x) = 3^x - 2, penyebut adalah 1 (karena tidak ada x di penyebut). Tidak ada nilai x yang membuat penyebut menjadi nol, jadi tidak ada asimtot vertikal.

Dengan demikian, pada grafik fungsi f(x) = 3^x - 2, tidak ada asimtot horizontal maupun asimtot vertikal. Grafiknya mungkin memiliki kurva yang melengkung tetapi tidak ada garis lurus horizontal atau vertikal yang menjadi asimtot.
