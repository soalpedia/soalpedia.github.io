4. varians dari data 8,7,10,12,9,4,6 adalah


Kelas           : 11
Mapel           : Matematika
Kategori      : Statistika
Kata kunci  : varians
Kode            : 11.2.1 (Kelas 11 Matematika Bab 1-Statistika)

Varians dari data 8,7,10,12,9,4,6 adalah ...

Pembahasan: 

Rata- rata dari data tunggal dapat dicari dengan menggunakan rumus:
[tex]\overline{x}= \frac{x_{1}+x_{2}+x_{3}+...+x_{n}}{n} \\ \overline{x}= \frac{8+7+10+12+9+4+6}{7} \\ \overline{x}= \frac{56}{7} \\ \overline{x}=8 [/tex]

Salah satu ukuran persebaran dalam statistika adalah varians. Varians adalah jumlah kuadrat dari selisih nilai data observasi dan nilai rata-ratanya, kemudian dibagi dengan jumlah data nya. Varians merupakan kuadrat dari standar deviasi. Varians biasa dilambangkan s². Varians dari data tunggal dapat dicari dengan menggunakan rumus:
 
[tex]s^2= \frac{\sum_{i=1}^{n} (x_i-\overline{x})^2}{n} } \\s^2= \frac{(x_1-\overline{x})^2+(x_2-\overline{x})^2+(x_3-\overline{x})^2+(x_4-\overline{x})^2+(x_5-\overline{x})^2+(x_6-\overline{x})^2+(x_7-\overline{x})^2}{7} \\ s^2= \frac{(8-8)^2+(7-8)^2+(10-8)^2+(12-8)^2+(9-8)^2+(4-8)^2+(6-8)^2}{7} \\ s^2= \frac{0^2+(-1)^2+2^2+4^2+1^2+(-4)^2+(-2)^2}{7} \\ s^2= \frac{0+1+4+16+1+16+4}{7} \\ s^2= \frac{42}{7} \\ s^2=6 [/tex]

Jadi, varians dari data 8,7,10,12,9,4,6 adalah 6.

13. Varians dari data 6,8,8,9,11,12 adalah


Varians dari data 6,8,8,9,11,12 adalah 4.

Pembahasan

Varians atau ragam adalah perbandingan antara jumlah dari kuadrat selisih setiap data dan rata-ratanya dengan banyaknya data.

Ragam (R) atau Varians :

[tex]\boxed{\bold{Ragam=\frac{\Sigma(x_i-\bar{x})^2}{n}}}[/tex]

Rata-Rata ([tex]\bar{x}[/tex]) :

[tex]\boxed{\bold{\bar{x}=\frac{\Sigma{x_i}}{n}}}[/tex]

n = banyaknya data

======================================================

Diketahui :

Data 6, 8, 8, 9, 11, 12

Ditanya :

Varians

Jawab :

Pertama, kita tentukan dulu rata-rata data di atas!

Data 6, 8, 8, 9, 11, 12 => n = 6

Rata-rata = [tex]\bar{x}[/tex]

[tex]\bar{x}=\frac{\Sigma{x_i}}{n}[/tex]

[tex]\bar{x}=\frac{6+8+8+9+11+12}{6}[/tex]

[tex]\bar{x}=\frac{54}{6}[/tex]

[tex]\bar{x}=9[/tex]

Jadi rata-rata data di atas adalah 9.

Selanjutnya kita masukkan nilai rata-rata di atas ke dalam rumus varians!

[tex]Ragam=\frac{\Sigma(x_i-\bar{x})^2}{n}[/tex]

[tex]R=\frac{(6-9)^2+(8-9)^2+(8-9)^2+(9-9)^2+(11-9)^2+(12-9)^2}{6}[/tex]

[tex]R=\frac{(-3)^2+(-1)^2+(-1)^2+(0)^2+2^2+3^2}{6}[/tex]

[tex]R=\frac{9+1+1+0+4+9}{6}[/tex]

[tex]R=\frac{24}{6}[/tex]

[tex]R=4[/tex]

Jadi varians dari data di atas adalah 4.

14. varians dari data : 20,22,22,24,24,26,26,28 adalah​


Varians dari data : 20, 22, 22, 24, 24, 26, 26, 28 adalah​ 6

Pendahuluan

Rata-rata adalah jumlah semua nilai data dibagi dengan banyak data.

[tex]\boxed {\overline {x} = \frac{x_1~+~ x_2~ +~ x_3~ +~ ...~ +~ x_n}{n}}[/tex]

Varian (ragam) data

[tex]\boxed {S^{2} = \frac{1}{n} \Sigma^n_{i = 1} (x_i - \bar{x})^2}[/tex]

Pembahasan

Diketahui :

data : 20, 22, 22, 24, 24, 26, 26, 28

Banyak data (n) = 8

Ditanya :

Varian data ?

Penyelesaian :

Menentukan rata-rata

[tex]\displaystyle \overline {x} = \frac{x_1~+~ x_2~ +~ x_3~ +~ ...~ +~ x_n}{n}[/tex]

[tex]\displaystyle \overline {x} = \frac{20 + 22 + 22 + 24 + 24 + 26 + 26 + 28}{8}[/tex]

[tex]\displaystyle \overline {x} = \frac{192}{8}[/tex]

x = 24

Menentukan ragam (Varian) dari data

[tex]\boxed {S^{2} = \frac{1}{n} \Sigma^n_{i = 1} (x_i - \bar{x})^2}[/tex]

S² = ¹/₈ [ (20 - 24)² + (22 - 24)² + (22 - 24)²+ (24 - 24)²+ (24 - 24)²+ (26 - 24)²+ (26 - 24)²+ (28 - 24)² ]

S² = ¹/₈ [(-4)² + (-2)² + (-2) ² + 0² + 0² + 2² + 2² + 4² ]

S² = ¹/₈ [16 + 4 + 4 + 0 + 0 + 4 + 4 + 16]

S² = ¹/₈ × 48

S² = 6

Jadi varians dari data : 20, 22, 22, 24, 24, 26, 26, 28 adalah​ 6.

Varian dan Simpangan Baku adalah Ukuran keragaman data statistik

Rumus :

s² = [tex]\frac{\Sigma^n_{i = 1} (x_i - \overline{x})^2}{n}[/tex]

Dimana :

s² = Varians

[tex]x_i[/tex] = nilai x ke i

[tex]\overline{x}[/tex] = rata-rata

n = Jumlah sample

Pembahasan :

Diketahui :

Data 17, 18, 20, 19, 22, 18

Ditanya :

Varians ?

Dijawab :

Pertama-tama kita cari dahulu nilai rata-rata dari data tersebut

Nilai rata-rata = (17 + 18 + 20 + 19 + 22 + 18) : 6 = 114 : 6 = 19

Kita cari Nilai [tex]\Sigma^n_{i = 1} (x_i - \overline{x})^2[/tex]

28. Diketahui data 2,6,7,1,4 varians data tersebut adalah


Varians dari data 2, 6, 7, 1, 4 adalah 5,2.

Varians merupakan salah satu dari ukuran keragaman (variasi) data statistik yang paling sering digunakan di mana dasar penghitungannya adalah keinginan untuk mengetahui keragaman suatu kelompok data dengan mengurangi setiap nilai data dengan rata - rata kelompok data tersebut dan dikuadratkan, selanjutnya semua hasilnya dijumlahkan dan hasil penjumlahan kuadrat (sum of squares) tersebut akan selalu bernilai positif. Nilai varian diperoleh dari pembagian hasil penjumlahan kuadrat (sum of squares) dengan jumlah data (n) yang dituliskan dengan S² = [tex] \frac{1}{n}[/tex]Σ(xi - x̄)².

Agar lebih jelasnya, simak pembahasan soal berikut.

PEMBAHASAN :

Diketahui beberapa data 2, 6, 7, 1, 4. Dari data - data tersebut, diketahui bahwa ada 5 data sehingga ditulis n = 5.

Sebelum kita menghitung variannya, kita akan hitung rata - rata data tersebut.

x̄ = [tex] \frac{jumlah \: data}{frekuensi \: data} [/tex]

x̄ = [tex] \frac{2 + 6 + 7 + 1 + 4}{5} [/tex]

x̄ = [tex] \frac{20}{5} [/tex]

x̄ = rata - rata data = 4

Kemudian kita hitung varians-nya.

S² = [tex] \frac{1}{n} [/tex]Σ(xi - x̄)²

S² = [tex] \frac{1}{5} [/tex][(2 - 4)² + (6 - 4)² + (7 - 4)² + (1 - 4)² + (4 - 4)²]

S² = [tex] \frac{1}{5}[/tex][4 + 4 + 9 + 9 + 0]

S² = [tex] \frac{1}{5} [/tex] × 26

S² = varians data = 5,2.

36. untuk data tunggal Menjelaskan rumus-rumus ukuran penyebaran (jangkauan, jangkauan antar kuartil, varians dan simpangan baku) untuk data tunggal​


Penjelasan dengan langkah-langkah:

Rumus-rumus ukuran penyebaran untuk data tunggal adalah sebagai berikut:

Jangkauan (range)

Jangkauan merupakan selisih antara nilai terbesar dan nilai terkecil dalam kumpulan data. Rumusnya adalah sebagai berikut:

Jangkauan = Nilai terbesar - Nilai terkecil

Jangkauan antar kuartil (interquartile range)

Jangkauan antar kuartil adalah selisih antara kuartil atas dan kuartil bawah pada data. Rumusnya adalah sebagai berikut:

Jangkauan antar kuartil = Q3 - Q1

dimana Q1 adalah kuartil pertama dan Q3 adalah kuartil ketiga.

Varians (variance)

Varians adalah ukuran yang menunjukkan seberapa jauh data tersebar dari rata-ratanya. Rumusnya adalah sebagai berikut:

Varians = (Σ(xi - x̄)²) / N

dimana Σ adalah simbol penjumlahan, xi adalah nilai observasi, x̄ adalah rata-rata dari nilai observasi, dan N adalah jumlah total observasi.

Simpangan baku (standard deviation)

Simpangan baku adalah akar kuadrat dari varians. Rumusnya adalah sebagai berikut:

Simpangan baku = √(Σ(xi - x̄)² / N)

dimana Σ adalah simbol penjumlahan, xi adalah nilai observasi, x̄ adalah rata-rata dari nilai observasi, dan N adalah jumlah total observasi.

Dalam rumus-rumus di atas, Σ(xi - x̄)² menunjukkan jumlah kuadrat selisih antara nilai observasi dengan rata-ratanya, dan N adalah jumlah total observasi.
