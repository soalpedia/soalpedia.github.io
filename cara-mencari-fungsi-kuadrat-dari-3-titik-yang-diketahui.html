7. Diketahui fungsi kuadrat mempunyai titik puncak (3,-2) dan melalui titik (0,16) . Tentukan persamaan fungsi kuadrat tersebut


Diketahui fungsi kuadrat mempunyai titik puncak (3,-2) dan melalui titik (0,16) . Tentukan persamaan fungsi kuadrat tersebut!Jawaban Pendahuluan

Soal matematika di atas merupakan materi dari fungsi dalam bentuk persamaan kuadrat, dimana terdapat grafik fungsi yang diketahui agar dapat menentukan fungsi kuadratnya.

Pembahasan  

Persamaan kuadrat mempunyai bentuk umum yaitu ax^2 + bx + c = 0, dimana a adalah koefisien dari variabel x^2, b adalah koefisien dari variabel x^1, dan c adalah koefisien dari variabel x^0.

[tex]ax^2 + bx + c = 0[/tex]

Fungsi dalam bentuk persamaan kuadrat atau fungsi kuadrat pada umumnya dinotasikan dengan f(x) = ax^2 + bx + c, dimana hasil penyelesaian dari fungsi ini bergantung pada himpunan nilai x yang diberikan. Himpunan penyelesaian fungsi kuadrat bisa dianggap sebagai nilai y sehingga f(x)=y dan pasangan nilai x dan y ditulis menjadi (x, y).

[tex]f(x) = ax^2 + bx + c\\y = ax^2 + bx + c[/tex]

Rumus untuk menentukan nilai x untuk grafik titik puncak pada fungsi bentuk persamaan kuadrat adalah x = -b/2a.

[tex]x = -b/2a[/tex]

Rumus untuk menentukan fungsi bentuk persamaan kuadrat dengan grafik titik puncak (p, q) dan titik sembarang (x, y) adalah y = a(x - p)^2 + q.

[tex]y = a(x - p)^2 + q[/tex]


Penyelesaian soal fungsi kuadrat dimana terdapat grafik koordinat yang diketahui adalah sebagai berikut.

Grafik fungsi kuadrat pada titik puncak P (3, -2) dan titik A (0, 16) perlu disubstitusikan dimana titik-titik tersebut merupakan nilai fungsi untuk (x, y).

Grafik pada titik puncak P (3, -2) atau x=3 dan y=-2

y = ax^2 + bx + c

-2 = a(3)^2 + b(3) + c

-2 = 9a + 3b + c  ...  (1)

Grafik pada titik A (0, 16) atau x=0 dan y=16

y = ax^2 + bx + c

16 = a(0)^2 + b(0) + c

c = 16

Sehingga nilai c adalah 16 atau c = 16

Rumus titik puncak pada fungsi kuadrat untuk x=3

x = -b/2a

3 = -b/2a

b = -6a  ...  (3)

Substitusi c = 16 dan b = -6a pada persamaan 1

-2 = 9a + 3b + c

-2 = 9a + 3(-6a) + 16

-2 - 16 = 9a - 18a

-18 = -9a

a =  2

Sehingga nilai a adalah 2 atau a = 2.

Substitusi a = -12 pada persamaan 3

b = -6a

b = -6(2)

b = -12

Sehingga nilai b adalah -12 atau b = -12.

Substitusi a = 2, b = -12, dan c = 16 pada bentuk umum fungsi kuadrat.

y = ax^2 + bx + c

y = 2x^2 - 12x  + 16

f(x) = 2x^2 - 12x + 16

Penyelesaian soal fungsi kuadrat ini juga dapat dilakukan dengan cepat dengan rumus berikut yaitu a(x - p)^2 + q.

Titik puncak P: (p, q) = (3, -2)

Titik A: (x, y) = (0, 16)

Grafik pada titik A (0, 16) atau x=0 dan y=16

y = a(x - p)^2 + q

16 = a(0 - 3)^2 + -2

16  = 9a - 2

18  = 9a

a  = 2

Grafik pada titik puncak P (3, -2) atau x=3 dan y=-2

y = a(x - p)^2 + q

y = 2(x - 3)^2 - 2

y = 2(x^2 - 6x + 9) - 2

y = 2x^2 - 12x + 18 - 2

f(x) = 2x^2 - 12x + 16

Sehingga grafik fungsi kuadrat adalah f(x) = 2x^2 - 12x + 16.

Kesimpulan

Persamaan grafik fungsi kuadrat yang memiliki titik puncak P (3,-2) dan melalui titik A (0,16) adalah f(x) = 2x^2 - 12x + 16 yang didapatkan dengan cara mengetahui bentuk umum fungsi kuadrat dan variabelnya dengan rumus titik puncak persamaan kuadrat.

Fungsi kuadrat memiliki beberapa istilah berikut ini:

Titik Puncak memiliki dua komponen yaitu sumbu simetri dan nilai maksimum. Titik puncak dapat disebut juga sebagai titik balik atau titik maksimum/minimum. Berikut ini adalah cara menggambar grafik fungsi kuadrat diantaranya adalah dengan menentukan nilai variabel x, didapatkan nilai y dari fungsi yang telah dikerjakan, mendapatkan koordinat titik pada sumbu x dan y, dan setiap titik koordinat digabungkan, serta grafik fungsi kuadrat yang terbentuk adalah parabolik.Pembahasan

Nilai a : x²Nilai b : 2xNilai c : -3

A. Titik Puncak

Untuk mencari titik puncak dapat menggunakan dua rumus yaitu ([tex]\frac{-b}{2a}[/tex]) digunakan untuk mencari titik puncak sumbu x dan ([tex]\frac{D}{-4a}[/tex]) digunakan untuk mencari titik puncak y. Perlu kamu ketahui, rumus untuk mencari (d = diskriminan) adalah [tex]{b}^{2}-4ac[/tex].

• Titik Puncak Sumbu X

x = [tex]\frac{-b}{2a}[/tex]

x = [tex]\frac{-2}{2.1}[/tex]

x = [tex]\frac{-2}{2}[/tex]

x = {[tex]-1[/tex]}

Untuk mencari titik puncak sumbu y terlebih dahulu mencari diskriminan sebagai berikut, dan lalu mengerjakannya sesuai dengan rumus:

d = b² - 4ac

d = 2² - 4.1.-3

d = 4 + 12

d = 16

y = [tex]\frac{D}{-4a}[/tex]

y = [tex]\frac{16}{-4.1}[/tex]

y = [tex]\frac{16}{-4}[/tex]

x = x² - 1x + 3x - 3

x = x (x - 1) + 3 (x - 1)

x = (x + 3) (x - 1)

x = x¹ = -3 dan x² = 1 atau (-3, 1)

• Titik Potong Sumbu Y

y = x² + 2x - 3

y = 0² + 2(0) - 3

y = 0 + 0 - 3

y = -3 (0, -3)

- Fungsi kuadrat tersebut adalah apa?

Dijawab:

Untuk mencari fungsi kuadrat, kita menggunakan bentuk umum fungsi kuadrat y = ax^2 + bx + c, di mana a, b, dan c adalah konstanta.

Dalam kasus ini, kita memiliki informasi tentang titik puncak dan titik yang dilewati oleh grafik.

Titik puncak (0, -4) memberikan informasi bahwa x = 0 dan y = -4. Dengan menggunakan ini, kita dapat mencari nilai dari c.

-4 = a(0)^2 + b(0) + c

-4 = c

Jadi, nilai c adalah -4.

Selanjutnya, kita menggunakan titik lain yang dilewati grafik yaitu (-1, -3) untuk mencari sisa konstanta yang tidak diketahui.

-3 = a(-1)^2 + b(-1) + -4

-3 = a - b - 4

1 = a - b

Dengan menggunakan nilai c dan persamaan di atas, kita dapat menentukan fungsi kuadrat tersebut.

Sehingga, jawaban dari pertanyaan adalah: Fungsi kuadrat tersebut adalah y = x^2 - x - 4.

Operasi pada bilangan kuadrat sama seperti aturannya dengan operasi pada bilangan biasa, hanya saja, kuadrat lebih dahulu dilakukan.
