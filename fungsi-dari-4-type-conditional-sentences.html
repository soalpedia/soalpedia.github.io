PENJELASAN

=> CONDITIONAL SENTENCE TYPE 1

First conditional sentence merupakan bentuk kalimat pengandaian yang digunakan ketika hasil atau konsekuensi memiliki kemungkinan terjadi di masa depan. Maka dari itu, bentuk kalimat dari conditional sentence tipe pertama ini memiliki bentuk kalimat simple future alias kalimat yang akan datang. Rumus dari kalimat ini adalah: if + simple present, simple future “will”.

=> CONDITIONAL SENTENCE TYPE 2

Second conditional sentences merupakan tipe kalimat pengandaian yang digunakan ketika hasil atau konsekuensi hanya memiliki harapan terwujud walaupun kemungkinannya sangat kecil. Dalam bentuk kalimat ini, rumus yang digunakan adalah if + simple past/were, would/could/might + bare infinitive. Penggunaan would/could/might berfungsi untuk menjelaskan seberapa besar sebuah kondisi itu akan terjadi.

=> CONDITIONAL SENTENCES TYPE 3

Third conditional sentence merupakan sebuah kalimat pengandaian ketika sebuah kondisi tidak mungkin terwujud sama sekali. Dalam tipe kalimat pengandaian yang ketiga ini, bentuk kalimat menggunaka past perfect yang dilengkapi dengan modal auxiliary seperti would, could, dan should. Rumus yang digunakan adalah if + past perfect, would/should/could/might + have + past participle.

Conditional Sentence Type 1 (Present Real Conditional)

Rumus: If + present simple, will + bare infinitive

Contoh: If I have a headache, I will go to the doctor.

Conditional Sentence Type 2 (Past Unreal Conditional)

Rumus: If + past simple, would + bare infinitive

Rumus: If + past perfect, would have + past participle

Contoh: If I had had a headache yesterday, I would have gone to the doctor.

Discussion

Ten sentences above use the conditional sentence of type 1. An activity will probably happen or will be done now or later because the requirements will be fulfilled soon, so, all the the speaker’s desire will be actualized in the present or future.

10 kalimat di atas menggunakan tipe ke 1 dari conditional sentence. Sebuah aktifitas yang akan mungkin terjadi atau akan di lakukan sekarang atau nanti-nanti karena persyaratannya akan di penuhi segera, sehingga, semua keinginan dari si pembicara akan di wujudkan saat ini atau nanti-nanti.

The pattern of conditional sentence of type 1 by using :

If+S+verb1 (s/es )+ O/C , Subject + will + verb 1 + O/C  ( use comma )

Or

Subject +will + verb 1 + O/C  If+S+verb1 (s/es) + O/C ( no comma )

The verb in IF clause uses the base verb. Meanwhile, the verb in the main clause (result clause ) uses modal auxiliary verb “will” plus “base verb” or we can say “simple future.”

Keep in mind, the verb form in IF clause depends on the subject whether it is singular or plural form.  

Explanation

CONDITIONAL SENTENCE  Conditional sentence tells about the condition that will be actualized ( possible condition ) or only dreaming ( intention ) because of the particular condition. Conditional sentence structure consists -If clause and result clause ( main clause ). Use Comma when the result clause (main clause ) is started at the beginning sentence. Otherwise, there is no comma when -IF clause is put after the result clause.

There are 4 types of Conditional Sentence  

1. CONDITIONAL ZERO   Conditional zero is used to describe the PHENOMENON,  and GENERAL TRUTH ( THE FACT/REALITY  ) Conditional zero uses the simple present tense form to reflect the truth.

Pattern :  

IF+S+base verb (s/es ) + O/C , S+base verb (s/es ) + O/C  

Example :  

If it rains , the roads get wet

Analysis  

It tells about the general truth or phenomenon.

It uses the simple present tense form to describe the truth. We can't use simple future in its main clause (result clause ), because it is not a prediction but it’s the fact. It is proven that the roads always get wet after raining.

2. CONDITIONAL TYPE 1   Conditional type 1 is used to describe the truth  in the present/future ( The planning (prediction)  will be probably happened ) In fact , the speaker will actualize his/her dream now or soon .

Pattern :  

If +S+ verb (s/es) + O, S+will+verb1+O/C  

Example :  

If the students study hard, they will pass the exam easily.

Analysis

Fact : it's probably/possible that they will be successful for the exam later because they study hard.

It uses simple future form in its main clause / result clause to describe the possible condition and state the prediction will be true.

3. CONDITIONAL TYPE 2   Conditional type 2 is used to describe  the untruth in the present / future It is Contrary to the fact in the present or it could be in the future The speaker has an intention ( dream ) but he/she can't afford his/her dream comes true in the present/future

Pattern:  

If+S+verb2+O/C , S+Would/Could+verb1+O/C  

Example :  

If I were a bird, I would fly around the world.

Analysis  

Fact in the present/future : I can't fly around the world because I'm not a bird now.

It uses the simple past form to reflect the untruth condition in the present and the speaker's intention will not come true because it is contrary to the real condition for now or future.

4. CONDITIONAL TYPE 3 Conditional type 3 Is used to describe  the untruth in the past It is Contrary to the fact in the past

Pattern :  

If+S+Had+Verb 3+O/C  , S+Would+have+Verb 3+O/C  

Example :  

If I had found my car key, I would have fetched my grandmother at the airport.

Analysis :

Fact in the past  : I didn't find my car key, so I didn't fetch my grandmother by my car at the airport

It reflects a regret for something that we couldn't afford to do in the past.

32. make 5 sentences in each type in conditional sentences​


CONDITIONAL SENTENCE

1. If you use a face mask, it will keep you from the dust. (type 1)

2. If I were a president, I would build the great library. (type 2)

3. If you hadn't asked somebody on the road, you would have got lost. (type 3)

4. If you freeze the water in the fridge, it changes into an ice. (Zero conditional)

5. If someone is infected by the coronavirus, she will get pneumonia. (type 1)

EXPLANATION

Conditional sentence adalah kalimat pengandaian/bersyarat, di mana suatu keinginan dapat terwujud apabila persyaratannya terpenuhi. Conditional sentence terdiri dari 2 klausa/kalimat, satu kalimat sebagai If clause dan kalimat lainnya sebagai main clause/result clause. If clause menggambarkan kondisinya, dan main clause menggambarkan konsekuensinya. Apabila kondisinya adalah pasti benar, maka konsekuensinya akan terwujud. Conditional sentence terbagi dalam empat tipe, yaitu 0, 1, 2 dan 3

Conditional tipe 0 [Zero conditional]

menjabarkan fakta dan kebenaran umum.

Rumus

If+simple present, simple present

If+S+verb1(s/es)....,S+verb1(s/es).....

Conditional tipe 1 [First conditional] menjabarkan keinginan yang kemungkinan terjadi saat ini ataupun nanti-nanti (real possibility) berdasarkan prediksi. [True in the present/future]

Rumus

If+Simple present, Simple future

If+S+verb1(s/es)..., S+will+verb1......

Conditional tipe 2 [Second conditional] menjabarkan keinginan yang kemungkinan kecil terwujud atau diragukan terjadi saat ini atau mendatang. [Untrue in the present/future]

Nominal sentence pada conditional tipe 2 menggunakan tobe "were" untuk subyek tunggal dan jamak.

Rumus :

If+Simple past, Past future

If+S+verb2 ......, S+would+verb1......

Conditional tipe 3 [Third conditional] menjabarkan keinginan yang tidak terwujud di waktu lampau. Pengulasannya saat ini hanyalah sebagai dampak dari kejadian di waktu lampau. [Untrue in the past]

Rumus :

If+Past perfect, Past future perfect

34. conditional sentences type 3 ?


Memakai past perfect tense.
Contoh kalimat:
If you had studied harder, you have passed the examuntuk menjelaskan kalimat dimana kejadian tersebut sudah terjadi.
polanya secara umum :
(+) if + subject + had + v3 + advreb (jika perlu) + subject + would have + v3 + adverb
(-) if + subject + hadn't + v3 + adverb + subject + woulan't have + v3 + adverb
ex : if i had sutdied harder, i wold have got higher score
di sini jelas bahwa ia menyesal karena nilainya rendah sehingga dia membuat pengandaian seperti kalimat di atas.
atau polanya bisa dibalik seperti :
i wouldn't have been scolded if i had done the exercises
