Struktur percabangan if bersarang / nested if merupakan struktur if yang paling komplek, karena merupakan perluasan dan kombinasi dari berbagai struktur if lainnya. Konsep dari percabangan ini adalah terdapat Struktur If yang berada didalam Struktur If lainnya. Artinya dalam pernyataan If bersarang jika kondisi If yang paling luar (paling atas) bernilai benar, maka kondisi If yang berada didalamnya baru akan dilihat (di cek).

      }else{

Pembahasan

Contoh program yang dapat dibuat menggunakan nested if adalah program menentukan angka tahun kabisat. Sebelum mencoba menerapkan algoritma ke dalam program C++, maka harus diketahui algoritmanya sebagai berikut

1. Program dimulai

2. Masukkan angka tahunnya

3. Jika angka tahun habis dibagi 100 (sisa baginya 0), maka

Jika angka tahun habis dibagi 400 (sisa baginya 0), cetak "tahun kabisat".Jika angka tahun tidak habis dibagi 400 (sisa baginya bukan 0), cetak "bukan tahun kabisat".

4. Jika angka tahun tidak habis dibagi 100 (sisa baginya bukan 0), maka

Jika angka tahun habis dibagi 4(sisa baginya 0), cetak "tahun kabisat".Jika angka tahun tidak habis dibagi 4(sisa baginya bukan 0), cetak"bukan tahun kabisat".

5. Program selesai.

Algoritma diatas dalam bentuk pseudocode dapat ditulis

Algoritma tahunKabisat

program start

var tahun : integer;

if(tahun % 100 == 0) then

   if(tahun % 400 == 0) then

       write "Tahun kabisat"

   else then

       write "Bukan tahun kabisat"

   end if

else then

   if(tahun % 4 == 0) then

       write "Tahun kabisat"

   else then

       write "Bukan tahun kabisat"

   end if

end if

program end

Setelah algoritma diketahui, maka sisanya adalah menerapkan algoritma kedalam kode program. pertama-tama import library iostream(input output stream) ke dalam program dengan perintah #include <iostream>. Fungsi library ini adalah untuk menerima input dan mencetak output. Kemudian buatlah procedure/function bernama tahunKabisat, function tersebut bertipe void (tidak mengembalikan nilai apapun) dan memiliki parameter Tahun(bertipe integer). Dalam function tahunKabisat, dibuat nested if sesuai petunjuk/alur dari algoritma. Kemudian untuk main function, deklarasikan variabel Tahun bertipe integer dan minta user untuk memasukkan angka tahunnya dengan menggunakan cin(c input). Langkah terakhir adalah memanggil function tahunKabisat dan memasukkan nilai variabel Tahun tadi kedalam function tahunKabisat. Kode program beserta output-nya dapat dilihat pada lampiran gambar.

16. Buatlah program Java nested loops..mohon bantuannya ka​


Kode Program Java

import java.util.Scanner;

public class LatihanNestedLoops {
   // Method untuk meminta masukan berupa angka
   private static int inputAngka(String teks, boolean closeScanner) {
       int tmp = 0;
       Scanner in = new Scanner(System.in);
       try {
           System.out.print(teks);
           tmp = in.nextInt();
       } catch (Exception e) {
           System.out.println("Terjadi kesalahan.");
       } finally {
           if (closeScanner) in.close();
       }
       return tmp;
   }

   // Method untuk menentukan karakter X atau O
   // dari parameter baris dan kolom
   private static char xAtauO(int b, int k) {
       if (b % 2 != 0)
           // Untuk baris ganjil, X pada kolom ganjil
           return (k % 2 != 0) ? 'X' : 'O';
       else
           // Untuk baris genap, X pada kolom genap
           return (k % 2 == 0) ? 'X' : 'O';
   }

   // PROGRAM UTAMA
public static void main(String args[]) {
       // Meminta masukan jumlah baris dan kolom
       int baris = inputAngka("Input jumlah baris: ", false);
       int kolom = inputAngka("Input jumlah kolom: ", true);
       System.out.println("=======================");
       for (int b = 1; b <= baris; b++) {
           for (int k = 1; k <= kolom; k++) {
               System.out.print(xAtauO(b, k) + " ");
           }
           System.out.println();
       }
   }
}
__________________

Pembahasan

Karena tidak ada spesifikasi yang lebih rinci, maka diasumsikan karakter X dan O dicetak bergantian setiap berganti nomor kolom, juga setiap berganti nomor baris. Jadi, pada baris bernomor ganjil, karakter X dicetak pada kolom bernomor ganjil. Pada baris bernomor genap, karakter X dicetak pada kolom bernomor genap.

Algoritma utama penentuan karakter X atau O yang akan dicetak pada baris dan kolom tertentu terletak pada method xAtauO, sehingga pada method main, kita hanya perlu memanggil method xAtauO pada loop for bagian dalam, dengan parameter variabel loopin baris dan kolomnya.

Contoh hasil eksekusi program dapat dilihat pada gambar yang disertakan. Terdapat 3 contoh kasus, yaitu untuk jumlah baris dan kolom berturut-turut 3 dan 5, 4 dan 6, serta 11 dan 15.

35. mengapa pada nested loop FOR- NEXT (perulangan di dalam perulangan) looping yang didalam akan lebih banyak dilakukan perulangan dibandingkan yang di luar ?


Nested Loop atau perulangan bersarang adalah sebuah perulangan yang terdapat di dalam perulangan. Bentuk nested loop for sendiri kurang lebih adalah seperti berikut

For i = 1 To 5

         For j = 1 To 5        


         Next

Next

For-Next di dalam akan memiliki lebih banyak perulangan karena perulangan di dalam akan diulang sebanyak jumlah perulangan yang ada di luar.

Sebagai contoh potongan kode di atas For-Next di luar hanya diulang sebanyak 5 kali sedangkan, For-Next di dalam akan diulang setiap kali perulangan di luar dilakukan sehingga terjadi 25 kali perulangan.

Jadi pada saat i = 1 terjadi 5 kali perulangan di For-Next dalam, kemudian i =2 terjadi 5 kali perulangan, begitu seterusnya hingga terjadi 5x5 perulangan untuk perulangan For-Next yang di dalam.
