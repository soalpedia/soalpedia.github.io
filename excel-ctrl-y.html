14. sebutkan semua fungsi ctrl a sampai z pada microsoft excel

Fungsi Ctrl + A adalah menyeleksi semua teks (select all)

Fungsi Ctrl + B adalah memberikan efek tebal (bold) pada huruf yang telah di blok atau pilih

Fungsi Ctrl + C adalah menyalin file atau huruf (copy)

Fungsi Ctrl + D adalah menampilkan form font, untuk memilih jenis huruf, warna, ukuran, dll

Fungsi Ctrl + E adalah membuat tulisan di tengah-tengah (center)

Fungsi Ctrl + F adalah menampilkan form pencarian (find)

Fungsi Ctrl + G adalah hampir sama dengan ctrl + F yaitu menampilkan pencarian di browser jika kita sedang membaca artikel. Pada Ms Word dapat digunakan sebagai next jika kita melakukan pencarian kata jika ditemukan beberapa kata

Fungsi Ctrl + H adalah fungsi untuk mengganti kata pada kolom pencarian dengan kata baru (replace), tetapi di Mac OS berfungsi untuk menyembunyikan aplikasi yang saat ini aktif (hide)

Fungsi Ctrl + I adalah menjadikan tulisan miring (italic)

Fungsi Ctrl + J adalah membuat paragraph/ text rata kiri dan kanan (justify)

Fungsi Ctrl + K adalah menambahkan link (hyperlink) tautan pada kata yang di blok, shape, atau object lainnya

Fungsi Ctrl + L adalah menjadikan paragraph atau text rata kiri (left)

Fungsi Ctrl + M adalah mengecilkan aplikasi (minimize) di Mac OS

Fungsi Ctrl + N adalah membuat dokumen baru (new)

Fungsi Ctrl + O adalah membuka dokumen (open)

Fungsi Ctrl + P adalah menampilkan form untuk print atau mencetak berkas

Fungsi Ctrl + Q adalah menutup aplikasi saat ini di Mac OS

Fungsi Ctrl + R adalah membuat tulisan atau paragraph rata kanan (right)

Fungsi Ctrl + S adalah menyimpan dokumen ketikan (save)

Fungsi Ctrl + T adalah Menggeser tabulasi bawah ke kanan

Fungsi Ctrl + U adalah memberikan efek garis bawah (underline)

Fungsi Ctrl + V adalah menempelkan berkas yang sudah di copy (paste)

Fungsi Ctrl + W adalah menutup file atau keluar dari aplikasi

Fungsi Ctrl + X adalah memindahkan file atau memotong berkas yang di block (cut)

Fungsi Ctrl + Y adalah mengembalikan perintah yang sebelumnya dibatalkan (redo)

Fungsi Ctrl + Z adalah membatalkan perintah yang sebelumnya dikerjakan (undo)
