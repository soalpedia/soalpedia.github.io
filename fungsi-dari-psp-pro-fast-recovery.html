bunga= tabungan awalx b/100x waktu/12

Fasilitas AutoRevovery pada MicrosoftWord adalah sebuah utilitas dimana file yang pengguna kerjakan akan di simpan secara periodik per 1 menit sehingga jika terjadi gangguan yang tiba-tiba seperti mati listrik dan hang dapat segera dikembalikan ke keadaan semula pada menit terakhir file itu disimpan.

Dalam penamaan, MSWord sendiri menghindari konflik penamaan berulang sehingga nama file dengan format *.asd (format file auto-recovery) akan berbeda dengan dengan format dengan *.wbk (format file auto-recovery).Pengguna dapat membuka file dengan format *.wbk pada destinasi penyimpanan default maupun custom.

28. Jelaskan apa fungsi backup, restore, dan recovery dalam pengaplikasiannya terhadap sistem operasi


Fungsi backup, restore, dan recovery dalam pengaplikasiannya terhadap sistem operasi adalah sebagai berikut yaitu:

Backup = adalah melakukan penyalinan data untuk mengamankan dari kehilangan data melalui penyimpanan fisik.Restore = artinya mengembalikan data yang telah hilang.Recovery = artinya adalah memulihkan sistem yang telah rusak pada sistem operasi.Penjelasan

Recovery itu hampir sama dengan backup dan restore. Ketiga kegiatan ini saling berkaitan, namun sebenarnya memiliki pengertian yang berbeda. Pencadangan data adalah tindakan pencegahan jika terjadi kerusakan atau kehilangan data.
