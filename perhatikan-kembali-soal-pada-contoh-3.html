Penjelasan:

Berdasarkan persamaan pada roket pertama: h(t) = -4(t - p)^2 + 64

Kita tahu bahwa waktu 0 detik memberikan ketinggian 0 kaki (feet), maka kita dapat menempatkan nilai t = 0 dan h = 0 ke dalam persamaan tersebut.

0 = -4(0 - p)^2 + 64

Sekarang kita dapat menyederhanakan persamaan tersebut:

0 = -4p^2 + 64

4p^2 = 64

p^2 = 16

p = ± 4

Dalam konteks ini, nilai p yang memenuhi adalah 4, karena kita mencari nilai p yang membuat roket kembali ke tanah setelah beberapa waktu dan memiliki tinggi yang sama pada waktu awal (0 detik).

Jadi, jawaban yang benar adalah D. 4.
