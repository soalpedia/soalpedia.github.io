4. berapakah hambatan pengganti pada rangkaian seri paralel dibawah ini​


Rs = 3Ω

Rt = Rs + Rp

Rt = 3Ω + 2Ω

34. dua hambatan masing-masing R1 = 4 Ohm R2 3 Ohm dan hambatan penggantinya apabila dirangkaikan paralel adalah​


1,71 Ohm

✍Pembahasan :

Diketahui :

R1 = 4 OhmR2 = 3 Ohm

Ditanya :

Besar hambatan penggantinya apabila dirangkaikan paralel adalah ??

Penyelesaian :

[tex] \frac{1}{Rtotal} = \frac{1}{R1} + \frac{1}{R2} \\ \frac{1}{Rtotal} = \frac{1}{4} + \frac{1}{3} \\ \frac{1}{Rtotal} = \frac{3 + 4}{12} = \frac{7}{12} \\ {Rtotal} = \frac{12}{7} = 1,71 \: Ohm[/tex]

jadi, hambatan total atau hambatan penggantinya adalah 1,71 ohm.
