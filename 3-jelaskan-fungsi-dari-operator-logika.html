Pengertian Operator Logika dalam PHP

Operator Logika adalah operator yang digunakan untuk membandingkan 2 kondisi logika, yaitu logika benar (TRUE) dan logika salah (FALSE). Operator logika sering digunakan untuk kodisi IF, atau untuk keluar dari proses perulangan (looping).

Untuk Penjelasan yang Lebih Lengkap ada di bawah ini.

Penjelasan:

Operator logika digunakan untuk membandingkan 2 inputan kondisi boolean, yakni true atau false. Dalam tutorial kali ini kita akan membahas tentang jenis dan cara penggunaan operator logika (Logical Operators) dalam PHP, serta memahami pengertian prinsip short-circuit dalam operasi logika PHP.

Jenis operand dalam operator logika ini adalah variabel dengan tipe boolean. Namun jika operand bukan boolean, akan “dikonversi” menjadi boolean oleh PHP (aturan “konversi” ini telah kita bahas pada tutorial tentang tipe data boolean PHP).

Saya akan membahas kode program diatas:

Pada baris 2, operasi logika yang dijalankan adalah $hasil1 = true and false, yang harusnya $hasil1 akan bernilai false (berdasarkan prinsip operator and: jika salah satu saja ada yang false, maka hasilnya adalah false)

Namun seperti yang terlihat dalam tampilan saat program dijalankan, variabel $hasil1 bernilai true!. Apa yang sebenarnya terjadi? Hal ini kembali kepada prinsip urutan prioritas operator.

Jika anda perhatikan tabel urutan operator pada tutorial urutan operator dalam PHP, operator assignment (pendefenisian variabel) yaitu menggunakan tanda sama dengan (=) memiliki priotitas lebih tinggi dari pada operator logika and. Sehingga yang sebenarnya diproses adalah ($hasil1 = true) and false, sehingga $hasil1 akan bernilai true.

Kesalahan pemograman seperti ini akan sulit dideteksi, sehingga anda disarankan menggunakan tanda kurung untuk menegaskan urutan program.

Pada baris ke-6, untuk variabel $hasil2 saya mengulangi operasi yang sama dengan baris 2, namun kali ini dengan menggunakan tanda kurung untuk memberitahukan kepada PHP bahwa operasi logikalah yang harus dijalankan pertama kali, baru setelah itu proses assigment yang kali ini berjalan sebagaimana harusnya (true and false menghasilkan false). Dan variabel $hasil2 bernilai false.

Pada baris ke-10, variabel $hasil3 bernilai true, karena operator xor akan menghasilkan true apabila salah satu operand bernilai true.

Untuk baris ke-14, $hasil4 = (false or true && false), urutan proses operasi dimulai terlebih dahulu pada operator && karena memiliki urutan prioritas lebih tinggi daripada operator or, sehingga yag diproses oleh PHP adalah (false or (true && false)), dan menghasilkan nilai false.

Pada contoh terakhir baris ke-20, operator || akan diproses terlebih dahulu, sehingga persamaanya menjadi $hasil5 = ($a and ($b || $a) or b).

Dalam membuat operasi logika ini, sedapat mungkin untuk mengujinya terlebih dahulu, karena kesalahan program pada logika akan sulit terdeteksi.

1. Operator logika Sama Dengan (=)

Penggunaan operator logika sama dengan (=) mengandung arti bahwa antara data yang ada dengan kriteria yang dimasukkan sesuai dengan penyataan logika (cocok)

2. Operator logika lebih besar (>)

Penggunaan operator logika ini mengandung arti bahwa antara data yang ada pasti lebih besar dari kriteria data yang dimasukkan. Jika sesuai maka hasilnya True dan jika tidak sesuai maka hasilnya False

3. Operator logika lebih Kecil (<)

Penggunaan operator logika ini mengandung arti bahwa antara data yang ada pasti lebih kecil dari kriteria data yang dimasukkan. Jika sesuai maka hasilnya True dan jika tidak sesuai maka hasilnya False

4. Operator logika lebih besar Sama Dengan (>=)

Penggunaan operator logika ini mengandung arti bahwa antara data yang ada sama besar atau lebih besar dari kriteria data yang dimasukkan. Jika sesuai maka hasilnya True dan jika tidak sesuai maka hasilnya False

1. Fungsi IF adalah fungsi logika dalam berbagai bahasa pemrograman dan perangkat lunak seperti Excel yang digunakan untuk melakukan pengujian kondisi. Fungsi ini memungkinkan program atau spreadsheet untuk menjalankan tindakan tertentu berdasarkan apakah suatu kondisi atau ekspresi bernilai benar (true) atau salah (false).

2. Kegunaan fungsi logika IF antara lain:

- Menggunakan kondisi untuk mengambil keputusan dalam program atau spreadsheet.

- Menampilkan hasil yang berbeda berdasarkan kondisi yang diberikan.

- Menghitung nilai berdasarkan kondisi tertentu.

3. Fungsi IF tunggal adalah penggunaan fungsi IF dengan satu kondisi atau ekspresi saja, seperti IF(kondisi, nilai_jika_benar, nilai_jika_salah).

4. Fungsi IF majemuk adalah penggunaan beberapa fungsi IF dalam satu rumus untuk mengatasi beberapa kondisi yang berbeda. Contohnya, IF(kondisi1, nilai1, IF(kondisi2, nilai2, nilai_default)).

5. Fungsi IF bertingkat adalah penggunaan beberapa pernyataan IF yang bersarang (nested) dalam satu rumus. Misalnya, IF(kondisi1, IF(kondisi2, nilai1, nilai2), nilai_default).

6. Lambang-lambang logika pada Microsoft Excel:

- "=" digunakan untuk perbandingan kesetaraan (equal).

- "<" digunakan untuk perbandingan kurang dari (less than).

- ">" digunakan untuk perbandingan lebih dari (greater than).

- "<=" digunakan untuk perbandingan kurang dari atau sama dengan (less than or equal to).

- ">=" digunakan untuk perbandingan lebih dari atau sama dengan (greater than or equal to).

- "<>" digunakan untuk perbandingan tidak sama dengan (not equal to).

7. Beberapa operator logika yang pernah dipelajari adalah "AND" (dan), "OR" (atau), "NOT" (bukan), "IF" (jika).

8. Fungsi logika adalah perangkat matematika atau perangkat lunak yang digunakan untuk mengambil keputusan berdasarkan kondisi atau perbandingan tertentu.

9. Operator logika adalah simbol atau kata kunci yang digunakan untuk menggabungkan kondisi atau ekspresi dalam suatu pernyataan logika. Lambang logika adalah karakter yang digunakan untuk menggambarkan operasi perbandingan seperti kesetaraan, lebih besar, atau kurang dari.

10. Contoh fungsi IF dalam Excel:

Dalam sebuah spreadsheet Excel, Anda dapat menggunakan fungsi IF untuk menentukan apakah suatu siswa lulus ujian dengan rumus seperti IF(nilai >= 60, "Lulus", "Tidak Lulus"). Rumus ini akan menghasilkan "Lulus" jika nilai siswa sama dengan atau lebih besar dari 60, dan "Tidak Lulus" jika nilai di bawah 60.
