Dari fungsi kuadrat y = 2x² -12x + 16, kita akan mencari nilai x dengan cara pemfaktoran, langkah selanjutnya adalah mencari titik puncak sehingga diperoleh panjang dan tinggi segitiga.

Pembahasan

2x² - 12x + 16

(2x - 4) (x - 4)

x = 0 dan x = 4

Jadi titik potong = (2, 0) dan (4, 0)

Langkah selanjutnya mencari titik puncak

2x² - 12x + 16 dimana a = 2, b= -12 dan c = 16

Xp = - b/2a = - (-12)/ (2x2) = 3

Yp = (b²-4ac)/-4a

    = {(-12)² - 4.2.16}/-4.2

    = (144 - 128)/-8

    = -2

jadi koordinat titik puncak = (3, -2)

Dari koordinat diatas (2, 0), (4,0) dan (3, -2), maka diketahui panjang sisi segitiga = 2 satuan dan tinggi = 2 satuan

Luas segitiga = 1/2 x alas x tinggi

                      = 1/2 x 2 x 2

                      = 2 satuan

Kesimpulan

Jadi luas segitiga tersebut adalah 2 satuan. Untuk lebih mudah mencari panjang sisi dan tinggi segitiga, kita bisa menggambarkan titik-titik koordinat yang diperoleh pada bidang datar cartesius.

Menghitung Titik Potong dan Titik Puncak Fungsi Kuadrat

Untuk menghitung luas segitiga yang dibentuk oleh dua titik potong dan titik puncak, kita perlu melihat koordinat titik potong sumbu x dan titik puncak fungsi parabola tersebut.

Jika diberikan suatu fungsi kuadrat, disebut juga fungsi parabola, kita dapat menghitung titik potong terhadap sumbu x dan titik puncaknya dengan rumus berikut.

Rumus menghitung titik potong sumbu x pada fungsi kuadrat:

Masukkan nilai y=0 pada fungsi kuadrat membentuk persamaan kuadrat f(x) = 0, akar-akar x adalah nilai x pada titik potong tersebut.

[tex]A(x,y), B(x,y)=(x_{1}, 0), (x_{2}, 0)[/tex]

Rumus menghitung titik puncak fungsi kuadrat:

[tex]P(x_{P}, y_{P}) = (\frac{-b}{2a}, -\frac{b^{2}-4ac }{4a})[/tex]

Jawaban

Titik potong:

[tex]2x^{2} -12x+16=0\\2(x^{2}-6x+8)=0\\2(x-4)(x-2)=0\\(2x-8)(x-2)=0\\x=4, x=2[/tex]

Maka, didapatkan dua titik potong: [tex]A(4, 0)[/tex] dan [tex]B(2, 0)[/tex]

Titik puncak:

Dari fungsi kuadrat [tex]y=2x^{2} -12x+16[/tex], didapatkan [tex]a=2, b=-12, c=16[/tex]

[tex]P(x_{P}, y_{P}) = (\frac{-(-12)}{2(2)}, -\frac{(-12)^{2}-4(2)(16) }{4(2)})[/tex]

[tex]P(x_{P}, y_{P}) = (\frac{12}{4}, -\frac{144-128}{8})[/tex]

[tex]P(x_{P}, y_{P}) = (3, 2)[/tex]

Untuk melihat grafik segitiga yang dibuat dapat lihat di bawah ya.

Luas Segitiga

Untuk menghitung luas segitiga diperlukan panjang alas dan tinggi. Bagaimana menghitung alas dan tinggi segitiga yang dibentuk titik potong dan titik puncak yang kita hitung sebelumnya?

Menghitung panjang alas

[tex]a = |x_{B} - x_{A}| \\a = |2-4|\\a = 2[/tex]

Menghitung panjang tinggi

[tex]t=y_{P}\\t=2[/tex]

Luas segitiga

[tex]L = \frac{a*t}{2} \\L = \frac{2*2}{2} \\L = 2 satuan[/tex]

Maka, luas segitiga tersebut adalah 2 satuan luas.

y = (-122 – 4. 2. 16) / 8  y = 2

 (Gambar 1 )

Tinggi segitiga = 2

Kesimpulan

30. Persamaan sumbu simetri dari grafik fungsi kuadrat f(x) = 2x2 – 12x + 3 adalah


Persamaan sumbu simetri dari grafik fungsi kuadrat f(x) = 2x² – 12x + 3 adalah x = 6.

Pembahasan

Fungsi  kuadrat mempunyai bentuk umum :

f(x) = ax² + bx + c

x adalah variabel. Sedangkan a, b, c adalah bilangan ∈ R dan a ≠ 0. Bila digambar, fungsi kuadrat membentuk grafik berupa parabola. Pada grafik parabola tersebut dapat dilihat titik puncak dan sumbu simetri.

Nilai a menentukan arah grafik.

Bila a > 0 maka grafik terbuka ke atas.

Bila a < 0 maka grafik terbuka ke bawah.

Persamaan sumbu simetri parabola dirumuskan dengan :

[tex]\boxed{x_p=-\frac{b}{2a}}[/tex]

Sedangkan koordinat titik puncak grafik dirumuskan dengan :

[tex]\boxed{(x_p,~y_p)=(-\frac{b}{2a},-\frac{b^2-4ac}{4a})}[/tex]

==================================================

Diketahui :

Grafik fungsi kuadrat f(x) = 2x² – 12x + 3

Ditanya :

Persamaan sumbu simetri

Jawab :

f(x) = 2x² – 12x + 3

a = 2, b = -12, c = 3

Persamaan sumbu simetri = [tex]x_p=-\frac{b}{2a}[/tex]

[tex]x=-\frac{-12}{2(1)}\\\\x=-\frac{-12}{2}\\\\x = 6[/tex]

Jadi persamaan sumbu simetri dari grafik fungsi kuadrat f(x) = 2x² – 12x + 3 adalah x = 6.

35. Dari fungsi kuadrat y = 2x² - 12x + 16 akan dibuat suatu segitiga. Titik - titik sudut segitiga tersebut merupakan titik potong sumbu x dan titik puncak. Tentukan luas segitiga tersebut.


Dari fungsi kuadrat y=2x²-12x+16 akan dibuat suatu segitiga. Titik sudut segitiga tersebut merupakan titik potong sumbu-x dan titik puncak. Tentukan luas segitiga tersebut!JawabanPendahuluan

Dari fungsi kuadrat y = 2x² -12x + 16, kita akan mencari nilai x dengan cara pemfaktoran, langkah selanjutnya adalah mencari titik puncak sehingga diperoleh panjang dan tinggi segitiga.

Pembahasan

2x² - 12x + 16

(2x - 4) (x - 4)

x = 0 dan x = 4

Jadi titik potong = (2, 0) dan (4, 0)

Langkah selanjutnya mencari titik puncak

2x² - 12x + 16 dimana a = 2, b= -12 dan c = 16

Xp = - b/2a = - (-12)/ (2x2) = 3

Yp = (b²-4ac)/-4a

    = {(-12)² - 4.2.16}/-4.2

    = (144 - 128)/-8

    = -2

jadi koordinat titik puncak = (3, -2)

Dari koordinat diatas (2, 0), (4,0) dan (3, -2), maka diketahui panjang sisi segitiga = 2 satuan dan tinggi = 2 satuan

Luas segitiga = 1/2 x alas x tinggi

                      = 1/2 x 2 x 2

                      = 2 satuan

Kesimpulan

Jadi luas segitiga tersebut adalah 2 satuan. Untuk lebih mudah mencari panjang sisi dan tinggi segitiga, kita bisa menggambarkan titik-titik koordinat yang diperoleh pada bidang datar cartesius.
