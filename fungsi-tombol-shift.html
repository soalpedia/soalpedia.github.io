6. jelaskan fungsi tombol shift pada keyboard?


Fungsi tombol shift pada tombol keyboard antara lain :

Membuat huruf kapital dan menjadikan lowercase pada kondisi capslock onSebagai shortcut membuka Recycle BinSebagai cara menghilangkan tampilan loading ketika membuka Microsoft OfficeSebagai shortcut mencari suatu kata/kalimat pada website dan dokumenSebagai cadangan shortcut jika tidak dapat melakukan klik kanan pada mouseSebagai shortcut pada browser untuk mengakses kembali link yang tertutup baru sajaSebagai sarana tindakan represif saat komputer hang/freeze/error

Pembahasan Membuat huruf kapital dan menjadikan lowercase pada kondisi capslock on sekaligus mengakses simbol diatas keyboard angka

Pada kondisi Capslock off, jika kita menekan shift lalu menekan salah satu huruf dikeyboard, maka akan muncul menjadi huruf kapital pada komputer. Hal tersebut akan berlaku sebaliknya jika capslock dalam kondisi on, maka huruf kapital akan menjadi lowercase.

Pada keyboard, simbol diatas angka dapat dimunculkan dengan menekan shift + angka maka akan muncul simbol yang ada diatas tombol angka di keyboard.

Sebagai shortcut menghapus permanen file

Dengan menekan tombol Shift + Delete / Del pada file yang sudah di select, file tersebut akan terhapus secara permanen dan tidak akan ditemukan di Recycle Bin

Sebagai cara menghilangkan tampilan loading ketika membuka Microsoft Office

Tampilan loading saat membuka microsoft office dapat dihilangkan selain menggunakan tombol minimize adalah dengan menekan shift beberapa lama sebelum membuka program.

Sebagai shortcut mencari suatu kata/kalimat pada website dan dokumen

Biasanya dalam mencari kata/kalimat kita menggunakan Ctrl + F, akan tetapi dapat dilakukan juga dengan tombol Shift yaitu kombinasi Shift + F3

Sebagai cadangan shortcut jika tidak dapat melakukan klik kanan pada mouse

Jika sewaktu - waktu mouse maupun touchpad kita memiliki kendala untuk mengakses 'klik kanan' maka dapat dilakukan dengan menekan tombol Shift + F 10

Sebagai shortcut pada browser untuk mengakses kembali link yang tertutup baru saja

Setelah menutup tab yang terdapat link secara sengaja maupun tidak dan ternyatak kita membutuhkan kembali dapat dilakukan dengan menekan Shift + Ctrl + T, maka link sebelumnya yang tertutup akan kembali

Sebagai sarana tindakan represif saat komputer hang/freeze/error

Saat komputer mengalami hang/freeze/error dapat diatasi dengan menekan Ctrl + Shift + Esc. Cara ini cukup mirip dengan penggunaan Ctrl + ALt + Del yang mengakses Task Manager maupun Alt + F4 dalam melakukan force close program

12. Apa fungsi dari tombol Shift +panah kanan dan shift+home? ​


Fungsi Shortcut Shift + Panah kanan:

Fungsinya adalah untuk select setiap huruf yang sudah diketik. Saat menekan tombol Shift + Panah Kanan secara bersamaan, tulisan yang kita ketik akan ter-select setiap hurufnya / satu-satu huruf.

Fungsi Shortcut Shift + Home:

Fungsinya adalah untuk select tulisan sampai ke awal tulisan / teks yang sudah diketik. Begitu pula sebaliknya, saat menekan tombol Shift + End, tulisan yang akan ter-select adalah sampai bagian akhir suatu teks.

27. Jelaskan fungsi kombinasi tombol keyboard “shift +F3”


Kombinasi tombol keyboard "shift + F3" biasanya digunakan untuk mengubah mode huruf dalam sebuah dokumen. Apabila tombol ini ditekan, akan terjadi salah satu dari tiga hal berikut:

Jika teks yang dipilih dalam dokumen adalah huruf kecil, maka teks tersebut akan diubah menjadi huruf besar.

Jika teks yang dipilih dalam dokumen adalah huruf besar, maka teks tersebut akan diubah menjadi huruf kecil.

Jika teks yang dipilih dalam dokumen adalah huruf besar dan kecil secara bergantian, maka teks tersebut akan diubah menjadi huruf kapital (huruf besar di awal kata dan huruf kecil di sisanya).

Perhatikan bahwa kombinasi tombol "shift + F3" hanya akan bekerja pada teks yang telah dipilih dalam dokumen. Jika tidak ada teks yang dipilih, tombol ini tidak akan memiliki efek apa pun.

Selain itu, kombinasi tombol "shift + F3" mungkin juga dapat digunakan untuk fungsi lain tergantung pada aplikasi yang sedang Anda gunakan. Misalnya, dalam aplikasi presentasi seperti Microsoft PowerPoint, tombol ini mungkin juga dapat digunakan untuk mengubah mode teks dari judul menjadi bullet points atau sebaliknya.

Jawaban:

Untuk mengubah huruf kecil (lowercase) menjadi huruf besar (uppercase) dan sebaliknya dalam aplikasi teks editor serupa Microsoft Word.

Penjelasan:

Shift + F3 adalah kombinasi tombol keyboard yang digunakan dalam program teks-editor seperti Microsoft Word. Saat menekan tombol ini, itu akan mengubah modus case (huruf besar atau kecil) dari teks yang dipilih atau dibungkus. Jika teks saat ini berada dalam mode huruf besar (uppercase), tombol ini akan mengubahnya menjadi huruf kecil (lowercase). Jika teks saat ini berada dalam mode huruf kecil, tombol ini akan mengubahnya menjadi huruf besar (uppercase).

Misalnya, jika memiliki teks "HELLO WORLD" yang dipilih dan menekan Shift + F3, teks tersebut akan berubah menjadi "hello world". Jika kemudian menekan Shift + F3 lagi, teks tersebut akan berubah menjadi "HELLO WORLD". Dan jika menekan Shift + F3 lagi, teks tersebut akan berubah menjadi "hello world" lagi, dan seterusnya...
