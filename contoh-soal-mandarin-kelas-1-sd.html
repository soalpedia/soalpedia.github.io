PEMBAHASAN Sedikit tambahan untuk angka

• 二 juga bisa jadi 两 untuk satuan puluhan. Contoh : 200 = 两白

Angka - angka dalam bahasan mandarin

一 Yī

Arti : Satu / pasangan untuk sebuahContoh : 我有一本书 (saya punya sebuah buku) 二 ÈrArti : duaContoh : 二年 (Bulan Febuari)

两

Arti : dua Contoh : 两只老虎 (dua ekor singa)

Untuk angka lainya dapat digunakan sama seperti contoh pada 二 dan 一 contoh :

三本书，四年，dll.

32. Di dalam kelas ada 35 murid, 19 mengambil Mandarin, 18 mengambil Inggris dan 3 tidak memilih. Hitung berapa memilih 1) Mandarin dan Inggris 2) Inggris 3) Mandarin. Ini soal venn diagram tlg dijawab


Himpunan

Total Murid = Mandarin + Inggris + Keduanya + Tidak Keduanya

35 = ( 19 - x ) + ( 18 - x ) + x + 3
35 = 19 + 18 + 3 - x
35 = 40 - x
x = 40 - 35
x = 5

1). Mandarin dan Inggris adalah 5 orang
2). Inggris adalah (18-5) = 13 orang
3). Mandarin adalah (19-5) = 14 orang
