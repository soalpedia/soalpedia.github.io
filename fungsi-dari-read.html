4. Meskipun sama-sama memiliki status sebagai penyimpanan data, secara teori RAM dan ROM memiliki fungsi, tugas, dan cara kerja berbeda. Apa saja perbedaan antara Random Access Memory (RAM) dan Read Only Memory (ROM)?​


Jawaban:

1.Random acces memory(ram)

RAM merupakan memori yang berfungsi menyimpan sementara perintah dan data pada saat sebuah program dijalankan.

Perintah dan data tersebut mencakup data yang akan dibaca dari hard disk, data-data yang dimasukan melalui alat input komputer, dan juga data-data hasil pemerosesan sebuah program. Data pada memori ini aksesnya secara acak dapat dibaca, ditulis, dan dihapus beberapa kali.

2.Read only memory (rom)

ROM (Read Only Memory) adalah memori non-volatile atau perangkat keras komputer yang digunakan untuk menyimpan perintah data secara tetap. Sesuai dengan namanya, komputer hanya dapat membaca data dan perintah yang terdapat di ROM. Komputer tidak dapat merubah data dan perintah yang disimpan pada ROM.

Jawaban:

1. Output dari algoritma "PERBANDINGAN" di atas akan menjadi 20.

Langkah-langkahnya sebagai berikut:

Read A: Nilai A diberikan sebesar 10.

Read B: Nilai B diberikan sebesar 20.

Read C: Nilai C diberikan sebesar 15.

MAX-A: Nilai MAX diisi dengan nilai A yaitu 10.

IF B-MAX THEN MAX-B: Karena 20 > 10, maka MAX diisi dengan nilai B yaitu 20.

IF C-MAX THEN MAX-C: Karena 15 < 20, maka nilai MAX tidak berubah.

WRITE MAX: Output dari algoritma adalah 20.

2. Tujuan atau fungsi dari algoritma "PERBANDINGAN" di atas adalah untuk menentukan nilai maksimal dari tiga bilangan A, B, dan C. Algoritma ini membaca tiga nilai, lalu membandingkan masing-masing nilai satu per satu dan memperbarui nilai maksimal sampai semua nilai sudah dibandingkan. Setelah semua nilai sudah dibandingkan, nilai maksimal akan dituliskan sebagai output.

3. BUAT FLOWCHART SENDIRI, MALES GUA. deskripsi dari flowchart dari algoritma "PERBANDINGAN" di atas:

Start: Menandai mulai dari algoritma.

Read A: Membaca nilai A.

Read B: Membaca nilai B.

Read C: Membaca nilai C.

MAX-A: Menentukan nilai A sebagai nilai maksimal.

IF B-MAX THEN MAX-B: Membandingkan nilai B dan MAX, jika B > MAX, nilai B akan menjadi nilai maksimal.

IF C-MAX THEN MAX-C: Membandingkan nilai C dan MAX, jika C > MAX, nilai C akan menjadi nilai maksimal.

Write MAX: Menuliskan nilai MAX.

End: Menandai akhir dari algoritma.

18. Jelaskan dan sebutkan definisi dan fungsi dari : 1. Uses Crt 2. Clrscr 3. Begin 4. Gotoxy 5. Read 6. Readln 7. Write 8. Writeln 9. Var 10. Const


1. Uses Crt : menggunakan library/unit Crt, yang di dalamnya terdapat beberapa fungsi yang dapat digunakan untuk penanganan screen dan keyboard. contoh-contoh fungsinya clrscr, gotoxy, textcolor, textmode, dll.

2. Clrscr : membersihkan layar

3. Begin : memulai program

4. Gotoxy : memindahkan kursor ke posisi (x,y)

5. Read : membaca atau menerima masukan sampai ditemukan spasi, sehingga kursor ke samping

6. Readln : membaca atau menerima masukan sampai ditemukan karakter newline, sehingga kursor ke bawah

7. Write : mencetak ke layar tanpa diikuti karakter newline, sehingga kursor berada di samping

8. Writeln : mencetak ke layar dengan diakhiri karakter newline '\n', sehingga kursor berada di bawah

9. Var : mendeklarasikan variabel (nilainya bisa berubah-ubah)

10. Const: mendeklarasikan konstanta (nilainya tidak bisa diubah)

f(n) = h(n)

[ ubah x menjadi n ]

3n - 20 = 7n + 12.

3n - 7n = 12 + 20

-4n = 32

n = 32 / (-4)

n = -8 ✔️

Jadi, nilai n adalah -8.
