2. Berikan contoh soal peluang tentang kartu bridge dengan pembahasan


 1.Terdapat 1 set kartu bridge sebanyak 52 Buah. Jika seorang mengambil 2 kartu secara acak Dua kali berturut-turut.
Tentukan peluang munculnya satu kartu As dan satu kartu Jack Jika kartu yang sudah di ambil tidak boleh dikembalikan lagi?

Jawab :

Jika A = Kartu pengambilan pertama
Maka P(A) = n(A)/n(S) = 4/52
Jika B|A = Kartu pengambilan kedua setelah kartu pengambilan pertama tidak dikembailikan lagi
Maka P(B | A) = 4/51

Jadi P(A n B) = P(A) x P(B|A)

= 4/52 x 4/51

= 16/2652

= 4/663
