2. Apakah fungsi dari MAC Access Filtering ,MAC Filtering, MAC APBlockir , NIC Filtering ,dan MAC Security ?


[jawaban]

Mata Pelajaran: Teknologi Informasi

• Jawaban pendek:

Apakah fungsi dari MAC Address Filtering ,MAC Filtering, MAC APBlockir , NIC Filtering ,dan MAC Security ?
1. MAC Address Filter berfungsi untuk membantu mencegah pengguna asing (tidak diinginkan) yang berniat untuk mengakses masuk ke jaringan router nirkabel.

2. MAC Filtering berfungsi untuk melakukan penyaringan (filter) pada MAC address tertentu dan membuat aturan apakah suatu MAC address dapat mengakses jaringan wireless atau tidak.

3. MAC APBlockir berfungsi untuk memblacklist / memblokir AP Tertentu

4. MAC Security berfungsi untuk melindungi PC dari serangan virus atau malware dari luar.


• Jawaban panjang: 

MAC Address Filter
MAC Address Filter ini berfungsi untuk membantu anda untuk mencegah pengguna asing (tidak diinginkan) yang berniat untuk mengakses masuk ke jaringan router nirkabel anda. Dengan menerapkan fitur ini, maka hanya perangkat nirkabel yang memiliki alamat MAC yang telah terdaftar (ditetapkan) saja yang dapat memperoleh akses ke router nirkabel

MAC Filtering
MAC filtering adalah suatu utility pada access point untuk melakukan filter pada MAC address tertentu dan membuat aturan apakah suatu MAC address dapat mengakses jaringan wireless atau tidak.

3. apakah kegunaan MAC filtering?


Fungsi utama dari MAC Address  

           memudahkan untuk router dan perangkat jaringan lainnya untuk secara unik mengidentifikasi setiap jenis perangkat yang melekat pada jaringan dalam beberapa cara.            MAC address filtering adalah memblokir client berdasarkan alamat MAC.

MAC adalah singkatan dari Media Access Control. Yaitu karakter atau alamat unik yang membedakan setiap perangkat jaringan yang terpasang dikomputer kita. MAC juga sering disebut sebagai alamat fisik wireless atau LAN card.

15. Ada TP-LINK tl-wr840n.1. Sebutkan cara setting IP Network!2. Sebutkan cara setting IP LAN!3. Sebutkan cara setting routing!​


1. Cara setting IP Network pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router dengan mengetik alamat IP router (biasanya 192.168.0.1 atau 192.168.1.1) di URL.Masukkan username dan password default yang tercantum pada manual atau label pada router.Pilih "Network" atau "LAN" dari menu utama.Ubah pengaturan IP Network sesuai kebutuhan, misalnya dengan mengubah alamat IP, subnet mask, dan gateway.Klik "Save" atau "Apply" untuk menyimpan perubahan.

2. Cara setting IP LAN pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router.Masukkan username dan password default.Pilih "LAN" dari menu utama.Ubah pengaturan IP LAN sesuai kebutuhan, misalnya dengan mengubah alamat IP dan subnet mask.Klik "Save" atau "Apply" untuk menyimpan perubahan.

3. Cara setting routing pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router.Masukkan username dan password default.Pilih "Routing" dari menu utama.Pilih tipe routing yang dibutuhkan, seperti Static Routing atau Dynamic Routing.Masukkan informasi yang dibutuhkan untuk setiap tipe routing, misalnya destination IP, subnet mask, gateway, dan metric.Klik "Save" atau "Apply" untuk menyimpan perubahan.

note: Langkah-langkah ini mungkin berbeda tergantung dari versi firmware router. Pastikan untuk membaca manual pengguna atau dokumentasi produk untuk informasi lebih rinci.

25. langkah konfigurasi Access Point tp-link​

Siapkanlah terlebih dahulu Access Point, kabel LAN, dan juga power dan setelah itu koneksikanlah kabel LAN pada komputer. Bukalah browser seperti Firefox, lalu ketikkanlah IP Address Access Point pada address bar lalu tekan enter.Masukkanlah username dan juga password dari Access Point. Kemudian, akan masuk ke halaman Access Point. Pada tempat ini telah digunakan untuk me-manage ataupun mengkonfigurasi segala macam pengaturan dan juga fungsi dari Access Point kemudian klik Next. 28. 4. Jika ingin melakukan filtering terhadap akses jaringan luar (internet) terhadap internet, maka kita bisa memblokir jaringan tersebut berdasarkan ... A. IP Address B. content C. MAC Address​


Jawaban:

A. IP Address.

Ketika ingin melakukan filtering terhadap akses jaringan luar (internet), kita bisa memblokir jaringan tersebut berdasarkan IP Address. Setiap perangkat yang terhubung ke jaringan internet memiliki alamat IP Address yang unik, sehingga dengan memblokir IP Address yang digunakan oleh jaringan yang ingin diblokir, maka akses dari jaringan tersebut ke internet bisa dibatasi.

Pilihan B, content, sebenarnya digunakan untuk melakukan filtering terhadap konten yang diakses oleh pengguna, seperti menghindari akses ke situs yang mengandung konten negatif atau berbahaya. Sedangkan pilihan C, MAC Address, digunakan untuk membatasi akses dari perangkat tertentu ke jaringan, seperti membatasi akses ke jaringan WiFi hanya untuk perangkat yang telah terdaftar (terdaftar dalam daftar MAC Address yang diperbolehkan).
