9. Bagaimana caranya mengatasi micro memory card yang write protected?


Pemecahannya:
.Kemungkinan MMC ataupun flash disk anda terkunci. Hal ini sering terjadi karena pada saat kita memasukkan atau mencabut MMC ataupun flash disk secara tidak sengaja menggeser tombol(kecil) slider ke posisi Lock (coba periksa apakah tombol slider pada posisi Lock atau Unlock). Geser slider ke posisi Unlock.
.Kemungkinan terinfeksi virus atau sebangsanya. Coba lakukan full scan terhadap MMC/ flash disk mengunakan antivirus
.Jika cara diatas tidak berhasil, kita coba masuk ke Registry Editor. 
Caranya: Klik Start >> Run>> Pada text box ketikkan regedit lalu klik Ok atau tekan Enter.Maka akan terbuka jendela registry editor. Arahkan ke path berikut ini:
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\StorageDevicePolicies
Jika key StorageDevicePolicies tidak ada , kita tambahkan secara manual. Arahkan ke path berikut:
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control
Klik kanan Control>> Pilih New > >Key> >ketikkan ( ataupun rename)> StorageDevicePolicies. 
Klik StorageDevicePolicies tadi, dan di area kosong sebelah kanan :
Klik kanan>>New>>DWORD value>>beri nama WriteProtect>> klik kanan WriteProtect>Modify>>DWORD value 0.
Tutup jendela registry editor, dan restart PC anda.Coba colokkan kembaliMMC ataupun flash disk anda tadi. Seharusnya masalah sudah bisa diselesaikan. 

Namun jika belum bisa, mau tidak mau anda harus memformatnya:
.Backup/copy file di MMC ataupun flash disk ke drive lain.
.Kita akan lakukan format dengan dua cara:
1. Download Apacer Formatting Utility, lalu extract/unzip ke folder lain, ke desktop misalnya.Klik dua kali file Start.bat dalam folder tadi , makaMMC/flash disk anda terformat
2. Forrmat menggunakan command line. 
.Klik Start>>Run>> ketikkan cmd pada text box>>klik ok atau tekan enter. Akan terbuka jendela command prompt
.Ketikkan DCHKDSK X: /F, dimana X adalah drive letter MMC ataupunflashdisk anda( misalnya: F: , G: dsb)

24. Sebuah hardware yang digunakan untuk menghubungkan jaringan antar komputer adalah. . . * Ethernet card Memory card VGA card Card reader Sound card.


Jawaban:

Ethernet card

Penjelasan:

Ethernet card adalah sebuah perangkat keras yang terpasang pada komputer, yang berfungsi untuk menghubungkan komputer dengan jaringan kabel Ethernet. Ethernet card juga dikenal dengan sebutan network interface card (NIC), yang berfungsi sebagai pengendali lalu lintas jaringan di komputer. Ethernet card biasanya terdiri dari sebuah chip kontrol yang terpasang pada sebuah kartu, yang kemudian dipasang pada slot perangkat keras komputer. Ethernet card memiliki port LAN yang digunakan untuk menghubungkan komputer dengan kabel Ethernet, sehingga komputer dapat terhubung dengan jaringan lokal atau internet. Ethernet card juga biasanya dilengkapi dengan sebuah LED yang menandakan status koneksi jaringan.

40. Bagaimana cara memformat memory card menjadi penyimpanan internal??


Masuk dengan aplikasi bertuliskan stroge/berkas lalu pindahkan memory card ke penyimpanan internal
Maaf klo salahCara mengubah MicroSD jadi penyimpanan internalUntuk mengubah MicroSD menjadi mediapenyimpanan internal sebenarnya sudah disiapkan (muncul dengan sendirinya) saat kamu pertama kali melakukan upgrade ke Android Marshmallow atau ketika kamu baru pertamakali memasukkan MicroSD ke smartphone.Disitu kamu akan diberikan opsi “Gunakan sebagai penyimpanan portabel”atau “Gunakan sebagai penyimpanan internal”. Bila ingin menjadikan sebagai memori internal, pilih opsti kedua yaitu “Gunakan sebagai penyimpanan internal”,setelah itu ikuti saja prosesnya hingga selesai (dalam prosesnya, migrasi data juga akan dilakukan).Bila opsi pengaturan diatas tidak muncul atau kamu ingin mengubah pengaturan yang sudah ditetapkan sebelumnya, cukup masuk saja ke “Pengaturan > Penyimpanan & USB > Pilih Kartu SD” setelah itu pilih “option (icon 3 titik dipojok kanan atas) > Format sebagai internal/portabel”.
