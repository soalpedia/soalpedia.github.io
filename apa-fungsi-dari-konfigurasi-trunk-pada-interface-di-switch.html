3. Apa fungsi dari IP ,router, switch, interface, access list​


IP address bisa digunakan sebagai alat atau media untuk mengidentifikasi host atau fitur antarmuka pada sebuah jaringan komputer.

Router memiliki fungsi untuk mengirimkan paket data atau suatu informasi melalui internet atau jaringan dari lokasi tertentu ke jaringan lainnya.

Switch mempunyai fungsi umum sebagai suatu concentrator yang dapat menerima serta membagikan data antara sejumlah perangkat komputer.

User Interface berperan sebagai penghubung yang memudahkan penggunanya agar dapat menjalankan suatu program, aplikasi, maupun mengunjungi website.

Access List merupakan sebuah tool yang digunakan di sisi AP (Access Point) untuk melakukan filtering koneksi dari client.

15. SOAL 3 (25%BOBOT) net 192.168.1.0 Switch 1 Switch 3 Router 1 Switch 2 net 192.168.2.0 net 192.168.3.0 jelaskan apa fungsi router pada gambar diatas dan apa baja yang di konfigurasi pada router tersebut agar semua perangkat bisa terhubung!​


Penjelasan:

Pada gambar yang diberikan, terdapat tiga jaringan yang terhubung melalui router. Fungsi utama router adalah menghubungkan jaringan yang berbeda dan mengarahkan paket data antara jaringan-jaringan tersebut. Router berfungsi sebagai penghubung antara jaringan lokal (LAN) dengan jaringan luar atau jaringan yang lebih besar, seperti Internet.

Untuk mengkonfigurasi router agar semua perangkat dapat terhubung, beberapa langkah yang dapat dilakukan antara lain:

1. Mengkonfigurasi antarmuka (interface): Setiap router harus memiliki konfigurasi antarmuka yang sesuai dengan jaringan tempatnya terhubung. Misalnya, pada Router 1, diperlukan konfigurasi pada antarmuka yang terhubung dengan Switch 1 dan antarmuka yang terhubung dengan Switch 3.

2. Memberikan alamat IP pada antarmuka router: Setiap antarmuka router yang terhubung ke jaringan lokal harus memiliki alamat IP yang sesuai dengan jaringan tersebut. Misalnya, pada Router 1, antarmuka yang terhubung ke Switch 1 dapat dikonfigurasi dengan alamat IP 192.168.1.1 dan antarmuka yang terhubung ke Switch 3 dapat dikonfigurasi dengan alamat IP 192.168.3.1.

3. Mengkonfigurasi routing: Router harus dikonfigurasi dengan tabel routing yang tepat untuk mengarahkan paket data antara jaringan yang terhubung. Misalnya, pada Router 1, perlu ditambahkan entri routing yang mengarahkan paket data dari jaringan 192.168.1.0/24 ke antarmuka yang terhubung dengan Switch 1 dan entri routing yang mengarahkan paket data dari jaringan 192.168.3.0/24 ke antarmuka yang terhubung dengan Switch 3.

4. Mengaktifkan protokol routing jika diperlukan: Jika ada lebih dari satu router dan jaringan yang kompleks, bisa digunakan protokol routing seperti OSPF (Open Shortest Path First) atau RIP (Routing Information Protocol) untuk mempermudah penyebaran informasi routing antar router.

Dengan mengkonfigurasi antarmuka, memberikan alamat IP, mengkonfigurasi routing, dan mengaktifkan protokol routing jika diperlukan, maka router dapat menghubungkan jaringan-jaringan yang berbeda dan memungkinkan semua perangkat terhubung dalam jaringan tersebut. Konfigurasi yang tepat akan bergantung pada perangkat keras dan perangkat lunak yang digunakan, serta kebutuhan jaringan yang spesifik.


16. Bagaimana cara konfigurasi interface mikrotik


Assalamualaikum wr. wb

Penjelasan:

1. Buka VirtualBox. Kemudian pilih File - Preference - Network - hilangkan centang pada DHCP Server

2. Sebelum konfigurasi interface dan IP,  pastikan disable server pada komputer dan pastikan ketiga lan card tidak ada IP addressnya. Setelah itu, buka Winbox. kemudian pilih New Terminal.

3. Untuk melihat konfigurasi lan card, ketik interface print kemudian Enter, maka akan muncul 3 jenis ethernet. Ubah nama Ether1 menjadi "Modem", Ether2 menjadi "APoint" dan Ether3 menjadi "Switch".

4. Untuk mengubah nama lan card gunakan perintah :

[admin@MikroTik] >  interface set 0 name=Modem

[admin@MikroTik] >  interface set 1 name=APoint

[admin@MikroTik] >  interface set 2 name=Switch

5. Untuk melihat hasil apakan sudah berhasil atau belum ketik interface print

6. Selanjutnya, untuk setting IP dapat menggunakan perintah :

[admin@MikroTik] > ip address add interface=Modem address=192.168.0.2 netmask=255.255.255.0  

[admin@MikroTik] > ip address add interface=APoint address=192.168.1.1 netmask=255.255.255.0

[admin@MikroTik] > ip address add interface=Switch address=192.168.2.1 netmask=255.255.255.0

Untuk melihat hasil dari ip address yang telah dibuat ketik perintah ip address print

Lakukan konfigurasi pada Router R1, sesuaikan IP Address dan interface yang digunakan serta berikan encapsulation dot1Q pada masing-masing interface guna mengaktifkan konfigurasi VLAN pada router R1, seperti gambar dibawah iniUntuk mengecheck konfigurasi pada Router R1 setelah melakukan konfigurasi di atas, maka ketikkan perintahshow running-configpada CLI Router Cisco anda dan pastikan sudah disetting IP Address dan Encapsulation dot1Q seperti gambar dibawah ini.

Selanjutnya melakukan konfigurasi VLAN / tagging VLAN pada switch manageable seperti gambar berikut:

Lakukan pengecekkan tagging vlan yang telah dilakukan tadi dengan mengetikkan perintahshow vlan briefpada CLI (Command Line Interface) swicth manageable Cisco yang dikonfigurasi tadi, dan pastikan vlan sudah di tagging aktif pada masing-masing interface seperti tampilan berikut.

Melakukan pengujian koneksi pada interface virtual yang sudah di tagging pada  interface fa0/0.x denganencapsulation dot1Q pada router R1 dengan cara melakukan ping terhadap masing-masing IP Address yang sudah dikonfigurasi sebelumnya dan pastikan success seperti gambar berikut ini.

Tahap selanjutnya pengujian koneksi dengan cara melakukan ping dari CLI Router R1 ke IP Address salah satu PC client yang terdapat pada masing-masing jaringan VLAN dan pastikan koneksi success seperti gambar di bawah ini.

PENGERTIAN: Trunk adalah komunikasi yang dirancang untuk membawa banyak sinyal secara bersamaan dengan tujuan untuk menyediakan akses jaringan antara dua titik. Jika mengacu pada istilah komunikasi, trunking atau trunk adalah sebuah konsep di mana sistem komunikasi mampu menyediakan akses jaringan untuk banyak klien dengan saling berbagi satu set garis (aturan) atau frekuensi, artinya tidak diberikan secara individual.

-------------------------------------------------

FUNGSI NYA : untuk menjadi penghubung antara dua switch yang telah terkonfigurasi VLAN. Trunk dapat diibaratkan sebagai batang pohon yang berfungsi menghubungkan ranting-ranting pada pohon. Sementara ranting-ranting tersebut adalah VLAN pada konsep trunk ini. Selain menjadi penghubung, fungsi lain dari trunk adalah untuk membatasi akses antar jaringan, sehingga tidak sembarang komponen dapat melakukan pengaksesan terhadap komponen lain di dalam jaringan lain.
