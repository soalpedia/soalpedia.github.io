26. 1. Perbedaan dedicated file server dengan non dedicated file server?.


Jawaban:

Perbedaan utama antara file server yang didedikasikan dan yang tidak didedikasikan adalah ketersediaan sumber daya. File server yang didedikasikan memiliki seluruh sumber daya yang ditentukan untuk tujuannya, sementara file server yang tidak didedikasikan berbagi sumber daya dengan tugas lain dalam jaringan.


Penjelasan:

File server yang didedikasikan hanya digunakan untuk menangani transaksi dan kegiatan file server, dan memiliki seluruh sumber daya yang diperlukan untuk melakukannya. Sementara itu, file server yang tidak didedikasikan berbagi sumber daya dengan aplikasi dan tugas lain dalam jaringan, membatasi performa dan pertumbuhan file server.

30. File yang digunakan sebagai etak file konfigurasi alamat IP di Debian adalah File yang digunakan sebagai tempat konfigurasi utama dari proxy server adalah File yang digunakan sebagai tempat konfigurasi utama dari mail server adalah File yang digunakan sebagai tempat konfigurasi utama dari DHCP server adalah


file untuk konfigurasi IP address bernama "interfaces", yang terletak di /etc/network/

file untuk konfigurasi proxy bernama "squid.conf", terletak di "/etc/squid/"

kalo untuk mail server, bukan webmail server maka file tersebut bernama "main.cf", filenya terletak di "/etc/postfix/"

file untuk DHCP yaitu "dhcpd.conf", file tsb terletak di "/etc/dhcp3/"

nah itu nama-nama file utama untuk service yang sering digunakan untuk server pada debian, semua file akan ada apabila service tersebut sudah terinstall, jadi pastikan servicenya sudah perinstall terlebih dahulu sebelum mengkonfigurasi semua file tersebut.

38. tugas dari file server


File server adaldah sebuah layanan penyimpanan file atau data yang terpusat dalam sebuah komputer server.
