Penjelasan:

1. GROUP TABLE OF CONTENTS  Pada Group Table Of Contents         ini terdapat 3 menu, yaitu:  • Table Of Contents.  Menu ini digunakan untuk pengolahan daftar isi secara otomatis. •  Add Text.  Menu ini digunakan untuk mengelola paragraf terpilih (sebagai inputan dalam daftar isi yang sudah dibuat). •  Update Table.  Menu ini digunakan untuk memperbarui daftar isi.  

2. GROUP FOOTNOTES Pada Group Footnotes ini terdapat 3 menu, yaitu:  •  Insert Footnote.  Menu ini digunakan untuk menyisipkan catatan kaki (rujukan yang berada dibagian bawah halaman atau footer). •  Next Footnote.  Menu ini digunakan untuk memantau footnote dan endnote yang terdapat diseluruh dokumen aktif. •  Show Notes.  Menu ini digunakan untuk memperlihatkan lokasi catatan, baik footnote maupun endnote.  

3. GROUP CITATION AND BIBLIOGRAPHY Pada Group Citation and Bibliography ini terdapat 4 menu, yaitu:  •  Insert Citation.  Menu ini digunakan untuk menyisipkan kutipan langsung pada daerah kursor aktif. •  Manage Sources.  Menu ini digunakan untuk mengelola seluruh seluruh sumber kutipan yang sudah disisipkan disemua bagian/segmen dkumen. •  Style.  Menu ini digunakan untuk memilih bentuk bilblography. •  Bilbliography.  Menu ini digunakan untuk menyisipkan daftar pustaka ke dalam file dokumen aktif.  

4. GROUP CAPTION Pada Group Caption ini terdapat 3 menu, yaitu:  •  Insert Table Of Figures.  Menu ini digunakan untuk menyisipkan daftar isi gambar yang disisipkan ke dalam dokumen. •  Update Table.  Menu ini digunakan untuk melakukan perubahan terhadap daftar katalog gambar. •  Cross-Reference.  Menu ini digunakan untuk menyisipkan referensi silang.

 5. GROUP INDEX Pada Group Index ini terdapat 3 menu, yaitu:  •  Mark Entry.  Menu ini digunakan untuk menandai masukan baru dalam dokumen. •  Insert Index.  Menu ini digunakan untuk menyisipkan daftar kata ke dalam dokumen. •  Update Index.  Menu ini digunakan untuk merubah data terkini.  

6. GROUP TABLE OF AUTHORITIES Pada Group Table of Authorities ini terdapat 3 menu, yaitu:  •Mark Citation.  Menu ini digunakan untuk menandai kutipan. •  Insert Table of Authorities. Menu ini digunakan untuk membuat daftar kutipan. •  Update Table.  Menu ini digunakan untuk memperbaharui daftar kutipan.

1.1 Group Table Of Contents
￼Gambar Table of Contents

1.5 Group Index

berikut keterangan gambar Table OF Authorities :
