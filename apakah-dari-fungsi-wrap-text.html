6. jenis" wrap text dan penggunaannya?


Wrap text berfungsi untuk mengatur letak gambar dan foto pada lembar kerja Microsoft Word. Fitur wrap text terdiri dari beberapa Jenis yakni.

In line with text. Square. Tight. Through. Top and bottom. Behind text. In front of text. Pembahasan

Wrap text merupakan salah satu fitur yang terdapat dalam Microsoft Word. Wrap text berfungsi untuk mengatur letak gambar dan foto pada lembar kerja Microsoft Word. Dalam Microsoft Word, fitur wrap text tidak akan langsung berfungsi. Fitur wrap text akan berfungsi ketika user memasukkan gambar dan mengklik gambar pada lembar kerja Microsoft Word.

Fitur wrap text terdiri dari beberapa Jenis. Berikut penjelasan dari jenis-jenis wrap text.

In line with text. In line with text merupakan perintah yang digunakan untuk memposisikan gambar sejajar dengan garis text.

Square. Square merupakan perintah yang digunakan untuk menempatkan text mengikuti posisi gambar.

Tight. Tight adalah perintah yang digunakan untuk membuat text mengelilingi gambar.

Through. Through memiliki fungsi yang sama dengan tight, yakni membuat text mengelilingi gambar.

Top and bottom. Top and bottom adalah perintah yang digunakan untuk membuat text berada di atas dan bawah gambar.

Behind text. Behind text adalah perintah yang digunakan untuk meletakkan gambar di belakang text.

In front of text. In front of text adalah perintah yang digunakan untuk meletakkan gambar di depan text yang membuat gambar menutupi text.

37. wrap text dalam Microsoft Word adalah​


Jawaban:

Tombol wrap text tergantung dari gambar di microsoft word. Maksudnya, baru aktif setelah Anda meng- klik gambar.

Kemudian pilihlah “page layout” di menu utama atau menu “format” yang akan muncul otomatis setelah meg- klik gambar.

Cari lah tab Arrange dan pilihlah “wrap text”

Penjelasan:

Penjelasan setiap fungsi- funsi “wrap text” di microsoft word sebagai berikut:

in line with text : pilihan tampilan “wrap text” ini merupakan bawaan awal saat membuka dokumen word baru di microsoft word.

square : tampilan word text akan mengikuti di sekitar gambar yang disisipkan ke microsoft word

tight : margin antara gambar dan text di dokumen microsoft word akan menjadi lebih kecil

through

top and bottom : teks pada microsoft word akan muncul di bagian atas dan bawah gambar

behind text : gamabar yang disisipkan ke microsoft word akan muncul di belakang text. Ini seperti gambar background pada dokumen microsoft office.

in front of text : berlawanan dengan konsep “behind text” yakni gambar benar- benar membuat teks yang ada di depannya menjadi kabur.

Mungkin di bawah ini salah satu contoh dari hasil pemakaian fungsi “wrap text” pada microsoft word
