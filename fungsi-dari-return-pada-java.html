15. jelaskan perbedaan fungsi void main dengan class dalam java


Perbedaan fungsi void main dengan class dalam java adalah, void main adalah main function atau fungsi main, di mana program pertama kali dieksekusi. Sedangkan class adalah kerangka  dari setiap object individual yang akan dibuat. Program dalam class hanya akan dieksekusi ketika objek dari class itu diakses melalui program dalam main function.

Pembahasan

Fungsi void main adalah sebuah fungsi utama dalam program java yang akan dijalankan oleh program ketika dieksekusi. Fungsi main ini berisi alur program yang dibuat, yang isinya akan memanggil fungsi fungsi lainnya.

class adalah kerangka  dari setiap object individual yang akan dibuat. Berikut adalah contoh  implementasi class dalam pemrograman Java:

class Bicycle

{

int cadence = 0;

int speed = 0;

int gear = 1;

}

Pada contoh di atas variabel cadence, speed, gear merupakan state atau

keadaan dari object yang akan dibuat. Class juga memiliki method yang

berfungsi sebagai alat interaksi class dengan dunia di luar class.

Contoh membuat objek dari class dalam fungsi main

public static void main(String[] args)

{

    // Membuat dua object sepeda yang berbeda

    Bicycle bike1 = new Bicycle();

    Bicycle bike2 = new Bicycle();

}

Class bisa memiliki fungsi yang disebut method.Method merupakan suatu operasi berupa fungsi-fungsi yang dapat dikerjakan oleh suatu object. Method yang didefinisikan pada class bisa dipanggil melalui object.

Method memiliki 4 (empat) bagian dasar :

Nama metode Kembalian yang dikembalikan setelah memanggil method. Input parameter. Badan atau isi method, berupa instruksi untuk melakukan sesuatu hal.

25. Apakah fungsi utama tanda sama dengan ganda (==) pada bahasa pemrograman seperti php,java,dan python ?


Jawaban:
Dalam bahasa manusianya itu berarti sama dengan

Penjelasan:

== di dalam bahasa pemrograman bisa di artikan sebagai sama dengan contoh

variable = "jawaban"

if variable == "jawaban" {
     print("variable adalah jawaban")
} else {
     print("variable bukan jawaban")
}

Fungsi utama tanda sama dengan ganda (==) pada bahasa pemrograman seperti PHP, Java, dan Python adalah sebagai operator logika kesamaan nilai dua variabel, yang mengembalikan nilai:

true (atau 1) jika nilai kedua variabel sama, atau false (atau 0) jika nilai kedua variabel berbeda.

Walaupun fungsi utamanya sama, terdapat perbedaan dalam penggunaannya, tergantung dari implementasi dan aturan yang ditetapkan pada setiap bahasa pemrograman tersebut.
 

Pembahasan

Untuk menentukan apakah dua variabel a dan b sama nilainya, bahasa pemrograman seperti PHP, Java, dan Python (bahkan C, C++, dsb.) menggunakan operator berupa tanda sama dengan ganda (==).
Alasan utamanya adalah untuk membedakan dengan operator atau tanda yang digunakan untuk memberi nilai (assignment) pada variabel, yaitu tanda sama dengan tunggal (=).

Agar lebih jelas, kita gunakan contoh ilustrasi.

Untuk contoh ini, kita akan mengevaluasi output dari algoritma sederhana berikut pada interpreter masing-masing bahasa pemrograman tersebut.

Deklarasi sekaligus inisialisasi variabel a dengan nilai integer 10.Deklarasi sekaligus inisialisasi variabel b dengan nilai string "10".Periksa apakah a sama dengan nilai integer 10.Periksa apakah a sama dengan nilai desimal/float 10.0.Periksa apakah a sama dengan hasil penjumlahan 5 + 5.Periksa apakah a sama dengan hasil penjumlahan 4.5 + 5.5.Periksa apakah b sema dengan nilai string "10".Periksa apakah a sama dengan b.Hasil Eksekusi Pada InterpreterPHP

Pada langkah terakhir, terdapat perbedaan, yaitu interpreter PHP menganggap integer 10 sama dengan string "10", sedangkan Java dan Python tidak. Hal ini terjadi karena perbandingan kesamaan nilai variabel dengan tanda sama dengan ganda (==) pada PHP dilakukan dengan mekanisme "type-juggling", artinya membandingkan "isi" dari variabel, tanpa memperhatikan tipe data.

Jika pada langkah terakhir kita ingin mendapatkan output serupa dengan Java atau Python, pada PHP kita gunakan tanda tripel sama dengan (===). Sebagai contoh:

php > echo $a === $b;

35. tipe data nilai pada fungsi function yang dikirim balik return value dapat dilihat pada​


Jawaban:

Tipe data nilai pada fungsi function yang dikirim balik return value dapat dilihat dari deklarasi fungsi tersebut. Dalam bahasa pemrograman, deklarasi fungsi biasanya menentukan tipe data yang dikembalikan oleh fungsi tersebut. Contohnya, dalam bahasa pemrograman C++, jika fungsi "foo" dideklarasikan sebagai berikut:

int foo() {

   // ...

}

Maka tipe data nilai yang dikembalikan oleh fungsi "foo" adalah integer (int). Tipe data yang dikembalikan oleh fungsi dapat digunakan untuk memastikan bahwa nilai yang dikembalikan oleh fungsi tersebut sesuai dengan yang diharapkan oleh program yang memanggil fungsi tersebut. Misalnya, jika fungsi "foo" diharapkan untuk mengembalikan integer, maka program yang memanggil fungsi tersebut harus menangani nilai yang dikembalikan sebagai integer.
