8. Fungsi alt pada komputer


fungsi tombol alt adalah :../
•alt +f4: menutup item aktif, atau keluar dari program aktif.
•alt + enter: menampilkan properti dari objek yg dipilih
•alt + spacebar: buka menu shortcut untuk jendela aktif...

semoga membantuAlt merupakan salah satu tombol pada keyboard yang ada pada komputer yang memiliki fungsi yang berhubungan dengan sistem perintah pada Windows. Dalam penggunaannya, Alt harus dipasangkan dengan tombol lainnya untuk bisa menghasilkan fungsi yang terbaik.nah dengan demikian berikut fungsi alt juga jika digabungkan dengan tombol lain:
 pada Komputer

1. Alt + A Berfungsi untuk menampilkan dan mengeluarkan menu Table.

2. Alt + B Untuk menampilkan daftar teman.

3. Alt + C Untuk mengeluarkan kotak chat.

4. Alt + E Berfungsi sebagai perintah untuk menampilkan menu Edit

5. Alt + I Untuk menampilkan menu Insert.

6. Alt + F4 Perintah keluar dari program yang sedang dijalankan dan untuk menutuip item yang sedang aktif.

7. Alt + H Membuka dialog Help

8. Alt + W Kombinasi ini berfungsi untuk memunculkan menu Window.

9. Left Alt + Left SHIT + Print Screen Untuk memunculkan menu High contrast on atau off. Menu ini berguna untuk mengatur contrast yang akan ditampilkan pada komputer ataupun laptop.

10. Alt + Enter

Menampilkan property dari item yang ditunjuk.

11. Alt + Spacebar Untuk menampilkan menu shortcut pada jendela yang dibuka atau yang sudah aktif.

12. Alt + Tab Membuka dialog jendela yang sudah terbuka kemudian bisa memilih manakah yang akan dibuka selanjutnya.

13. Alt + F Untuk menampilkan menu File

14. Alt + Home Untuk memunculkan menu Start.

15. Alt + Ctrl + Break Untuk membuat layar penuh menjadi lebih kecil.

16. Alt + (-) tanda minus Memunculkan konsol jendela yang aktif.

17. Alt + Q Keluar dari aplikasi game.

18. Alt + O Membuka Menu Favorites.

19. Alt + Esc Melakukan perintah minimize pada jendela atau aplikasi yang dibuka.

20. Ctrl + Alt + Del Kombinasi tombol yang satu ini bisa dikatakan menjadi kombinasi yang sangat penting untuk diketahui. Fungsi kombinasi ini berugna untuk membuka dialog Taks Manager dimana dari dialog ini berguna untuk melakukan perintah mematikan dan juga melihat property dari aplikasi yang berjalan.
