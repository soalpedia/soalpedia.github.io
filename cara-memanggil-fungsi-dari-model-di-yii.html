K = 2 (p+l)

19. Jelaskan Teknik pembuatan dan pemanggilan fungsi!


Buat dulu deklarasinya, misal anda ingin input nama, maka deklarasinya char, kalau input angka deklarasinya integer. Misal = int absen; char nama[30]; Lalu pemanggilan fungsinya kalau integer menggunakan %d kalau char menggunakan %c kalau string menggunakan %s. Nama yg hendak diinput termasuk string maksudnya terdiri dr beberapa karakter. Jadi pemanggilan fungsinya : printf("Input nama : "); scanf("%s", &nama"); printf("Input nomor absen : "); scanf("%d", &absen);
