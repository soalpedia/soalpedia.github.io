16. 2 jenis fungsi perulangan bahasa pemrograman python yaitu?


Jawaban:

perulangan 'for' dan 'while'

Penjelasan:

Dalam bahasa pemrograman Python, terdapat dua jenis fungsi perulangan yang umum digunakan, yaitu:

Perulangan for: Perulangan for digunakan untuk mengulang iterasi sejumlah elemen atau item dalam urutan tertentu seperti string, list, tuple, atau rangkaian angka. Setiap elemen dalam urutan tersebut akan diproses secara berurutan. Contoh penggunaan perulangan for:

for i in range(1, 6):

   print(i)

Pada contoh di atas, perulangan for mengulang iterasi dari 1 hingga 5 dengan menggunakan fungsi range(). Setiap nilai 'i' akan dicetak.

Perulangan while: Perulangan while digunakan untuk mengulang iterasi selama kondisi tertentu terpenuhi. Perulangan while akan terus berjalan selama kondisi yang diberikan bernilai True. Contoh penggunaan perulangan while:

i = 1

while i <= 5:

   print(i)

   i += 1

Pada contoh di atas, perulangan while akan terus berjalan selama nilai 'i' kurang dari atau sama dengan 5. Setiap nilai 'i' akan dicetak, dan nilai 'i' akan bertambah satu setiap kali iterasi dilakukan.

19. Tuliskan 10 bahasa pemrograman lengkap beserta arti dan fungsinya!​


                                  Bahasa Pemrograman dan Fungsinya

1. CSS (Cascading Style Sheet), adalah sebuah bahasa pemrograman yang fungsinya diibaratkan seperti kulit dan daging pada manusia. Yang mana komponen ini dapat meng indahkan suatu kerangka tepatnya HTML.

2. JS (JavaScript), adalah sebuah bahasa pemrograman interpreter yang tidak memerlukan compiler. Pada JS, biasanya digunakan untuk membuat website menjadi lebih interaktif dan dinamis.

3. PHP (PHP: Hypertext Preprocessor) adalah sebuah bahasa pemrograman yang biasanya digunakan pada backend karena ini digunakan untuk mengurus server pada website.

4. C, adalah bahasa pemrograman yang awalnya digunakan hanya untuk sistem komputer dan jaringan. Tetapi, seiring berjalannya waktu bahasa C digunakan untuk mengembangkan software atau perangkat lunak.

5. C++, adalah bahasa pemrograman yang mendukung Object-Orented Programming yang mana ini dapat sangat berguna untuk mengerjakan proyek besar. Sebagai info tambahan, C++ adalah versi lebih eksplisit dari bahasa C.

6. C#, adalah sebuah bahasa pemrograman yang termasuk pada jenis Object-Oriented Programming hasil pengembangan Microsoft sebagai bagian dari inisiatif kerangka .NET Framework . Biasanya bahasa ini digunakan untuk membangun game yang berukuran besar namun memilki grafis yang keren.

7. Python, adalah bahasa pemrograman yang biasanya digunakan untuk menambang data. Selain itu, Python pun digunakan untuk membuat aplikasi yang memiiki efisiensi tingkat tinggi sehingga untuk membuat aplikasi yang sederhana direkomendasikan menggunakan Python.

8. Java, adalah bahasa pemrograman yang cenderung lebih mudah untuk dipahami. Untuk pemula direkomendasikan untuk mempelajari bahasa ini. Selain itu, Java sudah mendukung OOP, mendukung multiplatform, dan memiliki open source library yang sangat lengkap.

9. Kotlin, merupakan bahasa pemrograman yang memiliki interoperability 100% dengan Java, maka apapun yang bisa menggunakan Java bisa menggunakan Kotlin. Sebagai contoh, kita gunakan Kotlin JS agar dapat menggunakan bahasa JavaScript pada Kotlin.

10. Pascal, merupakan bahasa pemrograman yang membantu mewujudkan pemecahan masalah yang lebih sederhana. Sehingga program semakin mudah dibaca dan lebih mudah diubah atau dimodifikasi. Bahasa ini cenderung bahasa yang sudah melegenda karena hadir pada 1971.

25. tuliskan ragam jenis bahasa pemrograman dan fungsinya​


Jawaban:

Jenis Bahasa Pemrograman

Seperti yang kita tahu bahwa setiap bahasa programming memiliki kegunaan masing-masing. Contoh dari bahasa komputer ini, yakni untuk membuat website, software desktop, aplikasi, antivirus, dan teknologi lainnya. Ada ratusan kode yang sebenarnya dimanfaatkan untuk mendukung pembuatan bahasa programming. Agar lebih jelasnya, berikut jenis-jenis bahasa programming atau komputer yang sering digunakan.

HTML

Anda mungkin sudah sering mendengar HTML? Hyper Text Markup Language (HTML) merupakan bahasa komputer yang dimanfaatkan untuk membuat situs website. Dimana HTML tersusun atas berbagai short codes yang nantinya dijadikan sebuah text untuk disimpan dalam file HTML tersebut. File inilah yang akan dibuka pada alat browser sehingga bisa membaca file untuk menerjemahkan kode HTML. Jenis editor sederhana HTM, misalnya Sublime, Atom, atau Notepad++.

PHP

Selain HTTP, ada juga PHP sebagai bahasa komputer paling popular dalam membuat website. Pasalnya PHP berfungsi untuk mempercantik tampilan dari situs halaman web. Hanya dengan memanfaatkan PHP Anda bisa membuat fitur menarik untuk website, seperti login username dan password, membuat survey, mengecek form, dan lainnya. Anda dapat mengetahui sebuah web ada PHP, jika mendapati alamat situs website yang diakhiri dengan PHP.

JavaScript

Ini dia jenis bahasa komputer yang sangat bermanfaat untuk membuat web baru. Bahkan berguna baik untuk melihat suatu animasi, tombol klik, dan pop-up pada website. Secara tidak langsung JavaScript memang berguna untuk menjadikan tampilan web semakin interaktif dan dinamis. Penggunaan JavaScript ini harus bersamaan dengan Node JS dan server-side language sebagai kode pemrograman.

CSS

CSS atau kepanjangan dari Cascading Style Sheets yang memiliki tujuan untuk menyederhanakan pembuatan halaman website. CSS disini lebih berfungsi untuk mengontrol aspek desain, seperti mengatur jenis font, warna tulisan, jarak spasi, dan lain sebagainya. 35. apa fungsi #! di beberapa bahasa pemrograman?​


Sebenarnya simbol "#!" tidak ada dalam bahasa pemrograman manapun yang saya tau. mungkin yang kamu maksud adalah:

tanda #, kalau di python tanda "#" adalah tanda Comment yakni baris tersebut tidak akan dieksekusi oleh interpreter.

Contoh Penggunaan #:

#Ini adalah komentar dengan python.

Namun "#!" bukan sesuatu dari python ataupun bahasa pemrograman lainnya., ini adalah baris hashbang (atau shebang) yang menunjukkan penerjemah mana yang harus memproses file.

contoh penggunaan #! :

#! /usr/local/bin/python3

ini berkaitan dengan hal Linux dan UNIX bukan bahasa pemrograman...

Penjelasan Lebih Lanjut:

Baris shebang(hashbang) dalam skrip apa pun menentukan kemampuan skrip untuk dieksekusi seperti yang dapat dieksekusi mandiri tanpa mengetik python sebelumnya di terminal atau ketika mengklik dua kali di pengelola file (bila dikonfigurasi dengan benar).

Jad dapat ditarik kesimpulan bahwa skrip:

#! /usr/local/bin/python3

atau sejenisnya dalam python hanya digunakan agar skrip python bisa berjalan mandiri tanpa mengetik python di terminal/cmd.
