Carilah wilayah (domain) dan jangkauan (range) dari setiap fungsi berikut ini: a. F (x) = 1/x-1 b. F (x) = x+1/x c. F (x) = akar 16-x² d. F (x) = akar 9-x²

Jawaban Pendahuluan

Soal matematika di atas merupakan materi dari fungsi invers dan bijektif.

Pembahasan

Dalam bidang matematika, fungsi adalah pemetaan setiap anggota suatu himpunan yang terpasang tepat dengan satu anggota himpunan yang lain.

Aturan dasar matematika dari suatu fungsi adalah setiap nilai masukan/input fungsi (dari domain) hanya bisa menghasilkan satu nilai hasil keluaran/output (ke kodomain), meskipun terdapat beberapa nilai domain yang menghasilkan nilai kodomain yang sama.

Pada suatu fungsi, domain adalah himpunan nilai atau anggota yang memungkinkan pada masukan/input, kodomain adalah himpunan nilai atau anggota yang memungkinkan pada keluaran/output, sedangkan range adalah himpunan penyelesaian suatu fungsi yang didapatkan dengan memetakan semua nilai atau anggota pada domain.

Bentuk notasi fungsi adalah f sehingga pemetaan antar himpunan dituliskan menjadi f : x→y yaitu fungsi yang memetakan himpunan anggota x sebagai domain ke himpunan anggota y sebagai kodomain.

Fungsi dengan himpunan bilangan rill dinotasikan dengan x ∈ R, sehingga domain fungsi dinotasikan menjadi Df = { x | x ∈ R } dan range fungsi dinotasikan menjadi Rf = { x | x ∈ R }.

Pada umumnya, fungsi dalam operasi bentuk aljabar dinotasikan dengan f(x), dimana hasil penyelesaian dari fungsi yang memetakan x ke y ini bergantung pada himpunan penyelesaian yaitu nilai x yang diberikan. Hasil dari himpunan penyelesaian fungsi bisa dianggap sebagai nilai y sehingga f(x)=y dan pasangan nilai x dan y ditulis menjadi (x, y).

Fungsi invers atau fungsi kebalikan adalah fungsi dengan pemetaan himpunan anggota berkebalikan dari fungsi aslinya, yaitu himpunan anggota pada domain yang berubah menjadi kodomainnya dan sebaliknya.

Bentuk notasi fungsi invers adalah f⁻¹ yang umumnya dituliskan menjadi f : y→x yaitu fungsi yang memetakan himpunan anggota x sebagai domain ke himpunan anggota y sebagai kodomain.

Pada umumnya, fungsi invers dalam operasi bentuk aljabar dinotasikan dengan f⁻¹(x), dimana fungsi invers sama dengan nilai x dari fungsi aslinya atau f⁻¹(x) = x. Rumus-rumus fungsi invers adalah sebagai berikut:

[tex]f(x) = ax+b \rightarrow f^{-1}(x) = \frac{x-b}{a} \quad (x \neq a)\\ f(x) = \frac{ax+b}{cx+d}  \rightarrow f^{-1}(x) = -(\frac{dx-b}{cx-a}) \quad (x \neq \frac{a}{c}) \\ f(x) =  ax^2+bx+c \rightarrow f^{-1}(x) = \frac{-b + \sqrt{b^2 \pm 4a(c-x)}}{2a}[/tex]


Penyelesaian soal menentukan wilayah (domain) dan jangkauan (range) dari setiap fungsi adalah sebagai berikut:

f(x) = x/2

Untuk persamaan berbentuk pecahan dimana terdapat x pada pembilang atau pengali, nilai x adalah himpunan bilangan rill, sehingga domain fungsi adalah Df = { x | x ∈ R } dan range fungsi adalah Rf = { y | y ∈ R }.

f(x) = 3

Untuk persamaan hanya berupa bilangan riil, maka hanya terdapat satu domain dan range yaitu 3, sehingga domain fungsi adalah Df = { x | x = 3 } dan range fungsi adalah Rf = { y | y = 3 }.

f(x) = 1/(x-1)

Untuk persamaan berbentuk pecahan dimana terdapat x pada penyebut atau pembagi, nilai penyebut tidak boleh bernilai nol.

x - 1 ≠ 0

x ≠ 1

Sehingga domain fungsi adalah Df = { x | x ∈ R; x ≠ 1 } dan range fungsi adalah Rf = { y | y ∈ R }.

f(x) = x + 1/x

Untuk persamaan berbentuk pecahan dimana terdapat x pada penyebut atau pembagi, nilai penyebut tidak boleh bernilai nol.

x ≠ 0

Sehingga domain fungsi adalah Df = { x | x ∈ R; x ≠ 0 } dan range fungsi adalah Rf = { y | y ∈ R }.

f(x) = √(16-x^2)

Untuk persamaan berbentuk akar dimana x adalah bilangan rill, nilai x harus lebih besar dari nol atau x ≥ 0.

16-x^2 ≥ 0

-x^2 ≥ -16

x^2 ≤ 16

x ≤ 4

Sehingga domain fungsi adalah Df = { x | x ≤ 4, x ∈ R } dan range fungsi adalah Rf = { y | y ∈ R }.

f(x) = √(9-x^2)

Untuk persamaan berbentuk akar dimana x adalah bilangan rill, nilai x harus lebih besar dari nol atau x ≥ 0.

9-x^2 ≥ 0

-x^2 ≥ -9

x^2 ≤ 9

x ≤ 3

Sehingga domain fungsi adalah Df = { x | x ≤ 3, x ∈ R } dan range fungsi adalah Rf = { y | y ∈ R }.

19. Tentukan domain dan range dari fungsi-fungsi yang diberikan berikut : t(x) = akar x2 + 2x


Penjelasan dengan langkah-langkah:

Untuk menentukan domain dan range dari fungsi t(x) = √(x^2 + 2x), kita perlu memperhatikan batasan-batasan dari akar kuadrat dan ekspresi di dalamnya.

Domain mengacu pada nilai-nilai x yang diperbolehkan dalam fungsi. Dalam hal ini, ekspresi di bawah akar kuadrat harus bernilai non-negatif, yaitu x^2 + 2x ≥ 0. Kita bisa mencari titik-titik di mana ekspresi ini sama dengan nol:

x^2 + 2x = 0

x(x + 2) = 0

Dari sini kita mendapatkan x = 0 dan x = -2 sebagai titik di mana ekspresi x^2 + 2x = 0. Jadi, domain fungsi t(x) adalah semua nilai x yang memenuhi x ≥ -2, karena seiring dengan nilai x yang lebih kecil dari -2 akan menghasilkan hasil negatif di dalam akar kuadrat.

Untuk range, kita perlu mencari nilai minimum dan maksimum dari fungsi t(x). Karena akar kuadrat selalu menghasilkan nilai non-negatif, nilai minimum dari t(x) adalah 0. Untuk mencari nilai maksimum, kita dapat memperhatikan bahwa x^2 + 2x adalah bilangan bulat positif atau nol (dalam domain yang telah ditentukan). Oleh karena itu, akar kuadrat dari bilangan bulat positif atau nol akan menghasilkan bilangan bulat positif atau nol. Jadi, range fungsi t(x) adalah semua nilai t yang diperoleh saat x ≥ -2. Secara formal, range dari fungsi ini adalah [0, ∞).

Untuk f(x) = 4x + 5, domainnya adalah {x | x ∈ R} dan rangenya adalah {y | y ∈ R}. Sementara itu, untuk f(x) = x² - 4, domainnya adalah {x | x ∈ R} dan rangenya adalah {y | y ≥ -4, y ∈ R}. Ilustrasi grafik fungsinya dapat kamu lihat di lampiran.

Penjelasan dengan langkah-langkah

Domain adalah himpunan variabel independen yang kita masukkan dalam sebuah fungsi untuk mendapat rangenya. Range adalah bagian dari kodomain fungsi yang benar-benar menghasilkan variabel dependen bagi fungsi.

Dalam matematika, variabel domain umumnya adalah x dan y adalah variabel umum range. f(x) juga dapat didefinisikan sebagai f(x) = y sehingga kita tahu bahwa nilai y merupakan nilai hasil fungsi x-nya.

Domain fungsi linear satu variabel seperti f(x) = 4x + 5 adalah {x | x ∈ R} karena semua nilai riil yang mungkin dari x merupakan anggota domain fungsi linear satu variabel, begitu juga dengan rangenya yang didefinisikan sebagai {y | y ∈ R}.

Domain fungsi kuadrat seperti f(x) = x² - 4 adalah {x | x ∈ R}. Rangenya ditentukan dengan mengetahui terlebih dahulu nilai minimum/maksimum dan arah bukaan parabola fungsi kuadratnya.

Berikut adalah cara menentukan range fungsi kuadrat f(x) = x² - 4.

Diketahui:

f(x) = x² - 4a = 1b = 0c = -4

Ditanya:

Arah bukaan parabolanya adalah?Nilai minimum/maksimumnya adalah?

Jawab:

(1)

Karena nilai a > 0, kurvanya (atau parabolanya) terbuka ke atas. Dengan demikian, kita perlu mencari nilai minimum, bukannya nilai maksimum.

(2)

x = -b/2a

x = -0/(2 · 1)

x = 0

f(0) = (0)² - 4

f(0) = 0 - 4

f(0) = -4

Karena nilai minimumnya adalah -4, maka range fungsi kuadrat f(x) = x² - 4 adalah ​{y | y ≥ -4, y ∈ R} yang berarti nilai y lebih dari atau sama dengan -4 dalam himpunan bilangan riil.

36. f (x) = 1 + x² cari domain Dan range​


Jika diketahui fungsi  f(x) = 1 + x² , maka domain dan range dari fungsi tersebut adalah Df = {x| x ∈ R} dan Rf = {y| y ∈ R}.

Pembahasan

Suatu fungsi dapat dinyatakan dalam bentuk: f : x ---> f(x).

Nilai fungsi untuk setiap nilai x yang diberikan dapat dihitung dengan cara mensubstitusikan nilai x ke dalam rumus fungsi f(x).

Domain (daerah asal) fungsi adalah himpunan nilai-nilai agar suatu fungsi terdefinisi.

Daerah hasil fungsi (range) adalah himpunan nilai-nilai fungsi dari setiap anggota daerah asal (domain).

Penyelesaian

diket:

f(x) = 1 + x²

ditanya:

domain dan range fungsi f(x)....?

jawab:

- mencari domain fungsi f(x)

 Karena fungsi f(x) berbentuk fungsi kuadrat yaitu f(x) = 1 + x² , maka domain untuk fungsi f(x) adalah semua anggota real.

  - mencari range fungsi f(x)

    Karena domain fungsi f(x) semua anggota real, maka daerah hasil (range) fungsi f(x) juga semua anggota real.

Kesimpulan

Jadi, domain dan range dari fungsi f(x) = 1 + x² adalah Df = {x| x ∈ R} dan Rf = {y| y ∈ R}.
