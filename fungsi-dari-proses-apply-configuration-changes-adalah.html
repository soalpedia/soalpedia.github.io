1. fungsi dari configuration manager adalah


Configuration management (CM) adalah suatu proses untuk memastikan adanya dokumentasi yang akurat dan efisien untuk berbagai versi dari infrastruktur baik software maupun hardware dimana kedudukan configuration management di dalam service management adalah core dalam organisasi. Dengan kata lain configuration management adalah proses mencatat dan memelihara secara up to date dari semua komponen infrastruktur IT (aset IT, status aset, dan konfigurasinya) serta korelasinya satu dengan yang lain. Penekanan dari configuration management adalah pada hubungan antar component IT. Configuration management tidak mengacu pada aplikasi sistem perangkat lunak atau tingkat versi pengembangan perangkat lunak pada saat pengembangan atau testing. Kompatibilitas hardware dan software secara tepat adalah bagian dari proses manajemen konfigurasi. Menurut sebuah laporan yang dikeluarkan oleh Enterprise Management Associates (EMA), disiplin ilmu seperticonfiguration managementIT menjadi semakin penting untuk mempertahankan tingkat pelayanan dan menjaga semua perangkat keras dan perangkat lunak berfungsi maksimal.

24. Apa perbedaan apply dan apply to all pada power point?


perbedaannya adalah jika apply kita hanya mengaplikasikan pada satu pilihan atau satu slide saja sedangkan apply to all biasanya kita akan mengaplikasinya pada semua pilihan atau semua slide biasanya apply muncul pada opsi pemilihan baground. Terima kasih semoga membantu ya. ✓jangan lupa klik gambar mahkotaKalo "apply", maka perubahan akan terjadi hanya pada slide itu (slide yang aktif).
Kalo apply to all, maka semua slide akan berubah (sesuai dengan apa yang dirubah).

Misal, kamu mau ganti background pada satu slide saja, maka pilih yang "apply". tapi kalo kamu mau ganti semua background (jadi sama semua), maka kamu pilih yang "apply to all"

Question.

M element atom has electrons configuration of (He) 2s² 2p5, while V element atom has electrons configuration of (Ar) 3d5 4s². Determine period and group of these element in periodic table.

Solution

M = [He]2s² 2p5

the number of electrons valance is 2 + 5 = 7

and the number of shell(n) = 2

so, this element is in second period and VIIA group

V = [Ar] 3d5 4s²

according to electrons configuration of V element atom, the number of shell(n) = 4. and number of valance electrons = 7, end to subshell d. So this element is in fourth period and VIIB group

31. Membuat bcdedit cukup rumit. Adalah banyaknya sub-perintah yang ada di dalamnya. Di bawah ini adalah beberapa sub-perintah, tuliskan fungsinya. 1. Createstore — 2. Configuration baru/create 3. Jawaban:

1.createstore — membuat penyimpanan data boot configuration baru

2.create — membuat entry baru di sebuah penyimpanan data boot configuration

3.deletevalue — menghapus sebuah elemen dari boot entry

4.bootdebug — mengaktifkan boot debugger dalam sebuah boot entry atau sebaliknya

5.Tracert—Lacak rute ke host jarak jauh dengan mengirimkan pesan permintaan ICMP

6.pathping—Info Latency dan Packet loss untuk setiap node di jalur jaringan

7.ADDUSERS—Digunakan untuk menambah dan daftar pengguna dalam file CSV

8.ARP—untuk mengelola ARP atau address resolution protocol.
