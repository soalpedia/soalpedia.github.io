4. tuliskan cara melakukan defragmenter


 1.Pada start menu pilih program->Acessories -> System Tools -> Disk defragmenter

2. Pilih drive yang berisi file windows anda (misalnya drive C), secara default windows diinstal di drive C

3. Klik tombol Analyze, tunggu hingga windows melakukan analisa terhadap file-file di hardisk anda. Jika ada file yang belum terfragmentasi maka kotak dialok untuk melakukan defrag. Tekan tombol defrag, proses ini butuh waktu sekitar 1 – 4 jam tergantung kapasitas hardisk yang di defrag.

15. Tuliskan fungsi organ ekskresi dalam sistim ekskresi??​


Penjelasan:

Fungsi sistem ekskresi yaitu membuang limbah yang tersisa atau tidak diperlukan lagi oleh tubuh. Jika tidak segera dikeluarkan maka akan dikeluarkan kondisi tubuh. Organ yang berperan dalam sistem ekskresi yaitu kulit, ginjal, paru-paru dan hati. Kita sebagai manusia harus mengetahui organ-organ yang terlibat dalam sistem ekskresi agar kita dapat menjaga kesehatan tubuh kita. Jika manusia tidak dapat mengaktifkan kesehatan tubuh atau sistem organ ekskresi maka akan muncul fungsi sistem ekskresi pada manusia . Fungsi sistem ekskresi akan diumumkan menurut organ ekskresi .

Jawaban:

Fungsi Sistem Ekskresi. Fungsi sistem ekskresi yaitu membuang limbah (sisa metabolime) berbahaya atau yang tidak dibutuhkan lagi oleh tubuh. Jika tidak segera dikeluarkan maka akan membahayakan kondisi tubuh. Organ yang berperan dalam sistem ekskresi yaitu kulit, ginjal, paru-paru dan hati.

-Sistem Endokrin = Memproduksi hormon hormon untuk mengatur aktivitas tubuh
-Sistem Saraf = Menerima dan merespon rangsang dari lingkungannya
-Sistem Reproduksi = Proses perkembangbiakan
