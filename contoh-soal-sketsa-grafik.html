10. Sketsakan grafik dari |x|+|y|=4


Grafik |x| + |y| = 4 dibagi jadi 4 bagian :

1. Untuk x < 0 dan y > 0 (di kuadran 2), maka :

|x| + |y| = 4

-x + y = 4

y = x+4  <= ini fungsi nya di kuadran 2

2. Untuk x > 0, y > 0 (kuadran 1), maka :

|x| + |y| = 4

x+y = 4

y = -x+4 <= fungsi di kuadran 1

3. Untuk x < 0 dan y < 0 (kuadran 3), maka :

|x| + |y| = 4

-x-y = 4

y = -x-4 <= di kuadran 3

4. Untuk x > 0 dan y < 0 (kuadran 4), maka :

|x| +  |y| = 4

x - y = 4

y = x-4

grafik dari |x| + |y| = 4 membentuk sebuah belah ketupat dengan diameter = 8 (dari y = -4 sampai y = 4, dan dari x = -4 sampai x = 4)
