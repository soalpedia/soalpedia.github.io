3. perbedaan cash register manual,elektronik,dan listrik?​


Jawaban:

Macam-macam Mesin Cash Register ada tiga jenis, yaitu:

A. Cash Register Manual

Terdapat beberapa merk mesin cash register manual salah satunya adalah mesin cash register manual merk Olivetti dari jenis berkunci sepuluh yang dapat mencetak dan berkapasitas 12 angka. Dapat digunakan untuk menjumlah, mengurangi, mengalikan dan memperoleh jumlah tombol atau jumlah kredit juga dilengkapi dengan kunci spasi mundur (back spacer).

B. Cash Register Elektronik

Mesin Cash Register mesin hitung elektronik yang biasa digunakan dalam perdagangan. Mesin ini selain dapat digunakan sebagai alat hitung yang dapat dijamin ketepatan hitungannya juga dapat digunakan sebagai alat penyimpan data barang (stock barang). Mesin ini dapat mencatat semua transaksi yang terjadi setiap kali transaksi. Ada beberapa perbedaan dalam sistem dan spesifikasi Kasir di dua sektor, untuk sektor ritel pada umumnya membutuhkan produk yang menekankan pada kecepatan pelayanan, oleh karena itu, biasanya untuk sektor ritel pada umumnya sudah dilengkapi dengan barcode scanner. Karena memasukkan kode secara manual dengan menggunakan keyboard akan memakan waktu yang lama. Seperti yang kita juga tahu sebagian besar produsen barang-barang umumnya memiliki barcode pada kemasannya.

Adapun kemampuan CRE adalah:

• Pencatatan penjualan dengan diskon.

• Pengurangan penjualan kredit/hutang.

• Pembatalan/penghapusan

• Penghitungan prosen

• Pencatatan nomor bon/transaksi pada kertas bon.

• Pencatatan persediaan uang kas.

• Penjumlahan penjualan/penerimaan cash.

• Pencatatan pengeluaran uang kas

C. Cash Register Listrik

Terdapat beberapa mesin cash register listrik salah satunya adalah NCR. Meskipun terdapat beberapa macam merk tetapi umumnya cara kerjanya sama.

27. sebutkan fungsi - fungsi dari tombol keyboard berikut tombol caps lock, tombol tab, tombol back pace, tombol delete, tombol entar, tombol esc, tombol home, tombol end, tombol Page up, tombol Page Down​


Jawaban:

Misalkan dipakai pada program pengolah data teks (seperti Ms. Word):

Caps Lock: untuk mengunci huruf kapital.

Tab: untuk memindah posisi kursor ke sebelah kanan 1 tabulasi.

Back space: untuk menghapus 1 karakter di sebelah kiri

Delete : untuk menghapus 1 karakter di sebelah kanan

Enter: untuk mengakhiri 1 baris dan mengawali pengetikan teks di baris baru (di bawahnya).

Esc: untuk menghilangkan tampilan menu yg muncul.

Home: untuk memindah posisi kursor ke awal baris teks.

End: untuk memindah posisi kursor ke akhir baris.

Page Up: untuk memindahkan posisi kursor 1 layar ke atas.

Page Down: untuk memindahkan posisi kursor 1 layar ke bawah.

Penjelasan:

Contoh pemakaian tombol Esc: misalkan mengklik kanan teks (dg mouse) lalu muncul suatu menu, maka tombol ini bisa dipakai untuk menghilangkan menu yg muncul tadi.

37. sebutkan fungsi tombol-tombol pada komputer​


Tombol keyboard:

Caps Lock = untuk membesarkan (mengkapitalkan) semua hurufWindows = untuk ke main source (untuk shutdown, pencaharian, dll)Shift = untuk membesarkan satu huruf dlm sebuah kalimatAlt = untuk commandCtrl = untuk command (contoh Ctrl + P untuk mengeprint, ctrl + v untuk paste, ctrl + c untuk copy)F1 - F12 = pengaturan komputer luarNum Lock = tombol yang memiliki fungsi ganda (contoh: Num Lock on, fungsi pengetikan angka-angka dan operator matematik aktif. Num Lock off, fungsi tombol navigasi aktif.)ESC (Escape): adalah tombol yang kegunaanya sama dengan pilihan CANCEL, yaitu untuk membatalkan satu tahap pekerjaan.Enter: adalah tombol yang kegunaanya sama dengan pilihan tombol OK, yaitu untuk menyatakan bahwa operasi yang dilakukan betul dan selesai. Enter juga dapat berarti menyisipkan baris kosong atau baris baru pada proses pemasukan text. TAB : Berarti pindah ke field atau daerah isian atau pilihan berikutnya.  Delete: akan mengakibatkan karakter(huruf) disebelah kanan atau ditempat dimana kursor berada akan terhapus.

 Home: akan mengakibatkan kursor berpindah ke awal baris dimana kursor berada.

 End: akan mengakibatkan kursor berpindah ke akhir baris dimana kursor berada.

Page Up: akan mengakibatkan kursor berpindah ke atas sejauh satu layar dari kursor berada.

Page Down: akan mengakibatkan kursor berpindah ke bawah sejauh satu layar dari kursor berada. Up    : Memindahkan kursor 1 baris ke atasDown : Memindahkan kursor 1 baris ke bawah Left : Memindahkan kursor 1 karakter ke kiriRight : Memindahkan kursor 1 karakter ke kanan

38. Tombol yang berfungsi mengubah fungsi tombol penunjuk menjadi tombol numerik adalah....


Jawaban:

Tombol NumLock

Penjelasan:

Perhatikan contoh gambar keyboard berikut ini (khususnya di bagian yg ditunjukkan tanda panah kuning):

- bila tombol NumLock dalam kondisi aktif, maka tombol penunjuk berubah menjadi tombol numerik.

- bila tombol NumLock dalam kondisi nonaktif, maka tombol numerik berubah menjadi tombol penunjuk.

Contohnya dapat diamati dengan seksama:

- tombol numeric 8 dapat berubah jadi tombol penunjuk panah atas

- tombol numeric 6 dapat berubah jadi tombol penunjuk panah kanan

- tombol numeric 4 dapat berubah jadi tombol penunjuk panah kiri

- tombol numeric 2 dapat berubah jadi tombol penunjuk panah bawah
