Materi: Sistem Administrasi Jaringan - OSI Layer

Jawaban:

Untuk membantu mendefinisikan alamat IP atau Internet Protocol sehingga tiap komputer dapat terhubung dengan satu jaringan adalah fungsi OSI A. Network Layer.

Penjelasan:

OSI memiliki kepanjangan Open System Interconnection, yang berarti interkoneksi dalam sistem terbuka. OSI adalah suatu lapisan struktur/kerangka jaringan yang difungsikan/diterapkan melalui protokol jaringan (network protocol). OSI terbagi dalam 7 layer, salah satunya adalah Network Layer.

Network Layer memiliki arti lapisan jaringan, yaitu salah salah lapisan OSI yang berfungsi untuk mendefinisikan alamat protokol internet atau Internet Protocol (dikenal akrab sebagai IP Address). Tujuannya adalah agar tiap-tiap perangkat komputer bisa terkoneksi dengan satu jaringan (network).

Jadi, opsi yang benar adalah A. Network Layer.

31. proses kerja layer pada perangkat lunak ialah?


Cara kerja Layer pada perangkat lunak   Pembentukan paket dimulai dari layer teratas model OSI. Application layer mengirimkan data ke presentation layer, di presentation layer data ditambahkan header dan atau trailer kemudian dikirim ke layer dibawahnya, pada layer dibawahnya pun demikian, data ditambahkan header atatu trailer kemudian dikirimkan ke layer dibawahnya lagi, terus demikian sampai ke physical layer. Di physical layer data dikirimkan melalui media transmisi ke host tujuan. Di host tujuan paket data mengalir dengan arah sebaliknya, dari layer paling bawah ke layer paling atas. Protocol pada physical layer di host tujuan mengambil paket data dari media transmisi kemudian mengirimkannya ke data link layer, data link layer memeriksa data link layer header yang ditambahkan host pengirim pada paket, jika host bukan yang dituju oleh paket tersebut maka paket itu akan di buang, tetapi jika host adalah yang dituju oleh paket tersebut maka paket akan dikirimkan ke network layer, proses ini terus berlanjut sampai application layer di host tujuan. Proses pengiriman paket dari layer ke layer ini disebut dengan “peer-layer communication”. Lebih singkatnya ketika data ditransfer melalui jaringan, sebelumnya data tersebut harus melewati ke‐tujuh layer dari satu terminal, mulai dari layer aplikasi sampai physical layer, kemudian di sisi penerima, data tersebut melewati layer physical sampai aplikasi. Pada saat data melewati satu layer dari sisi pengirim, maka akan ditambahkan satu “header” sedangkan pada sisi penerima “header” dicopot sesuai dengan layernya. Dan masing-masing layer mempunyai tugas tersendiri demi kelancaran data yang akan dikirimkan.

Jawaban:

Application Layer berhubungan langsung dengan aplikasi pengguna (seperti web browser atau email) dan menyediakan layanan komunikasi yang khusus untuk aplikasi tersebut.

Transport Layer mengelola pengiriman data antara perangkat, memecah data menjadi paket, memastikan pengiriman yang andal, dan mengatur aliran data dari sumber ke tujuan.
