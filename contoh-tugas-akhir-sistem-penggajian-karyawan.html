Jawab

Rata rata gaji karyawan pt rahmat anugerah adalah Rp4.020.000,- jika ditambah gaji seorang karayawan baru sebesar Rp2.400.000,- maka rata rata gaji karyawan menjadi sebesar Rp3.750.000,- banyak karyawan PT rachmat anugerah adalah 6 karyawan

Pembahasan Ingat Kembali

ok saya akan menjelaskan beberapa materi matematika yang berkaitan dengan soal ini

-Pindah ruas

{Pada saat bilangan pindah ruas bilangan positif menjadi negatif, bilangan negatif menjadi positif, di kali menjadi di bagi dan di bagi menjadi di kali, pangkat menjadi akar dan akar menjadi pangkat dll}

- Model Matematika

{Untuk membuat model matematika bisa dengan mengganti bendanya(objeknya) menjadi suatu variabel aljabar( a,b,x,y dll)}

- Sifat Distributif Pada Aljabar

{ab±ac = a(b±c), artinya jika ada a kali b ditambah/dikurang a kali c bisa disederhanakan menjadi a dikali dalam kurung (b ditambah/dikurang c) }

- Rata-rata(MEAN)

{rata-rata data adalah suatu data yang dikali banyak data hasilnya senilai dengan jumlah data.

untuk mencari rata-rata suatu data dapat dilakukan dengan membagi jumlah data dengan banyak data }

Penyelesaian

// agar mudah kita buat model matematikannya dahulu //

misal rata rata gaji karyawan setelah ada karyawan baru = x

misal rata rata gaji karyawan sebelum ada karyawan baru = y

misal banyak karyawan sebelum ada karyawan baru = b

// dari soal dapat kita simpulkan bahwa jumlah data dibagi banyak data harus sama dengan rata-rata data //

                                                    x = (y×b+Rp2.400.000,-×1)/(b+1)

                      Rp3.750.000,-(b+1) = Rp4.020.000,-b+Rp2.400.000,-

 Rp3.750.000,-b+Rp3.750.000,- = Rp4.020.000,-b+Rp2.400.000,-

Rp3.750.000,-b-Rp4.020.000,-b = Rp2.400.000,- - Rp3.750.000,-

                             -Rp270.000,-b = -Rp1.350.000,-

                                                    b = -Rp1.350.000,-/-Rp270.000,-

40. Gaji Karyawan yang belum/tidak dibayarkan pada akhir periode/masa maka....


PENDAHULUAN

Akun beban yang belum/tidak dibayarkan pada akhir periode/masa akan diakui sebagai utang oleh perusahaan sehingga perlu dibuatkan jurnal penyesuaian dengan cara melakukan debit (penambahan) pada akun beban terkait dan melakukan kredit (penambahan) pada akun utang terkait.

PEMBAHASAN

Gaji karyawan yang belum/tidak dibayarkan pada akhir periode/masa maka Harus disesuaikan di dalam jurnal penyesuaian dengan cara men-debit akun beban gaji dan men-kredit akun utang gaji
