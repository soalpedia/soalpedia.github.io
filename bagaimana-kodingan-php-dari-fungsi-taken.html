19. Fungsi dari Php fungsi dari sql


PHP dapat disebut dengan bahasa pemrograman umum yang berarti php dapat disematkan ke dalam kode HTML, atau dapat digunakan dalam kombinasi dengan berbagai sistem templat web, sistem manajemen konten web, dan kerangka kerja web.  

Sedangkan SQL sendiri merupakan singkatan dari Structured Query Language. Secara umum SQL dapat diartikan sebagai bahasa yang digunakan dalam mengakses data.  

Pembahasan

Fungsi dari PHP yakni membuat atau mengembangkan situs web statis atau situs web dinamis atau aplikasi Web. PHP digunakan karena untuk membuat website dinamis bisa dipakai untuk menyimpan data ke dalam database, membuat halaman yang dapat berubah-ubah sesuai dengan input  user, memproses form, dan yang lain-lain. Dalam membuat file PHP dapat digabungkan dengan menggunakan tag html, kemudian ketika tanpa menggunakan tag html apa pun disebut sebagai file PHP Murni. Server menginterpretasikan kode PHP lalu mengeluarkan hasilnya sebagai kode HTML ke browser web. Agar server mengidentifikasi kode PHP dari kode HTML, yang dilakukan adalah harus selalu menyertakan kode PHP dalam tag PHP.

Fungsi dari SQL yakni memungkinkan anda mengakses dan memanipulasi database, dapat mengeksekusi query terhadap database, dapat mengambil data dari database, dapat menyisipkan data dalam database, dapat memperbarui data dalam database, dapat menghapus data dari database, dapat membuat database baru, dapat membuat tabel baru dalam database, dapat membuat prosedur yang tersimpan dalam database, dapat menggabungkan beberapa tabel didalam database, dapat mengatur hak akses pada tabel, prosedur, dan yang lain-lain.

28. Blok program yang berfungsi sebagi badan sebuah alur kodingan dilambangkan dengan...? a. # b. {} c. <> d. *


Jawaban : b.{}

Pembahasan

Penjelasan singkat

Dalam beberapa bahasa pemrograman seperti HTML, simbol yang dipakai untuk sebuah alur kodingan adalah {} Biasanya setelah suatu fungsi, akan diawali oleh { dahulu, lalu setelahnya diakhiri dengan }. Jika pembukaan suatu koding tidak menggunakan {} maka akan terjadi error dan pemrograman tidak akan dapat dimulai. Namun, dalam beberapa bahasa pemrograman, ada pula yang menggunakan () dari pada {} sebagai badan sebuah alur

Gambar

Gambar foto merupakan contoh koding bahasa php yang menggunakan {} (lihat kotak untuk simbol {} pada satu jalur koding)

class - fungsi (bisa dilihat bahwa sesudah fungsi, untuk membuka suatu alur koding menggunakan { dan di bagian akhir diakhiri dengan })

Penjelasan simbol pada soal

# — Dalam program bahasa python, simbol ini digunakan untuk menambahkan kalimat dalam pemrograman yang nantinya tidak akan diproses dalam program (kalimat ditandai dengan warna hijau)<> — Simbol < dan > digunakan untuk suatu permisalan jika integrer lebih besar/kecil dari variabel yang ditulis. Atau dalam beberapa bahasa pemrograman, <> digunakan sebagai "tidak sebanding dengan" (seperti contoh If x <> 6)* — Simbol yang digunakan untuk perkalian dalam hampir semua bahasa pemrograman.

36. apa kegunaan dari koding dari website


Dapat membuat sebuah blog atau website sendiri tanpa bantuan orang lain. Ketika kita memutuskan untuk mulai membangun sebuah website tentu akan diperlukan kegiatan coding. Nah bagi yang sudah menguasainya tentu akan lebih mudah tanpa harus menggandeng orang lain.

Bisa dijadikan pekerjaan di masa depan sebagai seorang programmer. Seperti yang sudah dikatakan sebelumnya bahwa semakin hari kemampuan coding ini akan menjadi semakin penting. Maka di masa depan buat kamu yang punya minat dalam dunia programming tentu kesempatan masih terbuka lebar.

Banyak hal yang bisa dilakukan dengan coding maka dari itu kegiatan yang satu ini bisa dimanfaatkan sebagai ladang mencari uang. Misal saja dengan menerima pesanan untuk membuat website, perangkat lunak atau aplikasi dan bahkan memulai bisnis e-commerce.
