4.  - Pendeklarasian Fungsi:

void sapa(nama){

   cout << "Hello, " << nama << endl;

}

void = sebuah tipe dari fungsi, lebih tepatnya fungsi tanpa kembalian.

sapa = sebagai nama fungsi yang telah kita buat dan akan digunakan untuk memanggil fungsi itu nantinya di main program.

{} = menandakan kode yang ada di dalam tanda kurung kurawal ini sebagi bagian dari milik fungsi sapa.

nama = sebagai paramater.

cout = berarti console out, atau sederhannya menampilkan hasil dari program ke console/terminal.

<< = tanda yang berati akan mengeluarkan/menampilkan ke terminal

"Hello" = teks/string biasa.

nama = akan menampilkan nama(Parameter) ke console/terminal.

endl = bisa diartikan sebagia "\n" atau disebut juga dengan new line/baris baru.

; = titik dua/semicolom berarti sebagai penanda jika kode pada baris ini akan berhenti.

- Pendefinisian(pemanggilan?) :

#include <iostream>

using namespace std;

int main(){

string nama;

cout << "masukan nama kamu : " ; cin >> nama;

sapa(nama);

}

#include <iostream> = prosesor pengarah yang mengatakan kepada kompiler untuk meletakan kode dari header file iostream.

using namespace std; =  membuat kita tidak perlu menggunakan std:: lagi di dalam program.

int main() = sebagi main program.

{} = menandakan kode yang ada di dalam tanda kurung kurawal ini sebagi bagian dari milik main program.

str nama; = sebagai variabel yang menyimpan data/value yang dimasukan oleh user.

cout << =  berarti console out, atua sederhannya menampilkan hasil dari programnya ke console/terminal.

; = berarti sebagai penanda jika kode pada baris ini akan berhenti.

cin >> = console input, yang berarti user akan memasukan/input data ke variabel nama;.

sapa(nama); = memnaggil fungsi. kita akan mengoper data yang telah di-input-kan oleh user di variabel nama; ke dalam fungsi untuk natinya ditampilkan ke dalam console/terminal.

; = berarti sebagai penanda jika kode pada baris ini akan berhenti.

6. 1.sebut dan uraikan secara lengkap parameter-parameter berita?​


Jawaban:

Parameter-parameter berita adalah elemen-elemen yang digunakan untuk menentukan cakupan, fokus, dan karakteristik suatu berita. Berikut adalah beberapa parameter penting dalam menentukan berita:

1. Kepentingan Publik: Berita harus memiliki relevansi dan dampak pada masyarakat atau publik yang luas. Parameter ini menentukan apakah suatu peristiwa atau informasi dianggap berita yang penting untuk disampaikan kepada masyarakat.

2. Ketepatan Fakta: Berita harus didasarkan pada fakta yang dapat diverifikasi. Parameter ini menuntut keakuratan dalam pengumpulan informasi dan memeriksa sumber yang dapat dipercaya untuk memastikan kebenaran dan keabsahan informasi yang disampaikan.

3. Kebaruan: Berita harus memiliki elemen kebaruan atau aktualitas. Parameter ini menekankan pentingnya menyampaikan informasi yang baru, terkini, dan berbeda dari apa yang telah diketahui sebelumnya.

4. Relevansi: Berita harus relevan dengan isu-isu terkini, peristiwa yang sedang berlangsung, atau topik-topik yang menarik minat masyarakat. Parameter ini membantu menentukan apakah berita tersebut memiliki kaitan dengan kehidupan sehari-hari atau topik yang tengah dibicarakan.

5. Signifikansi: Berita harus memiliki tingkat signifikansi yang cukup tinggi dalam konteks sosial, politik, ekonomi, atau budaya. Parameter ini menilai sejauh mana suatu peristiwa atau informasi memiliki dampak yang signifikan bagi masyarakat atau kehidupan mereka.

6. Keseimbangan dan Keanekaragaman: Berita harus mencerminkan keseimbangan dan keanekaragaman pandangan. Parameter ini mengharuskan penyedia berita untuk memberikan sudut pandang yang beragam, menyajikan berbagai perspektif, dan memastikan tidak ada bias yang mencolok dalam penyampaian informasi.

7. Objektivitas: Berita harus bersifat objektif dan netral. Parameter ini menuntut penyedia berita untuk menyajikan informasi secara adil tanpa memihak atau terpengaruh oleh pandangan pribadi atau kepentingan tertentu.

8. Keadilan dan Etika: Berita harus mematuhi prinsip-prinsip etika jurnalisme dan menjaga keadilan dalam penyampaian informasi. Parameter ini menekankan pentingnya menjaga privasi, kebenaran, dan integritas dalam melaporkan berita.

Menggunakan parameter-parameter ini, penyedia berita dapat menghasilkan konten yang berkualitas, informatif, dan berdampak bagi masyarakat.

9. 1. Jelaskan apa yang dimaksud dengan parameter MYSQLI_ASSOC , MYSQLI_NUM, MYSQLI_BOTH !2. Jelaskan dan tuliskan parameter yang digunakan dalam fungsi mysqli_fetch_rows !


Yang dimaksud dengan parameter MYSQLI_ASSOC, MYSQLI_NUM, dan MYSQLI_BOTH adalah parameter yang digunakan dalam bentuk array. Perbedaan ketiga parameter tersebut hanya terletak pada cara array tersebut didefinisikan.


Pada MYSQLI_NUM = Item array akan menggunakan indeks key bertipe numerik.

Pada MYSQLI_ASSOC = Item array akan menggunakan nama kolom sebagai index key

Pada MYSQLI_BOTH = Item array akan diduplikasikan, yang satu menggunakan indeks key bertipe numerik, dan yang lainnya menggunakan nama kolom sebagai index key


Parameter yang digunakan dalam fungsi mysqli_fetch_rows adalah sebagai berikut

mysqli_fetch_row(result);


Result wajib diisi. Menandakan kumpulan hasil identifier yang dikembalikan oleh mysqli_query(), mysqli_store_result() atau mysqli_use_result().


mysqli_fetch_rows digunakan untuk mengambil sebuah baris dari kumpulan hasil dan mengembalikannya ke dalam bentuk array yang telah dicacah

ALT

   return 0;

38. jelaskan tentang delay parameter​


Jawaban:

parameter pengukuran kinerja dari sebuah jaringan adalah throughput, delay atau response time dan paket loss. Selain itu masih ada lagi parameter parameter yang lain.

Berikut ini adalah daftar definisi dan parameter untuk mengukur performa jaringan yang dapat digunakan ketika menganalisis kebutuhan dan hasil dengan tepat:

Kapasitas (bandwidth): merupakan kapasitas data yang dapat dibawa pada sebuah sirkuit atau jaringan, biasanya diukur dalam bit per detik (bps).

Pemanfaatan: proporsi yang digunakan dari total kapasitas yang tersedia.

Pemanfaatan Optimal: Nilai rata-rata penggunaan tertinggi sebelum jaringan dinyatakan bekerja penuh. Throughput: kuantitas dari data yang sukses di transfer antar node per satuan waktu, biasanya di ukur dalam detik.

Beban yang ditawarkan: jumlah dari semua data dari semua node di jaringan yang siap dikirimkan pada waktu tertentu.

Akurasi: Jumlah lalu lintas yang digunakan benar-benar ditransmisikan, biasanya terkait dengan jumlah keseluruhan lalu lintas.

Efficiency: Sebuah analisis dari seberapa banyak usaha yang diperlukan untuk menghasilkan sejumlah data throughput.

Delay (latency): waktu antara paket yang siap untuk ditransmisikan dari sebuah node dan di kirimkan ke tempat lain di dalam jaringan.

Variasi Delay: rata-rata dari jumlah delay yang bervariasi.

Response time: sejumlah waktu antara sebuah request untuk beberapa network service dan sebuah respon kepada requestor.
