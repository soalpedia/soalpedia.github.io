18. fungsi tool bar pada microsoft excel


Pada microsoft excel 2003, banyak sekali tool bar yang disediakan. namun yang umum dipakai adalah tool bar standard dan tool bar formatting:
- tool bar standard: berfungsi untuk menampung tombol-tombol standar (contohnya Open, Save, Print Preview)
- tool bar formatting: berfungsi untuk menampung tombol-tombol pemformatan dokumen (contohnya Font, Font Size, Bold, Italic)

21. Apa perbadaan antara Tool Bar dengan Quick Access Tool Bar?​


Jawaban:

Toolbar adalah sekumpulan tombol navigasi dalam bentuk icon yang terdapat pada bagian atas dan samping tampilan software, untuk memudahkan pengguna komputer dalam menjalankan fitur – fitur yang ada pada sebuah software.

Quick access toolbar adalah sebuah toolbar yang berisi sejumlah ikon atau shortcut/tools/tombol-tombol perintah yang dapat digunakan untuk mengakses secara cepat pada saat kita bekerja menggunakan Microsoft Word.

23. apa fungsi menu bar,tool bar,dan ribbon?


menu bar:mengatur file dokumen
tool bar:membuat dokumen baru
ribbon:menampilkan tan menu dari menu yg aktif.menu bar = bar yang berfungsi menampilkan daftar menu yang tersedia pada program tertentu (Contohnya: menu File, menu Edit, menu View)

toolbar = bar yang berfungsi menampilkan tombol-tombol yang dapat digunakan pada suatu program untuk mengedit suatu file (Contohnya: brush tool, font tool, marquee tool, crop)

ribbon= berfungsi menggabungkan menu bar dan toolbar dalam group-group tertentu sesuai jenisnya (mulai ada dalam Ms. Office 2007, contohnya: ribbon Home, ribbon Insert)

26. jelaskan fungsi tool bar pada layar autocad


1. Line = Membuat object dengan garis
2. Polyline = Membuat object dengan garis menyatu.
3.Splin = Membuat object dengan garis melengkung.
4.Elipse = Membuat object Elipse.
5. Cirxle = Membuat object Lingkaran.
6. Rectangle = Membuat object Persegi.
7.Region = Menyatukan object.
8.Extrude = Memberikan ketebalan pada object.
9.Move = Memindahkan object.
10.Mirror = Mengcopy object disebelah kiri, kanan, atas, dan bawah.
11.Trim = Memotong object.
12. Chamfer = Membantu object persegi menjadikan sudut object tidak lancip.
13. Fillet = Membuat object menjadikan sudut tidak lancip atau melengkung.
14. Union = Menggabungkan object.
15. Substract = Menghilangkan object 3D.
16. Ofset = Menggadakan object didalam atau diluar object.
17. Extrude Faces = Menambahkan ketebalan yang sudah dibuat.
