3. “Dibuat oleh” bener nya “created by” atau “create by” bantu jelasin juga ya


Kata dibuat dalam bahasa Inggris ditulis created by

Pembahasan

Dalam kalimat pasif (passive voice), subjek menjadi pihak yang menerima aksi. Rumus dasar passive voice adalah sbb :

be + verb₃

1. Simple present tense

The song is created by the famous lyricist (Lagu itu dibuat oleh penulis terkenal)

2. Simple present continuous tense

The product is being considered by the committee (Produk tersebut sedang dipertimbangkan oleh panitia)

3. Present perfect / past perfect tense

The new equipment has been ordered by the company (peralatan baru telah di order oleh perusahaan)

The new equipment had been ordered by the company (peralatan baru telah di order oleh perusahaan)

4. Modal

The contract should be signed by the manager (Kontrak itu seharusnya di tandatangani oleh manajer)

8. Jelaskan perbedaan fungsi delete entire row dengan delete entire column


Sebelum menjawab pertanyaan mari pahami terlebih dahulu pengertian dari opsi Delete entire. Delete entire merupakan sebuah cara untuk menghapus suatu data dalam satu rentang data baik baris maupun kolom dalam software Microsoft Excel.

Dibawah ini adalah penjelasan antara fungsi delete entire row dan delete entire column:

1. Delete Entire Row

Opsi entire row digunakan saat akan menghapus data di excel, satu atau lebih baris dari ujung kiri sampai ujung kanan. Jadi pada saat memilih opsi ini saat melakukan penghapusan data, semua data yang ada pada baris / rentang baris yang dipilih akan dihapus.

Kelebihan dari delete entire row ini adalah dapat menghapus rentetan data dalam suatu baris jika sudah tidak digunakan lagi secara sekaligus.


2. Delete Entire Columns

Opsi entire column ini hampir sama dengan cara delete menggunakan entire row. Hanya saja pada pilihan ini, penghapusan data dilakukan pada kolom dari ujung atas sampai ujung bawah.

Penggunaan opsi delete entire column ini akan dapat menghapus suatu data atau sekelompok data yang sudah tidak digunakan dalam data bentuk vertical.

30. apa fungsi tombol backspace dan delete?


Tombol backspace dan delete pada dasarnya memiliki fungsi yang sama, yakni untuk menghapus. Namun, dari keduanya memiliki perbedaan. Berikut penjelasannya.  Tombol backspace digunakan untuk menghapus huruf atau angka dari belakang ke depan. Selain itu, tombil backspace dapat digunakan sebagai perintah untuk kembali ke keadaan sebelumnya ketika digunakan untuk membuka dokumen komputer. Sedangkan tombol delete digunakan untuk menghapus dari depan ke belakang. Selain itu, tombol delete juga berfungsi untuk menghapus gambar, file, video, dokumen, dan lain-lain.

Pembahasan

Dalam sebuah perangkat komputer, pasti anda akan mendapatkan perangkat keras yang bernama keyboard. Keyboard adalah perangkat keras dari komputer yang berungsi untuk mengetik huruf, angka, dan memberikan perintah-perintah tertentu. Fungsi keyboard dalam sebuah komputer sangat penting.

Dalam sebuah keyboard, terdapat berbagai tombol dan berbagai perintah yang dapat digunakan oleh user computer. Beberapa diantaranya adalah tombol backspace dan delete. Tombol backspace dan delete pada dasarnya memiliki fungsi yang sama, yakni untuk menghapus. Namun, dari keduanya memiliki perbedaan. Berikut penjelasannya.

Tombol backspace digunakan untuk menghapus huruf atau angka dari belakang ke depan. Selain itu, tombil backspace dapat digunakan sebagai perintah untuk kembali ke keadaan sebelumnya ketika digunakan untuk membuka dokumen komputer. Sedangkan tombol delete digunakan untuk menghapus dari depan ke belakang. Selain itu, tombol delete juga berfungsi untuk menghapus gambar, file, video, dokumen, dan lain-lain.
