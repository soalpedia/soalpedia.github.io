Penjelasan:

1. Proofing, dapat anda gunakan untuk mengatur tata bahasa, pengejaan kata, dan menerjemahkan kata. Ada 4 ikon menu yang terdapat didalamnya, yaitu:

1) spelling, berfungsi untuk mengecek dari sebuah ketentuan range yang ditentukan

2) Research, berfungsi untuk melakukan pencarian seperti kamus atau ensiklopedi

3) Translate, berfungsi untuk menterjemahkan teks ke dalam bahasa yang lain

4) Theasaurus, berfungsi untuk mencari persamaan kata atau sinonim.

2.terdapat 5 toolgroup dalam menu view, yaitu:

a. Workbook Views

Worbook view, bisa anda gunakan untuk mengatur tampilan sheet agar mudah melihat data. Ada 5 ikon menu yang terdapat didalamnya, yaitu:

1) Normal, berfungsi untuk mengatur worksheet dalam bentuk normal, dengan pilihan normal seluruh isi sheet dampat ditampilkan.

2) Page layout, berfungsi untuk pengaturan worksheet/lembar kerja excel

3) Page break preview, berfungsi untuk mengatur tampilan worksheet dengan tampilan per halaman

4) Custom views, berfunsi untuk menampilkan dua layar yang berlainan

5) Full screen, berfungsi untuk mengatur model tampilan satu layar penuh

b. Show/Hide

Show/hide, bisa anda gunakan untuk menyembunyikan atau menampilkan beberapa opsi seperti Rules, Formula, Grid dan Headings.

c. Zooms

Zoom, bisa anda gunakan untuk memperbesar atau memperkecil tampilan. Ada 3 ikon menu yang terdapat didalamnya, yaitu:

1) zoom, berfungsi untuk mengatur presentasi perbesaran tampilan layar

2) 100%, berfungsi untuk menampilkan worksheet dengan perbesaran 100%.

3) Zoom to Selection, berfungsi untuk memilih perbesaran di cell yang aktif.

d. Window

Window, bisa anda gunakan untuk mengatur tampilan window. Ada 8 ikon menu yang terdapat pada winodw, yaitu:

1) New Window, berfungsi untuk membuka sebuah jendela baru

2) Arrange All, berfungsi untuk mengubah model tampilan jendela aplikasi dengan susunan tertentu.

3) Freeze Panes, berfungsi untuk membatasi gerakan layar worksheet pada saat digulung.

4) Split, berfungsi untuk membelah worksheet menjadi 4 bagian jendela

5) Hide Window, berfungsi untuk menyembunyikan worksheet

6) Unhide Window, berfungsi untuk menampilkan kembali worksheet yang disembunyikan

7) Save Workspace, berfungsi untuk menyimpan sebagai workspace

8) Switch Windows, berfungsi untuk menampilkan worksheet yang sedang dibuka.

e. Macros

Macros, merupakan fasilitas Ms. Excel yang berfungsi untuk membantu pekerjaan menjadi lebih sederhana dan cepat. Contohnya, bila anda ingin mengcopy suatu isi cell/sel B5 ke cell E5, anda tidak perlu melakukan banyak langkah, seperti menempatkan pointer, lalu copy data, menempatkan pointer kembali, dan mempaste di cell yang anda inginkan.

22. fungsi tab home, insert, page layout, references, mailings, review, dan view


Home
Home merupakan menu utama yang akan tampil pada saat pertama kali anda membuka word 2007. Menu home terdiri dari toolbar standar yaitu font, tool bar formatting dan paragraph, style dan editing serta clipboard. fungsinya untuk mengedit teks yang ingin diketik di paperwork

Insert Tab ini digunakan untuk memasukkan instruksi-instruksi ke dalam dokumen. Menu insert terdiri dari : Pages, Tables, Illustrations, Links, Header & Footer, Text, Symbol.

Page Layout Tab page Layout digunakan untuk mengatur tata letak data, yang meliputi Page Setup, memberiwarna background pada lembar kerja, mengatur spasi antar baris, membuat kolom pada teks dan mengatur indentasi. Menu page layout terdiri dari : Themes, Page Setup, Page Background, Paragraph, Arrange.

References Tab ini dugunakan untuk memasukkan berbagai perintah seperti menuliskan footnote (catatan kaki), menyisipkan gambar, dan membuat tanda citasi. Perhatikan tampilan tab references
Tab References. References terdiri dari : Table of Contents, Footnotes, Citations & Bibliography, Captions, Index, Table of Authorities.

Mailings Tab ini berguna untuk membuat mail merger (surat massal), mengetik teks pada amplop, dan membuat label. Mailings terdiri dari : create, start mail merge, write & insert Fields, preview results, finish.

Review Tab Review berguna untuk mengecek kesalahan tata bahasa serta memproteksi dokumen. Review terdiri dari : proofing Comments, Tracking, changes, Compare, Protect.

View Tab View digunakan untuk menampilkan atau mencetak dokumen pada layar, mengatur ukuran cetakannya, dan menampilkan garis bantu pada lembar kerja. Menu View berfungsi untuk mengatur tampilan layar dari dokumen yang kita kerjakan. Berikut ini merupakan berbagai fungsi dari tool-tool yang terdapat dalam menu View.


23. Tuliskan dan jelaskan fungsi yang terdapat pada tab home, tab page layout, tab references, tab mailings, tab review, tab view.​


Penjelasan:

1. Home

Tab ini memiliki fungsi untuk menyunting properti-properti tulisan seperti halnya mengganti font yang digunakan dalam lembar dokumen Microsoft Word 2007, ukuran font yang diinginkan atau bahkan hingga penempatan paragraf sehingga tulisan akan menjadi lebih rapi.

2. Insert

Tab Insert ini digunakan untuk mempercantik tampilan dokumen, untuk memasukkan gambar/foto ke dalam lembar kerja, Tab Insert ini bisa dibilang tab yang mendukung Tab Home sehingga lembar kerja yang kita buat menjadi lebih menarik dan rapi.

3. Page layout

Page Layout berbeda dengan Insert, Insert mengacu pada isi dokumen, sedangkan Page Layout mengacu pada lembar itu sendiri sehingga bisa dibuat semenarik mungkin agar lembar tidak terkesan polos.

4. Referances

Tab Referensi ini berfungsi sebagai tambahan tulisan/catatan penting pada sebuah lembar kerja, misal untuk menambahkan Daftar Isi, Catatan Penting di setiap lembar kerja, dan masih banyak lagi fungsi yang masih bisa digunakan.

5. Mailings

Tab Mailings ini dapat digunakn sebagai Mail Merger dan kebutuhan surat-menyurat lainnya sehingga akan lebih efisien jika menggunakan Mailings untuk membuat surat.

6. Review

Tab review ini sangat berguna untuk mereviewkan kembali seperti melihat ulang untuk mencari kesalahan penggunaan tata bahasa, memberikan informasi mengenai jumlah kata atau bahkan bisa lagi untuk memproteksi dokumen Anda.

7. View

Tab ini memberikan bantuan tampilan pada lembar kerja Microsoft Office Word, seperti perbesaran lembar kerja, memberikan tampilan buku, dan masih banyak lagi yang bisa digunakan untuk mencoba dokumen kita dalam berbagai bentuk tampilan.
