2. fungsi fungsi menu edit



Fungsi Menu Edit :

1.      Undo : Untuk membatalkan perintah terakhir

2.      Repeat : Untuk membatalkan Undo

3.      Cut : Untuk memotong suatu objek seperti kata, kalimat, gambar dan memasukkannya ke clipboard atau dokumen lain.

4.      Copy : Untuk meng-copy atau menyalin suatu objek ke dalam clipboard atau dokumen lain.

5.      Office Clipboard : Untuk mengatur pemunculan objek dari clipboard

6.      Paste : Untuk memunculkan suatu objek dari dalam clipboard atau dokumen lain.

7.      Paste Special : Paste dengan kriteria tertentu

8.      Paste as Hyperlink : Untuk memunculkan objek dari dalam clipboard namun tetap ada hubungannya walau berada di tempat yang berbeda.

9.      Clear : Untuk menghapus format atau objek yang dipilih.

10.  Select All : Untuk memilih/memberi tanda blok hitam pada seluruh lembar kerja

11.  Find : untuk mencari mencari kata yang dikehendaki

12.  Replace : untuk mengganti kata

13.  Go to : untuk berpindah halaman, baris, kolom, objek, dan lain-lain.

14.  Links : untuk membentuk hubungan

15.  Object : untuk mengedit object

Edit merupakan menu yang berhubungan dengan proses pengeditan dokumen yang sifatnya operasional. Dalam menu edit terdapat 15 sub menu yaitu : Undo, Repeat typing, Cut, Copy, Paste, Paste Special, Paste as Hyperlink, Clear, Select All, Find, Replace, Go To, Links dan Obyek. 
.
-Undo Typing untuk Membatalkan pengetikan
-Repeat Typing untuk Mengulang pengetikan
-Cut untuk Memotong
-Copy untuk Menggandakan
-Paste untuk Menampilkan hasil dari copy
-Paste Special untuk Menampilkan hasil copy dengan kondisi khusus
-Paste Hyperlink untuk Menampilkan hasil copy terhubung
-Clear untuk Menghilangkan format-Select All untuk Memilih seluruh dokumen-Find untuk Mencari kata-Repleace untuk Mengganti kata-Go To untuk Menuju halaman tertentu

4. Sebutkan submenu-submenu yang ada pada menu edit beserta fungsinya


Submenu yang ada pada menu edit Ms. Word 2003:

Undo / Undo Typing : Perintah untuk membatalkan perintah terakhir.

Redo : Perintah untuk membatalkan perintah undo.

Repeat Typing : Perintah untuk mengulangi perintah terakhir.

Cut : Perintah untuk mengopi teks dengan menghapus teks yang dicopy.

Copy : Perintah untuk mengopi teks tanpa menghapus teks yang dicopy.

Office Clipboard : Perintah untuk menampilkan task pane clipboard.

Paste : Perintah untuk menyalin teks yang sudah dicopy.

Paste Spesial : Perintah untuk menyalin teks dengan berbagai macam tipe teks.

Paste as Hyperlink : Perintah untuk menyalin teks dengan bentuk teks hyperlink.

Clear : Perintah untuk menghapus teks atau gambar yang salah atau yang tidak diinginkan.

Select All : Perintah untuk memilih semua teks (yang sudah di tulis) yang terdapat di dokumen.

Find : Perintah mencari teks atau kata yang terdapat di dokumen.

Replace : Perintah untuk mengganti teks.

Go To : Perintah untuk berpindah halaman, baris, kolom, objek, footnote, endnote, bookmark, comment, table, dan lainnya.

Links : Perintah untuk membuat hubungan dengan file / dokumen lain.

Object : Perintah untuk mengedit objek.

27. dalam menu bar terdapat menu file dan menu edit apakah fungsi dari ke dua menu


Menu file berfungsi untuk mengolah dokumen secara umum, seperti:

- Membuat dokumen baru (New)

- Membuka dokumen yang sebelumnya telah dibuat (Open)

- Mencetak dokumen (Print)

- Menyimpan dokumen (Save)

- Menutup dokumen (Close)

Menu edit berfungsi untuk mengolah dokumen secara spesifik, seperti:

- Menyalin teks di dalam dokumen (Copy)

- Memotong teks di dalam dokumen (Cut)

- Menempel teks yang sebelumnya telah disalin (Paste)

- Mencari teks tertentu dalam dokumen (Find)

33. Sebutkan 5 fungsi dari menu edit


Mungkin bagi kalian yang sudah banyak mengenal komputer tidak asing lagi pada saat mendengar kata Edit. Pada umumnya bukan hanya file dokumen saja yang bisa kita edit pada komputer, tapi beberapa jenis file lainnya pun bisa kita edit dan tentunya harus dibarengi dengan ilmu pengetahuan yang memadai untuk melakukannya. Namun pada saat kita ingin melakukan pengeditan pada dokumen atau yang masih dalam batas kemampuan kita, kita pun tak perlu repot-repot harus membuka Windows Explorer hanya cukup dengan menggunakan Aplikasi dari Windows yakni yang disebut Edit.

Bagaimana cara membuka Aplikasi Edit?

Pertama kamu bisa menekan Windows + R lalu tuliskan EDIT dan kamu juga bisa membukanya dari Windows Command Prompt, Windows + R =CMD lalu tuliskan Edit atau Start Menu/Accessories/Command Prompt.

Fungsi Edit tak ubah layaknya Notepad/Teks Dokumen atau pun aplikasi lainnya semisal Ms Words dan justru itu lebih lengkap lagi karena bisa digunakan untuk mengedit program, baik itu program sederhana seperti (*.bat, *.cmd, *.txt, *.dll) dan banyak lagi program-program yang bisa kamu edit di Edit program ini.
