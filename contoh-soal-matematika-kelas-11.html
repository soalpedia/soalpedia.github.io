Jawaban:

3(4^2) - 2(4) + 5 = 3(16) - 8 + 5 = 48 - 8 + 5 = 45.

Soal 2:

Sebuah segitiga memiliki panjang sisi a = 5 cm, b = 7 cm, dan sudut C = 60 derajat. Hitung panjang sisi c.

Jawaban:

Dengan hukum sinus, kita dapat menghitung panjang sisi c:

c = (a * sin(C)) / sin(B)

c = (5 * sin(60°)) / sin(180° - 60° - 90°)

c = (5 * √3/2) / √3/2

c = 5 cm.

Hitung integral dari f(x) = 2x^3 - 3x^2 + 4x - 1.

Jawaban:

Integral dari f(x) adalah:

F(x) = (2/4)x^4 - (3/3)x^3 + (4/2)x^2 - x + C

F(x) = (1/2)x^4 - x^3 + 2x^2 - x + C.

Luas trapesium dapat dihitung dengan rumus: L = (1/2)(a + b) * h

Hitung akar-akar persamaan kuadrat 2x^2 - 5x + 3 = 0.

x = (-b ± √(b^2 - 4ac)) / (2a)

x1 = (5 + √(5^2 - 4(2)(3))) / (2(2)) = (5 + √(25 - 24)) / 4 = (5 + 1) / 4 = 6/4 = 3/2

x2 = (5 - √(5^2 - 4(2)(3))) / (2(2)) = (5 - √(25 - 24)) / 4 = (5 - 1) / 4 = 4/4 = 1

Jadi, akar-akar persamaan tersebut adalah x1 = 3/2 dan x2 = 1.

Soal 6:

Hitung integral dari ∫(2x^2 + 3x - 4) dx.

(2/3)x^3 + (3/2)x^2 - 4x + C.

22. contoh soal turunan kelas 11 matematika f(x)=3/2x⁴+2/3x³-2​


Jika f(x) = 3/2x⁴ + ⅔x³ - 2, maka f'(x) = 6x³ + 2x².

Turunan atau diferensiasi adalah bentuk pengukuran terhadap bagaimana fungsi berubah seiring perubahan nilai yang dimasukkan dan menunjukkan bagaimana suatu besaran berubah akibat perubahan besaran lainnya. Turunan suatu fungsi dinyatakan dengan y' atau f'(x) di mana perumusannya memiliki beberapa ketentuan sebagai berikut.

• y = f(x) = axⁿ, maka turunan pertamanya adalah y' = f'(x) = a.n.xⁿ-¹

• y = f(x) = a, maka turunan pertamanya adalah y' = f'(x) = 0

Sehingga, untuk suatu fungsi aljabar yang dinyatakan dengan y = f(x) = axⁿ + bx + c memiliki turunan pertama y' = f'(x) = a.n.xⁿ-¹ + b.

Agar lebih jelasnya, simak pembahasan soal berikut.

PEMBAHASAN :

Perhatikan kembali soal di atas.

Diketahui suatu fungsi dinyatakan dengan f(x) = 3/2x⁴ + ⅔x³ - 2. Maka, turunan fungsi tersebut dapat dinyatakan sesuai dengan ketentuan yang telah tertulis di atas.

y = f(x) = 3/2x⁴ + ⅔x³ - 2

y' = f(x) = [3/2 . 4 . x⁴-¹] + [⅔ . 3 . x³-¹] - 0

y' = f'(x) = 6x³ + 2x²

Jadi, jika f(x) = 3/2x⁴ + ⅔x³ - 2, maka f'(x) = 6x³ + 2x².
