4. ada 2 cara by PC anda (maksut saya pc anda apa (window atau mac?))

1. Cara membuat nomor halaman di Windows

Pertama pilih tab Insert > Page Number lalu ilih lokasi dan gaya nomor halaman yang pengguna inginkan.

Jika pengguna tidak ingin nomor halaman ditampilkan di halaman pertama, pilih Different First Page.

Jika pengguna ingin penomoran dimulai dengan angka satu (1) di halaman kedua, buka Page Number > Format Page Numbers dan atur Start di angka nol (0).

Setelah selesai, pilih Close Header and Footer atau tekan Esc.

2. Cara membuat nomor halaman di macOS

Pertama pilih tab Insert, klik ikon Page Number, lalu klik Page Number.
