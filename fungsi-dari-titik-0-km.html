3x - 5 = 0

2x - 6 = 0

Untuk menentukan apakah ada titik potong, kita perlu memecahkan kedua persamaan ini dengan mencari nilai x yang memenuhi kedua persamaan. Jika kita memecahkan persamaan pertama, kita dapatkan:

3x = 5

x = 5/3

Namun, ketika kita mencoba memecahkan persamaan kedua, kita temukan:

2x = 6

x = 6/2

x = 3

Nilai x yang didapatkan berbeda, yaitu x = 5/3 dari persamaan pertama dan x = 3 dari persamaan kedua. Karena kedua persamaan tersebut tidak memiliki nilai x yang sama, maka garis yang dibentuk oleh fungsi tersebut tidak memiliki titik potong.
