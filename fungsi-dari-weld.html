2. apa saja fitur-fitur yang ada di dalam corel draw? apa fungsi dari weld dan distorsi?


Tool Bar

Top Bar merupakan sekumpulan tombol shortcut yang memiliki fungsi untuk menjalankan perintah dengan cepat. ShortCut. Ini adalah bagian yang paling sering digunakan oleh desainer grafis dalam melakukan sesuatu. CorelDraw memberikan fitur dalam toolbar yang powerfull namun mudah sekali digunakan.

Menu Bar

Menubar berisi tombol-tombol text yang memiliki banyak sekali fungsi yang berbeda sesuai dengan namanya masing-masing. Desainer pun sering menggunakan fasilitas ini biasanya dalam memberikan suatu effect berhubungan dengan vector dan bitmap.

Property Bar

Property Bar berisi sekumpulan tombol yang isinya itu menyesuaikan dengan tools yang sedang kita pilih di dalam tool box. Untuk lebih jelasnya kita akan pelajari lebih detil dipertemuan yang akan datang.

Docker

Docker merupakan kotak dialog yang di dalamnya berisi berbagai tombol perintah untuk mengatur, memberi effect kepada sebuah objek maupun teks.

Rulers

Rulers merupakan alat ukur berupa penggaris dibagian atas dari drawing page, dan di bagian kanan drawing page, rullers digunakan sebagai acuan yang mampu menunjukkan ukuran suatu objek atau dokumen, serta tata letak suatu desain.

Status Bar

Status Bar terletak dibagian bagian bawah area program sebelah kanan. Status Bar berisi informasi mengenai objek (teks) seperti ukuran, bentuk, warna, outline, format gambar, jenis huruf beserta  ukurannya. Anda bisa melakukan perubahan terkait bentuk, warna, outline dan format gambar menggunakan status bar ini.

Drawing Page

Drawing Page adalah adalah sebuah area kerja utama di dalam Corel Draw di mana Anda dapat mendesain suatu objek di dalamnya dan merupakan area cetak suatu desain.

Color Palette

Color Palette bagian dari CorelDraw yang berisi koleksi warna di dalam sebuah bar dan secara default/bawaan terletak di sebelah kanan dokumen / program.

Di pertemuan yang akan datang, kita akan bahas satu persatu fitur dari toolbox disertai dengan contoh kasusnya. Semoga bermanfaat.

7. Sebuah plat dengan lebar 75 mm dan tebal 12,5 mm di sambung dengan plat lain dengan single transverse weld and double parallel fillet seperti gambar. Tegangan Tarik maksimum 60 MPa dan tegangan geser 50 MPa. Hitung panjang setiap parallet fillet untuk beban statis dan fatigue​


Jawaban:

Untuk mengetahui panjang parallet fillet yang diperlukan untuk beban statis dan fatigue, kita perlu mengidentifikasi tegangan geser dan tegangan tarik pada plat ini.

Untuk beban statis, tegangan tarik terjadi pada plat atas, sedangkan tegangan geser terjadi pada plat bawah. Pada plat atas, tegangan tarik dapat dihitung menggunakan rumus:

Tte = (plata x L / (bh))^3

di mana:

- L adalah panjang plat atas (75 mm + 2x7,5 mm)

- b adalah tebal plat atas (12,5 mm)

- h adalah tinggi plat atas

- plata adalah koefisien tarik yang berbeda-beda menurut bahan plat yang digunakan.

Jika kita menggunakan bahan stahl, maka koefisien tariknya adalah 0,092, sedangkan jika bahan plat yang digunakan adalah aluminium yang punya koefisien tarik 0,29. Karena plat atas tidak memiliki tinggi, h = 0. Maka:

Tte(Stahl) = (0,092 x 122,5) 3

Tte(Stahl) = 1526,32 MPa

Tte(Alumunium) = (0,29 x 122,5)3

Tte(Alumunium) = 962,59 MPa

Untuk beban fatigue, tegangan tarik terjadi pada plat bawah, sedangkan tegangan geser terjadi pada plat atas. Pada plat bawah, tegangan geser dapat dihitung menggunakan rumus:

Tge = (pb x L / (bh))^3

di mana:

- L é a largura da peça (75 mm + 2xF)

- b é a espidez da peça (12,5 mm)

- h é a altura da peça

- pb é a carga de fatiga

- F é o comprimento da peça

21. Sebuah plat dengan lebar 120 mm dan tebal 15 mm di sambung dengan plat lain dengan single transverse weld and double parallel fillet weld seperti gambar. Tegangan tarik maksimum 70 MPa dan tegangan geser 56 MPa. Hitung panjang las parallel fillet untuk beban statis dan fatigue.​


Gambar plat yang disambung:

```

_______ <- plat A

| |

___ | | ___ <- plat B

| | | |

|___| |___|

```

Dalam perhitungan ini, diasumsikan bahwa tegangan tarik dan geser hanya bekerja pada plat A karena plat B menahan beban tegangan tarik dan geser secara merata.

Untuk beban statis, kita dapat menggunakan rumus tegangan sederhana:

```

tegangan tarik maksimum = F / A

tegangan geser maksimum = V / A

```

dengan `F` adalah gaya tarik maksimum, `A` adalah luas penampang plat sambungan (dalam mm^2), dan `V` adalah gaya geser maksimum. Kita dapat menghitung `A` dari lebar dan tebal plat:

```

A = lebar x tebal = 120 mm x 15 mm = 1800 mm^2

```

Maka, untuk tegangan tarik maksimum 70 MPa:

```

F = tegangan tarik maksimum x A = 70 MPa x 1800 mm^2 = 126000 N

```

Untuk tegangan geser maksimum 56 MPa:

```

V = tegangan geser maksimum x A = 56 MPa x 1800 mm^2 = 100800 N

```

Kita dapat menghitung kapasitas beban tarik dan geser dari fillet weld berdasarkan ukuran lasnya. Dalam hal ini, kita akan menghitung panjang las untuk beban tarik dan geser secara maksimum.

Untuk beban tarik maksimum, kita dapat menggunakan rumus sebagai berikut:

```

F = 0.707 x 2 x sigma_weld x l x s

```

dengan `sigma_weld` adalah kekuatan tarik las dalam MPa, `l` adalah panjang las, dan `s` adalah jarak antara las dan tepi plat. Kita dapat memperkirakan `sigma_weld` sebagai 80% dari kekuatan minimum kelas plat (dalam hal ini, menggunakan kelas plat baja sedang sekitar 400 MPa).

```

sigma_weld = 0.8 x 400 MPa = 320 MPa

```

Maka,

```

l = F / (0.707 x 2 x sigma_weld x s) = 126000 N / (0.707 x 2 x 320 MPa x s) = 221.3 / s mm

```

Untuk beban geser maksimum, kita dapat menggunakan rumus sebagai berikut:

```

V = 0.5 x sigma_weld x l x s

```

Maka,

```

l = 2 x V / (sigma_weld x s) = 2 x 100800 N / (320 MPa x s) = 628.125 / s mm

```

Jadi, panjang las parallel fillet untuk beban tarik maksimum menjadi 221.3 / s mm, dan untuk beban geser maksimum menjadi 628.125 / s mm. Panjang lasnya harus dihitung berdasarkan faktor keamanan tambahan yang diinginkan untuk beban fatigue.

37. Apa perbedaan dari tool WELD dan TRIM pada CorelDraw


Perbedaan antara alat "WELD" dan "TRIM" dalam CorelDRAW adalah sebagai berikut:

1. **WELD:**

  - **Deskripsi:** Alat "WELD" digunakan untuk menggabungkan dua objek atau lebih menjadi satu objek tunggal.

  - **Cara Kerja:** Saat Anda menggabungkan objek dengan "WELD", garis-garis pemotongan antara objek-objek tersebut dihilangkan, dan objek-objek tersebut bergabung menjadi satu objek tunggal. Hasilnya adalah objek tunggal yang tidak memiliki pemotongan atau jahitan di antara objek-objek yang digabungkan.

  - **Contoh:** Jika Anda memiliki dua objek berbentuk lingkaran yang tumpang tindih, penggunaan "WELD" akan menggabungkannya menjadi satu lingkaran besar tanpa bagian yang tumpang tindih.

2. **TRIM:**

  - **Deskripsi:** Alat "TRIM" digunakan untuk menghilangkan bagian dari objek yang tumpang tindih dengan objek lain.

  - **Cara Kerja:** Ketika Anda menggunakan "TRIM", Anda memilih objek yang menjadi "pisau pemotong" dan objek-objek lain yang akan dipotong atau dihilangkan sebagian. Bagian dari objek yang berada di luar area objek pemotong akan dihilangkan.

  - **Contoh:** Jika Anda memiliki dua objek yang tumpang tindih, dan Anda ingin menghilangkan bagian yang tumpang tindih dari salah satu objek, Anda dapat menggunakan "TRIM" dengan salah satu objek sebagai pemotong.

Dengan demikian, "WELD" digunakan untuk menggabungkan objek-objek menjadi satu, sementara "TRIM" digunakan untuk memotong atau menghilangkan bagian-bagian objek yang tumpang tindih. Kedua alat ini berguna dalam proses pengeditan dan perancangan grafis di CorelDRAW.
