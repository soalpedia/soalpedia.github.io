17. tentukan range dari fungsi f(x)=x^2+4x-36 jika diketahui domain df=(-1 <×<5)


Untuk menentukan range dari fungsi f(x) = x^2 + 4x - 36 dengan domain yang diberikan, yaitu -1 < x < 5, kita dapat melakukan langkah-langkah berikut:

1. Tentukan nilai minimum dan maksimum dari fungsi f(x) di dalam domain yang diberikan. Untuk mencari nilai ekstremum, kita akan mencari turunan pertama fungsi f(x) dan menyelesaikan turunan ini untuk mencari titik-titik kritis.

f(x) = x^2 + 4x - 36

f'(x) = 2x + 4

Untuk menemukan titik-titik kritis, kita atur f'(x) = 0:

2x + 4 = 0

2x = -4

x = -2

Jadi, titik kritisnya adalah x = -2.

2. Sekarang kita akan mengevaluasi f(x) pada batas domain, yaitu -1 dan 5, serta pada titik kritis x = -2:

a. f(-1) = (-1)^2 + 4(-1) - 36 = 1 - 4 - 36 = -39

b. f(-2) = (-2)^2 + 4(-2) - 36 = 4 - 8 - 36 = -40

c. f(5) = (5)^2 + 4(5) - 36 = 25 + 20 - 36 = 9

3. Jadi, rentang (range) dari fungsi f(x) dalam domain -1 < x < 5 adalah dari -40 hingga 9, atau secara matematis ditulis sebagai:

Range(f) = {-40 ≤ y ≤ 9}

Jadi, rentang dari fungsi f(x) adalah dari -40 hingga 9.

23. tentukan domain (Df) dan range fungsi (Rf) dari f(x) = x²-x, dengan menggunakan invers​


Jawaban:

Jadi, domain (Df) dari f(x) = x² - x adalah x ≥ -0.25 dan range (Rf) adalah y ≤ 0.5 atau y ≥ 0.

Penjelasan dengan langkah-langkah:

Untuk menentukan domain (Df) dan range (Rf) dari fungsi f(x) = x² - x menggunakan invers, kita perlu mengambil invers dari fungsi tersebut terlebih dahulu.

Langkah pertama adalah mengganti f(x) dengan y:

y = x² - x

Langkah kedua adalah menukar tempat antara x dan y:

x = y² - y

Langkah ketiga adalah menyelesaikan persamaan kuadrat tersebut untuk mendapatkan invers dari fungsi f(x).

x = y² - y

y² - y - x = 0

Kita dapat menggunakan rumus kuadrat atau melengkapi kuadrat untuk menyelesaikan persamaan ini:

(y - 0.5)² - (0.5)² - x = 0

(y - 0.5)² = x + 0.25

y - 0.5 = ±√(x + 0.25)

Menggunakan tanda plus:

y - 0.5 = √(x + 0.25)

y = √(x + 0.25) + 0.5

Menggunakan tanda minus:

y - 0.5 = -√(x + 0.25)

y = -√(x + 0.25) + 0.5

Sehingga invers dari fungsi f(x) adalah:

g(x) = √(x + 0.25) + 0.5 dan h(x) = -√(x + 0.25) + 0.5

Sekarang kita dapat menentukan domain (Df) dan range (Rf) dari fungsi f(x) menggunakan invers ini.

Untuk g(x) = √(x + 0.25) + 0.5, domainnya adalah x ≥ -0.25 karena nilai di dalam akar kuadrat harus ≥ 0.

Untuk Rf, tidak ada batasan pada akar kuadrat sehingga Rf adalah semua bilangan real positif y ≥ 0.

Untuk h(x) = -√(x + 0.25) + 0.5, domainnya juga adalah x ≥ -0.25.

Untuk Rf, akar kuadrat negatif memberikan nilai negatif, jadi Rf adalah semua bilangan real y ≤ 0.5.

Jadi, domain (Df) dari f(x) = x² - x adalah x ≥ -0.25 dan range (Rf) adalah y ≤ 0.5 atau y ≥ 0.

34. Domain fungsi (Df) dari f(x) = x² + 2x - 3agar mempunyai fungsi invers adalah....Mohon bantuannya ya​


Domain fungsi [tex]f(x)=x^2+2x-3[/tex] agar mempunyai fungsi invers adalah E. {x|x, x ≤ -1, x ∈ R} dan {x|x, x ≥ -1, x ∈ R}.

PEMBAHASAN

Fungsi invers adalah sebuah fungsi yang berkebalikan dari fungsi asalnya. fungsi invers dari [tex]f(x)[/tex] dilambangkan dengan [tex]f^{-1}(x)[/tex]. Suatu fungsi memiliki invers jika fungsi tersebut adalah fungsi bijektif (fungsi satu-satu), dimana setiap anggota di domain memiliki tepat satu pasangan di kodomain.

.

DIKETAHUI

[tex]f(x)=x^2+2x-3[/tex]

.

DITANYA

Tentukan domain fungsi f(x) agar mempunyai fungsi invers.

.

PENYELESAIAN

Kita cek terlebih dahulu apakah fungsi kuadrat f(x) termasuk fungsi bijektif atau bukan. Misal kita ambil x = -3 dan x = 1.

[tex]f(-3)=(-3)^2+2(-3)-3[/tex]

[tex]f(-3)=0[/tex]

.

[tex]f(1)=(1)^2+2(1)-3[/tex]

[tex]f(1)=0[/tex]

.

Karena f(-3) = f(1) maka f(x) bukan fungsi bijektif, sehingga kita perlu membatasi domain fungsinya agar menjadi fungsi bijektif. Kita batasi pada garis sumbu simetrinya. Mengapa? Karena sumbu simetri membagi parabola menjadi 2 bagian yang sama. Sehingga domain fungsi f(x) akan tepat memiliki satu pasangan di kodomain nya.

[tex]f(x)=x^2+2x-3[/tex]

a = 1

b = 2

c = -3

Diperoleh sumbu simetrinya :

[tex]x_p=-\frac{b}{2a}[/tex]

[tex]x_p=-\frac{2}{2(1)}[/tex]

[tex]x_p=-1[/tex]

.

Sehingga domain fungsinya kita batasi menjadi x ≤ -1 dan x ≥ -1.

.

KESIMPULAN

Domain fungsi [tex]f(x)=x^2+2x-3[/tex] agar mempunyai fungsi invers adalah E. {x|x, x ≤ -1, x ∈ R} dan {x|x, x ≥ -1, x ∈ R}.

.
