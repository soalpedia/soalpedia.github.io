2. apakah fungsi dari gambar potongan


Kategori Soal : Teknik Mesin
Materi : Gambar Teknik
Kata Kunci : gambar, potongan

Pembahasan :

Hai, saya coba bantu jawab yaa.. Sebelumnya saya akan jelaskan gambar potongan itu apa, kemudian fungsinya.

Gambar potongan merupakan gambar yang menunjukkan bagian-bagian yang tersembunyi dari suatu part atau komponen atau membuang bagian yang menutupi.

Nah, fungsi dari gambar potongan ini adalah memperlihatkan bagian-bagian dari benda yang tertutupi tadi (bisa berbentuk rumit dan berongga) untuk diperlihatkan dengan jelas, sehingga bagian yang berbentuk rumit/kompleks dan berongga atau bagian yang tersembunyi dapat terlihat dengan lebih jelas.

Biasanya bidang potong dibuat melalui sumbu dasar, kemudian potongannya disebut potongan utama.
Jika potongannya dibuat diluar sumbu dasar, maka potongannya harus diberi tanda dan arah penglihatannya dinyatakan dengan anak panah.

24. Fungsi kuadrat f(x)=x²-2x-8 tentukan Titik potong dengan sumbu x, tentukan Titik potong dengan sumbu y, nilai ekstrim dan titik ekstrim, gambar grafik fungsi kuadrat​


Diketahui fungsi kuadrat :

f(x) = x² - 2x - 8

Titik potong sumbu x jika f(x) = 0, maka :

f(x) = x² - 2x - 8

0 = x² - 2x - 8

x² - 2x - 8 = 0

( x - 4 )( x + 2 ) = 0

x1 = 4 , x2 = -2

Titik potong sumbu y jika x = 0, maka :

f(x) = x² - 2x - 8

y = x² - 2x - 8

y = 0² - 2(0) - 8

y = 0 - 0 - 8

y = -8

Sumbu simetrinya :

sumbu simetri = ( x1 + x2 ) / 2

sumbu simetri = ( 4 + (-2) ) / 2

sumbu simetri = 2/2

sumbu simetri = 1

Titik puncak x :

x puncak = sumbu simetri

x puncak = 1

x puncak = 1

Titik puncak y :

f(x) = x² - 2x - 8

f(x puncak) = (x puncak)² - 2(x puncak) - 8

f(1) = (1)² - 2(1) - 8

f(1) = 1 - 2 - 8

f(1) = -9

y puncak = -9

Titik ekstrim fungsi :

titik ekstrim = titik puncak

titik ekstrim = ( x puncak , y puncak )

titik ekstrim = ( 1 , -9 )

25. gambarkan fungsi f(x)=x2-3x-10 dan fungsi g(x)=2x-4. tentukan titik potong kedua fungsi tersebut


Karena yang diminta adalah titik potong, maka "samakan" kedua fungsi tersebut, sehingga:

f(x) = g(x)
x² - 3x - 10 = 2x - 4
x² - 3x - 2x - 10 + 4 = 0
x² - 5x - 6 = 0
(x + 1)(x - 6) = 0
x₁ = -1,  x₂ = 6 -> koordinat x

g(-1) = 2(-1) - 4
        = -2 - 4
        = -6
g(6) = 2(6) - 4
       = 12 - 4
       = 8
PS : g(x) sama dengan y
Titik potong kedua grafik terletak di (-1, -6) dan (6, 8)
