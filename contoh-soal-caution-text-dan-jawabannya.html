37. Soal-soal tentang caution, notice, warning


Explanation:

A notice is a short text which is used to give an information, give an instruction, or warn public. Notice will not make any personal injury if it's violated.
For example:
NOTICE
No smoking

A caution is a short text which is used to indicate a hazardous situation which could cause a minor or moderate injury if not avoided.
For example:
CAUTION
Wet floor

A warning is a short text which is used to indicate a hazardous situation which could cause a serious injury or death if not avoided.
For example:
WARNING
Unsafe for swimming
