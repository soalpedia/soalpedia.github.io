12. fungsi dari brigde, hub, router, Ethernet card​


Jawaban:

NIC ( Network Interface Controller )

NIC adalah perangkat keras yang berbentuk kartu dengan kegunaan untuk menjadi jembatan komputer ke sebuah jaringan komputer. Cara kerjanya mengubah aliran data paralel dalam bus komputer menjadi bentuk data serial yang dapat ditransmisikan kedalam media jaringan. Berikut ini fungsi-fungsinya:

Sebagai media pengirim data ke komputer lain di dalam jaringan.

Untuk mengotroldataflowantara komputer dan sistem kabel.

Untuk menerima data yang dikirim dari komputer melalui kabel, lalu diterjemahkan kedalam bitagar dapat dimengerti oleh komputer.

Repeater

Repeater adalah perangkat yang berfungsi memperluas jangkuan sinyalWIFIsehingga wilayah yang tidak dapat menangkap sinyal WIFI, bisa menangkap sinyal WIFI. Harus ada 2 alat untuk Repeater, yaitu alat untuk menerima sinyal dari server( Client ) dan alat untuk menyebarkan sinyal WIFI( Accespoin t). Berikut ini fungsi-fungsinya:

Untuk memperluas sinyal dari Server( pemancar ).

Untuk mempermudah akses sinyal WIFIdari Server.

Untuk memberika sinyal pada daerah yang jauh dari Server( pemancar ).

Hub

Hub adalah perangkat jaringan yang beroperasi di OSI layer 1. Physical Layer. Perangkat tersebut sebagai penyambung atau concentrator, dan menguatkan sinyal di kabel UTP. Menggunakan Hub dapat mengalami collisionkarena Hub tidak dapat mengenal MAC Address / Physical Addressyang mengakibatkan tidak dapat memilah data yang akan ditransmisikan. Berikut ini fungsi-fungsinya:

Tempat menambahkan atau mengurangi workstation.

Seperti Repeater yaitu menambah jarak network.

Bisa mendukung interfaceyang berbeda ( Ethernet, Token ring, dan FDDI).

Mempunyai fitur Fault Tolerance ( isolasi kerusakan )

Mempunyai pengelolaan yang tersentralisasi ( koleksi informasi, diagnostic)

Switch

Switch adalah perangkat jaringan yang beroperasi di OSI Layer 2, Data Link Layer.Perangkat tersebut sebagai penyambung atau concentrator dalam jaringan. Berbeda dengan Hub, Switch tidak dapat mengalami collision karena Switch dapat mengenal MAC Address / Physical Addresssehingga Switch dapat memilah data yang akan ditransmisikan. Berikut ini fungsi-fungsinya:

Sebagai penguat atau repeater.

Untuk menghubungkan kabel-kabel UTP( katagori 5/5e ) antar komputer.

Menghubungkan antar komputer dalam LAN.

Router

Router adalah perangkat jaringan yang beroperasi di OSI Layer 3, Network Layer. Layerini terkenal dengan pengalamatan jaringan menggunakan IP Address. Sehingga Router berperan sebagai penghubung  atau penerus paket data antara dua segmen jaringan atau lebih. Berikut ini fungsinya:

Sebagai penghubung antar dua atau lebih jaringan yang akan meneruskan data dari satu jaringan ke jaringan lainnya.

Bridge

Bridge adalah perangkat jaringan yang beroperasi di OSI Layer 2, data-link.Perangkat tersebut untuk membuat sebuah segmen jaringan atau memperluas jaringan. Bisa juga untuk menggabungkan dua buah media jaringan yang berbeda, contoh antara kabel UTPdengan kabel serat optik atau bisa juga dengan dua buah arsitektur jaringan yang berbeda. Berikut ini fungsi-fungsinya:

Untuk membagi jaringan menjadi dua buah jaringan.

Untuk mengatur informasi antar networkagar tertib.

17. cara memasang dan langkahnya:?-komputer server-client-lan card-kabel penghubung-router-hub dan switch-repeater-bridge-modulator Demodukator​


Jawaban:

Langkah 1: Persiapan dan Perencanaan

Sebelum memulai pemasangan, perlu dilakukan perencanaan jaringan. Tentukan bagaimana topologi jaringan Anda akan dibuat (misalnya, star, bus, atau mesh), identifikasi perangkat yang akan digunakan, dan tentukan lokasi fisik mereka.

Langkah 2: Pemasangan Server

Tempatkan komputer server di lokasi yang sesuai dan pastikan sumber daya seperti daya listrik dan pendingin udara tersedia.

Pasang sistem operasi server yang sesuai (seperti Windows Server atau Linux) dan konfigurasikan sesuai kebutuhan.

Instal perangkat keras tambahan seperti kartu jaringan (NIC), jika diperlukan.

Langkah 3: Pemasangan Client

Tempatkan komputer client di lokasi yang sesuai.

Pasang sistem operasi client (seperti Windows atau MacOS) dan konfigurasikan sesuai kebutuhan.

Instal kartu jaringan (NIC) jika belum terpasang.

Langkah 4: Instalasi LAN Card (NIC)

Pasang kartu jaringan (NIC) ke slot yang sesuai di komputer server dan client.

Pastikan driver NIC diinstal dan konfigurasikan alamat IP, subnet mask, dan pengaturan jaringan lainnya.

Langkah 5: Kabel Penghubung

Hubungkan komputer server dan client menggunakan kabel Ethernet yang sesuai (biasanya kabel UTP Cat 5e atau Cat 6).

Pastikan kabel terpasang dengan baik ke port NIC di kedua komputer.

Langkah 6: Pemasangan Router

Tempatkan router di lokasi yang strategis dalam jaringan Anda.

Hubungkan kabel WAN router ke koneksi internet (misalnya, kabel broadband atau DSL).

Hubungkan kabel LAN router ke salah satu port LAN pada server atau switch.

Konfigurasikan router sesuai dengan kebutuhan, termasuk pengaturan jaringan lokal (LAN) dan pengaturan keamanan.

Langkah 7: Hub dan Switch

Jika Anda menggunakan hub atau switch, hubungkan perangkat ini ke port LAN router atau server.

Hub atau switch akan digunakan untuk menghubungkan beberapa perangkat client ke jaringan.

Langkah 8: Repeater

Jika Anda memiliki jarak yang cukup jauh dalam jaringan Anda, pertimbangkan penggunaan repeater untuk memperluas jangkauan sinyal.

Tempatkan repeater di lokasi yang sesuai dan hubungkan dengan kabel ke jaringan utama.

Langkah 9: Bridge

Bridge digunakan untuk menghubungkan dua segmen jaringan yang berbeda.

Konfigurasikan bridge sesuai dengan kebutuhan untuk menghubungkan segmen jaringan yang berbeda.

Langkah 10: Modulator Demodulator (Modem)

Jika Anda menghubungkan jaringan Anda ke koneksi internet lewat DSL atau kabel, pasang modem yang sesuai.

Hubungkan modem ke port WAN router atau langsung ke komputer server, tergantung pada konfigurasi Anda.

Langkah 11: Pengujian

Setelah semua perangkat terpasang, uji jaringan Anda untuk memastikan bahwa semua perangkat terhubung dengan benar dan berfungsi dengan baik.

Lakukan pengujian konektivitas antara server dan client, serta pengujian internet jika koneksi internet tersedia.
