2. karakter Linux centos​


Jawaban:

Berikut adalah beberapa karakteristik atau fitur utama dari sistem operasi Linux CentOS:

1. Berbasis Open Source: Linux CentOS merupakan distribusi Linux yang sepenuhnya open-source dan gratis untuk digunakan. Ini berarti bahwa kode sumbernya dapat diakses dan dimodifikasi oleh siapa saja.

2. Stabilitas dan Keamanan: CentOS dirancang untuk memberikan tingkat kestabilan dan keamanan yang tinggi. Distribusi ini dikenal karena penyediaan patch keamanan yang konsisten dan dukungan jangka panjang, menjadikannya pilihan yang populer untuk server dan infrastruktur.

3. Kompatibilitas dengan Red Hat Enterprise Linux (RHEL): CentOS dibangun dengan menggunakan kode sumber yang sama dengan RHEL. Hal ini memungkinkan aplikasi dan perangkat lunak yang dikembangkan atau diuji di RHEL untuk berjalan dengan baik di CentOS.

4. Dukungan dan Komunitas: Meskipun CentOS tidak dikomersialkan dan tidak memiliki dukungan resmi, tetapi ada komunitas yang sangat aktif di sekitar distribusi ini. Komunitas ini menyediakan forum, dokumentasi, dan sumber daya lainnya untuk membantu pengguna CentOS.

5. Pengelolaan Paket: CentOS menggunakan sistem manajemen paket yum (Yellowdog Updater, Modified), yang memudahkan instalasi, pembaruan, dan penghapusan paket-paket perangkat lunak.

6. Lingkungan Desktop dan Server: CentOS dapat digunakan baik sebagai sistem operasi desktop maupun server. Desktop Environment seperti GNOME dan KDE tersedia untuk menghadirkan pengalaman desktop yang intuitif dan lengkap. Di sisi server, CentOS dapat digunakan sebagai platform yang andal untuk menjalankan berbagai layanan dan aplikasi server.

7. Performa Tinggi: CentOS dikenal karena performa tingginya, terutama dalam pengelolaan sumber daya dan keandalan.

8. Skalabilitas: CentOS mendukung berbagai arsitektur komputer, mulai dari x86 hingga ARM, dan dapat diimplementasikan di berbagai lingkungan komputasi, dari server tunggal hingga klaster.

9. Pembaruan Jangka Panjang: Salah satu kelebihan dari CentOS adalah penyediaan pembaruan jangka panjang, termasuk pembaruan keamanan. Ini memungkinkan pengguna CentOS untuk menjaga sistem mereka tetap aman dan terkini.

Ini hanya beberapa karakteristik utama dari Linux CentOS. Distribusi ini populer di kalangan administrator sistem dan pengembang aplikasi karena stabilitasnya, keandalannya, dan kesesuaian dengan infrastruktur berbasis Linux.


3. Gan,ane mau nanya.Cara install SSH di centos 6.4 gimana?Tolong pencerahannya,dan satu lg apakah ada repository di centos ? Jangan ngasal


1.Install Open SSH Server and SSH client
# yum -y install openssh-server openssh-clients
2.Start OpenSSH Service
# chkconfig sshd on
# service sshd start
3.OpenSSH Server configuration
# vi /etc/ssh/sshd_config
4.open port 22
# netstat -tulpn | grep :22
5.Edit /etc/sysconfig/iptables (IPv4 firewall),
# vi /etc/sysconfig/iptables
6.Add the line
-A RH-Firewall-1-INPUT -m state --state NEW -m tcp -p tcp --dport 22 -j ACCEPT
7.Save and close the file
# service iptables restart

perintah menggunakan yum install
