1. apa yang dimaksud dengan void setup dan void loop​


Berikut maksud dari void setup dan void loop:

VOID SETUP adalah keyword atau kata kunci yang berjalan hanya satu kali saja. Void setup ini berjalan satu kali di awal atau saat program dijalankan pertama kali. Void setup termasuk kode fungsi.VOID LOOP adalah keyword yang fungsinya melaksanakan ataupun mengeksekusi suatu perindah yang asalnya dari program yang berulang atau berjalan secara terus menerus (loops) selama program berjalan (board Arduino aktif).Pembahasan

Void berkaitan dengan Arduino ide. Void sendiri dikenal sebagai keyword atau kata kunci berupa kode khusus yang dipakai dalam mendeklarasikan suatu fungsi di setiap program. Tujuan void yakni agar fungsi ataupun program yang dideklarasikan tak membuat informasi atau perintah fungsi kembali pada program asal pemanggilan. Ada dua jenis voud yang sifatnay wajib ada yakniL

Void setup.Void loop.

Setup pada jenis void pertama mengindikasikan bahwa void ini hanya kita jumpai pada awal penggunaan suatu program. Awal maksudnya di sini adalah penggunaan paling pertama. Sedangkan apabila void berulang pada sebuah program maka sebutannya adalah void loop sebab kata loop sendiri bermakna pengulangan yang berlangsung terus menerus dalam bahasa Inggris.

17. 1)Tuliskan "urutan" pin digital pada arduino Uno 2)tuliskan "urutan" pin analog pada arduino3)tuliskan fungsi dari void setup()dari app tinkercad​


Jawaban:

1. Urutan Pin Digital pada Arduino Uno:

Arduino Uno memiliki 14 pin digital yang diberi label sebagai D0 hingga D13. Pin D0 dan D1 digunakan sebagai pin serial komunikasi UART (Universal Asynchronous Receiver Transmitter) untuk menghubungkan Arduino dengan perangkat lain, seperti komputer. Pin D2 hingga D13 dapat digunakan sebagai input atau output digital.

2. Urutan Pin Analog pada Arduino Uno:

Arduino Uno memiliki 6 pin analog yang diberi label sebagai A0 hingga A5. Pin-pin ini dapat digunakan sebagai input analog, yang memungkinkan pembacaan tegangan antara 0 hingga 5 volt. Selain itu, pin-pin analog ini juga dapat digunakan sebagai pin digital.

Nama metode Kembalian yang dikembalikan setelah memanggil method. Input parameter. Badan atau isi method, berupa instruksi untuk melakukan sesuatu hal.

}"
