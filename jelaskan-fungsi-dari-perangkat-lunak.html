9. tuliskan minimal semua perangkat lunak di komputer,dan jelaskan apa fungsinya?​


Casing komputer (case) adalah bagian yang menjadi “rumah” bagi sebagian besar komponen komputer. Mayoritas desktop PC memiliki casing box baik dari plastik ataupun metal. Namun untuk AIO (all-in-one) PC, casing komputer ini didesain ringkas dan seringkali langsung menyatu dengan layar monitor.

Komputer (case) adalah bagian yang menjadi "rumah" bagi sebagian besar komponen komputer. Mayoritas desktop PC memiliki kotak casing baik dari plastik maupun logam. Namun untuk AIO (all-in-one) PC, casing komputer ini dirancang lengkap dan dapat dipasang langsung menyatu dengan layar monitor.

Selain komponen komputer komponen, casing juga melindungi komponen tersebut dari debu dsb.

Sumber Daya listrik

Bagian-bagian Komputer Beserta Fungsinya

Power Supply Unit (PSU) adalah komponen yang mengalirkan listrik AC menjadi DC bervoltase rendah, sehingga dapat dibuat sebagai sumber daya untuk semua komponen komputer.

Papan induk

Bagian-bagian Komputer Beserta Fungsinya

Motherboard adalah papan sirkuit terpadu yang menjadi komponen utama dari sebuah komputer. Fungsi motherboard adalah menghubungkan seluruh komponen utama komputer seperti CPU, RAM, CD / DVD-ROM, hardisk / SSD, serta berbagai port dan slot ekspansi.

Komponen-komponen komputer yang ditempatkan secara langsung di motherboard antara lain:

CPU / Prosesor

Bagian-bagian Komputer Beserta Fungsinya

CPU (Central Processing Unit) atau yang lebih dikenal sebagai prosesor merupakan otak dari komputer. CPU mendesak melakukan kalkulasi setiap perintah yang dijalankan di komputer. Untuk menghindari overheat (panas berlebih), CPU biasanya dilengkapi heatsink dan kipas. Beberapa CPU juga dilengkapi sistem pendingin air. Berbagai CPU modern saat ini telah dilengkapi dengan GPU (Graphic Processing Unit).

Chipset

Bagian-bagian Komputer Beserta Fungsinya

Fungsinya adalah sebagai penghubung komunikasi antara CPU dan komponen komputer lainnya, seperti RAM.

RAM / Memori

Bagian-bagian Komputer Beserta Fungsinya

RAM (Random Access Memory) tugasnya adalah menyimpan data dan berbagai proses agar dapat diakses dengan sangat cepat oleh CPU atau komponen komputer lainnya. Idealnya, semakin besar jumlah RAM maka komputer akan semakin cepat meningkatkan jumlah RAM untuk menyimpan data dan proses.

Kartu Video / GPU / VGA / Kartu grafis

Bagian-bagian Komputer Beserta Fungsinya

Kartu grafis atau yang biasa disebut VGA / GPU / Kartu Video adalah komponen komputer yang menggunakan proses grafis. Mayoritas pengguna komputer hanya dapat menggunakan GPU internal (terintegrasi), tetapi gamer, animator, desainer, dsb yang didedikasikan GPU yang memiliki kapasitas besar sehingga dapat memproses grafis yang besar.

ROM

Bagian-bagian Komputer Beserta Fungsinya

ROM (Read-Only Memory) adalah tempat penyimpanannya firmware BIOS dan penyimpanannya. Komputer modern terbaru saat ini sudah menggunakan UEFI dan tidak lagi menggunakan BIOS.

Baterai CMOS

Bagian-bagian Komputer Beserta Fungsinya

baterai CMOS ini menyediakan daya untuk chip BIOS, sehingga mengatur BIOS dan tanggal tidak dihapus dan tetap berjalan dengan komputer dimatikan.

Bis

Bagian-bagian Komputer Beserta Fungsinya

Ini adalah slot yang dapat menghubungkan berbagai kartu seperti Kartu Grafis eksternal dan Kartu Suara ke motherboard.

Kartu suara

Bagian-bagian Komputer Beserta Fungsinya

Hampir semua motherboard memiliki kartu suara internal untuk pemrosesan suara. Namun demikian, pengguna dapat menambahkan juga kartu suara eksternal jika memang diperlukan untuk keperluan tertentu.

Hardisk (HDD) / SSD

Bagian-bagian Komputer Beserta Fungsinya

Hardisk adalah media penyimpanan yang dapat digunakan untuk menyimpan data, termasuk juga sistem operasi. Selain HDD (Hard Disk Drive) konvensional, saat ini SSD (Solid State Drive) juga sudah populer digunakan, terutama di laptop, netbook, atau PC portabel lainnya. HDD Dibandingkan, SSD memiliki kecepatan baca / tulis yang jauh lebih tinggi dan lebih hemat penggunaan daya karena tidak menggunakan komponen mekanik di dalamnya.

Drive Optik (CD / DVD-ROM)

Bagian-bagian Komputer Beserta Fungsinya

Meskipun masih populer dan banyak digunakan, CD / DVD-ROM sudah semakin jarang ditemui di komputer modern, khususnya di PC portabel seperti laptop dan netbook. Drive Optik seperti CD / DVD-ROM berfungsi untuk membaca dan menulis data ke disk optik seperti CD dan DVD.

Papan ketik

Bagian-bagian Komputer Beserta Fungsinya

Keyboard berfungsi sebagai input untuk berkomunikasi dengan komputer. Praktisnya, kamu bisa mengenkripsi dan memasukkan berbagai perintah ke komputer melalui keyboard.


10. Jelaskan fungsi perangkat lunak basis data?


Basis data atau data base adalah kumpulan informasi, serta data yang disimpan didalam suatu server komputer secara sistematis, dan diperiksa dan dirubah melalui komputer.

Pembahasan

Fungsi perangkat lunak basis data:

Meningkatkan untuk memelihara data

Perangkat lunak berbasis data dirancang dengan prosedur standar, biasanya bisa juga untuk menambah, mengedit, menghapus data dan lainya.

Dapat menghindari duplikasi data

Sering terjadi suatu hack, maka dari dibuatlah perangkat lunak basis data. Untuk menjaga keamanan data dari duplikasi, serta dapat juga mengatur serta memantau suatu data.

Menjamin kualitas data dan informasi dapat diakses  

Agar jaga tetap aman adalah tugas perangkat lunak, namun juga memberi keleluasaan mengunjungi suaut data dengan aman dan terjaga.

Mendukung kinerja aplikasi yang memerlukan penyimpanan data

Karena sudah banyak sistem penyimpanan data yang membutuhkan software, karena manfaatnya serta kegunaanya dalam menyimpan, serta menjaga data.

Pengelompokan data

Mempermudah untuk mengumpulkan jenis-jenis data, agar mudah dicari secara sejenis atau pengelompokkan

1.Definisi Perangkat Lunak

Program komputer berikut pedoman penggunaannya yang siap digunakan oleh pemakai (software)

Perangkat lunak atau peranti lunak (bahasa Inggris: software) adalah istilah khusus untuk data yang diformat, dan disimpan secara digital, termasuk program komputer, dokumentasinya, dan berbagai informasi yang bisa dibaca, dan ditulis oleh komputer. Dengan kata lain, bagian sistem komputer yang tidak berwujud.

2.Perangkat lunak atau software adalah sekumpulan data elektronik yang tersimpan dan dikendalikan oleh perangkat komputer. Data elektronik tersebut meliputi instruksi atau program yang nantinya akan menjalankan perintah khusus. Perangkat lunak juga disebut sebagai bagian sistem dalam komputer yang tidak memiliki wujud fisik yang diinstal dalam sebuah komputer atau laptop agar bisa dioperasikan.

3.Memproses data, perintah, atau instruksi khusus agar pengguna dapat mengoperasikan komputernya sesuai dengan hasil informasi yang diinginkan.

Sarana interaksi yang menghubungkan pengguna dengan perangkat kerasnya.

Mengidentifikasi suatu program yang ada di sebuah komputer.

Menyediakan fungsi dasar dari sebuah perangkat keras agar dapat dioperasikan. Seperti ketersediaan sistem operasi pada komputer.

Penerjemah suatu perintah software lainnya ke dalam bahasa mesin, agar dapat dimengerti oleh komputer.

4.Perangkat Lunak Berbayar – Software yang mengharuskan penggunanya untuk membayar dengan harga tertentu untuk bisa menggunakan perangkat lunak tersebut. Meskipun sudah dibeli, namun lisensi untuk menyebarluaskan tidak diberikan kepada pengguna, karena tindakan tersebut termasuk ilegal. Contoh software berbayar adalah Adobe Photoshop, Microsoft Office, Microsoft Windows dan sebagainya.

Freeware – Perangkat lunak ini gratis dan dapat digunakan tanpa batas waktu tertentu. Biasanya pengembang software membuat perangkat lunak ini untuk komunitas tertentu. Hak cipta tetap dipertahankan sehingga siapa saja dapat melakukan update software terbaru. Contoh software freeware adalah Google Chrome, Mozilla Firefox, dan sebagainya.

Free Software – Pengguna harus membeli perangkat lunak ini terlebih dahulu, setelah itu pengguna bebas untuk melakukan penggandaan, modifikasi, hingga distribusi.

Shareware – Perangkat lunak yang dibagikan secara gratis untuk keperluan tertentu. Biasanya sebagai uji coba dengan fitur terbatas, dan penggunaan dengan waktu yang terbatas (biasanya 15 atau 30 hari). Shareware dibagikan secara gratis untuk memberikan pengguna kesempatan untuk mencoba menggunakan program sebelum membeli lisensi versi lengkap dari perangkat lunak tersebut.

Malware – Perangkat lunak ini dianggap sebagai perusak, sehingga bisa berbahaya jika disalahgunakan. Tujuan perangkat lunak ini untuk menyusup, bahkan merusak sistem jaringan komputer. Contoh malware adalah spyware (perangkat lunak pengintai), adware (perangkat lunak untuk iklan yang tidak jujur), virus komputer, dan software lainnya yang dibuat dengan tujuan merugikan.

Open Source Software – Perangkat lunak yang bersifat terbuka, sehingga kode sumbernya dapat dipelajari, dimodifikasi, ditingkatkan dan disebarluaskan. Perangkat ini biasanya diperoleh secara gratis dan digunakan oleh komunitas tertentu untuk dikembangkan dengan lisensi GPL (General Public License). Contoh perangkat lunak ini adalah Linux, yang fungsinya setara dengan Microsoft Windows.

Firmware – Perangkat lunak penyimpanan yang hanya dapat dibaca, atau Memory Read Only. Software ini bersifat paten sehingga tidak bisa dilakukan modifikasi atau pengembangan meskipun terdapat masalah dalam fungsinya. Biasanya firmware telah menyatu dengan perangkat keras, sehingga dianggap bukan perangkat lunak seutuhnya.
