17. apa fungsi palet animation di potoshop


untuk memberikan warna kalo gak salah1. New
Layer : Digunakan untuk membuat layer baru

Layer From Background : Digunakan untuk membuat layer baru yang dialkukan dengan cara mengcopy paste objek yang ada pada layer background

Group : Digunakan untuk membuat group baru / layer yang berisi layer lain tapi masih kosong belum ada layernya

Group from layers : Digunakan untuk membuat group baru yang disi oleh layer-layer terpilih

Layer via copy : Digunakan untuk membuat layer baru dengan cara mengcopy paste objek yang ada pada layer aktif / terpilih

Layer via cut : Digunakan untuk membuat layer baru yang berisi objek dari layer yang terpilih sebelumnya. Layer terpilih akan otomatis kosong tidak ada objeknya.

2. Duplicate
Digunakan untuk mengakses tab duplicate, layer terpilih akan dicopy yang kemudian bisa didistribusikan ke dokumen baru, dokumen aktif, atau dokumen itu sendiri. Pada opsi ini bisa juga langsung direname atau dihanti nama leyer yang akan terpaste.

3. Delete
Layer : Digunakan untuk menghapus layer terpilih

Hidden layer : Digunakan untuk menghapus layer yang tidak terlihat(ditandai dengan ikon mata yang hilang pada pallete layer).


4. Layer Properties
Digunakan untuk mengakses properti yang ada pada sebuah layer tertentu / terpilih

5. Layer Style
Digunakan untuk mengubah style via blending options.

6. Smart Filter
Opsi filter yang digunakan pada smart objek (objek vector atau kumpulan objekphotoshop yangiubah menjadi satu bagian utuh)

7. New Fill Layer
Solid : Digunakan untuk membuat layer baru tergabung dalam satu masking (penutup / topeng) yang diisi oleh warna solid

Gradient : Digunakan untuk membuat layer baru tergabung dalam satu masking (penutup / topeng) yang diisi oleh warna gradasi

Pattern : Digunakan untuk membuat layer baru tergabung dalam satu masking (penutup / topeng) yang diisi oleh pattern atau background.

22. Apa fungsi costum animation??


Custon animasi memiliki fungsi yaitu menyediakan fasilitas untuk memilih dan mengatur efek animasi sesuai dengan kebutuhan. Custom animation digunakan untuk memberi efek animasi pada slide.

Pembahasan  

Animasi ialah kumpulan fungsi atau perintah yang dapat dipakai untuk mempercantik presentasi menggunakan powerpoint. Fungsi spesifik yang dimiliki ialah untuk mengendalikan animasi objek pada slide, misalnya mengendalikan teksbook, gambar, ataupun objek lainnya. Beberapa tombol maupun dialog yang ada pada panel tersebut akan aktif jika kita mengaktifkan objek. Untuk memahami beberapa perintah yang ada dalam custom animation disini kita mencoba membuat objek anak panah menggunakan Autoshapes.  

Animasi dibuat berdasarkan manfaatnya sebagai media yang digunakan untuk berbagai keperluan, diantaranya : media hiburan, Media presentasi, Media iklan, Media ilmu pengetahuan, Media bantu, Media pelengkap

Dalam media presentasi, animasi dipakai untuk membuat menarik perhatian para audien atau peserta presentasi terhadap materi yang disampaikan oleh presenter. Dengan penambahan animasi pada media presentasi, akan membuat suasana presentasi menjadi tidak kaku.

Fungsi animasi dalam presentasi antara lain :

Menarik perhatian dengan adanya pergerakan dan suara yang selaras Memperindah tampilan presentasi Memudahkan susunan presentasi Mempermudah penggambaran dari suatu materiPelajari lebih lanjut

37. 1.jelaskan fungsi duration pada animation Pane2.jelaskan mengenai start on clik pada animation Pane​


a.Group Preview

Pada toolgroup ini hanya ada satu tombol yaitu Preview  Animations, digunakan untuk untuk melihat hasil dari animasi yang digunakan pada teks atau gambar.

b.Group Animation

Toolgroup Animation berisikan 2 Icon, yaitu :

Ø  Animation, memiliki fungsi untuk menyediakan pilihan-pilihan efek animasi.

Ø  Effect option, memiliki fungsi untuk menambahkan variasi modifikasi pada efek animasi yang sudah dipilih di slide.

c.Group Advanced Animation

Toolgroup Advanced Animation berisikan 4 Icon, yaitu :

Ø  Add animation, berfungsi untuk menambahkan lagi efek animasi ke objek yang telah diberi efek animasi sebelumnya.

Ø  Animation Pane, memiliki fungsi untuk menampilkan panel (Jendela) penerapan efek animasi.

Ø  Trigger, berfungsi untuk menentukan objek yang telah diberi animasi yang mana yang akan tampil awal daripada objek-objek lain nya pada suatu slide.

Ø  Animation Painter, memiliki fungsi untuk menyalin efek animasi yang telah di terapkan pada objek yang di pilih kepada objek lain dalam presentasi.

d.Group Timing

Toolgroup Timing memiliki 5 icon, yaitu :

Ø  Animation Timing, berfungsi untuk menentukan kapan mulai aktif atau jalan animasi yang telah di terapkan pada objek.

Ø  Animation Duration, berfungsi untuk menentukan lamanya proses aktif efek animasi yang diterapkan pada objek yang dipilih.

Ø  Animation Delay, berfungsi untuk menentukan jeda waktu (Waktu Tunda) sebelum suatu animasi diaktifkan.

Ø  Move Earlier, memiliki fungsi untuk menyusun urutan animasi objek yang urutan akhir menjadi urutan yang awal.

Ø  Maove Later, memiliki fungsi untuk menyusun urutan tampil suatu animasi objek yang urutan awal menjadi urutan akhir.

Implementasi Toolsgroup pada Tab Animation

Pada Tab Animation kalian bisa menggunakan Fungsi dari toolgroup yang sudah kami jelaskan diatas,  Tab Aninmation memiliki fungsi yang kurang lebih sama dengan Tab Transitions, tetapi untuk Tab Animation ini lebih difungsikan untuk membuat animasi untuk teks atau gambar dikarenakan dalam toolgroup Animation terdapat 3 animasi yang berbeda yaitu, Enterance (Animasi Masuk), Emphasis (Animasi Tetap) dan Exit (Animasi Keluar).

             pada Tab ini tentunya akan membuat presentasi kamu semakin menarik untuk diamati oleh audience.
