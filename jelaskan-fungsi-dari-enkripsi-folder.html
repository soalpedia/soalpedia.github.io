16. tulislah fungsi enkripsi untuk tujuan keamanan ​


Jawaban:

Penjelasan:

Di bidang kriptografi, enkripsi adalah proses mengamankan suatu informasi dengan membuat informasi tersebut tidak dapat dibaca tanpa bantuan pengetahuan khusus. Dikarenakan enkripsi telah digunakan untuk mengamankan komunikasi di berbagai negara, hanya organisasi-organisasi tertentu dan individu yang memiliki kepentingan yang sangat mendesak akan kerahasiaan yang menggunakan enkripsi. Di pertengahan tahun 1970-an, enkripsi kuat dimanfaatkan untuk pengamanan oleh sekretariat agen pemerintah Amerika Serikat pada domain publik, dan saat ini enkripsi telah digunakan pada sistem secara luas, seperti Internet e-commerce, jaringan Telepon bergerak dan ATM pada bank.

Enkripsi dapat digunakan untuk tujuan keamanan, tetapi teknik lain masih diperlukan untuk membuat komunikasi yang aman, terutama untuk memastikan integritas dan autentikasi dari sebuah pesan. Contohnya, Message Authentication Code (MAC) atau digital signature. Penggunaan yang lain yaitu untuk melindungi dari analisis jaringan komputer.

33. Bagaimanakah cara mengelola file dan folder dengan menggunakan windows explorer jelaskan beberapa fungsi yang di gunakan dalam pengelolaan file dan folder


Berikut ini adalah beberapa cara mengelola file dan folder menggunakan windows explorer:

New folder, berfungsi untuk membuat folder baru.Rename, berfungsi untuk mengganti nama sebuah file atau folder tertentu.Cut, berfungsi untuk memindahkan file maupun folder.Copy, berfungsi untuk menyalin file maupun folder.Paste, berfungsi untuk meletakkan hasil pemindahan atau penyalinan file maupun folder.Delete, berfungsi untuk menghapus file maupun folder.Library, berfungsi untuk menampilkan seluruh folder dan file yang berada dalam drive tertentu.

Pembahasan

Windows explorer merupakan software dari windows yang digunakan sebagai sarana penjelajah file. Software ini berfungsi untuk menyajikan seluruh file maupun folder yang tersimpan dalam sebuah komputer.

Melalui windows explorer, user atau pengguna komputer dapat mengolah, mengelompokkan, hingga menyusun setiap file yang terdapat dalam komputernya. Windows explorer dapat digunakan untuk menggandakan, mengganti nama, mengelompokkan, dan menghapus file.

Folder merupakan suatu wadah yang menjadi tempat beberapa file. Biasanya, folder digunakan untuk mengelompokkan berbagai jenis file berdasarkan kategorisasi tertentu, seperti tanggal, isi, dan sebagainya.
