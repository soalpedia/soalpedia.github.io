9. diketahui fungsi c=100+0,8y. tentukan jumlah pendapatan saat break event point


Break Event Point adalah saat pendapatan (Y) di gunakan seluruhnya untuk consump (C), maka berlaku:
Y = C
Y = 100 + 0,8Y
Y - 0,8Y = 100
0,2Y = 100
Y = 100 : 0,2
Y = 500

jumlah pendapatan saat BEP adalah 500

24. Tuliskan 5 fungsi yang ada dicoding scratch selain event!​


Jawaban:

Berikut ini adalah 5 fungsi umum yang ada di Scratch selain event:

1. Motion: Fungsi Motion digunakan untuk mengatur pergerakan objek pada layar. Fungsi ini memungkinkan pengguna untuk menggerakkan objek ke kiri, ke kanan, ke atas, dan ke bawah, serta memutar objek atau mengubah ukurannya.

2. Looks: Fungsi Looks digunakan untuk mengatur tampilan objek pada layar. Fungsi ini memungkinkan pengguna untuk mengubah warna, ukuran, dan transparansi objek, serta memasukkan gambar atau teks pada objek.

4. Control: Fungsi Control digunakan untuk mengatur alur program dan mengontrol tindakan pengguna. Fungsi ini memungkinkan pengguna untuk mengatur perulangan, pengaturan kondisi (if-else), serta menjalankan program pada saat yang ditentukan.

5. Pen: Fungsi Pen digunakan untuk menggambar garis atau bentuk pada layar. Fungsi ini memungkinkan pengguna untuk mengatur ukuran, warna, dan gaya garis yang digunakan untuk menggambar. Fungsi Pen dapat digunakan untuk membuat animasi yang menarik atau untuk menunjukkan proses penghitungan pada program

25. Analyze the structure of text! Orientation : Event = Event 1 : Event 2 : Event 3 : Reorientation :


Recount text is a text which presents the writer's experience.

Purpose of this text is to retell the writer's experience.

Generic structure of this text :

1. Orientation

At the beginning of paragraph.

It tells time, place and with whom.

2. Event

It tells the sequence of activities.

Sequence adverbs ( first, second,

next , etc )

3. Reorientation :

At the last paragraph

The conclusion of the writer .



Kesimpulan dari teks recount

Recount text adalah teks yang menceritakan pengalaman si penulis.

Tujuannya adalah untuk menceritakan ulang pengalaman si penulis.

Generic structure :

1. Orientasi > ada di awal paragraf

Menjelaskan waktu , tempat kejadian


2. Event > kejadian berurutan . Bisa diidentifikasi keterangan nya ( kata first, second, next , dll )

3. Reorientasi > kesimpulan dari penulis tentang pengalaman nya. Biasanya ada di bagian penutup paragraf.
