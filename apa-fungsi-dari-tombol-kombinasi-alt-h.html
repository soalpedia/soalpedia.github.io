11. Tombol kombinasi Alt + F4 adalah perintah untuk


Tombol kombinasi Alt + F4 adalah perintah untuk menutup sebuah window (jendela) yang sedang aktif digunakan dan langsung menutupnya tanpa terkecuali.

1. shutdown = mematikan komputer
2. restart = memulai ulang komputer
3. sleep = menidurkan komputer
4. hibernate = tidur panjang
6. switch user = pindah user
7. log off = akun yang digunakan keluar dari sistem Windows

Jawaban:

A. Alt (tombol fungsi); F4 (tombol ketik)

Jika Alt menjadi tombol fungsi dan F4 menjadi tombol ketik akan susah mengoperasikan atau melakukannya dan tidak diperbolehkan atau diperkenankan.

B. Alt dan F4 sama-sama di kelompok tombol fungsi Opsi

Jika bersama-sama menjadi tombol opsi yang ada akan susah menggunakan ke2nya karena Alt di tombol opsi, sedangkan F4 tombol fungsi dan diperbolehkan atau diperkenankan juga memakai atau menggunakan seperti ini.

C. Alt (tombol ketik); F4 (tombol fungsi)

Ini baru diperbolehkan atau diperkenankan karena ini yang umu dipakai orang untuk memudahkan menutup item, aplikasi, dan program yang sedang aktif.

D. Alt dan F4 sama-sama di kelompok tombol ketik

Jika bersama-sama menjadi tombol ketik yang ada akan susah menggunakan ke2nya karena Alt dan F4 tidak ada di tombol ketik. Alt di tombol opsi, sedangkan F4 di tombol fungsi, jika ke2 itu pindah bersama-sama akan susah mengoperasikan atau melakukannya dan tidak diperbolehkan atau diperkenankan.
