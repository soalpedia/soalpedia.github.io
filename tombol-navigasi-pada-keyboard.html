Menu global

Saat Anda memindai layar, pilih tombol Menu untuk membuka menu global dengan opsi navigasi:

Kembali: Kembali ke halaman sebelumnya atau menyembunyikan keyboard jika keyboard terlihat.

Batal: Menutup menu global tanpa melakukan tindakan apa pun.

Opsi pada menu global berikut berfungsi sama seperti gestur lain di layar perangkat:

Kembali, Beranda, dan Ringkasan: Tombol pada menu navigasi perangkat

Notifikasi: Geser ke bawah dari bagian atas layar

Pilih otomatis  

Menu global menyertakan opsi untuk mengklik Aktifkan pilih otomatis atau Nonaktifkan pilih otomatis.

"Pilih otomatis" diaktifkan: Saat Anda mengklik item di layar, item tersebut akan otomatis dipilih.

"Pilih otomatis" dinonaktifkan: Jika tersedia lebih dari satu tindakan, Anda dapat mengklik item untuk membuka menu "Tindakan". Jika hanya tersedia satu tindakan, Anda dapat mengklik item untuk melakukan tindakan tersebut.

Menu Tindakan

Menu Tindakan terbuka jika Anda mengklik item yang mendukung beberapa tindakan dan Pilih otomatis tidak aktif. Menu ini mencantumkan opsi seperti Luaskan, Ciutkan, Pilih, Sentuh lama, Scroll maju, Scroll mundur, Potong, Salin, dan Tempel.  

Tips untuk menu Tindakan:

Jika item dapat di-scroll ke berbagai arah, menu Tindakan akan menyertakan opsi Scroll untuk setiap arah, seperti maju dan mundur.

Bila tindakan yang sama dapat diterapkan ke beberapa item yang berlokasi di tempat yang sama, menu Tindakan akan menyertakan tindakan ini beberapa kali. Jadi, satu kali untuk setiap item yang mendukung tindakan tersebut. Tindakan ini diberi nomor, seperti Scroll (1) dan Scroll (2). Jika Anda tidak yakin dengan tindakan yang akan dipilih, coba tindakan pertama. Jika perlu, Anda dapat membuka kembali menu dan mencoba tindakan kedua.

Dengan bidang teks, menu Tindakan menyertakan opsi seperti Potong, Salin, Tempel, Tandai, Sebelumnya, Berikutnya, Hapus, dan Urungkan.  

Pelajari lebih lanjut cara mengedit teks dengan Tombol Akses.

Untuk menutup menu tanpa melakukan tindakan apa pun, pilih Batal.

Pemindaian titik

Pemindaian titik memungkinkan Anda memilih tempat di layar untuk di-tap, digeser, atau di-zoom. Garis akan bergerak di layar untuk memungkinkan Anda memilih lokasi horizontal dan vertikal tertentu.

Mulai pemindaian layar dengan metode yang biasa Anda gunakan.

Pilih Menu lalu Pemindaian titik.

Untuk memulai pemindaian titik, tekan Pilih.

Jika Anda menggunakan pemindaian otomatis, gunakan tombol pindai otomatis untuk langkah ini dan 2 langkah berikutnya.

Saat garis pemindaian titik mencapai lokasi vertikal yang benar, tekan Pilih.

Saat garis pemindaian titik mencapai lokasi horizontal yang benar, tekan Pilih.

Untuk memilih tindakan, buka menu "Tindakan".

Untuk menonaktifkan pemindaian titik, pilih Menu lalu Keluar dari pemindaian titik.

20. Tombol apa saja yang merupakan tombol tombol khusus pada keyboard...


Penjelasan:

Secara umum, struktur tombol pada keyboard

terbagi atas 4, yaitu:

1. Tombol Ketik (typing keys)

Tombol ketik adalah salah satu bagian dari

keyboard yang berisi huruf dan angka serta

tanda baca. Secara umum, ada 2 jenis susunan

huruf pada keyboard, yaitu tipe QWERTY dan

DVORAK. Namun, yang terbanyak digunakan

sampai saat ini adalah susunan QWERTY.

2. Numeric Keypad

Numeric keypad merupakan bagian khusus dari

keyboard yang berisi angka dan sangat

berfungsi untuk memasukkan data berupa

angka dan operasi perhitungan. Struktur

angkanya disusun menyerupai kalkulator dan

alat hitung lainnya.

3. Tombol Fungsi (Function Keys)

Tahun 1986, IBM menambahkan beberapa

tombol fungsi pada keyboard standard. Tombol

ini dapat dipergunakan sebagai perintah khusus

yang disertakan pada sistem operasi maupun

aplikasi.

4. Tombol kontrol (Control keys)

Tombol ini menyediakan kontrol terhadap

kursor dan layar. Tombol yang termasuk dalam

kategori ini adalah 4 tombol bersimbol panah di

antara tombol ketik dan numeric keypad, home,

end, insert, delete, page up, page down, control

(ctrl), alternate (alt) dan escape (esc).

Keyboard merupakan salah satu perangkat keras dalam instalasi komputer. Keyboard berisikan huruf, angka, dan tombol-tombol lain yang berfungsi untuk menampilkan tulisan pada layar komuper. Print screen digunakan untuk mengcapture tampilan layar Pause break. Pause break biasanya digunakan pada sistem operasi DOS yang berungsi untuk menghentikan tampilan program berbasis text. End. End digunakan untuk memindahkan kursor pada baris paling belakang.

40. yang tidak termasuk struktur tombol tombol pada keyboard


Hai.

Pada keyboard, selain typing keys, numeric keypad, function key, dan control key bukanlah struktur keyboard.

Typing keys meliputi huruf abjad yang ada pada keyboard hingga tombol angka di atasnya.

Numeric keypad adalah bagian paling kanan pada keyboard, meliputi nomor dan sekitarnya.

Function keys meliputi tombol Fungsi1 (F1) dan seterusnya.

Control key meliputi tombol atas, bawah, kiri, kanan, dan sekitar, juga tombol Control (Ctrl), Alternative (Alt), dan sebagainya.

Intinya, selain tombol yang disebutkan di atas adalah bukan struktur keyboard.
