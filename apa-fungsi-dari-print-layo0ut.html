24. apa fungsi dari print preview


           Apa fungsi dari print preview

Print preview adalah menu yang berguna untuk melihat terlebih dahulu hasil dari lembar kerja dokumen yang telah dibuat sebelum di Print.

PembahasanPerbedaan Print dan Print Preview :

Print : adalah menu yang digunakan untuk mencetak file atau dokumen melalui alat Printer.

Print Preview : adalah menu yang digunakan untuk melihat tampilan file atau dokumen yang akan di print. Membantu untuk melihat bagaimana hasil yang akan dicetak sebelum proses pencetakan dilakukan.

LANGKAH LANGKAH MENCETAK DOKUMEN PENGOLAH ANGKA

A. Fasilitas Print Preview

Sebelum mencetak lembar kerja ke mesin printer, kita dapat melihat terlebih dahulu hasil percetakan di layar monitor. Tampilan layar di monitor akan sama dengan hasil percetakan mesin printer di kertas. Prosedur yang akan dilakukan adalah sebagai berikut.

Pilih dan klik tombol Office Button, pilih tobol Print, sehingga muncul menu print atau kotak pilihan print.
Pilih dan klik tombol Print Preview. Jendela print preview akan tampil lengkap bersama hasil percetakannya.
Hasil percetakan di Print Preview tersebut dapat langsung kita cetak di mesin printer dengan mengklik tombol perintah Print.
Untuk mengakhiri Print Preview dapat kita lakukan dengan mengklik tombol perintah Close.

Print Preview berfungsi untuk melihat dokumen yang sebelum diprint, ini bertujuan agar dokumen yang diprint sesuai dengan yang diinginkan. Selain itu print preview juga bisa digunakan untuk mengubah margin, ukuran kertas, Scalling, orientasi kerta (Portrait atau Landscape), menentuk halaman yang akan diprint, dan jumlah dokumen yang akan dprint.

31. apa fungsi dari print layout?


Kelas: X

Mata pelajaran: Teknologi Informasi

Materi: Microsoft Word

 

Saya akan mencoba menjawab dengan dua jawaban:

 

Jawaban pendek:


Print layout berguna menampilkan dokumen pada Microsoft WOrd sebagaimana akan tampak dokumen tersebut ketika dicetak.  

Jawaban panjang:  

Dalam Microsoft Word ada lima mode untuk melihat dokumen yaitu:  

Read Mode  

Dokumen mengisi jendela Microsoft Word dan diformat dalam bentuk seperti tampilan di tablet atau majalah. Tampilan ini sangat berguna untuk memeriksa dokumen, serta membaca yang telah Anda download atau terima di email.  

Print Layout  

Tampilan umum Microsoft Word. Dokumen umum, seperti memo, surat, dan laporan, sering ditulis dan diedit dalam tampilan Print Layout. Salah satu keuntungan bekerja dalam pandangan ini adalah kepatuhannya terhadap WYSIWYG (apa yang Anda lihat adalah apa yang Anda dapatkan). Margin, header / footer, dan format sesuai dengan hasil cetak. Halaman ditampilkan sebagai potongan kertas setara dengan jeda fisik antar halaman.  

Web Layout  

Tampilan Web Layout digunakan untuk membuat, melihat, dan mengedit halaman karena akan muncul secara online saat dibuka di browser. Dengan memilih Save As di mode ini, kitadapat menyimpan halaman dalam beberapa format yang kompatibel dengan web.  

Outline  

Outline akan menampilkan susunan dokumen seperti susunan bab, sub bab dan point penting dokumen.  

 Draft

 

Draft digunakan saat kecepatan sangat penting, misalnya saat menggunakan komputer yang lambat. Tampilan Draft menampilkan dokumen sebagai teks kontinu. Jeda halaman dilambangkan dengan garis putus-putus.
