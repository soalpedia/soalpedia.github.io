Tombol Tab

Fungsi tombol tab adalah untuk membuat kursor bergeser dengan jarak 1 tabulasi ke arah kanan. Fungsi tombol ini mirip dengan tombol space yang digunakan untuk memberi jarak antara kata 1 dan lainnya.  Letak tombol tab tepat diatas tombol caps lock.

Tombol Caps Lock

Fungsi tombol caps lock adalah untuk membuat huruf atau tulisan menjadi BESAR (UPPERCASE). Biasanya ada indikator atau lampu yang menyala begitu tombol ini dalam keadaan on.

Tombol Windows (Home)

Fungsi tombol home adalah untuk mengarahkan tampilan pada halaman awal atau halaman start secara cepat. Untuk mengembalikan pada halaman yang terakhir dibuka, maka cukup klik lagi tombol ini. Tombol home biasanya berupa logo windows atau jendela dan terletak antara CTRL dan ALT.

Tombol Space (Spasi)

Tombol space berbentuk panjang dan terletak dibagian paling bawah. Fungsinya adalah untuk menambahkan spasi atau jarak antara 1 kata dengan kata lainnya.

Tombol Delete

Tombol delete berfungsi untuk menghapus 1 karakter di kanan/depan kursor. Tombol ini akan menghapus kata yang ditulis setelah tanda kursos atau kata yang sebelumnya telah ditandai.

Tombol Enter

Fungsi tombol enter adalah untuk berpindah baris. Ketika mengetikan tombol enter maka tulisan di baris kursor akan berpindah ke baris baru. Tombol enter ada disebelah kanan keyboard dengan bentuk paling besar diantara tombol lainnya.

Tombol Backspace

Tombol Backspace ditandai dengan simbol <—- di keyboard. Fungsi tombol backspace adalah untuk menghapus satu karakter yang ditulis. Jika kamu ingin menghapus lebih cepat gunakan perpaduan tombol backspace dengan CTRL untuk menghapus sebuah kata.

Tombol PrtScr

Tombol print screen atau pada keyboard disingkat PrtScr/PrtSc adalah tombol yang digunakan untuk menangkap atau meng-capture layar. Tekan tombol ini 1 kali kemudian buka ms word ataupun paint. Ketik CTRL + P, maka gambar hasil screeshoot pun akan muncul.

Cara lain adalah dengan memadukan tombol windows/home dengan printscreen. Ketika di klik berbarengan maka akan muncul kedipan di layar yang menandakan layar sudah ter-screenshoot. Kamu dapat melihat hasilnya di folder PICTURES di my computer anda.

Tombol End

Fungsi tombol end adalah untuk memindahkan kursor ke baris akhir tulisan. Ini biasanya berjalan ketika anda sedang bekerja menggunakan ms word.

Tombol Page Up & Page Down

Tombol page up digunakan untuk menarik halaman ke atas (scrolling ke atas), dan page down untuk menarik halaman ke bawah (scrolling ke bawah). Dengan menggunakan tombol ini anda tak perlu repot ketika akan menurunkan atau menaikan posisi file di layar komputer.

Tombol Navigasi (Panah Kanan, kiri, atas, bawah)

Tombol navigasi berfungsi untuk menggerakan kursor ke atas, bawah, kanan ataupun kiri. Kursor sendiri adalah bagian di layar komputer yang berfungsi sebagai indikator posisi. Bentuknya lurus kecil dan berkedip-kedip.

Tombol Lain

Tombol NumLock -> tombol numlock berfungsi untuk mengubah bagian keyboard utama bertindak sebagai keypad numerik.

Tombol Scrooll Lock -> Pada microsoft excel tombol ini berfungsi untuk mengunci kolom agar tidak menggulung saat  tombol atas kanan kiri bawah ditekan.

Tombol Fn -> Tombol fn biasanya ada pada keyboard laptop ia hanya berfungsi ketika dikombinasikan dengan tombol lain. Misalnya untuk mencerahkan meredupkan layar, menaikan atau mengecilkan volume suara dll.

6. Tombol kombinasi pada keyboard komputer yang digunakan untuk proses menghidupkan komputer adalah...


Secara teknis menghidupkan komputer dekstop menggunakan keyboard bisa dilakukan melalui beberapa cara yang biasanya terdiri dari kombinasi tombol keyboard (contohnya dengan tombol CTRL + ESC) tapi dengan 2 catatan, yaitu motherboard dari komputer harus mendukung aktivitas tersebut dan yang kedua kamu harus melakukan konfigurasi atau seting pada BIOS terlebih dahulu.  

Pembahasan:

Untuk mempraktekan cara menghidupkan komputer dengan keyboard silahkan ikuti panduan/langkah-langkah dibawah ini:

BIOS

Langkah pertama yang harus dilakukan adalah memastikan motherboard dari komputer yang akan digunakan memang mendukung, karena jika tidak maka aktivitas ini akan mustahil dilakukan melalui keyboard.

Nah, satu-satunya cara untuk mengetahui adanya dukungan motherboard adalah dengan masuk ke sistem BIOS dengan menekan salah satu diantara tombol F1, F2, F10, F12 dan ESC (coba/pilih salah satu saja).

Jika sudah masuk pada sistem BIOS silahkan masuk ke bagian POWER lalu cari APM atau Advanced power management

Jika kamu menemukan "Power on by Keyboard" pada langkah kedua di atas maka bisa dipastikan komputer yang digunakan tersebut memang mendukung, dan sebaliknya jika tidak ada kemungkinan besar aktivitas menghidupkan komputer melalui keyboard tidak bisa kamu lakukan.

Konfigurasi  

Jika pada langkah sebelumnya memang mendukung maka selanjutnya kita akan melakukan sedikit konfigurasi dimana motherboard akan memberikan otoritas pada keyboard untuk mengirimkan sinyal agar komputer hidup, caranya:

Pada bagian APM atau Advanced power management silahkan seleksi "Power on by Keyboard" lalu untuk setingannya bisa kamu pilih dari yang sudah disediakan. Untuk pilihanya sendiri setiap komputer bisa berbeda, namun biasanya pilihannya berkisar pada

Disabled yaitu di non-aktifkanTombol powerCTRL + ESCTombol spasi (spacebar)

Karena kita akan mengaktifkan maka pilihannya adalah tombol power, CTRL + ESC dan spasi. Silahkan pilih yang kira-kira sesuai dengan yang kamu inginkan. Jika sudah Simpan konfigurasi tersebut dengan menekan tombol F10 lalu restart dan matikan komputernya.

 

Nah, sekarang kamu bisa menghidupkan komputer melalui tombol keyboard yang dipilih tadi.

Jawaban:

Untuk mematikan komputer melalui keyboard, tombol yang digunakan tergantung pada sistem operasi yang digunakan. Berikut adalah tombol yang dapat digunakan untuk mematikan komputer pada beberapa sistem operasi:

Windows: Tekan tombol Windows + X, kemudian pilih "Shutdown" atau "Shut down or sign out" dan pilih "Shut down".

Mac OS X: Tekan tombol Control + Option + Command + Power button (atau Control + Option + Command + Eject).

Linux: Tekan tombol Control + Alt + Delete atau Control + Alt + Backspace, kemudian pilih "Shutdown" atau "Power off".

Perlu diingat bahwa sebaiknya melakukan shutdown melalui sistem operasi dan tidak langsung mematikan komputer dengan tombol power karena hal ini dapat menyebabkan kerusakan pada sistem dan file yang sedang diproses.

40. apa saja nama tombol yang ada di keyboard komputer?​


Jawaban:

1. Tombol Esc

Fungsi tombol keyboard yang letaknya di ujung kiri atas ini dapat digunakan untuk membatalkan sebuah perintah yang sudah dilakukan. Semua perintah yang terlanjur diberikan akan batal jika kamu menekan tombol Esc ini.

2. Tombol Caps Lock

Tombol Capslock yang berada disisi kiri ini memilik fungsi untuk membuat tulisan menjadi huruf besar atau huruf kapital. Saat kamu menekan tombol Caps Lock ini maka ada lampu yang menyala pada keyboard sebagai penanda bahwa Caps Lock sedang aktif.

3. Tombol Tab

Tombol yang berada sedikit dibawah tombol capslock ini memiliki fungsi untuk memindahkan kursor dalam satu jeda atau tabulasi ke arah kanan. Saat kamu mengetik dan menekan tombol Tab, maka kursor akan bergeser ke kanan dengan jarak satu tabulasi.

4. Tombol Backspace

Tombol Backspace yang berada di sisi kanan huruf alphabet ini berfungsi untuk menghapus satu karakter yang berada di depan kursor. Jika kamu menekan tombol Backspace bersamaan dengan tombol Ctrl maka akan menghapus satu kata yang berada di depan kursor.

5. Tombol Delete

Fungsi tombol keyboard yaitu tombol Delete berfungsi untuk menghapus satu karakter yang berada di belakang kursor. Jika kamu menekan tombol Delete dan tombol Ctrl secara bersamaan, maka satu kata akan terhapus.

6. Tombol Home

Tombol Home pada keyboard memiliki fungsi untuk memindahkan kursor ke ujung sisi kiri pada satu baris saat kamu mengetik. Jika kamu menekan secara bersamaan antara tombol Home dan tombol Ctrl, maka kursor akan berpindah ke sisi kiri pada baris pertama paling atas pada lembar kerjamu.

7. Tombol End

Fungsi tombol keyboard, tombol End, adalah kebalikan dari tombol Home. Tombol End akan memindahkan kursormu ke sisi kanan paling akhir dalam satu baris dari tulisanmu. Jika kamu menekan tombol End berbarengan dengan tombol Ctrl, maka kursormu akan berpindah ke sisi kanan paling akhir dari lembar kerjamu.

8. Tombol Page Up dan Page Down

Fungsi tombol Page Up pada keyboard sebagai penggerak baris satu layar ke arah atas. Page Down merupakan kebalikan dari Page Up, yaitu menggeserkan satu baris ke bawah. Mudahnya, Page Up dan Page Down akan mengubah kursormu ke halaman atas atau ke halaman bawah.

9. Tombol Enter

Tombol Enter memiliki fungsi yang cukup beragam. Tombol Enter adalah tombol yang penting pada keyboard. Fungsi tombol keyboard ini untuk memindahkan kursor ke baris selanjutnya dan juga berfungsi sebagai suatu perintah setuju kepada suatu program yang ada pada komputer atau laptop.
