2. apa rumus rumus medan magnet pada kawat lurus dan kawat melingkar


Rumus Medan Magnet pada Kawat Lurus

B = (μ₀IN) / ([tex]2\pi r[/tex])

dimana

B = Medan Magnet (T)

dimana

I = Kuat Arus Listrik (A)

12. Jika kawat lurus dan kawat melingkar dialiri arus yang sama, maka kawat manakah yang menghasilkan medan magnet yang lebih kuat?jelaskan (kalau bisa pakai gambar)


Rumus untuk kawat lurus
B = (μ₀ * I) / (2πa)

Rumus untuk kawat melingkar
B = (μ₀ * I) / (2a)

Dengan menganggap I kawat lurus = I kawat melingkar
dan a kawat lurus = a kawat melingkar, maka perbandingan dari
B kawat lurus / B kawat melingkar = 1 / π
ibaratnya B kawat lurus = 1, B kawat melingkar = π = 3,14
ini berarti B kawat melingkar > B kawat lurus

22. Tolong berikan contoh soal dan cara penyelesaian di bawah inia. Medan magnet (induksi magnetik) di b = 0 dari kawat sejajar b. Medan magnet di titik p dari kawat melengkung dan kawat lurus ​


Jawaban:

a. Medan magnet (induksi magnetik) di titik p pada jarak r dari kawat sejajar dapat dihitung dengan menggunakan Hukum Ampere:

Rumusnya adalah: B = (μ₀ * I) / (2π * r)

Keterangan:

B = Induksi magnetik (tesla)

μ₀ = Permeabilitas vakum = 4π x 10^(-7) T m/A

I = Kuat arus dalam kawat (ampere)

r = Jarak dari titik p ke kawat (meter)

Contoh Soal:

Sebuah kawat sejajar dengan kuat arus 5 ampere terletak sejajar dengan sumbu x pada jarak 0,1 meter dari sumbu x. Tentukan medan magnet di titik p yang terletak pada jarak 0,5 meter dari kawat.

Cara Penyelesaian:

Dalam hal ini, I = 5 A dan r = 0,5 m.

Menggunakan rumus B = (μ₀ * I) / (2π * r), kita dapat menggantikan nilainya:

B = (4π x 10^(-7) T m/A * 5 A) / (2π * 0,5 m)

B = (5 x 10^(-7) T m) / (π * 0,5 m)

B = 10^(-6) T / (π * 0,5 m)

B = (2 x 10^(-6) T) / (π m)

Jadi, medan magnet di titik p yang terletak pada jarak 0,5 meter dari kawat adalah sebesar (2 x 10^(-6) T) / (π m).
