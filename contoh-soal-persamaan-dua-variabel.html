7. contoh soal sistem persamaan kuadrat dua variabel


Contoh soal sistem persamaan kuadrat dua variabel

Pendahuluan

Sistem persamaan kuadrat dua variabel (SPKDV) adalah sistem persamaan yang terdiri atas dua atau lebih persamaan kuadrat dua variabel yang mana variabel-variabel dalam persamaan-persamaan tersebut saling terkait.

Pembahasan

Contoh soal SPKDV)

Diketahui sistem persamaan

y = 2x² + x + 4

y = x² - 4x - 2

Penyelesaian sistem persamaan adalah ...

Penyelesaian :

Substitusikan persamaan y = 2x²  + x + 4 ke  dalam persamaan y = x ² – 4x – 2

2x²  + x + 4 = x ² – 4x – 2

2x²  – x ² + x  + 4x + 4 + 2 = 0

x ² + 5x + 6 = 0

(x + 2)(x + 3) = 0

(x + 2) = 0 atau (x + 3) = 0

       x = –2 atau      x = –3

Subtitusikan x₁ = -2 dan x₂ = -3 ke dalam persamaan y = x ² – 4x – 2

x₁ = -2 → y₁ = x ² – 4x – 2

                 = (-2)² – 4(-2) – 2

                 = 4 + 8 – 2

             y₁ = 10

Sehingga penyelesaiannya (-2 , 10)

x₂ = -3 → y₂ = x ² – 4x – 2

                  = (-3) ² – 4(-3) – 2

                  = 9 + 12 – 2

             y₂ = 19

Sehingga penyelesaiannya (-3, 19)

Jadi penyelesaian sistem persamaan tersebut adalah (-2, 10) dan (-3, 19).

Soal :

Jawaban :

Diketahui :

- Bilangan 1 = x

- Bilangan 2 = y

- x + y = 14

- x - y = 4

Ditanya :

- Nilai x dan y = .....?

Penyelesaian :

- Nilai x :

x + y = 14

y = 14 - x

x - y = 4

x - (14 - x) = 4

x - 14 + x = 4

2x = 4 + 14

2x = 18

x = 18/2

x = 9

- Nilai y :

y = 14 - x

y = 14 - 9

y = 5

HP = {(9, 5)}

32. contoh contoh soal sistem persamaan dua variabel


Nia membeli 5 pulpen dan 2 pensil dengan harga Rp. 23.000. Sedangkan Anggi membeli 2 pulpen dan 3 pensil dengan harga Rp. 18.000. Maka harga 1 pulpen dan 1 pensil masing masing adalah Rp. 3.000 dan Rp. 4.000.

Penyelesaian Soal :

Misalkan Harga pulpen = x

               Harga pensil = y

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Nia membeli 5 pulpen dan 2 pensil dengan harga Rp. 23.000" yaitu :

5x + 2y = 23.000   ... (Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "Anggi membeli 2 pulpen dan 3 pensil dengan harga Rp. 18.000" yaitu :

2x + 3y = 18.000   ... (Persamaan 2)

LANGKAH KETIGA (III)

Eliminasikan persamaan 1 dan persamaan 2 sehingga diperoleh nilai x dengan cara :

5x + 2y = 23.000   ║×3║   15x + 6y = 69.000

2x + 3y = 18.000    ║×2║   4x + 6y = 36.000

___________________________________ -

                                                   11x = 33.000

                                                      x =  [tex]\frac{33.000}{11}[/tex]

                                                      x = 3.000

LANGKAH KEEMPAT (IV)

Subtitusikan nilai x pada persamaan 2 sehingga diperoleh nilai y dengan cara :

2x + 3y = 18.000

2 (3.000) + 3y = 18.000

6.000 + 3y = 18.000

3y = 18.000 - 6.000

3y = 12.000

y =  [tex]\frac{12.000}{3}[/tex]

y = 4.000

∴ Kesimpulan harga 1 pulpen dan 1 pensil masing masing adalah Rp. 3.000 dan Rp. 4.000.

2x + y = 15.000  |×1|      2x + y = 15.000

                                   ------------------------  --
                                            -3y = -21.000
                                              y = -21.000 / -3
                                              y = 7.000
metode subtitusi 
2x + y = 15.000
2x + 7000 = 15.000
            2x = 15.000 - 7.000
            2x = 8.000
              x = 8000 / 2
              x = 4000

5x + 3y = 5 (4000) + 3 (7000)
             = 20.000 + 21.000
             = 41.000
Jadi, harga 5 kg mangga dan 3 kg apel adalah Rp 41.000

2)   Tiga tahun yang lalu, jumlah umur ayah dan umur ibu adalah 58 tahun. Lima tahun yang akan datang, umur ayah ditambah dua kali umur ibu adalah 110 tahun. Tentukan umur ayah dan umur ibu saat ini.
jawab :
misalnya, umur ayah = x
                  umur ibu = y 
Artinya, umur ayah tiga tahun yang lalu adalah (x – 3) tahun.Adapun umur ibu tiga tahun yang lalu adalah (y – 3) tahun. Umur ayah lima tahun yang akan datang adalah (x + 5) tahun dan umur ibu lima tahun yang akan datang adalah (y + 5) tahun.

(x - 3) + (y - 3) = 58
x + y = 58 + 3 + 3
x + y = 64  ... pers I

(x + 5) + 2(y + 5) = 110
x + 5 + 2y + 10 = 110
x + 2y = 110 - 5 - 10
x + 2y = 95  .... pers II

x +  y = 64
x + 2y = 95
-------------- --
    -y = -31
      y = 31

x + y = 64
x + 31 = 64
      x = 64 - 31
      x = 33

Jadi umur ayah dan ibu saat ini berturut-turut adalah 33 tahun dan 31 tahun

         tarif mobil = y

       -3y = -3000
          y = -3000 / -3
          y = 1000

2x + 1000 = 1500
             2x = 1500 - 1000
             2x = 500
               x = 500 / 2

Jadi tarif parkir sebuah motor Rp 250 dan tarif pakir sebuah mobil Rp 1000
