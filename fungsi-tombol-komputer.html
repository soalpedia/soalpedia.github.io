3. sebutkan fungsi tombol-tombol pada komputer​


Tombol keyboard:

Caps Lock = untuk membesarkan (mengkapitalkan) semua hurufWindows = untuk ke main source (untuk shutdown, pencaharian, dll)Shift = untuk membesarkan satu huruf dlm sebuah kalimatAlt = untuk commandCtrl = untuk command (contoh Ctrl + P untuk mengeprint, ctrl + v untuk paste, ctrl + c untuk copy)F1 - F12 = pengaturan komputer luarNum Lock = tombol yang memiliki fungsi ganda (contoh: Num Lock on, fungsi pengetikan angka-angka dan operator matematik aktif. Num Lock off, fungsi tombol navigasi aktif.)ESC (Escape): adalah tombol yang kegunaanya sama dengan pilihan CANCEL, yaitu untuk membatalkan satu tahap pekerjaan.Enter: adalah tombol yang kegunaanya sama dengan pilihan tombol OK, yaitu untuk menyatakan bahwa operasi yang dilakukan betul dan selesai. Enter juga dapat berarti menyisipkan baris kosong atau baris baru pada proses pemasukan text. TAB : Berarti pindah ke field atau daerah isian atau pilihan berikutnya. Pada saat proses pengetikan, TAB berarti lompat ke penghentian TAB (tab-stop) terdekat.

Backspace: akan mengakibatkan karakter (huruf) disebelah kiri kursor terhapus.

 Delete: akan mengakibatkan karakter(huruf) disebelah kanan atau ditempat dimana kursor berada akan terhapus.

 Home: akan mengakibatkan kursor berpindah ke awal baris dimana kursor berada.

 End: akan mengakibatkan kursor berpindah ke akhir baris dimana kursor berada.

Page Up: akan mengakibatkan kursor berpindah ke atas sejauh satu layar dari kursor berada.

Page Down: akan mengakibatkan kursor berpindah ke bawah sejauh satu layar dari kursor berada. Up    : Memindahkan kursor 1 baris ke atasDown : Memindahkan kursor 1 baris ke bawah Left : Memindahkan kursor 1 karakter ke kiriRight : Memindahkan kursor 1 karakter ke kanan

7. jelaskan 12 fungsi tombol komputer tolong di jawab besok dikumpulin.....


1. Fungsi Tombol F1

Menampilkan Help atau Bantuan pada semua aplikasi yang terpasang pada komputer

Membisukan suara perangkat komputer.

Tombol Windows + F1 akan menampilkan Microsoft windows help support center pada aplikasi Microsoft Office.

Ctrl + F1 akan menampilkan task panel pada Microsoft Office pada komputer.

2. Fungsi Tombol F2

Mengubah nama icon yang sudah terlebih dahulu disorot pada Windows

Mengurangi volume suara

CTRL + ALT + F2 akan membuka dokumen Microsoft Word yang baru

CTRL + F2 akan menampilkan menu print preview pada Microsoft Word

3. Fungsi Tombol F3

Shift + F3 untuk memperbesar ukuran teks.

Menghidupkan dan mematikan fungsi wifi.

Mengaktifkan fitur pencarian pada windows.

Memperbesar volume suara

4. Fungsi Tombol F4

Sebagai tombol yang berperan mengembalikan perintah sebelumnya atau istilahnya Undo.

Menghidupkan dan mematikan volume suara.

CTRL + F4 untuk menutup semua jendela aktif pada Windows

ALT + F4 untuk menutup semua program yang sedang berjalan

5. Fungsi Tombol F5

Me-refresh atau memuat ulang halaman pada browser.

Mengaktifkan tampilan slide show pada micsoroft power point.

Menurunkan tingkat kecerahan cahaya pada layar.

Membuka find and replace atau go to window microsoft word.

6. Fungsi Tombol F6

Mengarahkan kursor pada addres bar yang terdapat dalam browser.

Mematikan kursor agar tidak bergerak

Menaikkan tingkat pencahayaan layar.

CTRL + Shift + F6 untuk membuka dokumen lain pada microsoft word

7. Fungsi Tombol F7

Mengaktifkan spell check and grammar pada microsoft word.

Mengaktifkan mode penerbangan

Mengaktifkan layar monitor pada proyektor.

Menampilkan thesaurus pada microsoft word.

8. Fungsi Tombol F8

Memasukkan Windows Startup Menu saat mengakses Windows Safe Mode.

Menghentikan tampilan layar monitor dari proyektor.

9. Fungsi Tombol F9

Pada Quark 5.0 dapat membuka toolbar measurement.

Mematikan layar komputer

Menampilkan informasi pada sistem operasi.

10. Fungsi Tombol F10

Memeriksa kotak masuk di email.

Shift + F10 untuk menunjukkan icon pada file dan link yang tersorot di internet.

11. Fungsi Tombol F11

Membuat tampilan layar penuh saat sedang membuka browser di internet.

Mengecilkan kecerahan layar

Mengembalikan tampilan layar browser internet pada tampilan sebelumnya.

12. Fungsi Tombol F12

Shift + F12 untuk mengaktifkan tombol Save di Microsoft Word.

CTRL + F12 mengaktifkan perintah print dalam Microsoft Word.

Membesarkan kecerahan layar

17. fungsi tombol "stand by" pada waktu kita mau booting komputer adalah​


Stand By merupakan fitur pada komputer yang berfungsi untuk mematikan sejenak komputer namun tidak seluruhnya. Ketika sedang memasuki modus Stand By, biasanya layar monitor menjadi blank/hitam. Ini berguna untuk mengemat pasokan listrik ke dalam komputer. Ketika modus Stand By di non-aktifkan, maka layar desktop akan kembali menjadi seperti keadaan semula (sebelum Stand By diaktifkan).

18. sebutkan tombol tombol komputer sesuai Dibawah ini:1.sebutkan tombol tombol pada typing keys2.sebutkan tombol tombol pada numeric keypad3.sebutkan tombol tombol pada tombol fungsi4.sebutkan tombol tombol pada tombol control​


Jawaban:

1. Tombol tombol pada typing keys yaitu tombol abjad A-Z

2. Tombol tombol pada numeric keys yaitu tombol angka 0-9

3. Tombol tombol pada function keys yaitu tombol F1 sampai F12

Penjelasan:

1. Selain tombol abjad, ada juga tombol alfanumeric. Yaitu tombol yang menjadi gabungan antara angka dan simbol-simbol tertentu.

2. Selain tombol angka, yang termasuk dalam numeric keys ialah tombol /*-+.

3. Tombol tombol fungsi umumnya terletak di bagian paling atas di keyboard (berderet dari F1 sampai F12)

4. Selain contoh yang sudah disebutkan di atas, yang termasuk tombol control key dapat dilihat pada gambar berikut ini (peta keyboard).

Tombol Tab

Fungsi tombol tab adalah untuk membuat kursor bergeser dengan jarak 1 tabulasi ke arah kanan. Fungsi tombol ini mirip dengan tombol space yang digunakan untuk memberi jarak antara kata 1 dan lainnya.  Letak tombol tab tepat diatas tombol caps lock.

Tombol Caps Lock

Fungsi tombol caps lock adalah untuk membuat huruf atau tulisan menjadi BESAR (UPPERCASE). Biasanya ada indikator atau lampu yang menyala begitu tombol ini dalam keadaan on.

Tombol Windows (Home)

Fungsi tombol home adalah untuk mengarahkan tampilan pada halaman awal atau halaman start secara cepat. Untuk mengembalikan pada halaman yang terakhir dibuka, maka cukup klik lagi tombol ini. Tombol home biasanya berupa logo windows atau jendela dan terletak antara CTRL dan ALT.

Tombol Space (Spasi)

Tombol space berbentuk panjang dan terletak dibagian paling bawah. Fungsinya adalah untuk menambahkan spasi atau jarak antara 1 kata dengan kata lainnya.

Tombol Delete

Tombol delete berfungsi untuk menghapus 1 karakter di kanan/depan kursor. Tombol ini akan menghapus kata yang ditulis setelah tanda kursos atau kata yang sebelumnya telah ditandai.

Tombol Enter

Fungsi tombol enter adalah untuk berpindah baris. Ketika mengetikan tombol enter maka tulisan di baris kursor akan berpindah ke baris baru. Tombol enter ada disebelah kanan keyboard dengan bentuk paling besar diantara tombol lainnya.

Tombol Backspace

Tombol Backspace ditandai dengan simbol <—- di keyboard. Fungsi tombol backspace adalah untuk menghapus satu karakter yang ditulis. Jika kamu ingin menghapus lebih cepat gunakan perpaduan tombol backspace dengan CTRL untuk menghapus sebuah kata.

Tombol PrtScr

Tombol print screen atau pada keyboard disingkat PrtScr/PrtSc adalah tombol yang digunakan untuk menangkap atau meng-capture layar. Tekan tombol ini 1 kali kemudian buka ms word ataupun paint. Ketik CTRL + P, maka gambar hasil screeshoot pun akan muncul.

Cara lain adalah dengan memadukan tombol windows/home dengan printscreen. Ketika di klik berbarengan maka akan muncul kedipan di layar yang menandakan layar sudah ter-screenshoot. Kamu dapat melihat hasilnya di folder PICTURES di my computer anda.

Tombol End

Fungsi tombol end adalah untuk memindahkan kursor ke baris akhir tulisan. Ini biasanya berjalan ketika anda sedang bekerja menggunakan ms word.

Tombol Page Up & Page Down

Tombol page up digunakan untuk menarik halaman ke atas (scrolling ke atas), dan page down untuk menarik halaman ke bawah (scrolling ke bawah). Dengan menggunakan tombol ini anda tak perlu repot ketika akan menurunkan atau menaikan posisi file di layar komputer.

Tombol Navigasi (Panah Kanan, kiri, atas, bawah)

Tombol navigasi berfungsi untuk menggerakan kursor ke atas, bawah, kanan ataupun kiri. Kursor sendiri adalah bagian di layar komputer yang berfungsi sebagai indikator posisi. Bentuknya lurus kecil dan berkedip-kedip.

Tombol Lain

Tombol NumLock -> tombol numlock berfungsi untuk mengubah bagian keyboard utama bertindak sebagai keypad numerik.

Tombol Scrooll Lock -> Pada microsoft excel tombol ini berfungsi untuk mengunci kolom agar tidak menggulung saat  tombol atas kanan kiri bawah ditekan.

Tombol Fn -> Tombol fn biasanya ada pada keyboard laptop ia hanya berfungsi ketika dikombinasikan dengan tombol lain. Misalnya untuk mencerahkan meredupkan layar, menaikan atau mengecilkan volume suara dll.
