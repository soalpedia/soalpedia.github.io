3. Tombol tombol fungsi umumnya terletak di bagian paling atas di keyboard (berderet dari F1 sampai F12)

Misalkan dipakai pada program pengolah data teks (seperti Ms. Word):

Caps Lock: untuk mengunci huruf kapital.

Tab: untuk memindah posisi kursor ke sebelah kanan 1 tabulasi.

Back space: untuk menghapus 1 karakter di sebelah kiri

Delete : untuk menghapus 1 karakter di sebelah kanan

Enter: untuk mengakhiri 1 baris dan mengawali pengetikan teks di baris baru (di bawahnya).

Esc: untuk menghilangkan tampilan menu yg muncul.

Home: untuk memindah posisi kursor ke awal baris teks.

End: untuk memindah posisi kursor ke akhir baris.

Page Up: untuk memindahkan posisi kursor 1 layar ke atas.

Page Down: untuk memindahkan posisi kursor 1 layar ke bawah.

Penjelasan:

Contoh pemakaian tombol Esc: misalkan mengklik kanan teks (dg mouse) lalu muncul suatu menu, maka tombol ini bisa dipakai untuk menghilangkan menu yg muncul tadi.

5. tuliskan tombol tombol perintah yg terdapat pada tombol office buttom


New berfungsi untuk membuat sebuah dokumen baru atau menggunakan template.
Open berfungsi untuk membuka lembar dokumen yang telah disimpan (save) sebelumnya atau membuka berbagai jenis dokumen teks yang didukung oleh Word 2007.
Convert berfungsi untuk mengubah tipe file berbentuk *.doc (versi 2003 ke bawah) menjadi tipe file *.docx yang didukung oleh Word 2007.
Save berfungsi untuk menyimpang dokumen yang telah dibuat.
Save As berfungsi untuk menyimpan dokumen yang telah kita buat terlebih dulu dengan nama yang lain dan bisa juga menyimpan file ke dalam bentuk file yang lain selain *.docx, misalnya *.rtf dan *.xml. Hal itu bertujuan untuk menggandakan dokumen, sehingga isi file aslinya tetap terjaga
Print berfungsi untuk mencetak dokumen melalui printer
Publish berfungsi untuk mempublikasikan dokumen yang sedang terbuka ke dalam berbagai media, seperti Blog di internet dan Document Management Server serta Document Workspace di jaringan komputer lokal.
Close berfungsi untuk menutup dokumen yang sedang dibuka di jendela Word 2007.
Recent Document berfungsi menampilkan daftar-daftar file dokumen yang sedang dibuka terakhir kali menggunakan Word 2007.
Word Options berfungsi menampilkan jendela untuk mengatur opsi Word 2007.
Exit Word berfungsi untuk menutup program Word 2007

13. Tombol apa saja yang merupakan tombol tombol khusus pada keyboard!


Keyboard merupakan salah satu perangkat keras dalam instalasi komputer. Keyboard berfungsi untuk menampilkan tulisan pada layar komuper. Dalam sebuah keyboard, terdapat beberapa tombol khusus, yakni

DeletePage upPage downInsertPrint screenPause breakEndPembahasan

Keyboard merupakan salah satu perangkat keras dalam instalasi komputer. Keyboard berisikan huruf, angka, dan tombol-tombol lain yang berfungsi untuk menampilkan tulisan pada layar komuper. Pada umumnya, keyboard memiliki dua jenis, yakni keuboard QWERTY dan keyboard DVORAK. Kedua keyboard tersebut merupkana keyboard yang banyak digunakan oleh pengguna komputer.

Dalam sebuah keyboard, terdapat beberapa tombol khusus, yakni

Delete. Delete digunakan untuk menghapus huruf dari depan ke belakang. Page up. Page up digunakan untuk menggerakkan layar ke atas tanpa menggunakan mouse. Page down. Page down digunakan untuk menggerakkan layar ke bawah tanpa menggunakan mouse Insert. Insert digunakan untuk mensisipkan karakter pada posisi kursor. Print screen. Print screen digunakan untuk mengcapture tampilan layar Pause break. Pause break biasanya digunakan pada sistem operasi DOS yang berungsi untuk menghentikan tampilan program berbasis text. End. End digunakan untuk memindahkan kursor pada baris paling belakang.

37. Tombol apa saja yang merupakan tombol tombol khusus pada keyboard...


Penjelasan:

Secara umum, struktur tombol pada keyboard

terbagi atas 4, yaitu:

1. Tombol Ketik (typing keys)

Tombol ketik adalah salah satu bagian dari

keyboard yang berisi huruf dan angka serta

tanda baca. Secara umum, ada 2 jenis susunan

huruf pada keyboard, yaitu tipe QWERTY dan

DVORAK. Namun, yang terbanyak digunakan

sampai saat ini adalah susunan QWERTY.

2. Numeric Keypad

Numeric keypad merupakan bagian khusus dari

keyboard yang berisi angka dan sangat

berfungsi untuk memasukkan data berupa

angka dan operasi perhitungan. Struktur

angkanya disusun menyerupai kalkulator dan

alat hitung lainnya.

3. Tombol Fungsi (Function Keys)

Tahun 1986, IBM menambahkan beberapa

tombol fungsi pada keyboard standard. Tombol

ini dapat dipergunakan sebagai perintah khusus

yang disertakan pada sistem operasi maupun

aplikasi.

4. Tombol kontrol (Control keys)

Tombol ini menyediakan kontrol terhadap

kursor dan layar. Tombol yang termasuk dalam

kategori ini adalah 4 tombol bersimbol panah di

antara tombol ketik dan numeric keypad, home,

end, insert, delete, page up, page down, control

(ctrl), alternate (alt) dan escape (esc).
