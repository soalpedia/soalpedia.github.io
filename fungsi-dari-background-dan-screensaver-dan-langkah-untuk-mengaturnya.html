Jawaban:

Upload gambar yang mau kalian jadikan sebagai backgroud terlebih dahulu ke internet, bisa melalui blogspot, google drive, atau media penyimpanan lainnya. Setelah selesai jangan lupa ambil urlnya.

1. Login ke blogger

2. Pilih menu theme dan klik Edit HTML ( lihat gambar di bawah )

3. Carilah kode/script yang mirip dengan kode berikut ini.

body{background:#e4e4e4;margin:auto;padding:0;color:#333;font:normal normal 14px Arial, helvetica, sans-serif;

line-height:1.7em;}

Tips : supaya lebih mudah, klik asal di sembarang tempat dalam kode template Anda. Lalu tekan tombol CTRL+F pada keyboard laptop/pc yang sobat pergunakan. Dan lakukan pencarian menggunakan kode “background:” pada kolom yang tersedia. Bingung ? lihat gambar di bawah ini.

Cari kode yang di awali dengan kata body, setelah ketemu tambahkan kode “url('url link gambar yang ingin dijadikan background') repeat-x” di belakang script “background:#e4e4e4”. Sehingga menjadi seperti berikut.

body{background:#e4e4e4 url(url gambar yang telah kawan-kawan persiapkan sebelumnya') repeat-x;margin:auto;padding:0;color:#333;font:normal normal 14px Arial, helvetica, sans-serif;

line-height:1.7em;}

Catatan penting : setiap template berbeda kode dan penempatannya, jadi kode di atas belum tentu sama dengan script template blogger yang kawan-kawan pakai.

Untuk melihat perubahan, klik pratinjau (preview theme) dan jika sudah sesuai dengan harapan, langkah terakhir yang perlu Anda lakukan adalah mengklik tombol "Save theme".

#Note : Gambar yang dapat digunakan untuk menjadi background harus berformat jpg, gif, png dengan ukuran hingga 300kb. Tapi Admin sarankan dibawah 50kb supaya loading blog tidak berat. Jika kalian ingin mendapatkan latar belakang penuh bisa gunakan gambar dengan ukuran 1800x1600 atau lebih besar serta jika ingin posisi gambar tetap ketika di scroll maka harus memiliki panjang minimal 860px contoh 1400x880

21. untuk menganti background pada page colerdraw lankah langkah yang tepat adalah​


Tutorial Mengubah Background Halaman atau Drawing Page CorelDraw:

Mengubah Warna Background

1. Setelah menentukan ukuran halaman, yang pertama klik Menu Layout pada Menu Bar --> klik Page Background, kemudian akan muncul kotak dialog Option.

2. Setelah itu klik Solid kemudian pilih warna yang diinginkan. Untuk memilih warna juga disediakan alat Color Eyedropper Tool untuk memilih warna tertentu dari objek lain.

3. Terakhir klik Ok dan lihat hasilnya. (Saya memilih warna Cyan)

Membuat Background Berpola/Bergambar

1. Pertama klik Menu Layout pada Menu Bar --> klik Page Background

2. Kemudian klik Bitmap untuk mengganti background menjadi berpola/gambar. Ada beberapa hal yang perlu diperhatikan yaitu:

1. Source: Gambar yang dipilih akan terhubung dengan sumber data

2. Embbeded: Gambar yang dipilih tidak terhubung dengan sumber data.

3. Klik Default Size jika tidak ingin mengubah ukuran gambar yang akan digunakan sebagai background. atau Klik Costumize Size jika ingin merubah ukuran gambar yang akan digunakan untuk background. Mainten Aspect Ratio mengatur ukuran gambar agar sesuai dengan ukuran lembar kerja

4. Klik Ok untuk mengakhiri dan lihat hasilnya.

Tambahan:

untuk mencetak hasil desain beserta background centang Print and Export Background. Sebaliknya jika tidak ingin mencetak background jangan mencentang pilihan tersebut.

33. Sebutkan Langkah-langkah untuk mengubah foreground dan Background,​


Jawaban:

Buka Terminal

Untuk merubah foreground (warna tulisannya), ketikkan baris perintah berikut pada Terminal: $ gsettings set org.pantheon.terminal.settings foreground 'rgba(0, 0, 0, 1)'

Untuk merubah background (latar belakang Terminal), ketikkan baris perintah berikut pada Terminal: $ gsettings set org.pantheon.terminal.settings background 'rgba(255, 255, 255, 1)'

Dengan keterangan rgba(R, G, B, A) sebagai berikut:

R = Warna Merah (Red)

G = Warna Hijau (Green)

B = Warna Biru (Blue)

A = Tingkatan transparansinya, dengan range 0 sampai 1
