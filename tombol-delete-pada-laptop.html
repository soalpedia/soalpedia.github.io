24. apa fungsi tombol backspace dan delete?


Tombol backspace dan delete pada dasarnya memiliki fungsi yang sama, yakni untuk menghapus. Namun, dari keduanya memiliki perbedaan. Berikut penjelasannya.  Tombol backspace digunakan untuk menghapus huruf atau angka dari belakang ke depan. Selain itu, tombil backspace dapat digunakan sebagai perintah untuk kembali ke keadaan sebelumnya ketika digunakan untuk membuka dokumen komputer. Sedangkan tombol delete digunakan untuk menghapus dari depan ke belakang. Selain itu, tombol delete juga berfungsi untuk menghapus gambar, file, video, dokumen, dan lain-lain.

Pembahasan

Dalam sebuah perangkat komputer, pasti anda akan mendapatkan perangkat keras yang bernama keyboard. Keyboard adalah perangkat keras dari komputer yang berungsi untuk mengetik huruf, angka, dan memberikan perintah-perintah tertentu. Fungsi keyboard dalam sebuah komputer sangat penting.

Dalam sebuah keyboard, terdapat berbagai tombol dan berbagai perintah yang dapat digunakan oleh user computer. Beberapa diantaranya adalah tombol backspace dan delete. Tombol backspace dan delete pada dasarnya memiliki fungsi yang sama, yakni untuk menghapus. Namun, dari keduanya memiliki perbedaan. Berikut penjelasannya.

Tombol backspace digunakan untuk menghapus huruf atau angka dari belakang ke depan. Selain itu, tombil backspace dapat digunakan sebagai perintah untuk kembali ke keadaan sebelumnya ketika digunakan untuk membuka dokumen komputer. Sedangkan tombol delete digunakan untuk menghapus dari depan ke belakang. Selain itu, tombol delete juga berfungsi untuk menghapus gambar, file, video, dokumen, dan lain-lain.
