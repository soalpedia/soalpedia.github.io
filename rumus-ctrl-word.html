10. fungsi ctrl+l pada microsoft word adalah​


Jawaban:

ctrl + I = control + Italic

Fungsi: Untuk membuat teks bergaya miring.

Penjelasan:

• ctrl merupakan singkatan dari 'control'

• I atau Italic adalah untuk menerapkan gaya miring.

control lain yang biasa digunakan:

ctrl + B = control + Bold = menerapkan gaya Tebal pada tulisan

ctrl + U = control + Underline = menerapkan gaya Garis Bawah pada tulisan

Supaya kamu nggak lupa, kamu bisa hafalin kosa kata bahasa inggrisnya. Misal,

B, Bold = tebal

U, Underline = Garis Bawah

I, Italic = Miring

C, Copy = Salin

dan sebagainya

Kalo kamu mau liat bentuk perintah control lain, bisa baca di sumber berikut :)

https://support.microsoft.com/id-id/topic/pintasan-keyboard-di-word-95ef89dd-7142-4b50-afb2-f762f663ceb2
