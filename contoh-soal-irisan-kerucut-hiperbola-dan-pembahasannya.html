7. Contoh soal cerita himpunan irisan dan jawabanya


Contoh Soal :

Jika :

P = { x | 1 < x < 10, x ∈ bilangan cacah },

Q = { x | 2 < x < 10, x ∈ bilangan prima },

R = { x | 1 < x < 10, x ∈ bilangan ganjil }, maka P ∩ Q ∩ R adalah ...

Penyelesaian :

P = {2, 3, 4, 5, 6, 7, 8, 9}

Q = {3, 5, 7}

R = {3, 5, 7, 9}

Ingat, irisan himpunan adalah himpunan yang anggota-anggotanya juga merupakan anggota-anggota himpunan lainnya.

Jadi, P ∩ Q ∩ R = {3, 5, 7, 9}.

23. definisikan lah tentang irisan kerucut​


Jawaban:

Irisan kerucut adalah bidang yang terbentuk ketika sebuah kerucut dipotong oleh sebuah bidang datar. Irisan ini memiliki bentuk lingkaran di bagian dasarnya, mirip dengan bentuk lingkaran pada kerucut itu sendiri. Namun, irisannya juga bisa berbentuk elips atau bentuk lainnya tergantung pada sudut potongan dan posisi bidang potong.

Ada beberapa jenis irisan kerucut yang umum, yaitu:

1. Irisan tegak lurus: Ketika bidang potong tegak lurus terhadap sumbu kerucut, irisannya berbentuk lingkaran. Ini adalah jenis irisan kerucut paling umum.

2. Irisan miring: Ketika bidang potong membentuk sudut miring terhadap sumbu kerucut, irisannya berbentuk elips.

3. Irisan horizontal: Ketika bidang potong sejajar dengan dasar kerucut, irisannya berbentuk lingkaran dengan jari-jari yang sama dengan jari-jari dasar kerucut.

Irisan kerucut sering digunakan dalam matematika, fisika, dan rekayasa. Misalnya, dalam matematika, irisan kerucut digunakan untuk menghitung luas dan volume irisan, sedangkan dalam fisika dan rekayasa, irisan kerucut digunakan untuk mempelajari sifat optik, properti fluida, dan desain struktur.

31. irisan kerucut. misalkan terdapat suatu kerucut dengan jari-jari r cm dan panjang t cm. kemudian kerucut tersebut dijadikan irisan kerucut dengan memotong kerucut tersebut menjadi dua bagian dari atas ke bawah ( lihat gambar di atas ). tentukan rumus untuk menghitung luas irisan kerucut tersebut.


Irisan kerucut yang memotong kerucut menjadi dua bagian dari atas ke bawah dengan jari-jari r cm dan tinggi kerucut t cm. Maka rumus untuk menentukan luas irisan kerucut tersebut adalah [tex]\displaystyle \frac{1}{2}\pi r \ (r + \sqrt{r^2+t^2} ) + rt[/tex].

Penjelasan dengan langkah-langkah

Kerucut adalah limas yang bidang alasnya berbentuk lingkaran.

Rumus Kerucut

[tex]\boxed {V = \frac{1}{3} \times \pi \times r^2 \times t}[/tex]

[tex]\boxed {LP = \pi r(r+s)}[/tex]

Panjang garis pelukis (s) = [tex]\sqrt{r^2+t^2}[/tex]

Diketahui :

Irisan kerucut yang dipotong dua bagian.

Panjang jari-jari = r cm.

Tinggi = t cm.

Ditanya :

Luas irisan kerucut.

Jawab :

Menentukan luas irisan kerucut

Luas = L 1/2 lingkaran + L 1/2 selimut kerucut + L segitiga

        = [tex]\displaystyle \frac{1}{2}\pi r^2 + \frac{1}{2} \pi rs + \frac{1}{2}\ 2rt[/tex]

        = [tex]\displaystyle \frac{1}{2}\pi r \ (r + s) + rt[/tex]

        = [tex]\displaystyle \frac{1}{2}\pi r \ (r + \sqrt{r^2+t^2} ) + rt[/tex]

Kesimpulan :

Jadi luas irisan kerucut tersebut adalah [tex]\displaystyle \frac{1}{2}\pi r \ (r + \sqrt{r^2+t^2} ) + rt[/tex].
