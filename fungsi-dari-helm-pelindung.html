23. pelles bantuonHELM ME HELM ME HELM ME........ ​


Jadi, banyak siswa kelas 8A adalah 25 siswa.

Pembahasan

Rata-rata adalah jumlah data dibagi dengan banyak data.

Rumus rata-rata (mean) sebagai berikut.

[tex]\bar{x} = \frac{x_1 + x_2 + x_3 + ...+ x_n}{n}[/tex] atau [tex]\bar{x}[/tex] = ∑xifi / ∑fi

dengan

∑xifi = jumlah data dikali dengan frekuensi

∑fi = jumlah frekuensi semua data

Rumus yang digunakan untuk rata-rata gabungan sebagai berikut.

[tex]\bar{x}_{gab} = \frac{n_1 \bar{x_1} + n_2 \bar{x_2}}{n_1 + n_2}[/tex]

dengan

[tex]\bar{x}_{gab}[/tex] = rata-rata setelah digabung

n₁ = banyak data 1

n₂ = banyak data 2

[tex]\bar{x_1}[/tex] = rata-rata data 1

[tex]\bar{x_2}[/tex] = rata-rata data 2

Penyelesaian

diket:

rata-rata nilai kelas 8A = [tex]\bar{x_1} = 72[/tex]

rata-rata nilai kelas 8B = [tex]\bar{x_2} = 80[/tex]

jumlah siswa kelas 8B = n₂ = 15 siswa

rata-rata gabungan = [tex]\bar{x_{gab}} = 75[/tex]

ditanya:

banyak siswa kelas 8A = n₁ ...?

jawab:

[tex]\bar{x}_{gab} = \frac{n_1 \bar{x_1} + n_2 \bar{x_2}}{n_1 + n_2}[/tex]

[tex]75 = \frac{n_1 \: . \: 72 + (15) (80)}{n_1 +15}[/tex]

[tex]75 = \frac{72n_1 + 1200}{n_1 +15}[/tex]

75(n₁ + 15) = 72 n₁ + 1200

75 n₁ + 1125 = 72 n₁ + 1200

75 n₁ - 72 n₁ = 1200 - 1125

3 n₁ = 75

n₁ = 75 : 3

n₁ = 25

Kesimpulan

Jadi, banyak siswa kelas 8A adalah 25 siswa.
