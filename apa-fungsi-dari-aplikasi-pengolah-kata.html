21. Jelaskan fungsi Hyperlink pada software aplikasi pengolah kata!


Jawaban:

Artikel ini menguraikan sintaks rumus dan penggunaan fungsi HYPERLINK di Microsoft Excel.

Deskripsi

Fungsi HYPERLINK akan membuat pintasan yang menuju ke lokasi lain dalam buku kerja saat ini, atau membuka dokumen yang disimpan di server jaringan, intranet, atau Internet. Saat mengklik sel yang berisi fungsi HYPERLINK, Excel akan berpindah ke lokasi yang tercantum, atau membuka dokumen yang ditentukan.

Sintaks

HYPERLINK(lokasi_tautan, [nama_familier])

Sintaks fungsi HYPERLINK memiliki argumen berikut:

Link_location Diperlukan. Nama jalur dan file ke dokumen yang akan dibuka. Link_location bisa mengacu pada tempat dalam dokumen — seperti sel spesifik atau rentang nama dalam lembar kerja atau buku kerja Excel, atau ke bookmark dalam dokumen Microsoft Word. Jalur tersebut dapat berupa file yang disimpan di drive hard disk. Jalur juga bisa menjadi jalur konvensi penamaan Universal (UNC) di server (di Microsoft Excel untuk Windows) atau jalur uniform resource locator (URL) di internet atau intranet.

Catatan Excel untuk web fungsi hyperlink valid untuk alamat web (URL) saja. Link_location bisa berupa string teks yang diapit dalam tanda kutip atau referensi ke sel yang berisi link sebagai string teks.

Jika lompat yang ditentukan di link_location tidak ada atau tidak bisa dinavigasi, kesalahan muncul saat Anda mengklik sel.

Friendly_name Opsional. Teks pintasan atau nilai numerik yang ditampilkan di sel. Friendly_name ditampilkan dengan warna biru dan digarisbawahi. Jika friendly_name diabaikan, sel menampilkan link_location sebagai teks pintasan.

Friendly_name bisa berupa nilai, string teks, nama, atau sel yang berisi teks pintasan atau nilai.

Jika friendly_name mengembalikan nilai kesalahan (misalnya, #VALUE!), sel tersebut menampilkan kesalahan, bukannya teks pintasan.

Keterangan

Dalam aplikasi desktop Excel, untuk memilih sel yang berisi hyperlink tanpa harus menuju ke tujuan hyperlink, klik sel dan tahan tombol mouse hingga bentuk penunjuk menjadi tanda silang Kursor pilihan Excel , lalu lepaskan tombol mouse. Di Excel untuk web, pilih sel dengan mengkliknya ketika penunjuk adalah Panah; lompat ke tujuan hyperlink dengan mengklik saat penunjuk adalah tangan yang menunjuk.

35. 1. Fungsi dan fitur yang terdapat pada aplikasi perangkat lunak pengolah angka. 2. Fungsi dan fitur yang terdapat pada aplikasi perangkat lunak pengolah kata.


Software pengolah angka salah satunya yang terkenal adalah microsoft excel

Software pengolah kata salah satunya yang terkenal adalah microsoft word

Pembahasan

Fungsi Software pengolah angka adalah berfungsi untuk melakukan data penjumlahan serta pengolahan data

Fitur-fitur software pengolah angka :

Menggabungkan sel

Dengan pilih menu format-cells, setelah itu klik alignment, pilih dan tandai tabel yang ingin digabunggkan pilih merge cells dan oke

Adanya banyak-banyak formula untuk penjumlahan, rata-rata.

SUM, dengan menulis salah satu tabel 'SUM( pada kurung buka ini pilihlah kolom atau tabel yang ingin dijumlahkan(fungsi sum), setelah itu sudah dipilih tutup). Maka akan menghasilkan penjumlahan tambah dari fungsi SUM dari. Ada juga AVERAGE untuk mencari nilai rata-rata caranya sama dengan melakukan SUM, tadi.

Formula max

Formula ini untuk mencari nilai terbanyak  atau tertinggi dari, tabel tabel yang dipilih.

Fungsi Software pengolah kata adalah berfungsi yang berfungsi untuk mengolah kata agar mudah dan sederhana karena memiliki banyak fitur.

Fitur-fitur software pengolah kata :

Mail merge

Kita bisa membuat surat secara masal, dan diberi template nya dari microsoft word. Sehingga mempermudah kita membuat surat.  

Section

Berfungsi untuk memberikan jarak pemisah pada halaman dokumen

Page Layout-Breaks-Next Page lalu klik Insert-Page Number yang berguna untuk memberi jarak section dengan angka atau huruf pada header ataupun footer.

Page number

Insert - page number, adalah berfungsi untuk memberi nomor halaman pada dokumen.
