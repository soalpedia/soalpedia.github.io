1. apa perbedaan static link library, dynamic link library, dan import library?​


Jawaban:

Static library(juga dikenal sebagai archive) terdiri dari rutinitas yang disusun dan dihubungkan langsung ke program anda. Ketika anda mengkompilasi sebuah program yang menggunakan static library, semua fungsi dari static library menjadi bagian dari eksekusi. Pada Windows, static library biasanya memiliki ekstensi .lib, Sedangkan di linux, perpustakaan statis biasanya memiliki ekstensi .a (archive). Satu keuntungan dari static library adalah bahwa anda hanya harus mendistribusikan executable agar pengguna untuk menjalankan program anda. Karena library menjadi bagian dari program anda, ini memastikan bahwa versi yang tepat dari library selalu digunakan dengan program anda. Juga, karena static library menjadi bagian dari program anda, anda dapat menggunakannya sama seperti fungsi yang sudah anda tulis untuk program anda sendiri. Pada sisi negatifnya, karena salinan library menjadi bagian dari setiap executable, maka ini dapat menyebabkan banyak ruang terbuang. Static library juga tidak dapat diperbarui dengan mudah, untuk memperbarui library, seluruh executable perlu diganti.

Dynamic library (juga disebut shared library) terdiri dari rutinitas yang dimuat ke dalam aplikasi anda pada waktu dijalankan. Ketika anda mengkompilasi sebuah program yang menggunakan dynamic library, library tidak menjadi bagian dari eksekusi, tetap sebagai unit terpisah. Pada Windows, dynamic library biasanya memiliki ekstensi .dll (dynamic link library), Sedangkan di Linux, dynamic library biasanya memiliki ekstensi .so (shared object). Salah satu keuntungan dari dynamic library adalah bahwa banyak program dapat berbagi satu salinan, yang menghemat ruang. Mungkin keuntungan yang lebih besar adalah bahwa dynamic library dapat diperbarui ke versi yang lebih baru tanpa mengganti semua executable yang menggunakannya.

5. fungsi folder .link pada WhatsApp,Apakah kita boleh menghapus semua?​


Fungsi folder .link pada WhatsApp adalah untuk menyimpan tautan (link) yang dibagikan melalui aplikasi tersebut. Folder ini secara otomatis dibuat oleh WhatsApp untuk menyimpan tautan yang Anda terima atau bagikan.

Anda dapat menghapus tautan-tautan individual dari folder .link jika Anda tidak memerlukannya lagi. Namun, sebaiknya Anda tidak menghapus seluruh folder .link karena itu adalah bagian dari struktur penyimpanan WhatsApp dan dapat menyebabkan ketidakstabilan atau masalah lain pada aplikasi.

Jadi, boleh menghapus tautan-tautan individual dari folder .link, tetapi sebaiknya tidak menghapus seluruh folder tersebut.

20. apa itu minimum dynamic area ?​


Jawaban:

Minimum dynamic area adalah area minimum yang diperlukan untuk membuat sebuah transistor dalam suatu teknologi semikonduktor. Transistor pada umumnya terdiri dari tiga lapisan (source, drain, dan gate) yang dibentuk melalui proses lithography pada area tertentu pada permukaan semikonduktor. Minimum dynamic area menentukan seberapa kecil ukuran transistor yang dapat dibuat dalam suatu teknologi semikonduktor. Semakin kecil minimum dynamic area maka semakin kecil ukuran transistor yang dapat dibuat, sehingga semakin banyak transistor yang dapat dimuat pada sebuah chip sehingga meningkatkan performa dan efisiensi dari sirkuit elektronik.

28. pa fungsi Synchronous Dynamic Random Access Memory​


Jawaban:

Synchronous Dynamic Random Access Memory (disingkat menjadi SDRAM) merupakan sebuah jenis memori komputer dinamis yang digunakan dalam PC dari tahun 1996 hingga 2003. SDRAM juga merupakan salah satu jenis dari memori komputer kategori solid-state.

SDRAM, pada awalnya berjalan pada kecepatan 66 MHz untuk dipasangkan dengan prosesor Intel Pentium Pro/Intel Pentium MMX/Intel Pentium II, dan terus ditingkatkan menjadi kecepatan 100 MHz (dipasangkan dengan Intel Pentium III/AMD Athlon), hingga mentok pada kecepatan 133 MHz (dipasangkan dengan Intel Pentium 4 dan AMD Athlon/Duron). Popularitasnya menurun saat DDR-SDRAM yang mampu mentransfer data dua kali lipat SDRAM muncul di pasaran dengan chipset yang stabil. Setelah itu, akibat produksinya yang semakin dikurangi, harganya pun melonjak tinggi, dengan permintaan pasar yang masih banyak; dengan kapasitas yang sama dengan DDR-SDRAM, harganya berbeda kira-kira Rp. 150000 hingga 250000.

31. Apa perbedaan ip dynamic public dengan ip dynamic private?


Jawaban:

Dynamic IP Address

Disebut IP Dynamic / dinamis karena IP yang didapat perangkat komputer akan berganti setiap perangkat komputer direstart.

Seperti IP Address yang didapat modem speedy, IP yang kamu dapat saat konek ke hotspot akan berubah setiap kali perangkat direstart.

Penerapan IP dynamic dengan DHCP Server sangat memudahkan user karena perangkat akan mendapatkan IP secara otomatis saat terkoneksi ke jaringan.

Static IP Address

Static IP atau ip statis adalah IP yang dipasang secara manual atau  IP yang dialokasikan secara permanen khusus oleh router untuk suatu perangkat komputer (router,tablet,smartphone,smart device dll),sehingga ketika perangkat direstart IP tidak berubah.

33. 7. Komponen SNORT yang berfungsi mengambil data disisi layer 2 dari packet capture library dengan memisahkan data link, seperti Ethernet,token ring,dan 802. 11 adalah….


Jawaban:

preprocessor sistem link layer

Penjelasan:

Salah satu komponen dari SNORT yang berfungsi untuk mengambil data disisi layer 2 dari packet capture library dengan memisahkan data link seperti Ethernet, token ring, dan 802.11 adalah preprocessor sistem link layer. Preprocessor ini memungkinkan SNORT untuk mengakses header sistem link layer dari paket yang ditangkap, sehingga SNORT dapat memeriksa informasi yang terkandung dalam header tersebut dan mengambil tindakan yang diperlukan sesuai dengan aturan yang ditentukan. Preprocessor sistem link layer juga dapat membantu SNORT untuk mengidentifikasi jenis sistem link layer yang digunakan dan menangani setiap perbedaan dalam header sistem link layer yang mungkin terjadi antara jenis-jenis tersebut.
