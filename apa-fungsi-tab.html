9. jelaskan fungsi fungsi yang terdalat dalam tab home,tab insert Dan page layout pada Microsoft excel


1. Home
Home merupakan menu utama yang akan tampil pada saat pertama kali anda membuka word 2007. Menu home terdiri dari toolbar standar yaitu font, tool bar formatting dan paragraph, style dan editing serta clipboard.(klik untuk memperbesar gambar)
2. Insert
Tab ini digunakan untuk memasukkan instruksi-instruksi ke dalam dokumen. Menu insert terdiri dari : Pages, Tables, Illustrations, Links, Header & Footer, Text, Symbol.(klik untuk memperbesar gambar)
3. Page Layout
Tab page Layout digunakan untuk mengatur tata letak data, yang meliputi Page Setup, memberiwarna background pada lembar kerja, mengatur spasi antar baris, membuat kolom pada teks dan mengatur indentasi. Menu page layout terdiri dari : Themes, Page Setup, Page Background, Paragraph, Arrange.(klik untuk memperbesar gambar)
4. References
Tab ini dugunakan untuk memasukkan berbagai perintah seperti menuliskan footnote (catatan kaki), menyisipkan gambar, dan membuat tanda citasi. Perhatikan tampilan tab references
Tab References. References terdiri dari : Table of Contents, Footnotes, Citations & Bibliography, Captions, Index, Table of Authorities.(klik untuk memperbesar gambar)
5. Mailings
Tab ini berguna untuk membuat mail merger (surat massal), mengetik teks pada amplop, dan membuat label. Mailings terdiri dari : create, start mail merge, write & insert Fields, preview results, finish.(klik untuk memperbesar gambar)
6. Review
Tab Review berguna untuk mengecek kesalahan tata bahasa serta memproteksi dokumen. Review terdiri dari : proofing Comments, Tracking, changes, Compare, Protect.(klik untuk memperbesar gambar)
7. View
Tab View digunakan untuk menampilkan atau mencetak dokumen pada layar, mengatur ukuran cetakannya, dan menampilkan garis bantu pada lembar kerja. Menu View berfungsi untuk mengatur tampilan layar dari dokumen yang kita kerjakan. Berikut ini merupakan berbagai fungsi dari tool-tool yang terdapat dalam menu View.

14. Tuliskan dan jelaskan fungsi yang terdapat pada tab home, tab page layout, tab references, tab mailings, tab review, tab view.​


Penjelasan:

1. Home

Tab ini memiliki fungsi untuk menyunting properti-properti tulisan seperti halnya mengganti font yang digunakan dalam lembar dokumen Microsoft Word 2007, ukuran font yang diinginkan atau bahkan hingga penempatan paragraf sehingga tulisan akan menjadi lebih rapi.

2. Insert

Tab Insert ini digunakan untuk mempercantik tampilan dokumen, untuk memasukkan gambar/foto ke dalam lembar kerja, Tab Insert ini bisa dibilang tab yang mendukung Tab Home sehingga lembar kerja yang kita buat menjadi lebih menarik dan rapi.

3. Page layout

Page Layout berbeda dengan Insert, Insert mengacu pada isi dokumen, sedangkan Page Layout mengacu pada lembar itu sendiri sehingga bisa dibuat semenarik mungkin agar lembar tidak terkesan polos.

4. Referances

Tab Referensi ini berfungsi sebagai tambahan tulisan/catatan penting pada sebuah lembar kerja, misal untuk menambahkan Daftar Isi, Catatan Penting di setiap lembar kerja, dan masih banyak lagi fungsi yang masih bisa digunakan.

5. Mailings

Tab Mailings ini dapat digunakn sebagai Mail Merger dan kebutuhan surat-menyurat lainnya sehingga akan lebih efisien jika menggunakan Mailings untuk membuat surat.

6. Review

Tab review ini sangat berguna untuk mereviewkan kembali seperti melihat ulang untuk mencari kesalahan penggunaan tata bahasa, memberikan informasi mengenai jumlah kata atau bahkan bisa lagi untuk memproteksi dokumen Anda.

7. View

Tab ini memberikan bantuan tampilan pada lembar kerja Microsoft Office Word, seperti perbesaran lembar kerja, memberikan tampilan buku, dan masih banyak lagi yang bisa digunakan untuk mencoba dokumen kita dalam berbagai bentuk tampilan.

24. apa fungsi tab design


Telah terdapat fungsi dari tab design yaitu ialah untuk dapat mengatur format ataupun model design dari lembar kerja yang telah diciptakan.

Pembahasan

Tab design ini dapat digunakan untuk memberikan sebuah tema halaman, pewarnaan, format dokumen, dan lain-lain. Namun, Tab ini umumnya telah jarang sekali dignakan oleh para pengguna dari program pengolah kata tersebut.
