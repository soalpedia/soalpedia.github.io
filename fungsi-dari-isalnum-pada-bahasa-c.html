19. fungsi operator Increment Dan Decrement, dalam bahasa c++....tolong!!!!


Increment adalah operator yang digunakan untuk menambah nilai sebuah variable dengan 1 angka dan decrement digunakan untuk mengurangi nilai sebuah variable 1 angka. Contoh penggunaan increment dan decrement dalam C++:


int a = 10;

++a;

printf("%d\n",a); //Hasilnya menjadi 11

printf("%d\n",a++); //Hasilnya menjadi 11 karena nilai diprint terlebih dahulu lalu  ditambah 1 (Post-increment)

printf("%d\n",a); //Hasilnya menjadi 12 setelah diincrement

a--;

printf("%d",a); //Hasilnya menjadi 11

28. Fungsi-fungsi yang sama dengan perulangan for else pada Bahasa C, atau Bahasa Arduino UNO R3


Jawaban:

Pada dasarnya, kondisi IF ELSE merupakan modifikasi tambahan dari kondisi IF yang sudah kita pelajari sebelumnya.

Blok kode program IF tetap akan dijalankan ketika kondisi true (1), namun sekarang terdapat tambahan bagian ELSE akan dijalankan ketika kondisi false (0).

Berikut format dasarnya:

if (condition)

{

 //Kode program yang akan dijalankan jika condition berisi nilai True (1)

}

else

{

 //Kode program yang akan dijalankan jika condition berisi nilai False (0)

}

Bagian condition berperan sebagai penentu dari struktur percabangan ini. Jika condition terpenuhi (menghasilkan nilai TRUE atau 1), blok kode program milik IF akan dijalankan. Jika condition tidak terpenuhi (menghasilkan nilai FALSE atau 0), blok kode program bagian ELSE-lah yang akan diproses.

32. Apa fungsi simbol { dan ; dalam bahasa C++ ?


Penjelasan:

penyelesaian :

simbol { berguna untuk membuka suatu perintah/syntax yg akan dijalankan

simbol ; berguna untuk mengakhiri suatu perintah/syntax

Jawaban:

Fungsi simbol {} dalam bahasa pemrograman C++ adalah tanda dimulainya/diakhirinya eksekusi dari suatu program. Tanda ini disebut perintah /blok. Tanda { menandakan awal dari eksekusi program dan tanda } menandakan akhir dari eksekusi suatu program. Didalam tanda {} terdapat suatu perrintah untuk komputer agar menjalankan suatu program. Perintah ini disebut "statement". Contohnya adalah:

- Memerintah komputer untuk menampilkan string (gambar) ke layar

- Menghitung operasi matematika

- Membaca data dari keyboard, dll

Fungsi simbol ; (semicolon) dalam bahasa pemrograman C++ adalah menandakan akhir dari sebuah instruksi dalam sebuah script C++

Contoh:

Perhatikan script berikut ini:

<do - perintah>

: : = ' do ' <perintah> ' while ' ' (  ' <ekspresi> ' ) ' ' ; '

<goto - perintah>

: : = ' goto ' < label > ' ; '

<for - perintah>

: : = ' for ' ' ( ' < for - initialisasi > ' ; ' < for - kontrol > ' ; ' < for - iteration > ' ) '

< perintah >

< ekspresi - perintah >

: : = < ekspresi > ' ; '

< kembali - perintah >

: : = ' kembali ' < ekspresi > ' ; '

Perintah = variabel perintah

ekspresi = variabel ekspresi

initialisasi = variabel initialisasi

kontrol = variabel kontrol

do = perintah lakukan

kembali = perintah kembali

goto = perintah berpindah tempat ke < perintah >

38. fungsi dari bahasa programer C# dan C++? jelaskan!


Fungsi dalam artian penggunaan

Fungsi utama dari bahasa C++ dan C# adalah untuk mengembangkan software sistem, desktop application, dan video game. Namun ketika C++ dapat digunakan untuk semua sistem operasi, C# dibangun untuk mengembangkan software untuk sistem operasi Windows dan platform Microsoft.


Fungsi dalam artian logic in application

Fungsi adalah sebuah bagian prosedural yang berjalan secara rutin. Fungsi dapat mengembalikan nilai atau tidak. Dalam bahasa C, C# dan C++ secara structured programming sintaks yang digunakan sama namun ketika berbicara object-oriented programming language akan berbeda. Berikut contoh sintaks fungsi dari C, C++, dan C#.

//Structured Programming C dan C++

void HelloWorld()

{

   printf("Ini Function");

}

//C#

public void HelloWorld()

{

   Console.WriteLine("Ini Function");

}


//Object-Oriented Programming, C belum support OOP

//C++

Class Kom

{

   public:

        void HelloWorld();

}

void HelloWorld()

{

   printf("Ini Function");

}


//C#

public Class Kom

{

     public void HelloWorld()

     {

               Console.WriteLine("Ini Function");

     }

}
