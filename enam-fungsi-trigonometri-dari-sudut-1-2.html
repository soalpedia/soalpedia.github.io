sin x = 3/√13

    (2/3)² + 1 = cosec² x
      4/9   + 1 = cosec² x
            13/9 = cosec² x
      cosec x = √(13/9)
      cosec x = (1/3)√13

b) cosec x = 1/sin x
    (√13) /3 = 1/sin x
      sin x     = 3/√13
      sin x     = (3/13)√13
 
c)             sin² x + cos² x = 1
    {(3/13)√13)² + cos² x = 1
                 9/13 + cos² x = 1
                             cos² x = 1 - 9/13
                             cos² x = 4/13
                              cos x = √(4/13)
                              cos x = (2/13)√13
d) sec x = 1/cos x
               = 1/ (2/√13)

19. Tentukan nilai keenam perbandingan trigonometri sudut a pada setiap segitiga siku-siku berikut!​


Penjelasan dengan langkah-langkah:

Untuk menentukan nilai keenam perbandingan trigonometri sudut a pada setiap segitiga siku-siku, kita perlu menggunakan tiga perbandingan trigonometri dasar, yaitu sin, cos, dan tan. Nilai keenam perbandingan trigonometri tersebut adalah:

1. Sin(a) = Perbandingan antara panjang sisi miring yang berseberangan dengan sudut a dibagi panjang sisi miring terpanjang (hipotenusa).

2. Cos(a) = Perbandingan antara panjang sisi yang berdekatan dengan sudut a dibagi panjang sisi miring terpanjang (hipotenusa).

3. Tan(a) = Perbandingan antara panjang sisi yang berseberangan dengan sudut a dibagi panjang sisi yang berdekatan dengan sudut a.

Berikut adalah perbandingan trigonometri sudut a pada setiap segitiga siku-siku:

1. Segitiga Siku-siku Biasa:

- Sin(a) = Panjang sisi tegak / Panjang sisi miring terpanjang

- Cos(a) = Panjang sisi alas / Panjang sisi miring terpanjang

- Tan(a) = Panjang sisi tegak / Panjang sisi alas

2. Segitiga Siku-siku dengan Sudut 30-60-90:

- Sin(a) = 1/2

- Cos(a) = √3/2

- Tan(a) = 1/√3 atau √3/3

3. Segitiga Siku-siku dengan Sudut 45-45-90:

- Sin(a) = 1/√2 atau √2/2

- Cos(a) = 1/√2 atau √2/2

- Tan(a) = 1

Dalam setiap segitiga siku-siku, nilai perbandingan trigonometri tersebut akan tetap konsisten untuk sudut a yang sesuai.

Perlu diingat bahwa nilai-nilai ini berlaku untuk sudut a dalam radian. Jika ingin mengkonversi ke derajat, kita dapat menggunakan rumus konversi sudut.
