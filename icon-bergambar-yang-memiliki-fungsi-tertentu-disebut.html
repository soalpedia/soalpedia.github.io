8. Icon icon yg berada di pojok kanan atas jendela memiliki fungsi?? @ms.word


Jwaban :

Oke tidak perlu panjang lebar lagi, kita akan mulai saja pembahasan yang saya rasa akan cukup panjang ini, dimulai dari kita mengenal menu bar pada microsoft word. Perlu diketahui, karena Office yang saya miliki masih 2007 jadi kita akan gunakan software tersebut.

1. Home

Menu "Home" adalah toolbar menu yang berada pada ujung sisi kiri. Menu ini terdiri dari beberapa kelompok icon yang memiliki fungsi cukup penting. Oleh karena itu diletakan pada bagian pertama.

￼

Pada gambar tersebut kita bisa melihat ada lima kategori menu yang ada pada "Home". Mari kita pelajari satu - satu.

1# - Clipboard

￼

Pada bagian "Clipboard" terdapat beberapa Icon menu sebagai berikut.

Copy - Menu ini berfungsi untuk duplikasi atau mengcopy tilisan dari satu tempat

Cut - Sama seperti "Copy" hanya saja file yang kita "cut" akan hilang.

Paste - Menu paste berfungsi untuk meletakkan gambar setelah kita melakukan "Cut" atau "Copy"

Format Painter - Berfungsi untuk mengubah "painter"

2# - Font

￼

Menu "Font" ini berfungsi untuk mengatur format huruf yang digunakan seperti jenis huruf yang digunakan, jenis cetak (Bold, Italic, Underline, coret,), merubah warna huruf dan warna latar huruf dan beberapa format huruf lainnya.

3# - Paragraph

￼

Menu pada "Home" microsoft word selanjutnya adalah "Paragraph". Menu ini berisikan icon - icon yang berfungsi untuk mengatur sebuah paragraf yang sudah kita tulis pada microsoft word. Menu apa saja yang terdapat di sini, dan apa fungsinya

Align Text - Pengaturan format paragraf rata kanan, rata kiri, center, dan rata kanan kiri

Line Spacing - Menu ini berfungsi untuk mengatur jarak atau spasi baik antar paragraf, antar baris, dan antra sub judul dengan paragraf.

Shading - Berfungsi untuk memberikan shading pada tulisan

Border - Icon ini biasanya paling efektif pada saat mengatur tabel

Bullet & Numbering - Berfungsi untuk membuat opsi bullet atau opsi numbering

Baca Juga : Cara membuat bullet and numbering

4# - Styles

￼

Berikutnya adalah "Styles", berfungsi untuk mengatur styles huruf yang sudah tersedia template stylenya. Jadi anda tidak perlu mengatur secara manual tapi cukup untuk memilih salah satu style yang telah tersedia disana.

5# - Editing

￼

Menu "Home" selanjutnya yaitu "Editing" . Menu ini berfungsi untuk melakukan edit kata atau kalimat yang sudah kita tulis. Menu ini berisi beberapa fungsi yaitu :

Find - Find berfungsi untuk mencari kata atau kalimat agar cepat kita temukan

Replace - Menu ini berfungsi untuk mereplace atau merubah kalimat atau huruf secara otomatis pada lembar kerja microsoft word.

Select - Ini sla saja dengan "Blok" atau menandai, kata, tulisan atau objek lain. Sebab perintah edit pada microsoft word dapat dilakukan apabila kita sudah melakukan slect terhadap kata atau kalimat yang akan kita edit

Nah, lima macam opsi tersebut terdapat berada pada Toolbar menu "Home. Selanjutnya kita akan lanjutkan pembahasan fungsi menu dan ikon pada microsoft word 2007 beserta gambarnya ini dengan membahas menu selanjutnya.

2. Insert

Menu toolbar selanjutnya adalah "Insert". Fungsi toolbar ini seperti halnya namanya, yaitu untuk meng-insertkan berbagai macam tambahan seperti tabel, shape, garis, menyisipkan gambar, Header dan Footer dan lain sebagainya.

￼

Anda bisa lihat pada gambar menu "Insert" di atas. Ada banyak kelompok sub menu yang termasuk di dalam menu ini.

1# - Page

￼

Kelompok menu pertama pada menu "Insert" adalah "Page", fungsinya untuk insert cover page atau kertas cover, anda bisa menggunakan menu ini apabila anda anda ingin mendesain cover untuk tulisan yang anda buat. Kemudian insert blank page, yaitu untuk membuka lembar kerja baru (kosong). Sedangkan Page Break berfungsi untuk memberikan jeda dan jarak antar halaman.

2# - Table

￼

Anda masih ingat bagaimana cara membuat tabel pada microsoft word ? Inilah fungsi menu "table" yang ada di toolbar pada microsoft word. Yaitu berfungsi untuk membuat atau meng-insert tabel.

24. sebutkan fungsi fungsi dari tollbar icon


Line Spacing

Line Spacing berfungsi untuk mengatur jarak spasi pada baris. Jarak spasi saat kita membuka Microsoft Word pada umumnya adalah 1,15. Namun kita bisa merubahnya sesuai keinginan.

Shapes

Shapes berfungsi untuk menyisipkan berbagai pola seperti bangun datar, garis, panah, emoticon smile, dan lain sebagainya ke dalam kalimat.

Bold

Bold berfungsi untuk membuat teks menjadi cetak tebal.

Hyperlink

Icon ini berfungsi untuk membuat link ke suatu halaman web, dengan cara menandai teks yang ingin kita Hyperlink. Maka teks tersebut akan terhubung ke halaman web yang sudah kita pilih.

Cover Page

Cover Page ini berfungsi untuk membuat sebuah cover yang sudah tersedia di Microsoft Word. Kita hanya perlu memilih jenis cover yang sudah disediakan lalu memasukan kalimat sesuai keingingan kita.

Table

Table ini berfungsi untuk membuat table dengan banyak kolom dan baris sesuai keinginan.

Chart

Chart ini berfungsi untuk menyisipkan sebuah gambar dari grafik, bisa berupa Colom, Line, Bar, dan lain sebagainya.

Symbol

Symbol ini berfungsi untuk menyisipkan symbol seperti lambang Tak terhingga, Perkalian, Bembagian, Delta, OHM, dan lain sebagainya.

Clip Art

Icon ini berfungsi untuk menyisipkan gambar, video, rekaman suara, kedalam dokumen yang kita buat.

Paste berfungsi untuk menempatkan objek/teks yang ada di clipboard pada posisi titik sisip berada

Paste Special berfungsi untuk menempatkan objek/teks yang ada di clipboard pada posisi titik sisip berada dengan format yang berbeda

Paste As Hyperlink berfungsi untuk menempatkan teks sebagai hyperlink

d. Format Painter

Menyalin format pada Clipboard untuk diletakkan pada posisi yang diinginkan.

a. Fonts (CTRL+SHIFT+F)

Font berfungsi untuk memilih beberapa jenis atau bentuk font/huruf untuk digunakn pada teks.

b. Font Size (CTRL + SHIFT +T)

Font Size berfungsi untuk mengubah ukuran font/huruf pada teks.

c. Grow Font (CTRL +>)

Grow Font berfungsi untuk memperbesar font secara instan

d. Shrink Font (CTRL +<)

Shrink Font berfungsi untuk memperkecil font secara instan

e. Clear Formatting

Clear Formatting berfungsi untuk menghapus pengformatan teks terpilih

f. Bold (CTRL +B)

Bold berfungsi untuk mempertebal teks yang terpilih

g. Italic (CTRL +I)

Italic berfungsi untuk memirinkan teks yang terpilih

h. Underline (CTRL +U)

Underline berfungsi untuk memberikan garis bawah pada teks yang terpilih

i. Strikethrough

Strikethrough berfungsi untuk memberikan garis tengah (garis coret) pada teks yang terpilih

j. Subscript (CTRL +=)

Subscript berfungsi untuk mengetikan karakter pemangkatan (huruf kecil di bawah huruf sebelumnya)

k. Superscript (CTRL +SHIFT ++)

Superscript berfungsi untuk mengetikan karakter pemangkatan (huruf kecil di atas huruf sebelumnya

l. Change Case (SHIFT F3)

Change Case bersungsi untuk mengubah status huruf kapital/huruf kecil/besar kecil pada teks/kata yang terpilih.

m. Text Highlight Color

Text Highlight Color berfungsi untuk memberikan latar belakang warna pada text

n. Font Color

Font Color berfungsi untuk memberi warna teks yang terpilih

3. Paragraph

a. Bullets

Bullets berfungsi untuk memberikan tanda bullet di tiap paragraf terpilih

b. Numbering

Numbering berfungsi untuk memberikan/menghilangkan format penomoran pada paragraph yang terpilih

c. Multilevel List

Multilevel List berfungsi untuk membuat posisi penomoran berupa angka atau nomor urut

d. Decrease Indent

Decrease Indent berfungsi untuk menghapus atau menurunkan posisi paragraph ke kiri

e. Increase Indent

Increase Indent berfungsi untuk menambahkan atau memberi posisi paragraph ke kanan

f. Sort

Sort berfungsi untuk mengurutkan data pada sebuah tabel

g. Show/ Hide¶ (CTRL +*)

Show/ Hide ¶ berfungsi untuk menampilkan/ menyembunyikan koreksi paragraph

h. Align Text Left (CTRL+L)

Align Text Left berfungsi untuk mengatur perataan teks ke kiri.

i. Center (CTRL+E)

Center berfungsi untuk mengatur teks untuk rata tengah

j. Align Text Right (CTRL+R)

Align Text Right berfungsi untuk mengatur teks rata kanan

k. Justify (CTRL+J)

Justify berfungsi untuk mengatur perataan teks ke kanan

l. Line Spacing

Line Spacing berfungsi untuk mengatur jarak antara baris teks atau paragraph

m. Shading

Shading berfungsi untuk memberikan latar belakang warna pada teks yang terpilih

n. Buttom Border

Buttom Border berfungsi untuk meberikan garis tepi pada teks yang terpilih

4. Styles

a. Heading Style

Heading Style berfungsi untuk memberi style pada paragraph. Pada Heading Style secara otmoatis terdapat beberapa format style yang siap digunakan seperti : ¶ Normal, ¶ No Spacing, Heading 1, Title, Subtitle, Subtle Emphasis, Emphasis, dan lain-lain Sebagainya

b. Change Styles

Change Styles berfungsi untuk mengubah style paragraph yang sedang aktif

5. Editing

a. Find (CTRL +F)

Find berfungsi untuk mencari kata yang ada pada halaman tesebut

b. Replace (CTRL +H)

Replace berfungsi untuk menggati kata yang ada pada halaman dengan kata yang diinginkan

c. Select

Select berfungsi untuk menyeleksi teks, paragraph atau gambar pada halaman.
