7. Sebutkan dan jelaskan metode yang dapat digunakan untuk menentukan vektor resultan (besar dan arah) dari hasil penjumlahan vektor


Jawaban:

1. Metode Poligon

Penjumlahan vektor menggunakan cara poligon dilakukan dengan menggambarkan vektor-vektor yang garis nya digabungkan secara berurutan (diteruskan). Kemudian Vektor jumlahnya (Resultannya) digambarkan dengan menarik garis dari titik awal menuju titik akhir.

2. Metode Jajaran genjang

Penjumlahan vektor menggunakan metode jajaran genjang dibuat dengan cara menggambarkan garis-garis vektor yang akan digabungkan dari titik awal yang sama, kemudian buatlah garis yang sejajar dengan vektor ( dibuat dengan garis putus-putus) diambil dari kedua ujung vektor yang digabungkan sehingga didapat titik potong dan membentuk jajar genjang. Tahap terakhir gambarlah Vektor Jumlah (resultan) nya dengan menarik garis dari titik awal menuju titik potong garis yang dibuat putus-putus titik

9. besar dan arah vektor resultan tersebut adalah​


Besar vektor resultan adalah 10√2 N dan arah resultan searah F₂. Simak penjelasan berikut.

PembahasanMenentukan resultan vektor

1. Resultan 2 vektor yang membentuk sudut tertentu

F = √(F₁² + F₂² + F₁F₂ Cos α)

2. Resultan vektor komponen

* Vektor komponen-x

    Fx = F Cos α

* Vektor komponen-y

    Fy = F Sin α

* Resultan Vektor

    Fr = √(F₁² + F₂²)

* Arah vektor

    Tan α = Fy / FxDiketahui

F₁ = 10 N

F₂ = 8 N

F₃ = 10 N

F₄ = 8 N

Ditanya

Besar dan arah vektor resultan

Penyelesaian

Fx₁ = F₁ (-Cos 45°) = 10 (-1/2 √2) = -5√2

Fx₂ = F₂ Cos 90° = 8 (0) = 0

Fx₃ = F₃ Cos 45° = 10 (1/2 √2) = 5√2

Fx₄ = F₄ Cos 90° = 8 (-0) = 0

Vektor resultan dalam arah x, Fx

= Fx₁ + Fx₂ + Fx₃ + Fx₄

= -5√2 + 0 + 5√2 + 0

= 0

Fy₁ = F₁ Sin 45° = 10 (1/2 √2) = 5√2

Fy₂ = F₂ Sin 90° = 8 (1) = 8

Fy₃ = F₃ Sin 45° = 10 (1/2 √2) = 5√2

Fy₄ = F₄ (-Sin 90°) = 8 (-1) = -8

Vektor resultan dalam arah y, Fy

= Fy₁ + Fy₂ + Fy₃ + Fy₄

= 5√2 + 8 + 5√2 + -8

= 10√2

Vektor resultan, Fr

= √(Fx² + Fy²)

= √(0² + (10√2)²)

= √(10√2)²

= 10√2 N

Arah vektor resultan

Tan α

= Fy / Fx

= 10√2 / 0

= Tak terdefinisi

α = 90°

Arah vektor searah dengan F₂.

Kesimpulan

Jadi, besar vektor resultan adalah 10√2 N dan arah resultan searah F₂.

14. Bagaimana cara menggunakan rumus kosinus dalam menentukan besar vektor resultan


Untuk hal tersebut dapat menggunakan rumus :

[tex]R = \sqrt{ a^{2} + b^{2} + 2. a.b. cos \alpha } [/tex]

Sedangkan untuk selisih vektor menggunakan rumus :

[tex]R = \sqrt{ a^{2} + b^{2} - 2. a.b. cos \alpha } [/tex]
untuk mencari resultan vektor pada kosinus  , bisa di gunakan dengan cara : 

R = √x² + y² + 2xy. cos α

17. |M|=800n besar vektor resultan adalah 50n kearah selatan sedang arah vektor N ke barat.cari besar dan arah vektor N?​


Jawaban:

Diketahui |M| = 800n dan vektor resultan (R) sebesar 50n ke arah selatan. Arah vektor N ke barat, sehingga sudut antara vektor M dan N adalah 90 derajat. Kita dapat menggunakan hukum segitiga untuk mencari besar dan arah vektor N.

Dari hukum segitiga pada segitiga vektor M, R, dan N, kita dapatkan:

|R|^2 = |M|^2 + |N|^2 - 2|M||N|cosθ

Diketahui |M| = 800n, |R| = 50n, dan θ = 90 derajat, sehingga:

(50n)^2 = (800n)^2 + |N|^2 - 2(800n)|N|cos90

2500n^2 = 640000n^2 + |N|^2 - 0

|N|^2 = 640000n^2 - 2500n^2

|N|^2 = 637500n^2

|N| = √(637500n^2)

|N| = 750n

Sehingga besar vektor N adalah 750n.

Untuk mencari arah vektor N, kita dapat menggunakan trigonometri. Sudut antara vektor M dan N adalah 90 derajat, sehingga sudut antara vektor N dan sumbu x adalah 90 - arctan(800/50) = 74.5 derajat ke arah barat. Sehingga arah vektor N adalah 74.5 derajat ke arah barat.
