6. Misal pelanggan yang berbelanja di Pakuwan Mall sebanyak 10000 konsumen dan tentukan jumlah sampelnya berdasarkan rumus Slovin, coba hitung berapa jumlah sampelnya


Jawaban:

Rumus Slovin digunakan untuk menentukan jumlah sampel yang representatif dari populasi yang besar. Rumus tersebut dinyatakan sebagai berikut:

n = N / (1 + N * e^2)

Di mana:

n = Jumlah sampel

N = Jumlah populasi

e = Tingkat kesalahan (margin of error)

Dalam kasus ini, jumlah populasi adalah 10.000 konsumen (N). Untuk menentukan jumlah sampel, kita perlu menentukan tingkat kesalahan yang diizinkan (e). Misalkan kita anggap tingkat kesalahan yang diizinkan adalah 5% atau 0,05.

n = 10.000 / (1 + 10.000 * 0,05^2)

n = 10.000 / (1 + 10.000 * 0,0025)

n = 10.000 / (1 + 25)

n = 10.000 / 26

n ≈ 384

Jadi, jumlah sampel yang dibutuhkan berdasarkan rumus Slovin adalah sekitar 384 konsumen.

8. Penelitian yang akan mengukur kinerja karyawan sebuah organisasi memiliki populasi berupa jumlah karyawan sebanyak 175 orang. Jika peneliti menetapkan sampling error sebesar 10 %, maka jumlah sampel minimal dengan menggunakan rumus slovin sebesar...


Jawaban:

Rumus Slovin digunakan untuk menghitung ukuran sampel minimal dalam survei atau penelitian dengan menggunakan teknik sampling acak sederhana. Rumusnya adalah:

n = N / (1 + N * e^2)

Di mana:

n adalah ukuran sampel minimal yang diinginkan,

N adalah ukuran populasi, dan

e adalah tingkat kesalahan sampling (dalam desimal).

Dalam kasus ini, jumlah karyawan dalam populasi adalah 175 orang, dan peneliti menetapkan tingkat kesalahan sampling sebesar 10%, yang dalam desimal adalah 0.1. Substitusikan nilai-nilai ini ke dalam rumus:

n = 175 / (1 + 175 * 0.1^2)

n = 175 / (1 + 175 * 0.01)

n = 175 / (1 + 1.75)

n = 175 / 2.75

n ≈ 63.64

Jumlah sampel minimal yang dibutuhkan dengan menggunakan rumus Slovin adalah sekitar 63.64. Karena ukuran sampel haruslah bilangan bulat, peneliti harus membulatkannya ke atas menjadi 64. Oleh karena itu, jumlah sampel minimal yang diperlukan adalah 64 orang.

9. tentukan banyak sampel yang harus di ambil menggunakan rumus slovin dan tabel kricje jika. banyaknya populasi sebagai berikut : a).300, b)500, c) 2000 d)7000. e). 10000​


Penjelasan dengan langkah-langkah:

Rumus Slovin digunakan untuk menentukan jumlah sampel yang harus diambil dari populasi untuk mendapatkan tingkat kesalahan yang dapat diterima. Berikut adalah langkah-langkah untuk menggunakan rumus Slovin:

1. Tentukan tingkat kesalahan yang dapat diterima (e). Misalnya, e = 0,05 (5%).

2. Tentukan tingkat kepercayaan yang diinginkan (C). Misalnya, C = 0,95 (95%).

3. Tentukan jumlah populasi (N) yang akan diambil sampel.

Rumus Slovin:

n = N / (1 + N(e^2))

Untuk menggunakan Tabel Kricje, kita perlu menentukan level of precision (tingkat ketelitian) dan confidence level (tingkat kepercayaan) yang diinginkan. Tabel ini memberikan jumlah sampel minimum yang harus diambil berdasarkan ukuran populasi.

Berikut adalah hasil perhitungan jumlah sampel menggunakan rumus Slovin dan Tabel Kricje untuk setiap ukuran populasi:

a. Populasi = 300

Menggunakan rumus Slovin:

n = 300 / (1 + 300(0,05^2))

= 300 / (1 + 300(0,0025))

= 300 / (1 + 0,75)

= 300 / 1,75

≈ 171

Menggunakan Tabel Kricje:

Dengan level of precision sebesar 0,05 dan confidence level sebesar 95%, dengan populasi 300, kita dapat mengambil minimal 169 sampel.

b. Populasi = 500

Menggunakan rumus Slovin:

n = 500 / (1 + 500(0,05^2))

= 500 / (1 + 500(0,0025))

= 500 / (1 + 1,25)

= 500 / 2,25

≈ 222

Menggunakan Tabel Kricje:

Dengan level of precision sebesar 0,05 dan confidence level sebesar 95%, dengan populasi 500, kita dapat mengambil minimal 217 sampel.

c. Populasi = 2000

Menggunakan rumus Slovin:

n = 2000 / (1 + 2000(0,05^2))

= 2000 / (1 + 2000(0,0025))

= 2000 / (1 + 5)

= 2000 / 6

≈ 333

Menggunakan Tabel Kricje:

Dengan level of precision sebesar 0,05 dan confidence level sebesar 95%, dengan populasi 2000, kita dapat mengambil minimal 323 sampel.

d. Populasi = 7000

Menggunakan rumus Slovin:

n = 7000 / (1 + 7000(0,05^2))

= 7000 / (1 + 7000(0,0025))

= 7000 / (1 + 17,5)

= 7000 / 18,5

≈ 378

Jawab:

Untuk menentukan ukuran sampel minimal dengan menggunakan rumus Slovin, Anda dapat menggunakan rumus berikut:

n = (N x Z² x P x Q) : ((N - 1) x e² + Z² x P x Q)

di mana:

n adalah ukuran sampel minimal yang diperlukan

N adalah jumlah populasi

Z adalah tingkat kepercayaan yang diinginkan, dinyatakan dalam tingkat deviasi standar

P adalah proporsi populasi yang memenuhi kriteria yang Anda inginkan untuk sampel

Q adalah komplementer dari P, yaitu 1 - P

e adalah tingkat ketelitian yang diinginkan, dinyatakan sebagai proporsi dari populasi

Dengan menggunakan nilai Z sebesar 2 dan P sebesar 0.5, maka ukuran sampel minimal yang sebaiknya menggunakan metode non-probabilitas sampling dengan menggunakan rumus Slovin adalah sebagai berikut:

n = (5000 * 2^2 * 0.5 * 0.5) / ((5000 - 1) * 0.1^2 + 2^2 * 0.5 * 0.5)

= (5000 * 4 * 0.25) / ((4999 * 0.01) + 4 * 0.25)

= (20000 * 0.25) / (49.99 + 1)

= 5000 / 199.99

= approximately 25
