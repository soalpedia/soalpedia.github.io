12. 15 repetisi 3 set artinya​


repetisi sendiri dapat kita maknai sebagai suatu gerakan pengulangan pada gerakan yang sama tanpa melakukan istirahat atau memberikan jeda. salah satu contohnya ialah ketika kita melakukan olahraga sit up sebanyak 10 kali tanpa ada jeda maka itu dapat di maknai kita sedang melakukan repetisi sit up 10 kali. sedangkan set sendiri dapat kita maknai sebagai pengulangan dari repetisi, yang mana salah satu contohnya ialah ketika kita melakukan sit up sebanyak 10 kali lalu kita melakukan istirahat maka hal tersebut bisa di artikan 1 set dengan 10 repetisi.

maka untuk pertanyaan 15 repetisi 3 set dapat kita artikan sebagai 15 kali melakukan pengulangan gerakan sit up dan setelah itu memberikan jeda atau waktu istirahat lalu setelah itu di lakukan lagi 15 kali pengulangan gerakan sit up dan setelah itu memberikan jeda atau waktu istirahat dan setelah itu baru melakukan 15 kali pengulangan sit up lagi. dapat disimpulkan jumlah gerakan sit up pada 15 repitisi 3 set ialah 45 gerakan sit up.

26. Apa fungsi dari Set all four border


Fungsi dari set all four border adalah untuk memberikan garis tepi atau bingkai pada keseluruhan sisi paragraf. Fitur set all four border merupakan salah satu fitur yang terdapat pada aplikasi Open Office Writer. Open Office Writer sendiri merupakan aplikasi yang digunakan untuk membuat dokumen, aplikasi ini merupakan aplikasi padanan dari Microsoft Word dengan versi open source.

Pembahasan

Selain pilihan set all four border, pada Open Office Writer juga terdapat pilihan lainnya dalam pengaturan garis bingkai untuk paragraf. Pilihan lain yang tersedia adalah

No BorderSet Left and Right Borders OnlySet Top and Bottom Borders OnlySet Left Border Only

Pemberian garis pada paragraf bermaksud untuk menonjolkan paragraf tersebut atau untuk membuat dokumen agar terlihat semakin menarik untuk dibaca.

35. Jelaskan fungsi dari set instruksi secara umum


Keterangan
Mata Pelajaran          : Teknologi Informatika (T.I)

Pendidikan Jenjang    : Sekolah Menengah Pertama

Point yang diberikan  : 5+3 Point
Source Point               : “Set Instruksi Komputer”

 

[Question]

Jelaskan fungsi dari set instruksi secara umum


[Answer]

Berhubungan, tidak ada yang jawab pertanyaan anda, terlebih dahulu saya mencari apa itu “Set Instruksi”, yang saya dapatkan adalah

Set Instruksi

↪ sekumpulan lengkap instruksi yang dapat dimengerti oleh sebuah CPU dengan sebuah kamus berisi daftar perintah apa saja yang dapat dilakukan (didukung) oleh sebuah prosesor, dan biasanya terikat dengan sebuah keluarga arsitektur prosesor tertentu (misal x86, x64).

 

Kini saya ingin menyimpulkan kesimpulan saya tentang Set Instruksi Komputer

↪ sebuah perintah-perintah yang dikerjakan melalui brainware melalui hardware dan dilakukan pada software. Sehingga perintah-perintah tersebut mampu berjalan dengan baik. Tanpa adanya kesalahan.

 

Note

1.    Brainware  : orang yang menjalankan suatu perintah pada komputer

2.    Hardware   : perangkat keras yang terdapat diluar computer

↪ Contoh : Keyboard, Mouse, dll. *Mousepad tidak termasuk (setau saya)

3.    Software    : Perangkat lunak yang terdapat di dalam computer, yang telah dibuat oleh programmer-programmer.

↪ Contoh : Ms. Office, Mozilla Firefox, dll.
