10. 1. Diketahui suatu interpretasi I atas (over) domain D = himpunan semua bilangan bulat, dengan: I ={a ← 0, b ← 1, c ← 2, y ← 3, z ← 4, f ← fI(d1, d2) = 3d2 - d1, gI(d1, d2 , d3) = 2d1 + d3 – d2, p ← pI(d1, d2, d3): d1 > 2d2+ d3, q ← qI(d1, d2): d1 < d2} Lakukan perluasan terhadap interpretasi I dengan memberi nilai baru terhadap simbol predikat p


Perluasan terhadap interpretasi I dengan memberi nilai baru terhadap simbol predikat p pada I = {a ← 0, b ← 1, c ← 2, y ← 3, z ← 4, f ← fI(d1, d2) = 3d2 - d1, gI(d1, d2 , d3) = 2d1 + d3 – d2, p ← pI(d1, d2, d3): d1 > 2d2+ d3, q ← qI(d1, d2): d1 < d2} yaitu dengan memberikan nilai kebenaran (Benar atau Salah) pada predikat tersebut berdasarkan kondisi atau proposisi tertentu.

Pembahasan

Untuk memperluas interpretasi I dengan memberikan nilai baru pada simbol predikat p, kita dapat memberikan nilai kebenaran (Benar atau Salah) pada predikat tersebut berdasarkan kondisi atau proposisi tertentu.

Contohnya kita ingin memberi nilai baru pada predikat p. Kita dapat menetapkan kondisi tertentu yang menentukan apakah predikat itu benar atau salah. Sebagai contoh, kita dapat menetapkan p bernilai benar ketika d1 kurang dari atau sama dengan jumlah d2 dan d3, dan salah jika sebaliknya.

Jadi, interpretasi I yang diperbarui dengan nilai baru untuk predikat p adalah:

I = {a ← 0, b ← 1, c ← 2, y ← 3, z ← 4, f ← fI(d1, d2) = 3d2 - d1, g ← gI(d1, d2 , d3) = 2d1 + d3 – d2, p ← (d1 ≤ d2 + d3), q ← qI(d1, d2): d1 < d2}

Dalam interpretasi yang diperbarui ini, predikat p akan benar jika d1 kurang dari atau sama dengan jumlah d2 dan d3, dan salah jika sebaliknya.
