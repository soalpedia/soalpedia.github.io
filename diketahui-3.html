16. Diketahui persamaan 2^3x-3=64 nilai x yang diketahui adalah ?


Jawaban:

3

Penjelasan dengan langkah-langkah:

[tex]2 {}^{3x - 3} = 64 \\ 2 {}^{3x - 3} = 2 {}^{6} \\ 3x - 3 = 6 \\ 3x = 6 + 3 \\ 3x = 9 \\ x = \frac{9}{3} \\ x = 3[/tex]

Nilai x yang diketahui adalah x = 3.

Langkah Penyelesaian

[tex] {2}^{3x - 3} = 64[/tex]

[tex] {2}^{3x - 3} = {2}^{6} [/tex]

Karena angka yg dipangkatkan sudah sama, kita fokuskan ke pangkatnya sebagai persamaan.

[tex]3x - 3 = 6[/tex]

[tex]3x = 6 + 3[/tex]

[tex]3x = 9[/tex]

[tex]x = 9 \div 3[/tex]

[tex]x = 3[/tex]

Kesimpulan

Jadi, nilai x yang diketahui dari persamaan berikut adalah x = 3

18. Tentukan dx/dy1.diketahui : y = (x²-3)²2.diketahui : y = (x²-3)²³


Turunan pertama fungsi aljabar

Untuk fungsi y = (x² - 3)² adalah y' = 4x (x² - 3). Turunan pertama dapat dituliskan juga sebagai y' = 4x³ - 12x Untuk fungsi y = (x² - 3)²³ adalah y' = 46x (x² - 33)²²

Penjelasan dengan langkah-langkah:

Diketahui:

y = (x² - 3)² y = (x² - 3)²³

Ditanyakan:

[tex]\frac{dy}{dx}[/tex] ?

Jawaban:

Turunan pertama dari fungsi y atau [tex]\frac{dy}{dx}[/tex] untuk fungsi berpangkat y = uⁿ menggunakan aturan rantai  

y' = uⁿ⁻¹ × u'

dimana u' adalah turunan dari u.

1. y = (x² - 3)²

u = x² - 3
u' = [tex]2x^{2 \:-\: 1} \:-\: 0[/tex]
u' = 2x
[tex]\frac{du}{dx} \:=\: 2x[/tex] y = u²  
y' = [tex]2 u^{2 \:-\: 1}[/tex]
[tex]\frac{dy}{du} \:=\: 2u[/tex]

[tex]\frac{dy}{dx} \:=\: \frac{dy}{du} \: \frac{du}{dx}[/tex]

= [tex]2u \: 2x[/tex]

= [tex]2 \: (x^2 \:-\: 3) \: 2x[/tex]

= 4x (x² - 3)

atau

= 4x³ - 12x

2. y = (x² - 3)²³

u = x² - 3
u' = [tex]2x^{2 \:-\: 1} \:-\: 0[/tex]
u' = 2x
[tex]\frac{du}{dx} \:=\: 2x[/tex] y = u²³
y' = [tex]23 \: u^{23 \:-\: 1}[/tex]
y' = [tex]23 \: u^{22}[/tex]
[tex]\frac{dy}{du} \:=\: 23u^{22}[/tex]

[tex]\frac{dy}{dx} \:=\: \frac{dy}{du} \: \frac{du}{dx}[/tex]

= [tex]23u^{22} \: 2x[/tex]

= [tex]23 \: (x^2 \:-\: 3)^{22} \: 2x[/tex]

= 46x (x² - 33)²²

33. Diketahui |2-×| + |3-2×| <3


Jawaban, {x | 2/3 < x < 8/3, x ∈ R}

Pertama gunakan definisi nilai mutlak,
Untuk nilai |2 - x|
Jika 2 - x ≥ 0 atau x ≤ 2 maka |2 - x| = 2 - x
Jika 2 - x < 0 atau x > 2maka |2 - x| = -(2 - x)

Untuk nilai |3 - 2x|
Jika 3 - 2x ≥ 0 atau x ≤ 3/2 maka |3 - 2x| = 3 - 2x
Jika 3 - 2x < 0 atau x > 3/2 maka |3 - 2x| = -(3 - 2x)

Gabungkan semua kemungkinan syaratnya, dan pelajari kasus per kasus,

Kasus 1, x ≤ 2 dan x ≤ 3/2 => x ≤ 3/2
|2 - x| + |3 - 2x| < 3
2 - x + 3 - 2x < 3
5 - 3x < 3
-3x < -2
x > 2/3
Sehingga solusinya, 2/3 < x ≤ 3/2

Kasus 2, x ≤ 2 dan x > 3/2 => 3/2 < x ≤ 2
|2 - x| + |3 - 2x| < 3
2 - x - (3 - 2x) < 3
2 - x - 3 + 2x < 3
-1 + x < 3
x < 4
Sehingga solusinya, 3/2 < x ≤ 2

Kasus 3, x > 2 dan x > 3/2 => x > 2
|2 - x| + |3 - 2x| < 3
-(2 - x) - (3 - 2x) < 3
-2 + x - 3 + 2x < 3
-5 + 3x < 3
3x < 8
x < 8/3
Sehingga solusinya, 2 < x < 8/3

Gabungan dari ketiga solusi yang didapat dari tiap kasus yaitu 2/3 < x < 8/3. Jadi himpunan penyelesaian dari ketaksaman |2 - x| + |3 - 2x| < 3 adalah {x | 2/3 < x < 8/3}

Catatan, kasus x > 2 dan x ≤ 3/2 tidak kita masukkan karena tidak mungkin terjadi.
