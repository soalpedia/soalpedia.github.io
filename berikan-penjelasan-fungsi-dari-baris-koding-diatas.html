1. Terdapat Tag HTML

a. Tag ditandai dengan “<” dan “>”.

b. Dalam penulisan kode HTML harus diawali dengan <tag_pembuka> dan diakhiri dengan

</tag_penutup>.

c. Tag HTML tidak ‘case sensitive’, <b> sama dengan <B>.

d. Jika dalam suatu ag ada tag lagi maka penulisan tag akhir harus berurutan, misal <b><i> ...

e. Huruf Tebal Miring </i></b>.

2. Terdapat ELEMENT HTML, yang terdiri dari 2 bagian :

a. HEAD merupakan kepala dokumen dari HTML, yang berfungsi untuk memberikan informasi tentang dokumen.

b. BODY merupakan tubuh dari HTML, yang berfungsi untuk menentukan bagaimana isi suatu dokumen yang akan ditampilkan pada web browser.

13. Blok program yang berfungsi sebagi badan sebuah alur kodingan dilambangkan dengan...? a. # b. {} c. <> d. *


Jawaban : b.{}

Pembahasan

Penjelasan singkat

Dalam beberapa bahasa pemrograman seperti HTML, simbol yang dipakai untuk sebuah alur kodingan adalah {} Biasanya setelah suatu fungsi, akan diawali oleh { dahulu, lalu setelahnya diakhiri dengan }. Jika pembukaan suatu koding tidak menggunakan {} maka akan terjadi error dan pemrograman tidak akan dapat dimulai. Namun, dalam beberapa bahasa pemrograman, ada pula yang menggunakan () dari pada {} sebagai badan sebuah alur

Gambar

Gambar foto merupakan contoh koding bahasa php yang menggunakan {} (lihat kotak untuk simbol {} pada satu jalur koding)

class - fungsi (bisa dilihat bahwa sesudah fungsi, untuk membuka suatu alur koding menggunakan { dan di bagian akhir diakhiri dengan })

Penjelasan simbol pada soal

# — Dalam program bahasa python, simbol ini digunakan untuk menambahkan kalimat dalam pemrograman yang nantinya tidak akan diproses dalam program (kalimat ditandai dengan warna hijau)<> — Simbol < dan > digunakan untuk suatu permisalan jika integrer lebih besar/kecil dari variabel yang ditulis. Atau dalam beberapa bahasa pemrograman, <> digunakan sebagai "tidak sebanding dengan" (seperti contoh If x <> 6)* — Simbol yang digunakan untuk perkalian dalam hampir semua bahasa pemrograman.

24. tokoh tokoh dibalik komputer dan koding


Tokoh-tokoh dibalik komputer dan koding adalah sebagai berikut

- John Von Neumann. Mengembangkan 2 konsep penting yang mempengaruhi secara langsung jalur dari bahasa pemrograman komputer, yaitu share-program technique dan conditional control transfer

- Grace Hopper. Pembuat kompiler pertama. Kompiler ini mengubah pernyataan menjadi 0 dan 1 yang dapat dimengerti oleh komputer.

- IBM. Pembuat bahasa pemrograman FORTRAN ( FORmula TRANslating ).

- John McCarthy. Penemu bahasa LISP ( List Processing ).  

- Niklaus Wirth. Penemu bahasa pemrograman Pascal

- Dennis Ritchie. Penemu bahasa pemrograman C

- Bjarne Stroustroup. Penemu ekstensi pada bahasa pemrograman C

- Sun Microsystem. Penemu Java

- John Kemeny dan Thomas Kurtz. Penemu bahasa pemrograman BASIC

- Larry Wall. Penemu Perl

33. minta tolong kodingannya sih :(​


Penjelasan:

Berikut ini adalah salah satu implementasi dalam Python untuk mencetak transpose dari matriks:

```python

def transpose_matrix(matrix):

# Mengambil jumlah baris dan kolom dari matriks

rows = len(matrix)

cols = len(matrix[0])

# Membuat matriks kosong dengan ukuran kolom x baris

transposed_matrix = [[0] * rows for _ in range(cols)]

# Mengisi matriks transpose dengan nilai dari matriks awal

for i in range(rows):

for j in range(cols):

transposed_matrix[j][i] = matrix[i][j]

return transposed_matrix

# Membaca nilai n

n = int(input())

# Membaca matriks

matrix = []

for _ in range(n):

row = list(map(int, input().split()))

matrix.append(row)

# Mencetak matriks transpose

transposed_matrix = transpose_matrix(matrix)

for row in transposed_matrix:

print(*row)

```

Kode di atas membaca nilai n yang menunjukkan jumlah baris dan kolom dari matriks. Kemudian, matriks dibaca sebagai input dan disimpan dalam variabel `matrix`. Fungsi `transpose_matrix` digunakan untuk menghitung transpose dari matriks, dan matriks transpose dicetak satu per satu.

Misalnya, jika inputnya adalah:

```

3

5 4 5 6

8 6 5 7

8 7 1 4

```

Outputnya akan menjadi:

```

5 8 8

4 6 7

5 5 1

6 7 4

```
