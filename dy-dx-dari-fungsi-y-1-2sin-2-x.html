Mari selesaikan kedua persamaan diferensial ini:

1. x dy/dx = y + xy

Pertama, kita pisahkan variabel x dan y:

x dy = (y + xy) dx

Kemudian, bagi kedua sisi dengan xy untuk memisahkan variabel:

(1/y) dy = (1/x + 1) dx

Selanjutnya, kita integrasikan kedua sisi:

∫(1/y) dy = ∫(1/x + 1) dx

ln|y| = ln|x| + x + C₁

Kemudian, kita bisa menghilangkan ln dengan mengambil eksponen:

|y| = |x|e^(x + C₁)

Kita bisa mengecualikan nilai mutlak karena kita akan mendapatkan dua solusi, satu untuk y positif dan satu untuk y negatif:

y = ± x * e^(x + C₁)

2. xy dy/dx = x^2 + 1 / (y + 1)

Pertama, kita pisahkan variabel x dan y:

xy dy = (x^2 + 1) dx / (y + 1)

Kemudian, kita integrasikan kedua sisi:

∫(y) dy = ∫((x^2 + 1) / (y + 1)) dx

(y^2) / 2 = ∫((x^2 + 1) / (y + 1)) dx

Untuk melanjutkan, kita perlu mengekstraksi y dari integral, yang mungkin memerlukan teknik-teknik lebih lanjut. Selain itu, integrasi ini dapat menjadi rumit.
