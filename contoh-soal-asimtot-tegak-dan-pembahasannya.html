grafik 1/f(x) punya dua asimtot tegak dan satu asimsot datar.

ㅤ

untuk persamaan f(x) adalah

f(x) = a(x + 1)(x + 2)

2 = a(0 + 1)(0 + 2)

2 = 2a

a = 1

ㅤ

f(x) = (x + 1)(x + 2)

1/f(x) = 1/(x + 1)(x + 2)

ㅤ

asimtot tegak:

(x + 1)(x + 2) = 0

x = -1 atau x = -2

ㅤ

asimtot datar:

[tex]\displaystyle{ \sf{\lim_{x \to \infty } \dfrac{1}{(x + 1)(x + 2)} = 0}}[/tex]

ㅤ

Jadi 1/f(x) mempunyai 2 asimtot tegak dan 1 asimtot datar

14. asimtot tegak adalah?


asimtot adalah sumbu yang selalu didekati, tapi tidak pernah dipotong.

Asimtot tegak adalah garis tegak atau garis vertikal yang didekati oleh kurva lengkung dengan jarak semakin lama semakin kecil mendekati nol di titik jauh tak hingga. Asimtot tegak terjadi pada suatu fungsi rasional pecahan. Tetapi tidak semua fungsi rasional pecahan memiliki asimtot tegak. 

15. Tolong anda jelaskan mengenai asimtot datar, miring, dan tegak beserta dengan contohnya,,, terimakasih ditunngu jawaban secepat mungkin....


Asimtot datar adalah garis horizontal (Sejajar sumbu x) yang selalu mendekati grafik, tetapi tidak bersentuhan.
Seperti: y = 1/x (Asimtot datar : y = 0)
Asimtot miring adalah garis yang miring (bergradien) yang mendekati grafik, tetapi tidak bersentuhan juga.
Seperti, persamaan hiperbola:
x^2 - y^2 = 1, (Asimtot miring : y = x dan y = -x)
Asimtot tegak adalah garis tegak (Sejajar sumbu y) yang selalu mendekati grafik, tetapi tidak bersentuhan.
Seperti:
y = tan x, (Asimtot tegak : y = π/2 + n.π dengan n bilangan bulat)

21. Asimtot tegak dan asimtot datarhelpppp ygyppppppp​


Grafik [tex]y=f(x)[/tex] memotong sumbu [tex]Y[/tex] di titik:

[tex]\begin{aligned}\bf\left(0\,,\:{-}1\,\frac{1}{3}\right)\end{aligned}[/tex]
(opsi b)

 

PembahasanAsimtot Fungsi

Diketahui:

[tex]\begin{aligned}f(x)=\frac{8-ax}{2x-b}\end{aligned}[/tex]

Asimtot tegak dan asimtot datar [tex]f(x)[/tex] berpotongan di titik [tex](3, -2)[/tex].

Ditanyakan:

Titik potong antara grafik [tex]y = f(x)[/tex] dengan sumbu [tex]Y[/tex].

PENYELESAIAN

Asimtot tegak dan asimtot datar f(x) berpotongan di titik (3, –2), berarti:

Asimtot tegak [tex]f(x)[/tex] adalah [tex]x = 3[/tex].Asimtot datar [tex]f(x)[/tex] adalah [tex]y = -2[/tex].

Karena [tex]f(x)[/tex] tak terdefinisi untuk [tex]2x-b=0[/tex], maka asimtot tegaknya adalah:

[tex]\begin{aligned}2x&=b\implies x=\frac{b}{2}\end{aligned}[/tex]

Dengan asimtot tegak [tex]x = 3[/tex], diperoleh [tex]b = \bf6[/tex].

Sedangkan untuk asimtot datar [tex]f(x)[/tex], yaitu [tex]y=-2[/tex]:

[tex]\begin{aligned}-2&=\lim_{x\to\infty}\:\frac{8-ax}{2x-6}\\&=\lim_{x\to\infty}\left(\frac{8-ax}{2x-6}\times\frac{1/x}{1/x}\right)\\&=\lim_{x\to\infty}\:\frac{\frac{8}{x}-a}{2-\frac{6}{x}}\\&=\frac{0-a}{2-0}\\&=-\frac{a}{2}\\\therefore\ a&=-2\cdot(-2)=\bf4\end{aligned}[/tex]

Oleh karena itu, diperoleh:

[tex]\large\text{$\begin{aligned}f(x)=\frac{8-4x}{2x-6}\end{aligned}$}[/tex]

Grafik [tex]y=f(x)[/tex] memotong sumbu [tex]Y[/tex] ketika [tex]x=0[/tex].

[tex]\begin{aligned}f(0)&=\frac{8-0}{0-6}\\&=\frac{8}{-6}\ =\ -\frac{4}{3}\\&=\bf{-}1\,\frac{1}{3}\end{aligned}[/tex]

Diperoleh titik potong:

[tex]\begin{aligned}\bf\left(0\,,\:{-}1\,\frac{1}{3}\right)\end{aligned}[/tex]

[tex]\blacksquare[/tex]

 

KESIMPULAN

Grafik [tex]y=f(x)[/tex] memotong sumbu [tex]Y[/tex] di titik:

[tex]\large\text{$\begin{aligned}\bf\left(0\,,\:{-}1\,\frac{1}{3}\right)\end{aligned}$}[/tex]

25. Tentukan asimtot datar & asimtot tegak dari f(x) = -x / 2x+4 tidak sama dengan -2. Tolong yaa


Asimtot datar fungsi adalah y = - [tex]\frac{1}{2}[/tex]. Asimtot tegak dari fungsi adalah x = - 2. Asimtot datar membuat nilai x menjadi tak terhingga. Asimtot tegak fungsi akan membuat f(x) bernilai tak terhingga/

Pembahasan

ASIMTOT TEGAK DAN ASIMTOT DATAR PADA FUNGSI RASIONAL

Fungsi rasional adalah fungsi pecahan yaitu fungsi yang terdiri dari pembilang dan penyebut.

Asimtot tegak terjadi jika nilai fungsi mendekati tak terhingga.

f(x) = y = ∞Terjadi jika penyebut bernilai sama dengan nol.

Asimtot datar terjadi jika nilai variabel x mendekati tak terhingga.

x = ∞Dapat diperoleh menggunakan prinsip limit fungsinya.Jika m > n makay = [tex]\lim_{x \to \infty} \: \frac{ax^n}{bx^n} \:=\: \frac{a}{b}[/tex]y = [tex]\lim_{x \to \infty} \: \frac{ax^m}{bx^n}[/tex] = ∞y = [tex]\lim_{x \to \infty} \: \frac{ax^n}{bx^m}[/tex] = 0

Diketahui:

f(x) = [tex]\frac{- x}{2x + 4}[/tex] , x ≠ - 2

Ditanyakan:

Asimtot datar ?Asimtot tegak ?

Penjelasan

Menentukan asimtot datar.

y = [tex]\lim_{x \to \infty} \: \frac{- x}{2x + 4}[/tex]

Pangkat variabel x di pembilang sama dengan pangkat variabel x di penyebutnya.

y = [tex]\frac{- 1}{2}[/tex]

y = - [tex]\frac{1}{2}[/tex]

Asimtot datar fungsi adalah y = - [tex]\frac{1}{2}[/tex].

Menentukan asimtot tegak.

Penyebut = 0

2x + 4 = 0

2x = - 4

x = - [tex]\frac{4}{2}[/tex]

x = - 2

Asimtot tegak dari fungsi adalah x = - 2.

29. Diketahui fungsi rasional   . Asimtot tegaknya adalah​


Jawab:

Berdasarkan informasi yang diberikan, fungsi rasional tersebut adalah f(x) = (3x - 6)/(x - 2). Untuk mencari asimtot tegak (vertical asymptote), kita perlu mencari nilai x yang membuat penyebut (x - 2) menjadi nol.

Jika penyebut (x - 2) = 0, maka x = 2. Oleh karena itu, terdapat asimtot tegak pada x = 2.

Jadi, asimtot tegak dari fungsi f(x) = (3x - 6)/(x - 2) adalah x = 2.

39. Asimtot tegak f(x)= 2 tanx


Asimtot tegak f(x) = 2 tan x adalah x = 90ᵒ atau x = 270ᵒ, atau jika dalam bentuk π radian, asimtot tegak dari f(x) = 2 tan x adalah π/2 atau x = 3π/2. Grafik dari suatu fungsi rasional y = f(x)/g(x) memiliki asimtot tegak dan asimtot datar.

asimtot tegak jika y mendekati tak hingga (∞) atau g(x) = 0 asimtot datar jika x mendekati tak hingga (∞) atau y = [tex]\lim_{x \to \infty}[/tex]
Pembahasan

f(x) = 2 tan x

f(x) = 2 (sin x)/(cos x)

f(x) = (2 sin x)/(cos x)

f(x) memiliki asimtot tegak jika cos x = 0

cos x = 0

cos x = cos 90ᵒ  

x = 90ᵒ + k . 360ᵒ atau x = –90ᵒ + k . 360ᵒ

k = 0 ⇒ x = 90ᵒ             x = –90ᵒ (TM)

k = 1 ⇒ x = 450ᵒ (TM)    x = 270ᵒ

Keterangan: TM = Tidak Memenuhi


Jadi asimtot tegak dari f(x) = 2 tan x adalah x = 90ᵒ atau x = 270ᵒ


Jika diubah menjadi π radian,  

asimtot tegak dari f(x) = 2 tan x adalah x = π/2 atau x = 3π/2
