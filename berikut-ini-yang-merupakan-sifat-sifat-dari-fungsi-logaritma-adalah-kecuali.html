4. sifat sifat logaritma


Sifat Logaritma dari perkalian

Suatu logaritma merupakan hasil penjumlahan dari dua logaritma lain yang nilai kedua numerus-nya merupakan faktor dari nilai numerus awal. Berikut modelnya:

alog p.q = alog p + alog q

dengan syarat a > 0, ￼, p > 0, q > 0.

2. Perkalian Logaritma

Suatu logaritma a dapat dikalikan dengan logaritma b jika nilai numerus logaritma a sama dengan nilai bilangan pokok logaritma b. Hasil perkalian tersebut merupakan logaritma baru dengan nilai bilangan pokok sama dengan logaritma a, dan nilai numerus sama dengan logaritma b. Berikut model sifat logaritma nya:

alog b x blog c = alog c

dengan syarat a > 0, ￼.

3. Sifat Logaritma dari pembagian

Suatu logaritma merupakan hasil pengurangan dari dua logaritma lain yang nilai kedua numerus-nya merupakan pecahan atau pembagian dari nilai numerus logaritma awal. Berikut modelnya:

alog ￼ = alog p – alog q

dengan syarat a > 0, ￼, p > 0, q > 0.

Jawab:

Penjelasan dengan langkah-langkah:

Nomor 5

Misal [tex]\displaystyle ^{a^m}\log b^n=c\leftrightarrow \left ( a^m \right )^c=b^n[/tex], maka:

[tex]\begin{aligned}\left ( a^m \right )^c&=b^n\\a^{cm}&=b^n\\\sqrt[n]{a^{cm}}&=b\\\left ( a^{cm} \right )^{\frac{1}{n}}&=b\\a^{\frac{cm}{n}}&=b\\\frac{cm}{n}&=^a\log b\\c&=\frac{n}{m}~^a\log b\end{aligned}[/tex]

Diperoleh [tex]\displaystyle ^{a^m}\log b^n=\frac{n}{m}~^a\log b[/tex]

Penyelesaian

[tex]\begin{aligned}^2\log \left ( \frac{1}{64} \right )-^{\sqrt{3}}\log 81+^{25}\log \left ( \frac{1}{125} \right )&=^2\log \left ( \frac{1}{2^6} \right )-^{3^{\frac{1}{2}}}\log 3^4+^{5^2}\log \left ( \frac{1}{5^3} \right )\\&=^2\log 2^{-6}-\frac{4}{\frac{1}{2}}~^3\log 3+5^2\log 5^{-3}\\&=-6~^2\log 2-8~^3\log 3-\frac{3}{2}~^5\log 5\\&=-6(1)-8(1)-\frac{3}{2}(1)\\&=-\frac{31}{2}\end{aligned}[/tex]

Nomor 6

Karena [tex]\displaystyle ^a\log b=c\leftrightarrow a^c=b[/tex] maka [tex]\displaystyle a^{^a\log b}=b[/tex]

[tex]\begin{aligned}4^{^2\log 7}-9^{^3\log 2}&=\left ( 2^2 \right )^{^2\log 7}-\left ( 3^2 \right )^{^3\log 2}\\&=2^{2~^2\log 7}-3^{2~^3\log 2}\\&=2^{^2\log 7^2}-3^{^3\log 2^2}\\&=2^{^2\log 49}-3^{^3\log 4}\\&=49-4\\&=45\end{aligned}[/tex]

22. sifat-sifat logaritma​


Jawab:

1. 2

2. Coba kerjakan sendiri hitung dengan kalkulator ilmiah

3. -5

4. 3/2

Penjelasan dengan langkah-langkah:

Nomor 1

Misal [tex]\displaystyle ^a\log b=m\leftrightarrow a^m=b[/tex] dan [tex]\displaystyle ^a\log c=n\leftrightarrow a^n=c[/tex], maka:

[tex]\begin{aligned}\frac{b}{c}&=\frac{a^m}{a^n}\\\frac{b}{c}&=a^{m-n}\\m-n&=^a\log \left ( \frac{b}{c} \right )\\^a\log b-^a\log c&=^a\log \left ( \frac{b}{c} \right )\end{aligned}[/tex]

Penyelesaian

[tex]\begin{aligned}^9\log 243-^9\log 3&=^9\log \left ( \frac{243}{3} \right )\\&=^9\log 81\\&=2\end{aligned}[/tex]

Nomor 3

Misal [tex]\displaystyle ^a\log b=m\leftrightarrow a^m=b[/tex] dan [tex]\displaystyle ^a\log c=n\leftrightarrow a^n=c[/tex], maka:

[tex]\begin{aligned}bc&=a^ma^n\\\bc&=a^{m+n}\\m+n&=^a\log bc\\^a\log b+^a\log c&=^a\log bc\end{aligned}[/tex]

Berdasarkan sifat di atas dapat diturunkan sifat lainnya.

[tex]\begin{aligned}^a\log b^n&=^a\log (b\cdot b\cdot b...b)\\&=^a\log b+^a\log b+^a\log b+...+^a\log b\\&=n~^a\log b\end{aligned}[/tex]

Misal [tex]\displaystyle ^a\log b=m\leftrightarrow a^m=b[/tex] lalu kedua ruas dikalikan [tex]\displaystyle ^c\log[/tex], maka:

[tex]\begin{aligned}^c\log a^m&=^c\log b\\m~^c\log a&=^c\log b\\m&=\frac{^c\log b}{^c\log a}\end{aligned}[/tex]

Jika c = 10 diperoleh sifat [tex]\displaystyle ^a\log b=\frac{\log b}{\log a}[/tex]

Apa jadi nya jika [tex]\displaystyle ^a\log b~^b\log c[/tex]?

[tex]\begin{aligned}^a\log b~^b\log c&=\frac{\log b}{\log a}~\frac{\log c}{\log b}\\&=\frac{\log c}{\log a}\\&=^a\log c\end{aligned}[/tex]

diperoleh [tex]\displaystyle ^a\log b~^b\log c=^a\log c[/tex]

Penyelesaian

[tex]\begin{aligned}^3\log 6~^6\log \left ( \frac{1}{243} \right )&=^3\log \left ( \frac{1}{3^5} \right )\\&=^3\log 3^{-5}\\&=-5~^3\log 3\\&=-5(1)\\&=-5\end{aligned}[/tex]

Nomor 4

Perhatikan bahwa [tex]\displaystyle a^1=a\leftrightarrow ^a\log a=1[/tex], maka:

[tex]\begin{aligned}\frac{^3\log 64}{^3\log 16}&=\frac{^3\log 4^3}{^3\log 4^2}\\&=\frac{3~^3\log 4}{2~^3\log 4}\\&=\frac{3}{2}\end{aligned}[/tex]

1) ᵃlog a = 1

Contoh :

⁴log 2 + ⁴log 8

= ⁴log (2 × 8)

= ⁴log 16

= 2

(karena 4² = 16)

4) ᵃlog b - ᵃlog c = ᵃlog (b/c)

Contoh :

⁵log 100 - ⁵log 4

= ⁵log (100/4)

= ⁵log 25

= 2

(karena 5² = 25)

5) ᵃlog bⁿ = n . ᵃlog b

Contoh :

⁴log 64

= ⁴log 4³

= 3 . ⁴log 4

= 3 . 1

= 3
