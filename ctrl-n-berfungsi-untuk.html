F12 - Menyimpan kembali dengan nama yang sama atau berbeda (save as).

17. fungsi short cut ctrl+n .ctrl+b. ctrl+a . Alt× F4.shift+f3​


Ctrl + N = berfungsi untuk membuat dokumen baru
Ctrl + B = kombinasi antara tombol B dengan control Ctrl dapat membuat kalimat yang di blok menjadi huruf tebal.
Ctrl + A = kombinasi antara tombol A dengan control Ctrl dapat digunakan untuk memilih seluruh objek baik teks maupun gambar dalam Ms Word atau file lainnya.
Alt + F4 = akan menutup program sedang aktif di Microsoft Windows
Shift + F3 = Untuk merubah teks menjadi huruf besar besar semua, besar huruf depannya saja, atau juga merubah teks menjadi huruf kecil semua (MS Word). Contoh: MAKAN, Makan, makan.
