13. apa perbedaan blackbox testing boundary value analysis dan equivalence partitioning? sertakan sumber kalau bisa! Terima Kasih!


Jawaban:

Perbedaan Boundary Value Analysis Dan Equivalence Partitioning :

Boundary value analysis

Itu sebuah teknik yang di mana kita mengidentifikasi kesalahan pada proses batas data masukan untuk menemukan kesalahan itu dalam masukan pusat.

Boundary value Ada yang mengandung batas atas dan batas bawah suatu variabel.

Boundary value analysis Adalah pengujian batas antara partisi.

Equivalence partitioning

Itu sebuah teknik yang di mana input data dibagi menjadi partisi dari nilai. yang sah dan tidak valid

Dalam hal ini, input kepada perangkat lunak atau aplikasi yang dipisahkan ke dalam kelompok yang diharapkan untuk menunjukkan perilaku yang sama.

Hal ini memungkinkan kita membagi satu set kondisi dari tes ke partisi yang harus menjadi pertimbangan yang sama .

Penjelasan:

KOMPARASI PENGUJIAN SOFTWARE

Equivalence Partitioning ( EP )

adalah metode blackbox yg membagi domain input dari program menjadi beberapa partisi.

Boundary Value Analysis ( BVA )

adalah metode pelengkap dari Equivalence Partitioning penentu ketepatan nilai batas atas dan bawah.

perbedaannya :

EP dapat menguji tipe data jangkauan dan bebas namun sulit untuk menentukan nilai representatif, sedangkan BVA hanya dapat menguji tipe data jangkauan tetapi nilai representatifnya lebih jelas

sumber : Informasi Engineering (1807)
