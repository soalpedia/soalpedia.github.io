#include<stdio.h>

void printAllData(int listData[3][5],int row,int column)

{

printf("Print Data\n");

for(int i=0; i<row; i++)

{

 for(int j=0; j<column; j++)

 {

  printf("%-8d ",listData[i][j]);

 }

 printf("\n");

}

}

void multiplyAllDataByN(int listData[3][5],int row,int column, int n)

{

for(int i=0; i<row; i++)

{

 for(int j=0; j<column; j++)

 {

  listData[i][j] *=n;

 }

}  

}

int main()

{

int row = 3, column = 5;

int listData[3][5];

for(int i=0; i<row; i++)

{

 for(int j=0; j<column; j++)

 {

  scanf("%d",&listData[i][j]);

 }

}

int n = 3;

multiplyAllDataByN(listData,row,column,n);

15. Tulislah dalam bentuk definisi fungsi nilai mutlak![tex]\left. \begin{array} { l } { f ( x ) = | 7 - x | } \end{array} \right.[/tex]​


F(x) = |7 - x|

Definisi Untuk Fungsi Nilai Mutlak memiliki dua Jawaban, Jika lebih besar dari atau sama dengan 0 Maka Jawabannya sesuai dengan Fungsi awalnya. Namun jika Lebih kecil dari 0 maka jawabannya adalah Fungsi Awalnya dikali minus 1.

F(x) = |7 - x|

|7 - x| >= 0

7 - x >=0

7 >= x

x <=7

F(x) = -(7 - x)

7 - x < 0

7 < x

x > 7

Maka Kesimpulannya adalah :

F(x) = 7 - x, Jika x <= 7

F(x) = x - 7, Jika x > 7

F(x) = x - 7, Jika x < 7

Penjelasan dengan langkah-langkah:

f(x) = |7 - x|

Dalam bentuk definisi fungsi nilai mutlak :

(1) f(x) = 7 - x jika 7 - x  ≥ 0 atau -x  ≥ -7 atau x ≤ 7

(2) f(x) = -(7 - x) = -7 + x jika 7 - x < 0 atau -x < -7 atau x > 7

Jawaban dan Penjelasan:

Mungkin algoritma kedua fungsi berikut ini akan lebih "enak" dilihat pada gambar terlampir. Di sini, sulit untuk membuat format jawaban seperti itu.

Bagian I: Contoh algoritma menghitung banyaknya elemen array yang merupakan kelipatan 5 menggunakan notasi algoritmik

function Banyak_Elemen_Kelipatan_Lima (Input A: array[1..n] of integer) → integer

{ Menghitung banyaknya elemen kelipatan lima dalam array A. }

{ Prekondisi: array A dan n terdefinisi, n > 0 }

Kamus lokal:

   i: integer

   banyak: integer

Algoritma:

   i ← 1

   banyak ← 0

   while (i ≤ n) do

       if (A[i] mod 5 = 0) then         { Jika A[i] habis dibagi 5 }

           banyak ← banyak + 1        { increment “banyak” }

       i ← i + 1                                  { increment i }

   → banyak                                  { mengembalikan nilai “banyak” }

Bagian II: Contoh algoritma menghitung jumlah elemen array yang merupakan kelipatan 5 menggunakan notasi algoritmik

function Jumlah_Elemen_Kelipatan_Lima (Input A: array[1..n] of integer) → integer

{ Menghitung jumlah dari semua elemen kelipatan lima dalam array A. }

{ Prekondisi: array A dan n terdefinisi, n > 0 }

Kamus lokal:

   i: integer

   jumlah: integer

Algoritma:

   i ← 1

   jumlah ← 0

   while (i ≤ n) do

       if (A[i] mod 5 = 0) then          { Jika A[i] habis dibagi 5 }

           jumlah ← jumlah + A[i]       { jumlah = jumlah + A[i] }

       i ← i + 1                                   { increment i }

   → jumlah                                    { mengembalikan nilai “jumlah” }

36. Jika diketahui array A= {12,10,4,14,16,7,18,5,13,9} maka nilai tukar dari algoritma pengurutan cepat pada pivot 18 adalah..​


Untuk dapat menjawab pertanyaan ini, diperlukan informasi yang lebih spesifik tentang implementasi algoritma pengurutan cepat (quick sort) yang digunakan dan bagaimana pivot dipilih dalam kasus tersebut. Tanpa informasi tersebut, tidak dapat memberikan jawaban yang tepat.

Secara umum, algoritma pengurutan cepat (quick sort) adalah sebuah algoritma pengurutan yang mengurutkan sebuah array dengan cara memilih elemen array sebagai pivot, membagi array menjadi dua bagian berdasarkan pivot, dan kemudian mengurutkan bagian-bagian tersebut secara rekursif. Biasanya, pivot dipilih secara acak dari array, atau dipilih sebagai elemen tengah dari array.

Jika pivot dipilih sebagai elemen 18 dari array A di atas, maka proses pengurutan cepat akan membagi array A menjadi dua bagian, yaitu bagian pertama yang berisi semua elemen yang lebih kecil dari 18, dan bagian kedua yang berisi semua elemen yang lebih besar dari 18. Bagian pertama akan berisi elemen-elemen 12, 10, 4, 14, 16, 7, 5, 13, 9, sedangkan bagian kedua akan berisi hanya satu elemen yaitu 18. Setelah itu, bagian-bagian tersebut akan diurutkan secara rekursif menggunakan algoritma pengurutan cepat.

Nilai tukar dari algoritma pengurutan cepat adalah jumlah perpindahan elemen yang terjadi saat proses pengurutan. Namun, karena tidak ada informasi yang lebih spesifik tentang implementasi algoritma pengurutan cepat yang digunakan, tidak dapat memberikan jawaban yang tepat tentang nilai tukar dari algoritma pengurutan cepat pada pivot 18.
