2. fUNGSI LIST DALAM MS EXCEL


Fungsi

Deskripsi

Fungsi AND

Mengembalikan TRUE jika seluruh argumen adalah TRUE

Fungsi FALSE

Mengembalikan nilai logika FALSE

Fungsi IF

Menentukan uji logika yang dijalankan

Fungsi IFERROR

Mengembalikan nilai yang Anda tentukan jika rumus mengevaluasi kesalahan; jika tidak, mengembalikan hasil rumus

Fungsi IFNA 
￼

Mengembalikan nilai yang Anda tentukan jika ekspresi menetapkan #N/A, jika tidak, mengembalikan hasil dari ekspresi

Fungsi IFS 
￼

Memeriksa apakah satu persyaratan atau lebih terpenuhi dan mengembalikan nilai yang sesuai dengan persyaratan TRUE yang pertama.

Fungsi NOT

Membalikkan logika argumennya

Fungsi OR

Mengembalikan TRUE jika setiap argumen adalah TRUE

Fungsi SWITCH 
￼

Mengevaluasi ekspresi terhadap daftar nilai dan mengembalikan hasil yang sesuai dengan nilai cocok yang pertama. Jika tidak ada kecocokan, nilai default opsional dapat dikembalikan.

Fungsi TRUE

Mengembalikan nilai logika TRUE

Fungsi XOR 
￼

Mengembalikan eksklusif logika OR dari semua argumen

23. apa yang dimaksud dengan print out?


Print out berasal dari 2 kata print dan out. Print memiliki pengertian cetak, out keluar. Biasanya kata print out digunakan oleh teknologi sekarang, seperti Microsoft Office, yang gunanya untuk membuat nyata data yang ingin di print, dalam bentuk kertas.  

Pembahasan

Kata Print out memiliki pengertian mencetak, dari bahasa asalnya inggris, tujuanya bahasa Indonesia, dan katanya print out, memiliki pengertianya mencetak. Pengertian kata lain print seperti cetakan, membuat copy, potret, namun tetap umumnya mencetak. Namun biasanya kata print out digunakan pada teknologi, sistem informasi zaman sekarang, untuk menunjukan perintah mengeprint suatu data yang akan d print. Menurut Glosarium bahwa print out dari internet memiliki pengertian cetakan, dari definisi print out juga memiliki pengertian cetakan. Menurut kamuslengkap.id, print out memiliki pengertian mencetak. Menurut pertanyaan.com, bahwa print out memiliki pengertian the output of a computer in printed form, terjemahanya keluaran dari komputer daftar printer atau form yang berisikan yang akan data di print. Namun dari refernsi beberapa diatas, bahwa print out tetap memiliki pengertian mencetak, secara perkata terjemahan yaitu cetak keluar.

25. Jelaskan fungsi print preview dan page setup pda ms. Excel


Jelaskan fungsi print preview dan page setup pada ms. Excel, fungsi print preview adalah untuk melihat terlebih dahulu sebuah dokumen sebelum kita cetak. Dengan melihat dokumen terlebih dahulu sebelum kita cetak, kita dapat mengetahui dokumen mana saja yang kita perlukan untuk dicetak sehingga tidak ada kertas yang terbuang. Hal ini tentu sangat bermanfaat apabila kita memiliki usaha rental pengetikan atau rental komputer. 

Sedangkan fungsi page setup adalah untuk mengatur dokumen kita, yang bisa kiata atur pada dokumen kita melalui page setup adalah margin halaman, layout kertas hingga ukuran kertas. Dengan melakukan pengatura pada page setup kita dapat memilih ukuran kertas yang akan kita gunakan.

Demikian pembahasan mengenai fungsi print preview dan page setup pada microsoft excel. Fungsi print preview untuk melihat dokumen sebelum dicetak sedangkan page setup untuk melakukan pengaturan terhadap dokumen kerja kita. Page setup dan print preview juga terdapat pada aplikasi microsoft office lainnya.

39. JelaskAn fUngsi print previev pada microsoft word dan excel


Untuk melihat printed version dokumen atau halaman sebelum benar-benar dicetak dalam bentuk kertas.

Print Preview Edit Mode adalah Menu Ribbon lengkap yang menampilkan Print Preview.
Fungsi Print Preview Edit Mode adalah untuk menampilkan print out di layar monitor dengan menu yang lebih lengkap. Sebagian dari Menu Ribbon Print Preview Edit Mode juga dapat disetting dengan Menu Page Layout atau  melalui menu Print ( File --> Print ). Tetapi tidak ada salahnya jika ingin lebih mempercepat proses print out dengan menampilkan Print Preview Edit Mode pada Customize Quick Accsess Toolbar.

Bagaimana caranya?
Cara untuk menampilkan Print Preview Edit Mode pada Customize Quick Accsess Toolbar sama dengan menampilkan icon Print Preview and Print atau menampilkan icon Camera pada Microsoft Excel atau juga sama dengan menampilkan icon Save All.
