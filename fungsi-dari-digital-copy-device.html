4. jelaskan tentang Input device,Process device, Out put device beri masing² contoh beserta fungsinya​


Input device mengirimkan informasi ke sistem komputer untuk diproses, dan Output device mereproduksi atau menampilkan hasil pemrosesan tersebut. Input device hanya mengizinkan input data ke komputer dan Output device hanya menerima output data dari perangkat lain. Process device adalah semua perangkat di dalam komputer yang menafsirkan dan memanipulasi data yang masuk selama tahap ini.

Contoh Input device yaitu: Keyboard, mouse, microfon, dan webcam.

Contoh Output device yaitu: Monitor, proyektor, dan speaker.

Contoh Process device yaitu: CPU, memory and motherboard.

Pembahasan

Sebagian besar perangkat hanya merupakan Input device atau Output device, karena masing-masing peragkat hanya dapat menerima input data dari pengguna atau data output yang dihasilkan oleh komputer. Namun, beberapa perangkat dapat menerima input dan menampilkan output, dan disebut sebagai perangkat I/O (perangkat input/output).

Contohnya keyboard mengirimkan sinyal listrik, yang diterima sebagai input. Sinyal tersebut kemudian ditafsirkan oleh komputer dan ditampilkan atau dikeluarkan pada monitor sebagai teks atau gambar. Di bagian bawah gambar, komputer mengirim, atau mengeluarkan data ke printer. Kemudian, data tersebut dicetak ke selembar kertas, yang juga dianggap sebagai output.

Process device merupakan perangkat apa pun di dalam komputer yang menafsirkan dan memanipulasi data yang masuk selama tahap ini.

Process device adalah komponen yang bertanggung jawab untuk pemrosesan informasi di dalam sistem komputer. Ini termasuk perangkat seperti CPU, memori dan motherboard. Perangkat penyimpanan adalah komponen yang memungkinkan data disimpan dalam sistem komputer.

28. 9. Fungsi alat ini ialah untuk memindai data atau dokumen dengan cara menduplikasi dalam bentuk digital atau file dalam komputer. Alat ini termasuk jenis hardware ... a. input device b. output device C. storage deviced. processing device​


Alat yang digunakan untuk melakukan pemindaian data ataupun dokumen dengan cara membuat salinannya yang berbentuk digital (file) di dalam komputer disebut scanner dan ia termasuk hardware yang berjenis input device (komponen masukan).

Penjelasan:

Komputer merupakan alat yang tujuan penggunaannya adalah pengolahan data sesuai prosedur yang telah dirumuskan. Untuk mendukung kinerja komputer, terdapat lima jenis komponen yang membantunya, yaitu:

input device (komponen masukan)output device (komponen keluaran)processing device (komponen pemrosesan)storage device (komponen penyimpanan)additional device (komponen tambahan)

Berikut akan disebutkan contoh-contoh dari masing-masing jenis komponen.

Input Device:

keyboardmousewebcambarcode scannerjoystickscannermikrofonpemindai sidik jarikinectflash disk/CD ROM/DVD ROM/external hard disk

Output Device:

monitorprinterproyektorheadphoneheadsetplotterFlat Panel Display (FDP)speakersound cardVirtual Graphic Array (VGA) card

Processing Device:

processor, Central Processing Unit (CPU)motherboard/mainboard/baseboardpower supply

Storage Device:

memori Random Access Memory (RAM)hard diskoptical driveflash disksolid state disk (SSD)

Additional Device:

Local Area Network (LAN) cardWireless Local Area Network (WLAN) cardfirewire cardbluetooth

Pada kalimat soal, terdapat alat yang berguna untuk melakukan pemindaian data atau dokumen. Caranya yaitu menduplikasi data atau dokumen tersebut dalam bentuk digital atau file dalam komputer. Karena hasil akhirnya memberikan file ke dalam komputer, dengan kata lain memasukkan file, jelas bahwa hardware tersebut merupakan input device. Nama alat tersebut adalah scanner. Macam-macam bentuk scanner di antaranya:

scanner drum (bisa memindai kertas besar dan dokumen atau data kuno)scanner flatbed (menyerupai tempat tidur, kompatibel untu berbagai ukuran kertas, memakai teknologi Charge Coupled Double atau CCD)film scanner (mengonversi film negatif, seperti foto dan video, menjadi bentuk digital)roller scanner (memindai dokumen atau gambar yang fisiknya berupa lembaran, caranya memakai dokumen yang digerakkan)
