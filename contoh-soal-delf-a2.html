20. Bagaimana cara mencari nilai a+b dari soal a4 +a2 b2 +b4 =6 dan a2 +ab +b2 =4


a^4 + a²b² + b^4 = 6 dan a² + ab + b² = 4

a^4 + a²b² + b^4 = 6
⇒a^4 + a²b² + a²b² + b^4 = 6 + a²b²
⇒a^4 + 2a²b² + b^4 = 6 + a²b²
⇒(a² + b²)² = 6 + a²b²
⇒a² + b² = √(6 + (ab)²)

a² + ab + b² = 4
⇒ a² + b² = 4 - ab
⇒ √(6 + (ab)²) = 4 - ab
⇒6 + (ab)² = (4 - ab)²
⇒6 + (ab)² = 16 - 8ab + (ab)²
⇒-10 = -8ab
⇒ab = -10/-8 = 5/4

a² + ab + b² = 4
⇒a² + ab + ab + b² = 4 + ab
⇒a² + 2ab + b² = 4 + ab
⇒(a + b)² = 4 + ab
⇒(a + b)² = 4 + 5/4
⇒(a + b)² = 21/4
⇒(a + b) = √(21/4) = (1/2)√21

27. bentuk sederhana sifat eksponen dari soal (a2-1)(a+1)-1


Jawab:

Penjelasan dengan langkah-langkah:

Untuk menyederhanakan sifat eksponen dari ekspresi (a^2 - 1)(a + 1)^{-1}, kita perlu menerapkan beberapa aturan eksponen. Berikut langkah-langkahnya:

1. (a^2 - 1)(a + 1)^{-1}

2. Kita ketahui bahwa (a + b)(a - b) = a^2 - b^2. Maka, kita dapat menerapkan aturan ini pada (a^2 - 1) untuk menyederhanakan bagian pertama ekspresi:

  (a^2 - 1) = (a)^2 - (1)^2

            = a^2 - 1

3. Setelah kita menyederhanakan bagian pertama, ekspresi menjadi:

  (a^2 - 1)(a + 1)^{-1} = (a^2 - 1)(1/(a + 1))

4. Kita tahu bahwa 1/(a + b) = 1/a + 1/b. Maka, kita dapat menerapkan aturan ini pada ekspresi di atas:

  (a^2 - 1)(1/(a + 1)) = (a^2 - 1)(1/a + 1/1)

                        = (a^2 - 1)(1/a + 1)

Jadi, bentuk sederhana sifat eksponen dari ekspresi (a^2 - 1)(a + 1)^{-1} adalah (a^2 - 1)(1/a + 1).
