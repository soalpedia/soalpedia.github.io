6. a. jelaskan definisi dari puts() b. sebutkan kegunaan dari puts() c. jelaskan aturan (sintaks) penggunaan keyword puts() dalam pemrograman dasar bahasa C


A. puts itu mengirim String ke stdout untuk ditampilkan di layar
B. puts() mengirim String ke layar output dan menambahkan newline setelah string tersebut
Contoh
puts("baris pertama"); //ada enter setelah puts
printf("kalimat-"); // tidak ada enter setelah printf
printf("sambung");

tampilan di layar:
baris pertama
kalimat-sambung

int puts(const char *str)
*str adalah string yang ingin di tulis

Jika terjadi error akan return EOF





7. sabila(puts/put) the bag ob the table, jadi jawabannya puts/put?


Simple Present Tense

Sabila puts the bag on the table.

Penjelasan: Karena subjek yang digunakan adalah tunggal (Sabila/She), maka verb harus diakhiri dengan s/es.

Pembahasan

Berikut adalah formula yang digunakan untuk membuat kalimat dalam bentuk simple present tense:

Positive form

Subject + V1 (-s, -es, -ies)

He goes to schoolMy daughter lives in South AfricaIt snows a lot during the winter

Subject: (He, She, It)

Negative form

Subject + does/do NOT + V1

My mother doesn’t speak good englishIt doesn’t rain in the summerShe doesn’t study Math on Monday

Subject: (Do => I, You, We, They) (Does => He, She, It)

Question form

Does/Do + Subject + V1

Does he go to school?Does she live in Paris?Do you like apple?

Subject: (Do => I, You, We, They) (Does => He, She, It)

Jawaban:

Fungsi yang paling umum digunakan dalam menampilkan data adalah A. PRINTF().

Fungsi PRINTF() merupakan fungsi standar dalam bahasa pemrograman C yang digunakan untuk mencetak (menampilkan) data ke layar atau output. Fungsi ini memungkinkan kita untuk mencetak teks, angka, atau nilai variabel dengan format yang diinginkan.


18. apa perbedaan pengguna puts dan printf ?


PUTS ( )
Fungsi ini digunakan khusus untuk menampilkan data string ke layar. Sifat fungsi ini, string yang ditampilkan secara otomatis akan diakhiri dengan \n (pindah baris). Dibandingkan dengan printf(), perintah ini mempunyai kode mesin yang lebih pendek.PRINTF ( )
Merupakan fungsi yang paling umum digunakan dalam menampilkan data. Berbagai jenis data dapat ditampilkan ke layar dengan fungsi ini.
Bentuk penulisan : printf(“string kontrol”, argumen1, argumen2, …);
- String kontrol dapat berupa keterangan yang akan ditampilkan pada layar beserta penentu format seperti %d, %f. Penentu format dipakai untuk memberi tahu kompiler mengenai jenis data yang akan ditampilkan
- Argumen adalah data yang akan ditampilkan ke layar. Argumen ini dapatr berupa variabel, konstanta atau ungkapanprintf()
- untuk menentukan data string yaitu %s
- untuk mencetak pindah baris memerlukan notasi ' \n'
puts()
- tidak perlu penentu tipe data string, karena fungsi ini khusus tipe data string
- untuk mencetak pindah baris tidak perlu notasi ' \n', karena sudah diberikan secara otomatis
