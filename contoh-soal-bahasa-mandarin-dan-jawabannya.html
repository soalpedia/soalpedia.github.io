19. bahasa mandarinnya apa tanggal lahirmu jawab di bahasa mandarin ya


你的出生日期什么 ？ // Nǐ de chūshēng rìqí shénme?

_____________________

Pembahasan:

出生日期 (chūshēng rìqí) adalah penyebutan tanggal lahir dalam Bahasa Mandarin. Partikel de (的) menyatakan kepemilikan atas sesuatu. Nah dalam kalimat itu subjeknya adalah 你 (ni) maka setelah dibubuhi kepemilikan maka menjadi tanggal lahirmu. Untuk menanyakan apa maka dapat menggunakan shénme (什么).

Arti per kata:

你 (ni) = kamu.的 (de) = partikel kepemilikan.出生日期 (chūshēng rìqí) : tanggal lahir.什么 (shénme) : apa.

30. tolong kerjain soal mandarin saya plsTuliskan dalam bahasa mandarin!"seekor kucing kecil "​


Bahasa mandarin dari "seekor kucing kecil" adalah :

Seekor kucing kecil = Yī zhǐ Xiǎo māo = 一只小猫

Mengapa saya kata seekor saya menggunakan kata satuan 只, itu karena khusus untuk hewan kata satuan nya menggunakan zhī

Penjelasan :

只 = zhī adalah kata satuan untuk seekor hewan.

支 = Zhī adalah kata satuan untuk sebuah pensil/pulpen.

件 = jiàn adalah kata satuan untuk sebuah baju.

张 = Zhāng adalah kata satuan untuk sehelai kertas.

本 = Běn adalah kata satuan untuk sebuah buku.

双 = shuāng adalah kata satuan untuk benda sepasang misalnya sepasang sepatu,sepasang kaos kaki.

Dan masih ada beberapa kata satuan lainnya.
