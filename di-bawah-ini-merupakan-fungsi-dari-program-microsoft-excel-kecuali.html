9. Berikut ini adalah formula yang terdapat di dalam program aplikasi Microsoft Excel kecuali


Microsoft Excel adalah sebuah program aplikasi pengolah angka (spreadsheet) yang dikembangkan oleh Microsoft. Excel umumnya digunakan untuk mengelola, menganalisis, dan memvisualisasikan data dalam bentuk tabel atau grafik.

Karena pada soal tidak lengkap, maka saya akan menuliskan data program aplikasi Microsoft Excel.

Formula dalam Ms. Excel :

1. =SUM           : Penjumlahan

2. =AVERAGE  : Rata-Rata

3. =MAX          : Nilai Maksimum

  =MIN           : Nilai Minimum

4. =IF               : menguji syarat sudah benar atau tidak

Jadi selain di atas, berarti bukan termasuk Microsoft Excel.

Pembahasan

Microsoft Excel adalah sebuah program aplikasi pengolah angka (spreadsheet) yang dikembangkan oleh Microsoft. Excel umumnya digunakan untuk mengelola, menganalisis, dan memvisualisasikan data dalam bentuk tabel atau grafik.

Dalam Excel, pengguna dapat membuat dan mengedit tabel data dengan berbagai rumus matematika dan fungsi logika, memformat tampilan tabel dengan warna, font, dan bingkai, serta membuat grafik untuk membantu memvisualisasikan data. Excel juga memiliki kemampuan untuk mengimport dan mengekspor data dari dan ke berbagai format file, termasuk CSV, TXT, dan XML.

Excel banyak digunakan di berbagai bidang seperti keuangan, bisnis, pendidikan, dan ilmu pengetahuan. Pengguna dapat menggunakan Excel untuk membuat laporan keuangan, menghitung anggaran, menganalisis data penjualan, mengatur jadwal kerja, dan banyak lagi.
