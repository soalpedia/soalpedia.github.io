1. pengertian parameter dan contohnya​


Parameter adalah nilai numerik yang menggambarkan karakteristik seluruh anggota populasi. Berikut ini adalah contoh parameter, ketika kita ingin mengetahui tinggi badan rata-rata seluruh penduduk Indonesia. Pada kasus tersebut yang menjadi parameternya adalah populasi seluruh tinggi badan penduduk Indonesia.

Pembahasan

Parameter adalah nilai numerik yang menggambarkan karakteristik seluruh anggota populasi. Fungsi dari parameter adalah untuk memudahkan dalam memahami dan mendapatkan informasi penting tentang populasi tertentu. Informasi ini akan membantu untuk menyelesaikan permasalahan dalam populasi dan mencegah masalah yang mungkin terjadi di masa depan.

Berikut ini adalah contoh parameter, ketika kita ingin mengetahui tinggi badan rata-rata seluruh penduduk Indonesia. Pada kasus tersebut yang menjadi parameternya adalah populasi seluruh tinggi badan penduduk Indonesia.

Parameter yang digunakan pada contoh kasus diatas terlalu sulit karena banyaknya jumlah penduduk di Indonesia. Sehingga harus mengukur tinggi badak 500 orang secara acak yang tersebar di seluruh Indonesia. Kemudian data yang didapatkan digunakan membuat data statistik untuk mendapatkan kesimpulan rata-rata tinggi badan seluruh populasi penduduk Indonesia.

function rangeWithStep(startNum, stopNum, step) {

   if (typeof stopNum == 'undefined') {

       // one param defined

       stopNum = startNum;

       startNum = 0;

   }

   if (typeof step == 'undefined') {

       step = 1;

   }

   if ((step > 0 && startNum >= stopNum) || (step < 0 && startNum <= stopNum)) {

       return [];

   }

   var result = [];

   for (var i = startNum; step > 0 ? i < stopNum : i > stopNum; i += step) {

       result.push(i);

   }

   return result;

};

15. Uraikanlah parameter-parameter yang ada dalam getaran dan gelombang​


Jawaban:

Periode: T

Frekuensi: f

Amplitudo: A

Cepat rambat: v

Panjang gelombang: lambda

Penjelasan:

Periode:

Periode adalah waktu yang dibutuhkan gelombang untuk melakukan sebuah siklus. Periode dihitung dalam satuan detik dan dilambangkan sebagai huruf T (Time) yang berarti waktu. Periode diukur dengan satuan detik/siklus.

Frekuensi:

Frekuensi gelombang adalah jumlah siklus yang diselesaikan dalam waktu tertentu. Frekuensi dilambangkan dengan huruf f kecil dan kita ukur dalam satuan siklus/detik atau sama saja dengan Hertz (Hz).

Amplitudo:

Amplitudo adalah simpangan terjauh sebuah gelombang. Amplitudo bisa juga dibilang sebagai jarak antara garis puncak dan puncak atau palung dari suatu gelombang. Semakin besar amplitudo, semakin banyak energi yang dibawa.

Cepat rambat:

Cepat rambat gelombang adalah kecepatan gelombang. Kecepatan gelombang dapat diukur dalam meter/detik (m/s). Cepat rambat gelombang diukur berdasarkan jarak gelombang itu bergerak dalam kurun waktu tertentu. Kecepatan gelombang memberitahu kita berapa banyak siklus yang diselesaikan suatu gelombang per detiknya.

Panjang gelombang (lambda):

Panjang gelombang adalah jarak untuk setiap siklus gelombang (wavelength). Panjang gelombang bisa didapat dengan mengukur jarak antara dua puncak berurutan.

Buat lah sebuah 3 fungsi overloading yang memiliki 1 parameter - Jika parameternya int maka outputnya “Tipe data int, hasilnya [parameter * 100]” - Jika parameternya double maka outputnya “Tipe data double, hasilnya [parameter * 1000]” - Jika parameternya string maka outputnya “Tipe data string, nilainya Hello [parameter]” contoh pada fungsi main : int main() { overloadingFunc(10; overloadingFunc(10.0); overloadingFunc("World); return 0; } outputnya : Tipe data int, hasilnya 10000 Tipe data float, hasilnya 1000 Tipe data string, nilainya Hello World


Jawaban dan Penjelasan:

Overloaded Functions

Saya buat 2 versi. Yang pertama, argumen bertipe string menggunakan tipe "string" dari namespace std. Karena ada baris "use namespace std;", maka nama tipenya "string" saja, tidak perlu "std::string".

Versi kedua, argumen bertipe string menggunakan "char*". Pada saat passing argumen ke fungsi, harus menggunakan casting berupa "(char*)", agar tidak ada warning pada saat kompilasi.

Versi 1: tipe data string menggunakan string dari namespace std (std::string)

#include <iostream>

using namespace std;

void overloadingFunc(int i) {

   cout << "Tipe data int, hasilnya " << (100*i) << endl;

}

void overloadingFunc(double d) {

   cout << "Tipe data double, hasilnya " << (1000*d) << endl;

}

void overloadingFunc(string s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc("World");

   return 0;

}

Versi 2: tipe data string menggunakan char*

Bagian di atas ini persis sama seperti di atas.

void overloadingFunc(char* s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc((char*)"World");

   return 0;

}

33. Jelaskan parameter parameter tingkat pelaksanan demokrasi dalam suatu negara


PARAMETER DEMOKRASI
Prinsip-prinsip Negara demokrasi yang telah tersebutkan di atas kemudian dituangkan dalam konsep yang praktis untuk dapat diukur dan dicirikan. Ciri-ciri inilah yang kemudian dijadikan parameter untuk mengukur tingkat pelaksanaan demokrasi suatu Negara dalam menjalankan tata pemerintahanya sehingga dikatakan demokratis atau tidak, ada 4 aspek dalam mengukur hal ini, yaitu:
1) masalah pembentukan Negara.
2) dasar kekuasaan Negara.
3) susunan kekuasaan Negara.
4) masalah kontrol rakyat.
Menurut Djuanda Wijaya parameter kehidupan demokratis adalah sebagai berikut:
a. dinikmati dan dilaksanakan hak serta kewajiban politik oleh masyarakat berdasarkan prinsip-prinsip dasar HAM yang menjamin adanya kebebasan, kemerdekaan, dan rasa merdeka.
b. penegakan hukum yang mewujud pada supremasi hukum.
c. kesamaan hak dan kewajiban anggota masyarakat.
d. kebebasan pers yang bertanggung jawab.
e. pengakuan pada hak minoritas.
f. pembuatan kebijakan Negara yang berlandaskan asas pelayanan, pemberdayaan, dan pencerdasan.
g. system kerja yang kooperatif dan kolaboratif.
h. keseimbangan dan keharmonisan. 
i. tentara yang professional.
j. lembaga peradilan yang independent.
Amien Rais menambahkan kriteria lain dalam parameter demokrasi, yaitu:
a) adanya partisipasi dalam pengambilan keputusan. 
b) distribusi pendapatan secara adil.
c) kesempatan memperoleh pendidikan.
d) ketersediaan dan keterbukaan informasi.
e) mengindahkan fatsoen politik.
f) kebebasan individu.
g) semangat kerjasama.
h) hak untuk protes.
Pendapat berikutnya adalah pendapat dari Sri Soemantri yang menyatakan bahwa:
1. hukum diterapkan dengan persetujuan wakil rakyat yang dipilih secara bebas.
2. hasil pemilu dapat menyebabkan pergantian orang-orang dalam pemerintahan.
3. pemerintahan harus terbuka.
4. kepentingan minoritas harus dipertimbangkan.
Frans Magnis Suseno juga berpendapat bahwa parameter Negara demokrasi adalah:
1. Negara terikat hukum.
2. kontrol efektif pemerintahan oleh rakyat.
3. pemilu yang bebas.
4. adanya jaminan terhadap hak-hak minoritas.
Sedangkan menurut G. Bingham Powell Jr. menurutnya parameter Negara demokratis adalah:
1. pemerintah mengklaim mewakili hasrat para warganya.
2. klaim berdasarkan pada adanya pemilihan kompetitif secara berkala antara calon alternative.
3. partisipasi warga dewasa sebagai calon dipilih dan pemilih.
4. pemilihan bebas.
5. adanya kebebasan dasar warga negaranya.
Selanjutnya Affan Ghofar (pakar politik UGM) sejumlah prasyarat untuk mengamati apakah sebuah political order (pemerintahan) merupakan system yang demokratik atau tidak melalui ukuran:
1. akuntabilitas.
2. rotasi kekuasaan. 
3. recruitment politik.
4. pemilihan umum. 
5. adanya pengakuan dan perlidungan hak-hak dasar.
Kelima elemen tersebut berlaku secara universal di dalam melihat demokratis tidaknya suatu rezim pemerintahan (political order).
