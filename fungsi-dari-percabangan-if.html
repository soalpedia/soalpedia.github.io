1. jelaskan penggunaan fungsi nested IF (IF bercabang)!​


Jawaban:

Struktur percabangan if bersarang / nested if merupakan struktur if yang paling komplek, karena merupakan perluasan dan kombinasi dari berbagai struktur if lainnya. Konsep dari percabangan ini adalah terdapat Struktur If yang berada didalam Struktur If lainnya. Artinya dalam pernyataan If bersarang jika kondisi If yang paling luar (paling atas) bernilai benar, maka kondisi If yang berada didalamnya baru akan dilihat (di cek).


2. fungsi IF bercabang sering disebut dengan istilah Nested IF ​


Yoi, nested if ini sendiri biasanya digunakan untuk mencari kondisi dalam kondisi apabila kondisi pertama terpenuhi, contoh nya saya ingin mengecek apabila suatu input merupakan integer dan jika iya, saya ingin tahu jika integer tersebut lebih dari sama dengan 10 atau kurang.

penulisan dalam python nya seperti ini:

n = adalah input dari user

if n >= 10:

print("bilangan lebih dari 10")

else:

print("bilangan kurang dari 10")

else:

print("input bukanlah sebuah angka")

12. Fungsi IF bercabang dalam penerapannya dapat disederhanakan menggunakan fungsi .....


Fungsi IF bercabang dalam penerapannya dapat disederhanakan menggunakan fungsi switch. Switch menggunakan case pada percabangannya.

Pembahasan

Fungsi SWITCH adalah fungsi yang digunakan untuk memilih satu dari beberapa opsi yang mungkin berdasarkan nilai atau ekspresi yang diberikan. Fungsi SWITCH bekerja dengan cara mengevaluasi sebuah ekspresi atau nilai, kemudian memeriksa nilai tersebut terhadap beberapa opsi yang telah ditentukan sebelumnya. Jika nilai tersebut cocok dengan salah satu opsi, maka fungsi akan mengembalikan nilai yang sesuai dengan opsi tersebut.

Fungsi SWITCH dapat digunakan untuk menggantikan pernyataan IF bercabang yang lebih kompleks, terutama ketika ada banyak kondisi yang harus dipenuhi. Dengan menggunakan fungsi SWITCH, kita dapat membuat kode yang lebih mudah dibaca dan dipahami, serta lebih efisien dan singkat.
