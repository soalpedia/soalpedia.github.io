6. diketahui f(x)=√1-x dan g(x)=x². tentukan (a) domain dan range fungsi f(x). (b) domain dan range fungsi g. (c) domain FoG​


Jika diketahui fungsi  [tex]f(x) = \sqrt{1- x}[/tex] dan g(x) = x² , maka

a. domain dan range dari fungsi [tex]f(x) = \sqrt{1 - x}[/tex] adalah Df = {x| x ≤ 1, x ∈ R} dan Rf = {y| y ≥ 0, y ∈ R}.

b. domain dan range dari fungsi g(x) = x² adalah Dg = {x| x ∈ R} dan Rg = {y| y ≥ 0, y ∈ R }.

c. domain fungsi [tex]fog(x) = \sqrt{1 - x^2}[/tex] adalah [tex]D_{fog(x)}[/tex] = {x| -1 ≤ x ≤ 1, x ∈ R}.

Pembahasan

Suatu fungsi dapat dinyatakan dalam bentuk: f : x ---> f(x).

Nilai fungsi untuk setiap nilai x yang diberikan dapat dihitung dengan cara mensubstitusikan nilai x ke dalam rumus fungsi f(x).

Domain (daerah asal) fungsi adalah himpunan nilai-nilai agar suatu fungsi terdefinisi.

Daerah hasil fungsi (range) adalah himpunan nilai-nilai fungsi dari setiap anggota daerah asal (domain).

Fungsi komposisi adalah suatu penggabungan fungsi dari operasi dua jenis fungsi yaitu f(x) dan g(x) sehingga menghasilkan fungsi baru.

Operasi fungsi komposisi dinotasikan dengan "o" dibaca sebagai bundaran. Fungsi baru yang terbentuk dari fungsi f(x) dan fungsi g(x) sebagai berikut.

(fog(x)) = f(g(x)) ---> maksudnya adalah nilai x pada fungsi f(x) diganti dengan fungsi g(x) (gof(x)) = g(f(x)) ---> maksudnya adalah nilai x pada fungsi g(x) diganti dengan fungsi f(x)

Penyelesaian

diket:

[tex]f(x) = \sqrt{1 - x}[/tex]

g(x) = x²

ditanya:

a. domain dan range fungsi f(x)....?

b. domain dan range fungsi g(x)....?

c. domain fungsi fog(x)...?

jawab:

a. - mencari domain fungsi f(x)

Karena fungsi [tex]f(x) = \sqrt{1 - x}[/tex] berbentuk [tex]\sqrt{a}[/tex] , maka agar fungsi tersebut dapat terdefinisi adalah dengan syarat a ≥ 0, sehingga

1 - x ≥ 0

1 ≥ x

x ≤ 1

- mencari range fungsi f(x)

 Karena domain fungsi f(x) adalah x ≤ 1, maka range dapat ditentukan dengan memasukkan nilai x ≤ 1, yaitu 1, 0, -1, -2, -3 .... dan seterusnya.

misal

 x = 1, maka  [tex]f(1) = \sqrt{1 - 1} = 0[/tex]

 x = 0, maka  [tex]f(0) = \sqrt{1-0} = 1[/tex]

 dan seterusnya

 Karena nilai range paling kecil adalah 0, maka range dapat dituliskan sebagai berikut.

Rf = {x| x ≥ 0, x ∈ R}

b. - mencari domain fungsi g(x)

     Karena fungsi g(x) berbentuk fungsi kuadrat yaitu g(x) = x², maka domain untuk fungsi g(x) adalah semua anggota real.

  - mencari range fungsi g(x)

    Karena domain fungsi g(x) semua anggota real, maka daerah hasil (range) fungsi g(x) juga semua anggota real.

c. - mencari fungsi fog(x) lebih dulu

     fog(x) = f(g(x))

               = [tex]\sqrt{1 - x^2}[/tex]

   

    - mencari domain fungsi fog(x)

      Karena fungsi [tex]fog(x) = \sqrt{1 - x^2}[/tex] berbentuk [tex]\sqrt{a}[/tex] , maka agar fungsi tersebut dapat terdefinisi adalah dengan syarat a ≥ 0, sehingga

      1 - x² ≥ 0

      x² - 1 ≤ 0

      (x - 1)(x + 1) ≤ 0

       x - 1 = 0     atau x + 1 = 0

           x = 1                     x = -1

   Tuliskan nilai x pada garis bilangan

   ++++++++--------------+++++++++

                 -1    ≤ x ≤    1

   sehingga domain fungsi fog(x) adalah [tex]D_{fog(x)}[/tex] = {x| -1 ≤ x ≤ 1, x ∈ R}.

Kesimpulan

Jadi,

a. domain dan range dari fungsi [tex]f(x) = \sqrt{1 - x}[/tex] adalah Df = {x| x ≤ 1, x ∈ R} dan Rf = {y| y ≥ 0, y ∈ R}.

b. domain dan range dari fungsi g(x) = x² adalah Dg = {x| x ∈ R} dan Rg = {y| y ≥ 0, y ∈ R }.

c. domain fungsi [tex]fog(x) = \sqrt{1 - x^2}[/tex] adalah [tex]D_{fog(x)}[/tex] = {x| -1 ≤ x ≤ 1, x ∈ R}.

20. Saya mau bertanya matematika tentukan domain dan range dri f(x)= 2x/1-x^2 dan f(x)= akar 2x-4 f(x)= akar x/x+1


Domain, yang penting apapun itu tidak akan menghasilkan bilangan real jika x adalah bilangan tersebut.

Untuk:
[tex]\displaystyle f(x)=\frac{2x}{1-x^2}[/tex]
Syarat domain:
Penyebut tidak boleh sama dengan nol.
0 ≠ 1 - x²
0 ≠ (1+x)(1-x)
Sehingga, x ≠ -1 atau, x ≠ 1

Serta untuk range:
Adalah seluruh bilangan real, kecuali untuk x mendekati tak hingga atau negatif tak hingga.
Dengan limit:
lim_x⇒∞ 2x/(1-x²)
= lim_x⇒∞ 2/(1/x - x)
= 2/(0 - ∞)
= 2/-∞
= 0

Tetapi untuk x = 0 akan menghasilkan f(x) = 0 sehingga untuk range 0 ada di x = 0

Sehingga:
Domain = {x | x ≠ {-1,1}, x E R}
Range = {f(x) | f(x) E R}

Fungsi kedua:
f(x) = √[2x- 4]

Domainnya adalah bagaimana yang di dalam akar tidak boleh negatif.
0 ≤ 2x - 4
4 ≤ 2x
2 ≤ x
x ≥ 2

Dengan asal absis minimumnya dari x = 2
Substitusikan:
f(2) = √[2(2) - 4] = 0

Dan untuk f(∞) akan menghasilkan tak hingga.

Sehingga:
Domain : {x | x ≥ 2, x E R}
Range : {f(x) | x ≥ 0, x E R}

Untuk fungsi ketiga:
[tex]\displaystyle f(x)=\sqrt{\frac{x}{x+1}}[/tex]

Domain, agar yang di dalam akar tidak negatif.
0 ≤ x/(x+1)
Dengan garis bilangan diperoleh:
Domain : {x | x < -1 atau x ≥ 0, x E R}

Karena untuk x < -1 atau x ≥ 0 menghasilkan nilai real.
Tentukan nilainya untuk x mendekati tak hingga atau negatif tak hingga:
lim_x⇒∞ √x/(x+1)
= √[lim_x⇒∞ x/(x+1)]
= √[lim_x⇒∞ 1/(1+1/x)]
= √[1/(1+0)]
= 1
Mengingat akar kuadrat selalu menghasilkan bilangan tak negatif di domainnya

Maka, rangenya:
Range : {f(x) | 0 ≤ x < 1 atau x > 1, x E R}

21. tentukan domain dan range dari fungsi f(x)=√3-|x-2|​


Jawaban:

Untuk menentukan domain dan range dari fungsi f(x) = √3 - |x - 2|, kita akan menguraikannya satu per satu:

1. Domain adalah himpunan semua nilai x yang memenuhi fungsi tersebut. Dalam hal ini, akar kuadrat (√) akan memiliki nilai nyata hanya jika argumennya (nilai di dalam akar) non-negatif. Artinya:

x - 2 ≥ 0

x ≥ 2

Jadi, domain dari fungsi ini adalah semua nilai x yang lebih besar atau sama dengan 2, yaitu [2, ∞).

2. Range adalah himpunan semua nilai f(x) yang dihasilkan oleh fungsi tersebut. Untuk mengidentifikasi range, kita perlu mempertimbangkan dua kasus:

a. Ketika x < 2: Dalam hal ini, |x - 2| akan negatif atau nol, dan f(x) = √3 - |x - 2| akan menghasilkan nilai f(x) tertinggi ketika |x - 2| = 0. Oleh karena itu, f(x) akan mencapai nilai maksimum √3 ketika x < 2.

b. Ketika x ≥ 2: Dalam hal ini, |x - 2| akan positif, sehingga f(x) = √3 - |x - 2| akan menghasilkan nilai minimum 0 (ketika |x - 2| = 0). Selain itu, f(x) tidak akan pernah lebih besar dari √3.

Jadi, range dari fungsi ini adalah [0, √3).

Jadi, domain fungsinya adalah [2, ∞) dan range fungsinya adalah [0, √3).

30. Tentukan domain dan range dari fungsi f(x) = -2 + x²


Jika diketahui fungsi  f(x) = -2 + x² , maka domain dan range dari fungsi tersebut adalah Df = {x| x ∈ R} dan Rf = {y| y ∈ R}.

Pembahasan

Suatu fungsi dapat dinyatakan dalam bentuk: f : x ---> f(x).

Nilai fungsi untuk setiap nilai x yang diberikan dapat dihitung dengan cara mensubstitusikan nilai x ke dalam rumus fungsi f(x).

Domain (daerah asal) fungsi adalah himpunan nilai-nilai agar suatu fungsi terdefinisi.

Daerah hasil fungsi (range) adalah himpunan nilai-nilai fungsi dari setiap anggota daerah asal (domain).

Penyelesaian

diket:

f(x) = -2 + x²

ditanya:

domain dan range fungsi f(x)....?

jawab:

- mencari domain fungsi f(x)

 Karena fungsi f(x) berbentuk fungsi kuadrat yaitu f(x) = -2 + x² , maka domain untuk fungsi f(x) adalah semua anggota real.

  - mencari range fungsi f(x)

    Karena domain fungsi f(x) semua anggota real, maka daerah hasil (range) fungsi f(x) juga semua anggota real.

Kesimpulan

Jadi, domain dan range dari fungsi f(x) = -2 + x² adalah Df = {x| x ∈ R} dan Rf = {y| y ∈ R}.
