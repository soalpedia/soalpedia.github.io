14. tombol bertanda logo windows pada keyboard adalah tombol shortcut yang berfungsi untuk...

Windows Logo Key berada di antara tombol Alt dan Fn/ Ctrl yang berada di sebelah kiri bawah bagian keyboard. Nah, berikut beberapa fungsi shortcut yang bisa digunakan dengan bantuan Windows Logo Key. 

1. Windows: untuk menampilkan Start menu.

2. Windows Logo + D: untuk menampilkan desktop, meminimasi aplikasi windows yang terbuka atau mengembalikan (restore) semua aplikasi windows yang terbuka.

3. Windows Logo + E: untuk membuka Windows Explorer.

4. Windows Logo + F: untuk menampilkan menu pencarian file (Search Results).

5. Windows Logo + Ctrl + F: untuk menampilkan menu pencarian "Search Results - Computer".

6. Windows Logo + F1: untuk membuka menu "Help and Support Center".

7. Windows Logo + R: untuk membuka kotak dialog "Run".

8. Windows Logo + break: untuk membuka kotak dialog "System Properties".

9. Windows Logo + M: untuk meminimasi semua aplikasi windows yang sedang dibuka atau dengan kata lain kembali ke desktop.

10. Windows Logo + shift + M: kebalikan dari Windows Logo + M, yakni untuk meng-undo semua windows yang diminimasi.

11. Windows Logo + L: untuk mengunci workstation

12. Windows Logo + U: untuk membuka menu "Utility Manager". 

13. Windows + tab, layar komputer akan terganti menjadi 3D.

15. Setelah selesai, klik ikon/tombol-tombol berikut dan tentukan kegunaan objek-objek dalam windows calculator.Mohon jelaskan kegunaan/fungsinya ​


Icon atau tombol-tombol yang terdapat pada Windows calculator berdasarkan gambar, yaitu:

Tombol X berfungsi untuk melakukan operasi perkalian.Tombol - berfungsi untuk melakukan operasi pengurangan. Tombol yang berupa persegi empat, berfungsi untuk membuka jendela kalkulator yang sedang digunakan.Tombol yang berupa persegi empat yang berhimpitan, berfungsi untuk memperlihatkan semua jendela kalkulator yang ada.Tombol √, berfungsi untuk untuk melakukan operasi pengakaran.Tombol tanda silang dalam tanda panah, berfungsi untuk menghapus data yang telah diinput sebelumnya.Tombol %, berfungsi untuk melakukan operasi persentase.Tombol ÷, berfungsi untuk melakukan operasi pembagian.Tombol =, berfungsi untuk menampilkan hasil operasi perhitungan.Tombol [tex]x^2[/tex], berfungsi untuk melakukan operasi perpangkatan.

Pembahasan

Windows calculator adalah aplikasi yang terdapat pada windows yang berfungsi sama dengan kalkulator biasanya. Dalam windows calculator ada beberapa mode calculator yang dapat digunakan, yaitu:

Mode Standar untuk operasi matematika dasar seperti menambah, mengurangi, mengalikan, dan membagi, akar kuadrat, menghitung persentase, dan pecahan. Mode ilmiah berisi fungsi seperti log, modulo, eksponen, trigonometri derajat, dan SIN, COS, dan TAN.Mode pemrogram untuk berbagai sistem angka-biner, desimal, heksadesimal, dan oktal.Mode perhitungan tanggal untuk menghitung perbedaan antara dua tanggal tertentu.

18. Jelaskan fungsi-fungsi dari bagian-bagian windows dibawah ini? a. Shortcut b. Tombol start c. Tombol pencarian d. Taskbar e. Background deskop​


Jawaban:

Misalkan pada Windows 10:

a. Shortcut berfungsi untuk memasuki program/ aplikasi secara cepat. Pengguna windows dapat membuka program/ aplikasi dg melakukan dobel klik mouse di shortcut yg tampil di desktop windows

b. Tombol Start berfungsi sebagai tombol yg mulai diklik paling awal bila pengguna Windows ingin membuka program tertentu yg tidak tersedia shortcutnya di desktop windows.

c. Tombol pencarian berfungsi untuk menampilkan kotak pencarian data/ file tertentu (baik file dokumen maupun file sistem) yg diinginkan.

d. Taskbar berfungsi sebagai tempat untuk menampilkan program yg sedang aktif sekaligus sebagai tempat bagi program/ aplikasi dalam kondisi minimize.

e. Background desktop berfungsi sebagai gambar latar belakang tampilan utama Windows.

Penjelasan:

Untuk lebih jelasnya, bisa memperhatikan gambar lampiran:

a. Shortcut dapat ditambah oleh pengguna dg cara klik kanan area desktop dan klik New - Shortcut (ditunjukkan tanda panah merah)

b. Tombol Start ditunjukkan tanda panah putih

c. Tombol Pencarian ditunjukkan tanda panah kuning

d. Taskbar adalah bagian yg ditunjukkan tanda panah coklat

e. Background desktop bisa diatur dg cara klik kanan area desktop dan klik Personalize sehingga akan tampil kotak dialog settings (ditunjukkan tanda panah hitam).
