diketahui :

a = 38

b = -4

Un = 26

ditanyakan :

n = ...?

jawab :

Un = a + (n - 1) x b

26 = 38 + (n - 1) x -4

26 = 38 + -4n + 4

26 - 38 - 4 = -4n

-14 = -4n

n =14-4

n = 10

10. Tuliskan operasi aritmatik menghitung luas lingkaran ​


Jawaban:

Rumus Luas Lingkaran, Contoh Soal dan Penjelasannya

Luas lingkaran (L) = πr2 atau π x r x r.

Diameter lingkaran (d) = 2 x jari-jari = 2r.

Jari-jari lingkaran (r) = ½ x diameter lingkaran.

Keliling lingkaran (K) = 2 x π x r = 2πr atau Keliling lingkaran = π x diameter lingkaran.
