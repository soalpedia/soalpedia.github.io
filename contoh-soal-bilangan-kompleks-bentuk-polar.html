3. nyatakan bilangan kompleks 2+2i dalam bentuk polar​


Bentuk polar dari bilangan kompleks a + bi dapat dinyatakan dengan:

[tex]r(\cos \theta +i \sin \theta )[/tex] . Bentuk polar dari bilangan kompleks 2 + 2i adalah [tex]2\sqrt{2} (\cos \frac{\pi }{4} +i\sin \frac{\pi }{4} )[/tex].

Penjelasan dengan langkah-langkah:

Diketahui:

Bilangan kompleks 2+2i

Ditanya:

Nyatakan dalam bentuk polar!

Pembahasan:

Bentuk polar dari bilangan kompleks a + bi dapat dinyatakan dengan:

[tex]r(\cos \theta +i \sin \theta )[/tex] dengan

[tex]r=\sqrt{a^2+b^2}[/tex]

[tex]\tan \theta = \frac{b}{a}[/tex]

Dari bilangan komplek 2 + 2i diperoleh

a = 2

b = 2

Maka,

[tex]r=\sqrt{2^2+2^2}\\ r=2\sqrt{2}[/tex]

[tex]\tan \theta = \frac{2}{2}=1\\\theta = \arctan 1=\frac{\pi }{4}[/tex]

Bentuk polar:

[tex]2\sqrt{2} (\cos \frac{\pi }{4} +i\sin \frac{\pi }{4} )[/tex]

Bentuk Eksponen: z = [tex]r {e}^{j \theta}[/tex]

dimana

r adalah modulus dari z (rumus: r = |z| = √(x² + y²))

θ adalah argumen dari z, diperoleh dari tan θ = [tex]\frac{y}{x}\\[/tex]

[tex]\\[/tex]

Bentuk polar:

z = 2√2 (cos 315° + j sin 315°) atau

θ = arc tan ([tex]\frac{4}{3}\\[/tex]) = 233° = [tex]\frac{233\pi}{180}\\[/tex]

Bentuk polar:

z = 5 (cos 233° + j sin 233°) atau

z = 5 (cos [tex]\frac{233\pi}{180}\\[/tex] + j sin [tex]\frac{233\pi}{180}\\[/tex])

Bentuk eksponen: z = [tex]5 {e}^{j\frac{233\pi}{180}}[/tex]

[tex]\displaystyle\\\bullet~\theta = \tan^{-1}\left(\frac{y}{x}\right), x > 0\\\\\bullet~\theta = \tan^{-1}\left(\frac{y}{x}\right) + \pi, x < 0[/tex]

13. Dapatkan bentuk Polar dan bentuk eksponensial dari bilangan kompleks Z1 = 1-i dan Z2 =2 √ 3 +2i. Terletak di kuadran berapa sudut o bilangan kompleks tersebut​


Terdapat dua bilangan kompleks: z₁ = 1-i dan z₂ = 2√3 +2i. Bentuk polar dari kedua bilangan kompleks tersebut adalah:

[tex]z_1=\sqrt{2}(\text{cos }\frac{7\pi}{4}+i\text{sin }\frac{7\pi}{4})\\z_2=4(\text{cos }\frac{\pi}{6}+i\text{sin }\frac{\pi}{6})[/tex]

sedangkan bentuk eksponensial dari kedua bilangan kompleks tersebut adalah:

[tex]z_1=\sqrt{2}\text{ exp}(\frac{7\pi i}{4})\\z_2=4\text{ exp}(\frac{\pi i}{6})[/tex]

Sudut dari bilangan kompleks z₁ berada di kuadran IV, sedangkan sudut dari bilangan kompleks z₁ berada di kuadran I.

Penjelasan dengan langkah-langkah

Diketahui:

z₁ = 1-i

z₂ = 2√3 +2i

Ditanya: bentuk polar, bentuk eksponensial, dan letak sudut dari z₁ dan z₂

Jawab:

Dari z₁ dan z₂, diperoleh:

x₁ = 1y₁ = -1x₂ = 2√3y₂ = 2

Dari tanda positif dan negatif dari masing-masing nilai x dan y bilangan kompleks, diperoleh bahwa z₁ berada di kuadran IV dan z₂ berada di kuadran I. Mari hitung nilai modulus dari masing-masing bilangan kompleks.

[tex]r_1=\sqrt{1^2+(-1)^2}=\sqrt{1+1}=\sqrt{2}\\r_2=\sqrt{(2\sqrt{3})^2+2^2}=\sqrt{12+4}=\sqrt{16}=4[/tex]

Lalu, tentukan sudut masing-masing bilangan kompleks.

[tex]\theta_1=\text{tan}^{-1}(\frac{-1}{1})=\text{tan}^{-1}(-1)=\frac{7\pi}{4}\text{ (karena ada di kuadran IV)}\\\theta_2=\text{tan}^{-1}(\frac{2}{2\sqrt{3}})=\text{tan}^{-1}(\frac{1}{\sqrt{3}})=\text{tan}^{-1}(\frac{1}{3}\sqrt{3})=\frac{\pi}{6}\text{ (karena ada di kuadran I)}[/tex]

Dari nilai modulus dan sudut tersebut, mari tentukan bentuk polarnya.

[tex]z_1=\sqrt{2}(\text{cos }\frac{7\pi}{4}+i\text{sin }\frac{7\pi}{4})\\z_2=4(\text{cos }\frac{\pi}{6}+i\text{sin }\frac{\pi}{6})[/tex]

Setelah itu, mari tentukan juga bentuk eksponensialnya.

[tex]z_1=\sqrt{2}\text{ exp}(\frac{7\pi i}{4})\\z_2=4\text{ exp}(\frac{\pi i}{6})[/tex]

19. bilangan kompleks Z = 2 + j4 rubah ke bentuk polar


Untuk mengubah bilangan kompleks Z = 2 + j4 ke dalam bentuk polar, kita perlu menghitung magnitudo (r) dan sudut fase (θ) dari bilangan tersebut.

Magnitudo Z (r) dapat dihitung menggunakan rumus:

r = √(a² + b²)

dimana a dan b adalah bagian real dan imajiner dari bilangan kompleks Z, yaitu:

a = 2 dan b = 4

maka r = √(2² + 4²) = √20 = 2√5

Sudut fase Z (θ) dapat dihitung menggunakan rumus:

θ = tan⁻¹(b/a)

maka θ = tan⁻¹(4/2) = tan⁻¹(2) = 63,43°

Sehingga Z dalam bentuk polar adalah:

Z = 2√5 ∠ 63,43°


20. tuliskan bentuk polar dari bilangan kompleks berikut : Z = [tex] - 1 - \sqrt{3} [/tex]​


Jawaban:

Bentuk polar dari bilangan kompleks Z = -1 - √3 dapat dicari dengan menghitung modulus dan argumen bilangan tersebut.

Modulus bilangan kompleks Z dapat dihitung dengan menggunakan rumus:

|Z| = √(Re(Z)² + Im(Z)²)

Dalam hal ini, Re(Z) = -1 dan Im(Z) = -√3, sehingga:

|Z| = √((-1)² + (-√3)²) = √(1+3) = √4 = 2

Argumen bilangan kompleks Z dapat dihitung dengan menggunakan rumus:

Arg(Z) = arctan(Im(Z) / Re(Z))

Dalam hal ini, Im(Z) = -√3 dan Re(Z) = -1, sehingga:

Arg(Z) = arctan((-√3) / (-1)) = arctan(√3)

Untuk menghitung arctan(√3), kita dapat menggunakan identitas segitiga 30-60-90. Dalam segitiga tersebut, rasio antara sisi yang bersebrangan dengan sudut 30° dan sudut 60° adalah √3 : 1. Maka, arctan(√3) = 60°.

Jadi, bentuk polar dari bilangan kompleks Z = -1 - √3 adalah:

Z = 2cis(60°) atau Z = 2(cos 60° + i sin 60°)

sedangkan..

Penjelasan dengan langkah-langkah:

Untuk mendapatkan bentuk polar dari bilangan kompleks Z1 = 3+4i, kita dapat menggunakan rumus:

r = √(a^2 + b^2)

θ = arctan(b / a)

dengan a dan b adalah bagian real dan imajiner dari bilangan kompleks.

Diketahui a = 3 dan b = 4. Sehingga:

r = √(a^2 + b^2) = √(3^2 + 4^2) = √(9 + 16) = √25 = 5

θ = arctan(b / a) = arctan(4 / 3)

Kita dapat mencari nilai θ dengan menggunakan kalkulator. Sehingga, θ ≈ 0.93 radian.

Maka, bentuk polar dari Z1 adalah (5, 0.93).

Untuk mendapatkan bentuk eksponensial dari Z1, kita dapat menggunakan rumus:

Z = r x e^(iθ)

dengan r dan θ adalah nilai yang sudah kita dapatkan sebelumnya.

Sehingga, bentuk eksponensial dari Z1 adalah:

Z1 = 5 x e^(i0.93)

Untuk mendapatkan bentuk polar dari bilangan kompleks Z2 = -1+i, kita dapat menggunakan rumus yang sama:

r = √(a^2 + b^2)

θ = arctan(b / a)

Diketahui a = -1 dan b = 1. Sehingga:

r = √(a^2 + b^2) = √((-1)^2 + 1^2) = √2

θ = arctan(b / a) = arctan(-1)

Kita dapat mencari nilai θ dengan menggunakan kalkulator. Sehingga, θ ≈ -0.79 radian.

Maka, bentuk polar dari Z2 adalah (√2, -0.79).

Z2 = √2 x e^(-i0.79)
