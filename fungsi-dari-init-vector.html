2. apa perbedaan perintah init 0 dan init 6 ?


Penjelasan:

Perkembangan teknologi di era digital seperti zaman  sekarang ini bertumbuh semakin cepat dari waktu ke waktu. Secara tidak langsung penggunaan teknologi ini meningkat secara tajam di kehidupan modern seperti sekarang.

Teknologi adalah suatu sarana atau sistem yang berfungsi untuk memberikan kenyamanan serta kemudahan bagi manusia. Oleh karena itu teknologi menjadi sangat penting di era sekarang, apalagi dengan adanya teknologi digital yang berkembang pesat setiap harinya.

Dalam penggunaan teknologi sistem komputer terdapat istilah init 0 dan init 6, berikut penjelesan mengenai keduanya :

init 0 => Merupakan perintah shutdown melalui consol  yang mana artinya adalah mematikan suatu sistem komputer. Mematikan secara keseluruhan semua komponen dalam komputer seperti RAM, Hardisk, Prosesor dan yang lainya dinonaktifkan karena aliran listrik terputus. Pada saat komputer dihidupkan kembali akan membutuhkan waktu yang lama sebab pada saat Shutdown RAM akan dikosongkan, Komputer akan load semua sistem operasi saat dihidupkan kembali. Shutdown cocok digunakan bila seseorang tidak akan menggunakan komputer lagi dalam jangka waktu yang lama.

init 6 => Merupakan perintah untuk merestart sistem melalui consol jika menggunakan perintah ini, maka komputer akan di matikan (secara shutdown) kemudian dihidupkan kembali

32. Diketahui vector a = 4i – 2j + 2k dan vector b = 2i – 6j + 4k. Proyeksi vector orthogonal vector a pada vector b adalah .....


vektor a = 4i – 2j + 2k

vektor b = 2i – 6j + 4k

a . b = (4).(2) + (-2).(-6) + (2).(4) = 8 + 12 + 8 = 28

|b|² = 2² + [-6]² + 4² = 4 + 36 + 16 = 56

Proyeksi vector orthogonal vector a pada vector b :

= ( [a . b] / |b|² ) . vektor b

= (28 / 56) . vektor b

= ½ . vektor b

= ½ . (2i – 6j + 4k)

= i - 3j + 2k

38. Diketahui persamaan parameter suatu fungsi vector y='2t, x=6t+2 dan -∞​


JAWABAN DAN PENJELASAN :

Karena y='2t dan x=6t+2, maka bentuk parameter dari fungsi vector tersebut adalah v(t) = <6(t+1), 2t>

Dengan menggunakan bentuk ini, kita dapat menentukan nilai vector pada suatu nilai t tertentu. Misalnya, jika t=0, maka v(0) = <6(0+1), 2(0)> = <6,0>. Jadi pada nilai t=0, vector tersebut memiliki nilai (6,0).

Jika nilai t berubah, maka nilai vector juga akan berubah sesuai dengan persamaan parameter v(t). Sebagai contoh, jika t=1, maka v(1) = <6(1+1), 2(1)> = <12,2>. Jadi pada nilai t=1, vector tersebut memiliki nilai (12,2).

Dengan begitu, kita dapat mengetahui bentuk dan nilai vector pada suatu nilai t tertentu menggunakan persamaan parameter v(t) = <6(t+1), 2t>.
