4. menentukan nilai variabel jika diketahui 3 bilangan dengan variabel adalah tripel pythagoras


Contoh soal menentukan variabel jika diketahui tiga bilangan dengan variabel merupakan triple phytagoras. Berikut akan diberikan tiga contoh soal menentukan variabel pada panjang sisi segitiga siku-siku. Simak pembahasan berikut.

Pembahasan

Pada segitiga siku-siku berlaku teorema phytagoras: " kuadrat sisi miring (hipotenusa) sama dengan jumlah kuadrat sisi siku-sikunya".

Teorema phytagoras dapat dirumuskan sebagai berikut:

x² + y² = r²r² - y² = x²r² - x² = y²

dengan:

r = sisi miring

x, y = sisi siku-siku

Contoh soal triple phytagoras dengan variabel.

1. Apabila bilangan a + 4, 4a, dan 4a + 2 membentuk triple phytagoras, tentukan nilai a!

Jawab:

a + 4, 4a, dan 4a + 2 membentuk triple phyatgoras. Karena 4a + 2 merupakan sisi aling panjang, maka diperoleh sisi miring segitiga siku-siku tersebut adalah 4a + 2.

r = 4a + 2, x = a + 4, y = 4a

x² + y² = r²

(a + 4)² + (4a)² = (4a + 2)²

(a + 4)(a + 4) + 16a² = (4a + 2)(4a + 2)

a² + 4a + 4a + 16 + 16a² = 16a² + 8a + 8a + 4

a² + 4a + 4a + 16 + 16a² - 16a² - 8a - 8a - 4 = 0

a² + 8a  - 16a + 16 - 4 = 0

a² - 8a + 12 = 0

(a - 6)(a - 2) = 0

a = 6 atau a = 2

∴ Maka nilai a yang memenuhi adalah a = 6 atau a = 2.

2. Tiga bilangan t, 2t + 2, dan 2t + 3 membentuk triple phytagoras, tentukan nilai t!

Jawab:

t, 2t + 2, dan 2t + 3 membentuk triple hytagoras. Karena 2t + 3 merupakan sisi yang paling panjang, maka sisi miring diantara ketiga bilangan tersebut adalah 2t + 3. Sehingga diperoleh:

r = 2t + 3, x = t, y = 2t + 2

x² + y² = r²

t² + (2t + 2)² = (2t + 3)²

t² + (2t + 2)(2t + 2) = (2t + 3)(2t + 3)

t² + t² + 4t + 4t + 4 = t² + 6t + 6t + 9

2t² + 8t + 4 = t² + 12t + 9

2t² + 8t + 4 - t² - 12t - 9 = 0

2t² - t² + 8t - 12t + 4 - 9 = 0

t²  - 4t  - 5 = 0

(t - 5)(t + 1) = 0

t = 5 atau t = -1

Ingat! nilai x = t, artinya salah satu panjang sisi segitiga siku-siku tersebut adalah t, maka t tidak boleh bernilai negatif. Sehingga nilai t yang memenuhi adalah t = 5.

∴ Maka nilai t adalah 5.

3. Diketahui bilangan - bilangan 2p, 3p + 3, 5p - 3 membentuk triple phytagoras, tentukan nilai p!

Jawab:

5p - 3 merupakan sisi terpanjang, maka sisi miring dari bilangan triple phytagoras tersebut adalah 5p -3. Sehingga diperoleh:

r = 5p - 3, x = 2p, y = 3p + 3

x² + y² = r²

(2p)² + (3p + 3)² = (5p - 3)²

4p² + (3p + 3)(3p + 3) = (5p - 3)(5p - 3)

4p² + 9p² + 9p + 9p + 9 = 25p² - 15p - 15p + 9

13p² + 18p + 9 = 25p² - 30p + 9

13p² + 18p + 9 - 25p² + 30p - 9 = 0

13p² - 25p² + 18p + 30p + 9 - 9 = 0

- 12p² + 48p = 0   : -12

p² - 4p = 0

p(p - 4) = 0

p = 0 atau p = 4

Ingat! karena x = 2p, artinya salah satu panjang sisi segitiga siku-siku tersebut adalah 2p, sehingga p ≠ 0. Maka nilai yang memenuhi tiga bilangan tersebut adalah p = 4.

∴ jadi nilai p = 4.

10. Tuliskan tiga contoh dari Tripel Pythagoras


Berikut adalah 3 contoh Tripel Pythagoras :

3, 4, 55, 12, 137, 24, 25

Pembahasan

Pada segitiga siku-siku berlaku teorema Pythagoras yang menyatakan bahwa kuadrat sisi miring (hipotenusa) merupakan penjumlahan dari kuadrat sisi-sisi lainnya. Misal a dan b adalah sisi siku-siku dan c adalah sisi miring maka dapat dirumuskan :

c² = a² + b²

Tripel Pythagoras adalah pasangan 3 bilangan asli yang memenuhi teorema Pythagoras. Bilangan terbesar merupakan hipotenusa.

Berikut adalah contoh tripel Pythagoras :

3, 4, 55, 12, 137, 24, 25

Pembuktian :

# 3, 4, 5

5² = 3² + 4²

25 = 9 + 16

25 = 25 (terbukti)

# 5, 12, 13

13² = 5² + 12²

169 = 25 + 144

169 = 169 (terbukti)

# 7, 24, 25

25² = 7² + 24²

625 = 49 + 576

625 = 625 (terbukti)

15. contoh soal dalam kehidupan sehari-hari yang menggunakan tripel Pythagoras​


Berikut adalah beberapa contoh soal dalam kehidupan sehari-hari yang menggunakan teorema Pythagoras:

1. Seorang petani ingin membangun sebuah pagar di sepanjang kedua sisi lapangan sepak bola di kebunnya yang berbentuk persegi panjang. Jika lebar lapangan adalah 40 meter dan panjang lapangan adalah 60 meter, berapa panjang diagonal pagar yang harus dia pasang?

Jawaban: Menggunakan teorema Pythagoras, kita dapat menghitung panjang diagonal dengan rumus d = √(a^2 + b^2). Jadi, panjang diagonal pagar adalah √(40^2 + 60^2) = √(1600 + 3600) = √5200 = 72,11 meter (sekitar).

2. Seorang pelajar ingin memeriksa apakah sebuah lemari berukuran 80 cm x 120 cm dapat masuk melalui pintu dengan lebar 90 cm. Untuk mengetahui ini, dia perlu mengukur diagonal lemari. Apakah lemari itu dapat masuk melalui pintu?

Jawaban: Menggunakan teorema Pythagoras, kita dapat menghitung panjang diagonal lemari dengan rumus d = √(a^2 + b^2). Jadi, panjang diagonal lemari adalah √(80^2 + 120^2) = √(6400 + 14400) = √20800 = 144,22 cm. Karena panjang diagonal lemari (144,22 cm) lebih besar dari lebar pintu (90 cm), maka lemari tidak dapat masuk melalui pintu.

3. Seorang penjahit ingin membuat segitiga siku-siku dengan panjang kaki-kaki 5 cm dan 12 cm. Berapa panjang sisi miring segitiga tersebut?

Jawaban: Menggunakan teorema Pythagoras, kita dapat menghitung panjang sisi miring dengan rumus d = √(a^2 + b^2). Jadi, panjang sisi miring segitiga tersebut adalah √(5^2 + 12^2) = √(25 + 144) = √169 = 13 cm.

4. Seorang arsitek ingin memasang tangga yang akan membentang di antara dua lantai. Jika tinggi langkah tangga adalah 18 inci dan panjang langkah tangga adalah 24 inci, berapa panjang tangga yang harus dia pasang?

Jawaban: Menggunakan teorema Pythagoras, kita dapat menghitung panjang tangga dengan rumus d = √(a^2 + b^2). Jadi, panjang tangga yang harus dipasang adalah √(18^2 + 24^2) = √(324 + 576) = √900 = 30 inci.

5. Seorang surveyor ingin mengukur jarak horizontal antara dua titik di atas bukit. Jika jarak vertikal antara kedua titik adalah 100 meter dan jarak diagonal antara kedua titik adalah 120 meter, berapa jarak horizontal yang harus diukur?

Jawaban: Menggunakan teorema Pythagoras, kita dapat menghitung jarak horizontal dengan rumus a = √(c^2 - b^2). Jadi, jarak horizontal yang harus diukur adalah √(120^2 - 100^2) = √(14400 - 10000) = √4400 = 66,33 meter (sekitar).
