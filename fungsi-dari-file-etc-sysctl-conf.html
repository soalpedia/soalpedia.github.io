Jawaban:

a. /etc/default/isc-dhcp-server.

Penjelasan:

Pengaturan jenis interface yang digunakan dalam layanan DHCP server sebagai hardware yang secara langsung menerima request DHCP Ack dari klien dapat diubah pada file `/etc/default/isc-dhcp-server` (jawaban a).
