1. Sebutkanenu menu perintah yang terdapat pada control menu dan fungsinya


Menu perintah pada control menu dapat dilihat di pojok kanan atas program, yaitu:
1. Minimize: menu untuk meminimalkan tampilan program sehingga hanya aktif di taskbar.
2. Maximize: menu untuk memaksimalkan tampilan program sehingga memenuhi layar, serta mengaktifkan menu Restore.
3. Restore: menu untuk menampilkan program secara biasa (tidak sampai memenuhi layar) dan mengaktifkan menu Maximize.
4. Close: menu untuk menutup tampilan program.

12. Tuliskan 3 perintah yang terdapat pada tab menu file


1. New.
New adalah submenu dalam menu file yang di dalam submenu tersebut terdapat beberapa jenis pemilihan file mana yang akan digunakan. Misalnya saja, blank document, blog post, recent templates, sample templates, my templates dan new from existing. Dari semua hal tersebut, terdapat jenis atau bentuk yang berbeda-beda untuk menulis dokumen. Jadi, tinggal pilih jenis mana yang akan digunakan. Fungsi dari submenu new adalah untuk membuat dokumen baru.
2. Open.
Open adalah salah satu submenu dalam menu file yang pengguna mengklik submenu tersebut maka akan muncul beberapa dokumen atau beberapa file yang sudah disimpan sebelumnya. Selanjutnya, tinggal memilih saja file atau dokumen mana yang akan dibuka. Fungsi dari submenu open dalam menu file ini adalah untuk membuka dokumen atau file yang sudah disimpan sebelumnya.
3. Save.
Save adalah salah satu submenu dalam menu file yang di jika pengguna mengklik submenu tersebut maka pengguna berarti akan menyimpan file atau dokumen mereka. Kemudian, pengguna harus memilih di mana file atau dokumen tersebut akan diletakkan. Pengguna juga dapat memilih dalam bentuk atau tipe apa file atau dokumen mereka akan disimpan. Fungsi dari submenu save dalam menu file ini adalah untuk menyimpan dokumen atau file.

29. jelaskan fungsi dari menu edit pada program Corel draw dan Adobe Photoshop CS​


**CorelDRAW:**

- **Undo/Redo (Ctrl+Z, Ctrl+Y):** Memungkinkan Anda untuk membatalkan tindakan terakhir atau mengembalikannya kembali.

- **Cut (Ctrl+X), Copy (Ctrl+C), Paste (Ctrl+V):** Digunakan untuk memotong, menyalin, dan menempelkan elemen grafis.

- **Clone (Ctrl+K):** Membuat salinan objek yang sedang dipilih.

- **Group (Ctrl+G) dan Ungroup (Ctrl+U):** Mengelompokkan dan mengurai objek untuk mengedit bersama atau secara terpisah.

- **Transformations:** Berisi perintah seperti "Rotate," "Scale," "Mirror," dan lainnya untuk mengubah bentuk atau orientasi objek.

- **Find and Replace (Ctrl+F):** Memungkinkan Anda mencari elemen dan menggantinya dengan elemen lain dalam desain.

- **Object Properties:** Menyediakan akses ke pengaturan dan properti objek yang sedang dipilih.

- **Special Effects:** Mengandung perintah untuk memberikan efek khusus seperti "Drop Shadow," "Blur," dan lainnya.

**Adobe Photoshop CS:**

- **Undo/Redo (Ctrl+Z, Ctrl+Y):** Untuk membatalkan tindakan terakhir atau mengembalikannya.

- **Cut (Ctrl+X), Copy (Ctrl+C), Paste (Ctrl+V):** Digunakan untuk memotong, menyalin, dan menempelkan bagian gambar.

- **Transform (Ctrl+T):** Untuk mengubah ukuran, rotasi, atau transformasi lainnya pada layer atau seleksi.

- **Image Adjustments:** Mengandung perintah untuk menyesuaikan warna, kecerahan, kontras, dan banyak aspek lain dari gambar.

- **Layers:** Memungkinkan Anda mengelola lapisan (layers) yang berisi elemen-elemen gambar yang bisa diatur dan dimanipulasi secara terpisah.

- **Crop (C):** Untuk memotong gambar sesuai dengan area yang diinginkan.

- **Select (Ctrl+A, Ctrl+D, Ctrl+Shift+I):** Untuk memilih bagian tertentu dari gambar.

- **Fill (Shift+F5) dan Stroke (Ctrl+L):** Digunakan untuk mengisi area dengan warna atau pola tertentu, serta mengatur garis tepi.
