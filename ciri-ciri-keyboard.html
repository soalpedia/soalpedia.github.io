12. Jelaskan ciri ciri keyboard yang baik


1. seluruh key berfungsi dengan baik
2. Kondisi keyboard baik
3. Tidak ada kabel yang terkelupas (Dapat menyebabkan keyboard connect / disconnect dengan sendirinya) dan yang terakhir
4. Keyboard tersebut kompatibel dengan OS yg ada di komputer tersebut
1. Seluruh tombol dapat berfungsi sebagaimana mestinya
2. Seluruh tombol lengkap dan kondisi keyboard masih bagus dipandang
3. Jika digunakan, keyboard yang baik akan melakukan tugasnya dengan baik.
4. Kabel USB yang menghubungkan keyboard dengan PC masih dalam keadaan baik sehingga tidak terjadi auto-connect / disconnect
5. Cocok dengan OS yang digunakan

14. Tuliskan dan jelaskan menurut anda, keyboard serial, keyboard ps/2, keyboard wireless, keyboard usb ! bantu jawab ya kak ​


Jawaban:

keyboard serial : keyboard yang memakai koneksi port berjenis serial untuk koneksinya biasa digunakan pada perangkat lama.

Keyboard Ps/2 : Keyboard yang menggunakan konektsi port berjenis ps/2, dalam media koneksi ini keyboard ada yang digabung dengan mouse dan ada yang tidak.

keyboard wireless : keyboard yang menggunakan media nirkabel seperti bluetooth, radiowave,dll untuk berfungsi, keyboard ini memerlukan recharge ketika habis baterai.

keyboard usb : keyboard yang paling umum digunakan saat ini, keyboard ini menggunakan port usb sebagai konektor.
