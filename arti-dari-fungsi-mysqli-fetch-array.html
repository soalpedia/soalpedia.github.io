17. Diberikan sebuah array integer A yang berukuran N elemen. Array A sudah terdefinisi elemen-elemennya. Buatlah fungsi untuk menghitung banyaknya elemen yang kelipatan 5 dan jumlah elemen array yang kelipatan 5.


Jawaban dan Penjelasan:

Mungkin algoritma kedua fungsi berikut ini akan lebih "enak" dilihat pada gambar terlampir. Di sini, sulit untuk membuat format jawaban seperti itu.

Bagian I: Contoh algoritma menghitung banyaknya elemen array yang merupakan kelipatan 5 menggunakan notasi algoritmik

function Banyak_Elemen_Kelipatan_Lima (Input A: array[1..n] of integer) → integer

{ Menghitung banyaknya elemen kelipatan lima dalam array A. }

{ Prekondisi: array A dan n terdefinisi, n > 0 }

Kamus lokal:

   i: integer

   banyak: integer

Algoritma:

   i ← 1

   banyak ← 0

   while (i ≤ n) do

       if (A[i] mod 5 = 0) then         { Jika A[i] habis dibagi 5 }

           banyak ← banyak + 1        { increment “banyak” }

       i ← i + 1                                  { increment i }

   → banyak                                  { mengembalikan nilai “banyak” }

Bagian II: Contoh algoritma menghitung jumlah elemen array yang merupakan kelipatan 5 menggunakan notasi algoritmik

function Jumlah_Elemen_Kelipatan_Lima (Input A: array[1..n] of integer) → integer

{ Menghitung jumlah dari semua elemen kelipatan lima dalam array A. }

{ Prekondisi: array A dan n terdefinisi, n > 0 }

Kamus lokal:

   i: integer

   jumlah: integer

Algoritma:

   i ← 1

   jumlah ← 0

   while (i ≤ n) do

       if (A[i] mod 5 = 0) then          { Jika A[i] habis dibagi 5 }

           jumlah ← jumlah + A[i]       { jumlah = jumlah + A[i] }

       i ← i + 1                                   { increment i }

   → jumlah                                    { mengembalikan nilai “jumlah” }

29. Tulislah dalam bentuk definisi fungsi nilai mutlak![tex]\left. \begin{array} { l } { f ( x ) = | 7 - x | } \end{array} \right.[/tex]​


F(x) = |7 - x|

Definisi Untuk Fungsi Nilai Mutlak memiliki dua Jawaban, Jika lebih besar dari atau sama dengan 0 Maka Jawabannya sesuai dengan Fungsi awalnya. Namun jika Lebih kecil dari 0 maka jawabannya adalah Fungsi Awalnya dikali minus 1.

F(x) = |7 - x|

|7 - x| >= 0

7 - x >=0

7 >= x

x <=7

F(x) = -(7 - x)

7 - x < 0

7 < x

x > 7

Maka Kesimpulannya adalah :

F(x) = 7 - x, Jika x <= 7

F(x) = x - 7, Jika x > 7

F(x) = x - 7, Jika x < 7

Penjelasan dengan langkah-langkah:

f(x) = |7 - x|

Dalam bentuk definisi fungsi nilai mutlak :

(1) f(x) = 7 - x jika 7 - x  ≥ 0 atau -x  ≥ -7 atau x ≤ 7

(2) f(x) = -(7 - x) = -7 + x jika 7 - x < 0 atau -x < -7 atau x > 7

36. apa perbedaan php mysql dengan php mysqli ?


Perbedaan Antara mysql extension Dengan mysqli extension

Sepanjang tutorial PHP MySQL Part 4 sampai 11 di duniailkom ini, kita menggunakan extension mysql dari PHP untuk mengakses MySQL. Fungsi-fungsi yang kita pelajari tersebut (seperti fungsi mysql_connect, mysql_query, dan mysql_fetch_array) memang sudah tidak disarankan lagi, tetapi saya tetap membahasnya sebagai dasar bagi kita untuk masuk ke extension yang lebih baru: mysqli dan PDO.

Terhitung dari PHP versi 5.5, extension mysql tidak lagi disarankan penggunaannya dan berstatus deprecated, yang berarti mungkin akan dihapus pada PHP versi berikutnya. Kita disarankan untuk mulai beralih menggunakan mysqli atau PDO. Mari kita bahas tentang mysqli terlebih dahulu.

Mysqli merupakan kependekan dari MySQL Improved Extension. Seperti yang terlihat dari namanya, extension ini merupakan versi perbaikan dan penambahan dari extension mysql sebelumnya yang umum digunakan. Extension mysqli dibuat untuk mendukung fitur-fitur terbaru dari MySQL Server versi 4.1 keatas.
