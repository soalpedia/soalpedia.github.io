13. Buatlah Rangkuman Perangkat Keras Komputer dan Perangkat Input - Output Komputer​


Perangkat Keras Komputer dan Perangkat Input-Output Komputer

Perangkat Keras komputer secara umum adalah seluruh perangkat fisik yang digunakan untuk menjalankan sebuah operasi komputasi pada komputer. Terdiri dari 3 kelompok, yaitu:

Kelompok Perangkat InputKelompok Perangkat Unit Pemroses (procesing unit)Kelompok Perangkat Output.

1. Kelompok Perangkat Input

Kelompok perangkat input berfungsi untuk memfasilitasi proses input data. Jenis jenisnya bergantung kepada jenis data maupun cara yang dapat dilakukan untuk menginput data, diantaranya:

Keyboard Komputer, untuk menginput data yang paling umum seperti data teks maupun angka.Mouse, untuk menginput data dengan cara mengklik pilihan yang tertera pada layar.Microphone, untuk menginput data suara pada komputer multimediaDrawing Pad, untuk menginput data sketsa gambar yang digoreskan pada drawing pad agar dapat masuk ke unit pemroses.Scanner, untuk menginput data citra hasil pemindaian.Camera, untuk menginput data citra hasil jepretan foto.

Kelompok Perangkat Unit Pemroses

Kelompok perangkat unit pemroses atau biasa juga disebut Central Processing Unit (CPU) adalah kumpulan rangkaian yang minimal terdiri dari Motherboard, Processor, Random Access Memory (RAM), Storage, beragam input dan output port (audio, display, USB, HDMI, serial dan paralel), beragam controller, power supply, beragam slot untuk Network Interface Card, Display Card/Graphic Processing Unit (GPU), serta serangkaian cooling system dan tentunya sebuah chassis yang menjadi media diletakkannya seluruh komponen central processing unit.

Unit ini berfungsi untuk melakukan proses komputasi terhadap objek input dan kemudian mengirimkan hasilnya ke perangkat keras output.

Kelompok Perangkat Output

Kelompok perangkat output berfungsi untuk menampilkan informasi hasil proses komputasi dari unit pemroses. Tampilan informasi ini bisa disajikan dalam beberapa bentuk, diantaranya: tampilan visual di layar, audio pada speaker dan tampilan tercetak. Untuk tampilan di layar perangkat keras yang digunakan monitor. Untuk output berbentuk audio digunakan speaker. Sedangkan untuk tampilan tercetak maka yang digunakan perangkat printer.

32. sebutkan 4 perangkat keras keluaran/output komputer beserta fungsinya


Berikut adalah empat contoh perangkat keras keluaran komputer dan fungsinya:

Monitor. Perangkat ini berfungsi untuk menampilkan hasil proses yang dilakukan oleh komputer dalam bentuk visual.Printer. Perangkat ini berfungsi untuk mencetak data dari komputer menjadi bentuk fisik. Hasil cetak tersebut dapat berupa gambar, tulisan, dan sebagainya.Speaker atau headset. Perangkat ini berfungsi untuk menyalurkan hasil proses yang dilakukan oleh komputer dalam bentuk suara atau audio.Proyektor. Perangkat ini berfungsi untuk memproyeksikan program atau monitor. Biasanya, perangkat ini digunakan untuk aktivitas presentasi.

Pembahasan

Dalam sebuah komputer, ada yang bernama software dan hardware. Software merupakan sebuah perangkat lunak pada jaringan komputer. Software tersebut berupa data yang bertugas untuk mengeksekusi setiap perintah yang ingin dilakukan oleh user. Selain itu, software juga berfungsi sebagai:

Penghubung antara perangkat lunak dan perangkat keras.Pengindentifikasi suatu program.Pengatur hardware agar dapat terus bekerja sesuai fungsinya.

Perangkat keras atau hardware adalah perangkat fisik, termasuk bagian-bagian atau komponen dari komputer. Output device sendiri merupakan salah satu jenis hardware yang berguna untuk menghasilkan keluaran proses perngolahan data, berupa tampilan, suara atau cetakan.

33. 5. Jelaskan hubungan output data komputer dengan perangkat output komputer dalam sistem komputer!​


Jawaban:

Input dan Output sangat berpengaruh dalam interaksi manusia dan komputer, hal ini dikarenakan input dan output merupakan masuk dan keluarnya data melalui bahasa pemrograman yang mana hasil data tersebut (output) memiliki ketergantungan terhadap input.

Penjelasan:

Sebagai contoh, saya masukkan string variabel nama = "Davis". Ketika saya melakukan output yang menampilkan variabel nama, selama tidak diubah pasti "Davis" akan dimunculkan dalam bentuk output.

Dari sini dapat kita lihat bahwa kesimpulannya adalah output dihasilkan dari proses oleh komputer, output memiliki ketergantungan terhadap input, dan input memiliki ketergantungan terhadap manusia. Sehingga, terbentuklah suatu pengaruh ketergantungan antara komputer dengan manusia.
