36. tombol yg berfungsi untuk menscrool layar ke bawah adalah....​


Jawaban:

Page Down/ Page Up

Pembahasan:Page Up

Page Up memiliki fungsi menaikkan atau berpindah halaman ke arah atas. Seperti namanya tombol ini hanya menggerakan halaman atau seluruh tampilan layar. Selain itu Page Up hanya bisa digunakan untuk menaikkan.

Dari pada menggunakan tombol panah atau tombol navigasi kalian akan lebih mudah berpindah halaman dengan Page Up dan Page Down.

Page Down

Kebalikan dari Page Down, Page Down berfungsi untuk berpindah atau menggeser halaman ke arah bawah. Sama persis dengan Page Up hanya saja Page Down ke arah bawah sedangkan Page Up ke arah atas.

Sekilas fungsi tombol Page Up dan Page Down hampir sama seperti tombol Tab. Yaitu dapat menggeser atau berpindah halaman ke atas dan ke bawah lebih jauh dari pada menggunakan tombol navigasi atas dan bawah.
