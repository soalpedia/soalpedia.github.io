37. Perintah untuk menekan tombol kiri mouse satu kali secara cepat dinamakan …


Jawaban:

Perintah untuk menekan tombol kiri mouse satu kali secara cepat dinamakan klik ganda (double click).

Pembahasan:

Double click adalah istilah yang digunakan untuk menggambarkan proses menekan tombol mouse dua kali dengan cepat sambil menahannya. Dalam kebanyakan kasus, klik dua kali dilakukan dengan tombol kiri mouse dan digunakan untuk membuka atau menjalankan file, folder, atau program perangkat lunak.

Misalnya, program dan file di Microsoft Windows dibuka dengan mengklik dua kali ikon. Untuk membuka browser Internet kamu, kamu akan mengklik dua kali pintasan ikon browser.
