14. langkah konfigurasi Access Point tp-link​


Telah terdapat langkah-langkah dalam konfigurasi Access Point TP-LINK yaitu ialah sebagai berikut:

Siapkanlah terlebih dahulu Access Point, kabel LAN, dan juga power dan setelah itu koneksikanlah kabel LAN pada komputer. Bukalah browser seperti Firefox, lalu ketikkanlah IP Address Access Point pada address bar lalu tekan enter.Masukkanlah username dan juga password dari Access Point. Kemudian, akan masuk ke halaman Access Point. Pada tempat ini telah digunakan untuk me-manage ataupun mengkonfigurasi segala macam pengaturan dan juga fungsi dari Access Point kemudian klik Next. Selanjutnya, pilihlah mode Access Point dan klik Next. Pada bagian selanjutnya yaitu ialah tempat untuk mengubah SSID dan juga mengisi password. Pada bagian network setting tidak perlu ada yang dirubah. Setelah itu, akan muncul halaman pemberitahuan maka dari itu pastikanlah semuanya telah benar dan pilih reboot. Kemudian, tunggulah hingga proses reboot telah selesai. Setelah proses reboot telah selesai maka dari itu kita dapat melihat laptop ataupun handphone untuk dapat mengetahui apakah wifinya telah berhasil dibuat. Apabila SSID yang telah diciptakan tersebut telah muncul maka dari itu proses konfigurasinya telah berhasil.Pembahasan

Pengertian dari TP-LINK yaitu ialah suatu perusahaan penyedia dunia untuk perangkat jaringan dan juga aksesoris yang dinilai terpercaya untuk dapat memenuhi segala macam aspek di dalam kehidupan kita sehari-hari.

16. bagaimana cara melakukan sharing data dengan jaringan wireless menggunakan TP-LINK TL-MR3420


Persiapkan Access point TP Link 3420, modem eksternal, kabel jaringan UTP, dan PC/Notebook. Pada TP Link MR 3420 terdapat 4 port berwarna kuning yang dapat disambungkan ke jaringan lokal LAN. Hubungkan Power Supply pada access point ke soket listrik. Hubungkan Access point ke PC dengan menggunakan kabel UTP. Tekan tombol ON pada Access point. Lakukan konfigurasi pada access point.
