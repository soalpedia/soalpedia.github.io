10. Buatlah sketsa grafik fungsi kuadrat y = x² – x – 6 ​


Diketahui :

fungsi kuadrat y = x² -x -6

Ditanya :

Sketsa grafik ... ?

Jawab :

Untuk memulai menggambar sketsa grafik, hal yang harus diperhatikan adalah mencari titik potong sumbu x dan titik potong sumbu y terlebih dahulu.

Titik Potong dengan sumbu x

Karena motong sumbu x, artinya ( y = 0 )

y = x² -x -6

0 = x² -x -6

(x +2)(x -3) = 0

x = -2 → ( -2,0 )x = 3 → ( 3,0 )

Titik potong dengan sumbu y

Karena memotong sumbu y, artinya ( x = 0 )

y = x² -x -6

y = (0)² -(0) -6

y = -6 → ( 0,-6 )

Dengan demikian, kita sudah memperoleh 3 titik ( -2,0 ) , ( 3,0 ) , ( 0,-6 ) sebagai titik awal dari sketsa grafik.

Selanjutnya menentukan titik belok / titik maksimum / titik minimum fungsi. Misal titik belok tersebut adalah [tex](x_1 , y_1)[/tex]

[tex]x_1 = -\frac{b}{2a} = -\frac{-1}{2(1)} = -\frac{1}{2}[/tex]

Subtitusi [tex]x_1[/tex] kedalam fungsi kuadrat y

[tex]y_1 = (x_1)^2 -(x_1) -6\\y_1 = (-\frac{1}{2})^2 - (-\frac{1}{2}) -6\\y_1 = -\frac{25}{4}[/tex]

Diperoleh [tex](x_1,y_1) = (-\frac{1}{2} , -\frac{25}{4})[/tex]

Selanjutnya hubungkan titik-titik yang telah diperoleh secara eksponensial (Melengkung).

titik 1 → [tex](-2,0)[/tex]titik 2 → [tex](3,0)[/tex]titik 3 → [tex](0,-6)[/tex]titik 4 → [tex](-\frac{1}{2},-\frac{25}{4})[/tex]

Maka, didapat suatu sketsa grafik dari fungsi kuadrat y = x² – x – 6 .
