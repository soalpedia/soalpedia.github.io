1. apa perbedaan antara paket aplikasi Squid dengan squid2 pada linux debian


Perbedaan antara squid dengan squid2 pada linux adalah perbedaan beberapa fitur. Pada squid2, kemampuan squid sudah bisa untuk melakukan proxy terhadap protokol https. Selain itu pada squid2 proses caching website lebih baik daripada squid. Dengan menggunakan proxy, kita bisa membatasi akses website mana saja yang bisa kita akses melalui jaringan komputer kita. Selain itu fungsi cache pada proxy akan membuat kita semakin cepat ketika mengakses sebuah website yang sudah sering kita kunjungi sebelumnya.

4. Directory pada linux yang berisi aplikasi-aplikasi executable


Pada Linux terdapat beberapa direktori yang terletak tepat dibawah root direktori, dan merupakan direktori yang penting untuk menjalankan dan mengoperasikan sistem. Beberapa direktori tersebut adalah sebagai berikut

- /bin

- /dev

- /etc

- /home

- /mnt

- /opt

- /sbin

- /tmp

- /usr

- /var


Directory pada Linux yang berisi aplikasi-aplikasi executable adalah directory /bin. Directory /bin mengandung program program yang dibutuhkan untuk menjalankan dan mengatur sistem. Beberapa perintah yang sering digunakan yang terdapat pada direktori ini adalah date ( menampilkan tanggal hari ini ), ls ( menampilkan daftar isi dari direktori ), dan cp ( menghasilkan copy-an dari sebuh file ). Nama bin dipakai untuk direktori ini karena aplikasi executable di UNIX dan Linux disebut dengan binary files.

28. Linux Debian termasuk aplikasi​


Debian adalah salah satu sistem operasi yang berbasis kernel linux. Sehingga sistem operasi debian juga bersifat open source sebagaimana linux.

Pembahasan

Sistem operasi adalah software pusat atau utama untuk menjalankan berbagai aplikasi atau program pada komputer ataupun server. Sistem operasi yang dikenal masyarakat saat ini seperti Windows dari microsoft kemudian mac OS sistem operasi dari apple dan ada juga Linux.

Kemudian Debian sendiri merupakan sistem operasi yang berbasis kernel yang berfungsi untuk menjalankan dari berbagai aplikasi pada perangkat keras komputer seperti proses, memori dan penyimpanan. Sehingga jika dianalogikan dengan sebuah mesin bus dan supirnya, maka Linux adalah busnya itu sendiri, kemudian kerne adalah mesin dan debian adalah supir dari bus tersebut. Mudah-mudahan dapat dipahami.

Demikian, semoga membantu!

36. aplikasi yang di gunakan untuk file sharing pada linux adalah


Samba merupakan aplikasi Unix yang memanfaatkan protokol SMB (Server Message Block). Sebagian sistem operasi memanfaatkan SMB dalam komunikasi client-server-nya, termasuk Windows dan OS/2. Samba memungkinkan mesin Linux berkomunikasi dengan mesin Windows/Dos mapupun OS/2. Samba dapat digunakan untuk :
• Berbagi (share) file dan printer pada semua komputer yang terhubung ke jaringan.
• Client unix/Linux bisa mengakses file system windows
• Client windows bisa mengakses file system Linux dan sebaliknya
• Mengintegrasikan authentikasi antara linux dan Windows ke dalam suatu domain pada suatu jaringan.
• Menempatkan mesin Unix (Linux) sebagai Primary Domain Controller sebagaimana yang dilakukan Windows NT / 2000 server.
