4. pada gambar di atas diketahui POQ=105° dan ROS=45° jika 87,5 cm² luas juring ROS adalahtolong dijawab ya

POQ=105° dan ROS=45° 

Buat perbandingannya
jika α = luas ROS, berarti
87,5 cm² / 105° = α / 45°

Dikali silang
87,5 x 45 = 105 x α

Singkatnya,
α = 87,5 x 45 / 105
α = 37,5

22. bu ros memiliki sejumlah uang.Sepertiganya ia belanjakan di pasar dan seperempat dari sisanya ia gunakan untuk ongkos.Jika sekarang uang bu ros tinggal Rp18.000,maka uang bu ros mula mula adalah...


Permisalan =
X = Uang mula-mula
Y = Sisa uang mula-mula

X - ( 1/3 X ) = Y ( Y = 2/3 X )
Y - ( 1/4 Y ) = 18.000 , dengan kata lain, 18.000 = 3/4 Y

1/4 Y = 18.000 / 3
= 6.000

Y = 18.000 + 6.000
= 24.000 ( Sisa uang belanja yang belum dikurangi ongkos )

Y = 2/3 X
24.000 = 2/3 X

1/3 X = 24.000 / 2
= 12.000

X = 12.000 dikali 3
X = 36.000

Jadi, uang bu Ros mula-mula sejumlah Rp. 36.000, 00

23. Bu Ros membeli gula bertuliskan bruto 50 kg dan tara 2% dengan harga Rp580.000,00. Jika Bu Ros ingin menjual gula dengan harga Rp12.500,00 per kilonya, berapakah keuntungan bu Ros?


Jawab:

Keuntungannya adalah Rp.32.500,-

Penjelasan dengan langkah-langkah:

Keterangan :

Bruto = Berat total

Tara = Berat kemasan

Netto = berat bersih

Diketahui :

Bruto = 50 kg

Tara = 2%

Harga beli = Rp.580.000,-

Harga jual = Rp.12.500,- per kg

Ditanya :

Keuntungan harga jual

Proses Pengerjaan

1. Cari netto (berat bersih)

50 kg × 2% = 1 kg

Netto = 50 kg - 1 kg

= 49 kg

2. Cari keuntungan

(Harga jual × netto ) - harga beli

= (12.500 × 49) - 580.000

= 612.500 - 580.000

= Rp.32.500,-
