1. contoh fungsi genap dan ganjil


Sebuah fungsi bisa dikatakan genap bila bagian kiri mencerminkan bagian kanan:
atau [tex]f(x) = f(-x)[/tex]

Contoh" fungsi:
[tex]f(x) = x^2 \\ f(x) = \cos x[/tex]

Sedangkan fungsi ganjil memiliki simetri putar bila diputar pada posisi (0,0)
atau [tex]f(-x) = -f(x)[/tex]

Contoh" fungsi:
[tex]f(x) = x^3 \\ f(x) = x \\ f(x) = \sin(x)[/tex]

7. Tentukan fungsi genap atau ganjil atau bukan keduanya​


a. f(x) = (x² + 1)/(|x| + x⁴)

f(-x) = ((-x)² + 1)/(|-x| + (-x)⁴)

= (x² + 1)/(|x| + x⁴)

= f(x)

Hint! |-x| = x = |x|

Karena f(-x) = f(x), maka f(x) merupakan fungsi genap

b. g(x) = (sin x) + sin x

g(-x) = (sin (-x)) + sin (-x)

= - sin x - sin x

= -((sin x) + sin x)

= -g(x)

Karena g(-x) = -g(x), maka g(x) merupakan fungsi ganjil

c. F(x) = √x² + 4

F(-x) = √(-x)² + 4

= √x² + 4

= F(x)

Karena F(-x) = F(x), maka F(x) merupakan fungsi genap

14. fungsi genap dan fungsi ganjil tentukan f(×)=-x²+5 merupakan fungsi genap ​


Jawaban:

Untuk menentukan apakah suatu fungsi termasuk fungsi genap atau fungsi ganjil, kita perlu memeriksa apakah f(-x) sama dengan f(x) untuk setiap nilai x di domain fungsi tersebut.

Jika f(-x) = f(x) untuk setiap x, maka fungsi tersebut merupakan fungsi genap. Jika f(-x) = -f(x) untuk setiap x, maka fungsi tersebut merupakan fungsi ganjil. Jika tidak memenuhi keduanya, maka fungsi tersebut bukan fungsi genap maupun fungsi ganjil.

Mari kita periksa f(x) = -x² + 5:

f(-x) = -( -x² ) + 5 (kita ganti x dengan -x)

= x² + 5

Karena f(-x) tidak sama dengan f(x), maka f(x) bukan fungsi ganjil.

Namun, jika kita perhatikan, f(x) juga bukan fungsi ganjil, karena f(-x) bukan sama dengan -f(x).

Dengan demikian, kita dapat menyimpulkan bahwa fungsi f(x) = -x² + 5 tidak termasuk fungsi ganjil maupun fungsi genap.

36. tentukan apakah fungsi f(x) = 3 cos x - 5 termaksud fungsi genap, fungsi ganjil, atau tidak genap dan tidak ganjil???


fungsi genap merupakan fungsi yang memenuhi : [tex]f(x)=f(-x)[/tex]
[tex]f(x)=3 cos x - 5[/tex]
jika [tex]x[/tex] pada fungsi tersebut diganti dengan [tex]-x[/tex], maka didapatkan :
[tex]f(-x)=3 cos (-x) - 5 = 3 cos x - 5[/tex]
karena [tex]f(x) = f(-x)[/tex], dengan demikian fungsi tersebut merupakan fungsi genap

37. apa yang di maksud fungsi genap dan fungsi ganjil​


fungsi genap = fungsi f(x) = [tex]x^{n}[/tex], jika n adalah sebuah interger genap

fungsi ganjil = fungsi f(x) = [tex]x^{n}[/tex], jika n adalah sebuah integer ganjil

integer = adalah tipe data yang terdiri dari angka bulat (tidak mengandung nilai pecahan atau nilai desimal). Nilai ini bisa berbentuk angka positif maupun negatif.
