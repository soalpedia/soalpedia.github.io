1. Suatu batang baja menerima gaya tekan sebesar 500 N, baja tersebut mendapat tegangan normal 20N/m². Berapa luas penampang baja tersebut


Fisika - Elastisitas Bahan


Diketahui :

Gaya tekan (F) = 500 N

Tegangan (τ) = 20 N/m²

Luas Penampang (A) = ...


Kita gunakan rumus tegangan :


τ = F/A


Masukan datanya :


τ = F/A

20 = 500 / A

20 A = 500

A = 500 / 20 = 25 m²

12. Sebuah batang baja berongga berdiameter bagian luarnya 150 [mm], sedangkan tebal batang adalah 1/8 dari diameter luarnya mengalami gaya tekan sebesar 1200 [kN], Tentukan tegangan yang terjadi pada batang tersebut​


Jawaban:

Penjelasan:

Persamaan tegangan adalah:

σ = F/A

Karena gaya sudah diketahui, perlu selanjutnya ditentukan luas area dimana gaya tersebut bekerja. Untuk itu diperlukan menentukan diameter dalam dari batang tersebut.

Untuk menjawab akan digunakan notasi

d = diameter dalam

D = diameter luar

t = tebal

t = (D-d)/2

t = 1/8 D

menggunakan dua persamaan diatas dapat ditulis persamaan

(D-d)/2 = 1/8 D

D - d = 1/4 D

d = 3/4 D

d = 3/4 x 150 = 112,5mm

Luas permukaan tempat gaya bekerja adalah luas bagian luar dikurang luas bagian dalam

A = π(D/2)² - π(d/2)²

A = π/4 (D² -d² )

A ≈ 7730 mm²

Kembali ke persamaan awal untuk mencari tegangan

σ = F/A

σ = 1200 kN / 7730mm²

σ ≈ 155kPa

20. Sebuah batang baja panjangnya (L0) = 1m dipanaskan sehingga batang baja mengalami perubahan suhu (∆T) sebesar 50°C. Berapakah pertambahan panjang batang baja (∆L), jika koefisien muai panjang baja (a) = 11x10-6 (°C)-1


• SUHU DAN PEMUAIAN

→ pemuaian panjang

Data di soal

Panjang awal (Lo) = 1 m

Perubahan suhu (∆T) = 50°C

Koefisien muai panjang (α) = 11 × 10⁻⁶ /C

Jadi berapa perambahan panjangnya?

Pertambahan panjang besi memenuhi persamaan berikut,

∆L = Lo × α × ∆T

∆L = 1 × 11 × 10⁻⁶ × 50

∆L = 550 × 10⁻⁶

∆L = 5,5 × 10⁻⁴ meter

atau

∆L = 0,00055 meter

Jadi, pertambahan panjang besi tersebut adalah 0,00055 meter.

Pertambahan panjang batang baja (ΔL) adalah 5,50 x 10⁻⁴ m atau 0,55 mm sesuai dengan pembahasan sebagai berikut.

PEMBAHASAN

Diketahui

Lo = 1 meter

ΔT = 50°C

α = 11 x 10⁻⁶ /°C

Ditanya

ΔL = ?

PENYELESAIAN

      Pertambahan panjang logam akibat pemuaian panjang dapat dicari menggunakan rumus

        [tex]\boxed{\Delta \text{L} = \text{L}_o \times \alpha \times \Delta \text{T}}[/tex]

      Maka, berdasarkan kondisi pada soal, didapatkan nilai pertambahan panjang batang baja adalah

ΔL = Lo • α • ΔT

ΔL = 1 m • 11 x 10⁻⁶ /°C • 50°C

ΔL = 550 x 10⁻⁶ m

ΔL = 5,50 x 10² x 10⁻⁶ m

ΔL = 5,50 x 10⁻⁴ m

     Jadi, pertambahan batang baja tersebut adalah 5,50 x 10⁻⁴ m.

32. Batang baja pada suhu 0°C panjangnya 35 cm. kemudian batang tersebut dipanaskan hingga suhu 95 °C. jika koefisien muai panjang baja 0.000011/ °C, berapakah panjang batang baja setelah dipanaskan?​


Diketahui:

panjang mula-mula (L0) = 35 cm

koefisien muai panjang (α) = 0,000011 /°C

perubahan suhu (ΔT) = 95°C

Ditanya:

panjang batang baja setelah dipanaskan (L)

Jawab:

ΔL = L0 x α x ΔT

ΔL = 35 x 0,000011 x 95

ΔL = 0,035 cm

sehingga

L = 35 + 0,035 = 35,035 cm

34. batang kuningan hendak dimasukan ke dalam cicin baja. pada suhu 20°C, diameter batang kuningan adalah 8,755 cm dan diameter bagian dalam cicin baja adalah 8,743 cm. agar batang kuningan dapat masuk ke dalam cicin baja, kuningan dan baja tersebut harus dinaikan suhu nya sampai sekitar ...


ini menurutku semoga membantu :)
asumsi : suhu awal kedua benda sama.

misalkan dimuaikan sampai suhu t,

L = Lo' (1 + α' ΔT')

L = Lo (1 + αΔT)

L - L = 0

Lo' (1 + α' ΔT') - (Lo (1 + αΔT)) = 0

8.743(1 + 0.000019(t - 20)) - ( 8.755 (1 + 11 * 10^-6(t - 20))) = 0

t = 191.89°C
