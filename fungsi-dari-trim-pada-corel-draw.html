1. Jelaskan fungsi weld trim dan intersect dalam Corel draw​


Jawaban:

ada di penjelasan

Penjelasan:

A. Weld adalah sebuah alat yang digunakan untuk menggabungkan dua objek atau lebih menjadi satu bidang atau satu outline. Jika dua bidang atau lebih telah diproses weld sudah berhasil, maka jika diberikan warna atau efek lain bidang tersebut akan memiliki satu efek layaknya objek tunggal meskipun bidang tersebut hasil dari gabungan dua objek.

B. Trin digunakan untuk memotong sebagian dari objek menggunakan objek lain. Yang harus diperhatikan adalah waktu menyeleksi bidang untuk di trim, bidang yang pertama adalah berfungsi sebagai pemotong dan yang lain bagian yang dipotong.

C. Intersect digunakan untuk membuat area baru dari dua objek atau lebih yang saling menimpa. Jika weld dan trim tadi tidak menambah objek lain maka intersect ini memiliki kecenderungan menambah objek baru.

35. Jelaskan fungsi toolbar yang ada di dalam corel draw


Maap klo salah
1.Menu Bar : berisikan menu utama yang ada diatas dan banyak digunakan untuk membantu saat bekerja dalam CorelDraw, dimana menu - menu tersebut dikelompokkan sesuai dengan kategorinya. Menu Bar ini terdiri dari File, Edit, View, Layout, Arrange dll
2.Property Bar : menu ini berfungsi untuk menunjukkan properti / nilai dari objek yang ada di tampilan lembar kerja CorelDraw sehingga Property Bar akan selalu berubah - ubah tergantung dari objek yang ditampilkan. Misal lembar kerja kertas secara default adalah letter, coba ubah menjadi A4 maka nilai Property Bar akan berubah
3.Toolbar : menu yang standar dimunculkan dalam ikon - ikon menu seperti undo, redo, copy, paste, save.
4.Title Bar : menu ini berada paling atas dan sebagai penunjuk nama file yang sedang dibuka. Secara default saat membuka awal program CorelDraw X3 maka nama Title Bar'nya yaitu CorelDraw X3 -[Graphic 1]
5.Ruler : sebagai penggaris yang sangat berguna sebagai alat ukur saat bekerja dengan CorelDraw
6.Toolbox : menu sangat penting dalam CorelDraw karena berisikan ikon - ikon menu yang berguna melakukan desain kotak, lingkaran, gradasi, dll. Menu ini biasanya ada disamping kiri dari lembar kerja
7.Drawing window : area diluar media kerja Corel Draw. Biasanya klik kanan di menu Drawing window dengan Drawing Page (media / lembar kerja) akan berbeda hasilnya.Serta objek gambar yang ada di area ini akan masuk dalam Print
8.Drawing page : area media / lembar kerja CorelDraw yang dipakai untuk menggambar / menulis. Jika objek yang digambar sampai keluar area Drawing page maka akan terjadi perintah trouble saat Print
9.Color Pallete : berisikan kumpulan warna yang bisa dipakai untuk mewarnai objek gambar / tulisan di media kerja CorelDraw. Secara default letak Color Pallete ada disamping kanan dari media kerja CorelDraw
10.Docker : menu ini berisikan tampilan - tampilan menu dari Menu Bar. Coba saja klik Text pilih Insert Symbol Character, maka akan ada menu baru disamping kanan media kerja CorelDraw
11.Status Bar : menu yang berada di paling bawah dan berisikan informasi text dari objek yang aktif / diklik
12.Document navigator : berfungsi sebagai navigasi untuk menambah jumlah halaman dari media kerja, biasanya ditulis dalam Page 1, Page 2 dll
13.Navigator : berfungsi untuk membantu mengarahkan tampilan di media kerja CorelDraw.

36. Fungsi transformation pada Corel draw ?


Untuk melakukan pergeseran objek atau gambartransformation terdiri dari :
1. Position,digunakan untuk mengubah posisi objek secara terskala berdasarkan koordinat horizontal dan vertikal. Selain mengubah posisi objek dapat juga dilakukan perubahan posisi sekaligus penduplikasian objek ke destinasi perubahan.
2. Rotation,digunakan untuk mengubah sudut objek dengan nilai tertentu berdasarkan koordinat horizontal dan vertikal. Selain mengubah sudut objek dapat juga dilakukan perubahan sudut sekaligus penduplikasian objek ke destinasi perubahan.
3. Scale & Mirror,digunakan untuk membalik sisi objek berdasarkan garis vertikal dan horizontal.  Selain membalik/flip objek dapat juga di lakukan penduplikasian objek.
4. Size,digunakan untuk mengubah ukuran objek berdasarkan garis horizontal dan vertikal. Nilai plus dan minus dapat di sisipkan untuk menentukan posisi perubahan ukuran objek. Selain perubahan ukuran biasa dapat juga di lakukan penduplikasian objek.
5. Skew,digunakan untuk mengubah bentuk objek berdasarkan sudut dan garis horizontal vertical.

37. Apa fungsi Polygon tool di corel draw


Fungsi Polygon tool  yaitu untuk menggambar dengan bentuk persegi secara banyak, sekedar dengan melakukan atau mengaktifkan drag dan klik mouse. Memiliki turunan 4 macam tools, yaitu :

Star tool : berfungsi untuk menggambar berbagai macam bentuk bintang.

Complex Star tool : berfungsi untuk menggambar bentuk bintang yang mempunyai banyak potongan sisi.

Graph Paper tool : berfungsi  untuk menggambar bentuk rangkaian  susunan kotak-kotak seperti jala-jala.

Spiral tool : berfungsi  untuk menggambar bentuk lingkaran spiral secara simetris atau logaritmis

Corel draw mempunyai fungsi  yaitu untuk mengedit  gambar, oleh sebab itu banyak dipakai pada sebuah pekerjaan dalam bidang percetakan ataupun pekerjaan pada bidang lain yang menginginkan proses pencitraan atau visualisasi.

Fungsi polygon tool

Fungsi Polygon tool  yaitu untuk menggambar dengan bentuk persegi secara banyak, sekedar dengan melakukan atau mengaktifkan drag dan klik mouse. Memiliki turunan 4 macam tools, yaitu :

Star tool : berfungsi untuk menggambar berbagai macam bentuk bintang.

Complex Star tool : berfungsi untuk menggambar bentuk bintang yang mempunyai banyak potongan sisi.

Graph Paper tool : berfungsi  untuk menggambar bentuk rangkaian  susunan kotak-kotak seperti jala-jala.

Spiral tool : berfungsi  untuk menggambar bentuk lingkaran spiral secara simetris atau logaritmis

Penggunaan polygon tool

Langkah – langkah membuat bintang adalah sebagai berikut :

Klik Polygon tool pada toolbox.

Sesuaikan  beberapa sisi objek polygon pada property bar, misalnya sembilan sisi

Tekan kemudian tahan dan ubah posisi mouse pada lembar kerja untuk menggambar objek polygon
