37. sebutkan 3 aplikasi pengolah kata pada komputer jelaskan pula kelebihan tiap apliaksi dibandingkan dengan aplikasi lainnya ? ​


Jawaban:

Misalkan pada komputer dg sistem operasi Windows:

1. Notepad

2. Wordpad

3. Ms. Word

Penjelasan:

1. Kelebihan Notepad ialah:

- aplikasi pengolah kata yg sangat ringan

- tidak membutuhkan kapasitas harddisk yg besar

- tidak perlu memory komputer yg tinggi.

2. Kelebihan Wordpad ialah:

- aplikasi pengolah kata yg cukup sederhana karena ada beberapa fitur seperti Ms. Word

- mampu membuka dan menyimpan data dg format .doc

- pemakaiannya mudah karena memiliki interface (antar muka) mirip Ms. Word.

3. Kelebihan Ms. Word ialah:

- aplikasi pengolah kata yg sangat lengkap fiturnya.

- Pada versi terbaru, mendukung penyimpanan data dg format PDF

- dokumen yg disimpan, dapat diamankan memakai password

Amatilah lampiran:

1. Notepad ditunjukkan tanda panah merah

2. Wordpad ditunjukkan tanda panah hijau

3. Ms. Word ditunjukkan tanda panah kuning
