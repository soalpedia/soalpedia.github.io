3. F8 adalah shortcut pada keyboard untuk memunculkan menu ?


Fungsi dari F8 adalah salah satunya sebagai menu shortcut pada keyboard untuk memunculkan text tool. Soal tersebut pada dasarnya merupakan soal pilihan ganda dimana opsi pilihan jawabannya adalah A.berzier tool, pilihan jawaban B. shape tool, pilihan jawaban C. freehand tool, pilihan jawaban D. text tool dan pilihan jawaban E. artistict media tool. Maka pilihan jawaban yang benar untuk soal tersebut adalah D.

Pembahasan

Fungsi  F8 sebagai menu shortcut pada keyboard untuk memunculkan text tool tersebut merupakan bagian dari fungsi tombol F8 yang digunakan pada aplikasi Corel Draw. Adapun fungsi dari tombol F1 hingga tombol 10 pada aplikasi Corel Draw antara lain yaitu:

38. Shortcut yang di gunakan untuk melihat jurnal sementara adalah


Penjelasan:

Shortcut yang umum digunakan untuk melihat jurnal sementara (temporary journal) dapat bervariasi tergantung pada perangkat dan aplikasi yang digunakan. Namun, berikut adalah beberapa shortcut yang umum digunakan untuk melihat jurnal sementara di beberapa aplikasi:

1. Microsoft Word: Tidak ada shortcut khusus untuk melihat jurnal sementara di Microsoft Word. Anda dapat menggunakan fitur "Undo" (Ctrl + Z) untuk membatalkan langkah terakhir dan melihat perubahan yang dilakukan.

2. Google Docs: Tidak ada shortcut khusus untuk melihat jurnal sementara di Google Docs. Anda dapat menggunakan fitur "Undo" (Ctrl + Z) untuk membatalkan langkah terakhir dan melihat perubahan yang dilakukan.

3. Sublime Text: Untuk melihat jurnal sementara di Sublime Text, Anda dapat menggunakan shortcut "Ctrl + Z" untuk membatalkan langkah terakhir. Untuk melihat perubahan yang dilakukan pada jurnal sementara, Anda dapat menggunakan shortcut "Ctrl + Y" untuk mengulangi langkah terakhir.

4. Visual Studio Code: Untuk melihat jurnal sementara di Visual Studio Code, Anda dapat menggunakan shortcut "Ctrl + Z" untuk membatalkan langkah terakhir. Untuk melihat perubahan yang dilakukan pada jurnal sementara, Anda dapat menggunakan shortcut "Ctrl + Shift + Z" atau "Ctrl + Y" untuk mengulangi langkah terakhir.

Harap dicatat bahwa shortcut ini mungkin berbeda tergantung pada sistem operasi yang Anda gunakan dan pengaturan aplikasi yang sedang digunakan. Pastikan untuk memeriksa dokumentasi atau menu bantuan aplikasi yang spesifik untuk mencari tahu shortcut yang tepat untuk melihat jurnal sementara di aplikasi yang Anda gunakan.
