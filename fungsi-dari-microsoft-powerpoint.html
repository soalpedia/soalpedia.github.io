25. jelaskan macam-macam fungsi arrange dalam microsoft powerpoint​


Jawaban:

A. Clipboard:

1.Paste : untuk menempelkan hasil copy atau cut(sama dengan ctrl+v)

2.Copy : untuk membuat salinan dari teks yang diblok atau objek terpilih

3.Cut : untuk memotong teks yang ditandai atau objek terpilih(bila ingin dipindah ke tempat lain,terakhir dengan perintah paste)

4.Format Painter : untuk meniru format suatu bagian, kemudian diterapkan ke bagian lainnya

B. Slides:

1.New Slide : berfungsi untuk menambahkan halaman slide baru

2.Layout : berfungsi untuk memilih jenis tata letak yang tersedia dalam power point

3.Reset : berfungsi untuk menghapus format yang telah diterapkan dalam halaman slide hingga kembali ke format default

4.Section : berfungsi untuk membagi halaman slide dalam beberapa versi

C. Font:

1.Font Box : untuk memilih jenis huruf yang tersedia dalam sistem

2.Font Size : untuk untuk menguabah ukuran huruf

3.Increase Font Size : untuk memperbesar ukuran huruf secar bertahap dengan klik mouse

4.Decrease Font Size : untuk mengecilkan ukuran huruf dengan klik mouse

5.Clear All Formatting : untuk menghapus seluruh format yang diterapkan dalam bagian teks agar terlihat standar

6.Bold : untuk menebalkan ukuran huruf yang sudah tersedia

7.Italic : untuk membuat huruf miring

8.Underline : untuk memberikan garis bawah pada teks terpilih

9.Texr Shadow : untuk memberi efek bayangan yang telah tersedia

10.Strikethrough : untuk memberi garis coretan dalam bagian tengah teks terpilih

11.Character Spacing : untuk mengatur jarak spasi /renggangan antar huruf

12.Change Case : untuk mengubah teks terpilih menjadi huruf besar/kecil

13.Font Color : untuk mengubah warna huruf dari teks terpilih

D. Paragraf:

1.Bullets : berfungsi untuk membuat paragraf penomoran dengan titik bulat

2.Numbering berfungsi untuk membuat paragraph penomoran dengan angkaatau huruf

3.Decrease Indent : berfungsi untuk menggeser paragraf ke sebelah kiri

4.Increase Indent : berfungsi untuk menggeser paragraph ke sebelah kanan

5.Line Spacing : berfungsi untuk mengatur jarak spasi antar baris teks

6.Align Text Right : berfungsi untuk mengatur teks ke sebelah kiri

7.Center : berfungsi untuk mengatur teks ke sebelah tengah

8.Aligh Text Right : berfungsi untuk mengatur teks menjadi ke sebelah kanan

9.Justify : berfungsi untuk mengatur teks menjadi ke sebelah kanan-kiri

10.Left To Right Text Direction : berfungsi untuk mengubah arah penulisan teks dari kiri ke kanan dalam penulisan latin

11.Right to  left text direction : berfungsi untuk mengubah arah penulisan teks ke kanan ke kiri dalam penulisan arab

12.Columns : berfungsi untuk mengubah jumlah kolom teks

13.Text Direction : berfungsi mengatur arah penulisan horizontal

14.Align Text : berfungsi mengatur paragraf berada di tengah kotak, ke sebelah atas,atau ke sebelah bawah kotak

15.Convert To Smartart : berfungsi untuk mengubah teks ke dalam objek shape sehingga berbentuk ilustrasi visual

E. Drawing:

1.Shapes Box : untuk membuat objek shape di area halaman slide

2.Arrange : untuk mengatur susunan objek bersama objek lainnya

3.Quick Style : untuk membuat gambar objek shape dengan yang menarik

4.Shape Fill : untuk mengatur warna bagian tengah objek

5.Shape Outline : untuk mengatur warna, bentuk, dan ketebalan garis

6.Shape Effect : untuk mengatur efek visual objek

F. Editing:

1.Find : berfungsi untuk mencari kata/karakter tertentu dalam dokumen yang sedang dibuka

2.Replace : berfungsi untuk menganti kata tertentu dalam dokumen yang sudah dibuka

3.Select : berfungsi untuk menyeleksi objek dan mengatur jenis menyelesi

Smoga membantu yaa...

Jawaban:

Microsoft Power Point

PowerPoint merupakan salah satu aplikasi yang ada dalam paket Microsoft Office. PowerPoint adalah aplikasi yang digunakan untuk membuat sebuah bahan presentasi dalam bentuk slide. Aplikasi ini bisa digunakan untuk keperluan mengajar, presentasi dan bisa juga untuk membuat animasi sederhana, PowerPoint berjalan di atas komputer PC berbasis sistem operasi Microsoft Windows dan juga Apple Macintosh yang menggunakan sistem operasi Apple Mac OS, meskipun pada awalnya aplikasi ini berjalan di atas sistem operasi Xenix.

Kehadiran powerpoint membuat sebuah presentasi berjalan lebih mudah dan menggantikan cara presentasi kuno yaitu dengan transparasi proyektor atau biasa disebut OHP (Over Head Proyector). Dengan adanya Ms Powerpoint, membuat presentasi menjadi sangat mudah karena didukung dengan fitur fitur yang canggih dan menarik. Di dalam terdapat berbagai macam tools dan template yang memberikan efek tertentu pada sebuah presentasi. Dengan menggunakan program yang satu ini, maka presentasi yang disajikan akan lebih menarik.

1.Fungsi Microsoft Power Point

-Membuat presentasi dalam bentuk slide-slide

Menambahkan audio, video, gambar, dan animasi dalam presentasi sehingga presentasi lebih menarik.

-Mempermudah dalam mengatur dan mencetak slide,

-Membuat presentyasi dalam bentuk softcopy sehingga dapat dengan mudah diakss dan di share melalui perangkat computer.

2.Kelebihan Microsoft Power Point

-Mudah untuk dioperasikan

-Tersedia berbagai macam desain dan animasi

-Tersedia berbagai macam template menarik

-Dapat dibuat dalam berbagai format
