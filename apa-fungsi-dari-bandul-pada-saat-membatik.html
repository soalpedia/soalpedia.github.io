19. Jika bandul A dilepas dan menyentuh bandul B maka bandul C akan terdorong sejauh jarak bandul A terhadap B, ketika bandul C kembali dan menyentuh bandul B, bandul A akan terdorong sejauh 3/4 dari jarak terjauh bandul C dari bandul B dan demikian seterusnya. Total jarak seluruh persimpangan bandul A dan bandul C sampai ketiga bandul tersebut berhenti adalah ... cm A. 112 D. 56 B. 98 E. 14 C. 84


a = 14

r = 3/4

s = a/1-r

= 14/(1-(3/4))

= 14/1/4

= 56 cm

Totak jarak

= 2 × s

= 2 × 56

= 112

Jadi Total jarak seluruh persimpangan bandul A dan bandul C sampai ketiga bandul tersebut berhenti adalah

A. 112 cm

34. jika panjang bandul adalah L maka frekuensi bandul 2 Hz jika panjang bandul dikurangi 50% tentukan frekuensi bandul


GH
- Ayunan bandul

ℓ₁ = L, f₁ = 2 Hz
ℓ₂ = ½L, f₂ = __ ?

Frekuensi ayunan bandul
f = [tex] \frac{1}{2 \pi } \sqrt{ \frac{g}{l} } [/tex]
frekuensi berbanding terbalik dengan akar panjang tali

f₂ / f₁ = √ℓ₁ : √ℓ₂
f₂ / f₁ = √(ℓ₁ / ℓ₂)
f₂ / 2 = √(L / ½L)
f₂ / 2 = √2
     f₂ = 2√2 Hz  ← jawaban
