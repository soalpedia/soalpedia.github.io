19. find the equation of the circle whose radius is 5 which touches the circle x² + y² - 2x -4x -20 = 0​


Penjelasan dengan langkah-langkah:

x² + y² - 2x -4y -20 = 0

cari titik pusatnya dulu

x = –(–2)/2 = 1

y = –(–4)/2 = 2

jadi pusatnya (1, 2)

dengan r = 5

kita cari titik potongnya

dengan :

(a+1)/2 = 5

a + 1 = 2(5)

a = 10-1

a = 9

(b+2)/2 = 5

b+2 = 2(5)

b = 10-2

b = 8

Persamaannya menjadi :

(x - a)² + (y - b)² = r²

(x - 9)² + (y - 8)² = 5²

x² - 18x + y² - 16y + (-9)² + (-8)² - 5² = 0

x² + y² - 18x - 16y + 81 + 64 - 25 = 0

x² + y² - 18x - 16y + 120 = 0
