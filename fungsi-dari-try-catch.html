17. Apa fungsi Try-Except pada bahasa pemrograman python?Minta tolong dijawab​


Try-Except adalah fitur pada bahasa pemrograman Python yang digunakan untuk menangani error atau exception yang mungkin terjadi saat program dijalankan. Struktur Try-Except digunakan untuk mengeksekusi blok kode dalam "try" dan jika terjadi error atau exception, maka kode dalam "except" akan dijalankan.

Contohnya seperti ini:

try:

# kode yang mungkin menyebabkan exception

result = x / y

except ZeroDivisionError:

# kode yang akan dijalankan jika terjadi exception ZeroDivisionError

print("Error: Tidak bisa membagi dengan nol")

except Exception as e:

# kode yang akan dijalankan jika terjadi exception lainnya

print("Error: ", e)

Dalam contoh diatas, kode dalam blok try akan mencoba untuk membagi x dengan y. Jika terjadi exception ZeroDivisionError, karena pembagian dengan 0, maka kode dalam blok except pertama akan dijalankan dan menampilkan pesan "Error: Tidak bisa membagi dengan nol". Jika terjadi exception lainnya, maka kode dalam blok except kedua akan dijalankan dan menampilkan pesan "Error: ", diikuti dengan deskripsi dari exception yang terjadi.

Try-Except sangat membantu dalam menangani error yang mungkin terjadi saat program dijalankan, sehingga program dapat tetap berjalan dengan stabil meskipun terjadi error.
