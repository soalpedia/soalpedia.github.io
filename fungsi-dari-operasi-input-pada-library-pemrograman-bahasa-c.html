13. Buatlah program untuk menghitung jumlah digit dari sebuah bilangan yang diinput oleh user menggunakan Bahasa pemrograman C!


Penjelasan:

Yang saya tangkap dari soal tersebut adalah meminta kita untuk membuat sebuah program yang menghitung jumlah digit dari sebuah bilangan yang diinput user, yaitu seperti: apabila diinput bilangan 234 maka outputnya adalah 3 karena ada 3 digit angka dalam bilangan tersebut. Benar?Bila demikian, saya akan anggap bilangan tersebut sebagai karakter karena sifatnya mirip dengan mengecek jumlah karakter yang diinput.

Kode Program:

//Program dimulai

#include <stdio.h>

#include <string.h>

int main() {

   char bilanganYangDiinputkan[100000]; //Deklarasi bilanganYangDiinputkan

   int jumlahDigitAngka; //Deklarasi jumlahDigitAngka

   

   printf("Masukkan sebuah bilangan: " );

   fgets(bilanganYangDiinputkan, sizeof(bilanganYangDiinputkan), stdin);  //Proses input bilangan

   jumlahDigitAngka = strlen(bilanganYangDiinputkan) - 1;  //Menghitung jumlah digit angka

   printf("Jumlah digit dari bilangan tersebut adalah %d\n", jumlahDigitAngka);  //Output

   return 0;  

}

//Program selesai

22. apa fungsi #! di beberapa bahasa pemrograman?​


Sebenarnya simbol "#!" tidak ada dalam bahasa pemrograman manapun yang saya tau. mungkin yang kamu maksud adalah:

tanda #, kalau di python tanda "#" adalah tanda Comment yakni baris tersebut tidak akan dieksekusi oleh interpreter.

Contoh Penggunaan #:

#Ini adalah komentar dengan python.

Namun "#!" bukan sesuatu dari python ataupun bahasa pemrograman lainnya., ini adalah baris hashbang (atau shebang) yang menunjukkan penerjemah mana yang harus memproses file.

contoh penggunaan #! :

#! /usr/local/bin/python3

ini berkaitan dengan hal Linux dan UNIX bukan bahasa pemrograman...

Penjelasan Lebih Lanjut:

Baris shebang(hashbang) dalam skrip apa pun menentukan kemampuan skrip untuk dieksekusi seperti yang dapat dieksekusi mandiri tanpa mengetik python sebelumnya di terminal atau ketika mengklik dua kali di pengelola file (bila dikonfigurasi dengan benar).

Jad dapat ditarik kesimpulan bahwa skrip:

#! /usr/local/bin/python3

atau sejenisnya dalam python hanya digunakan agar skrip python bisa berjalan mandiri tanpa mengetik python di terminal/cmd.

Bahasa Pemrograman merupakan sebuah instruksi untuk memerintah komputer agar bisa menjalankan fungsi tertentu, namun hanya instruksi standar saja. Seperti yang sudah Saya singgung di atas, bahasa pemrograman merupakan sebuah perhimpunan dari aturan sintaks dan semantik yang tugasnya untuk mendefinisikan program komputer. Seseorang yang bisa memahami bahasa pemrograman dapat menentukan mana data yang akan di simpan / diteruskan, data mana yang akan di olah, dan langkah apa saja yang harus di ambil dalam berbagai situasi.
