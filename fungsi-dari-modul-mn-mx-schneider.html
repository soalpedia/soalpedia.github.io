18. turunan fungsi g(x)=(mx-4) (x-2) adalah.......​


Penjelasan dengan langkah-langkah:

kalikan dulu kedua komponen yg ada didalam kurung.

[tex]g(x) = (mx - 4)(x - 2) \\ g(x) = {mx}^{2} - 2mx - 4x + 8 [/tex]

rumus turunan fungsi nya adalah

[tex]g(x) = a \times {x}^{n} \\ g'(x) = a \times n \times {x}^{n - 1} [/tex]

jadi turunannya

[tex]g'(x) = m \times 2 {x}^{2 - 1} - 2m {x}^{1 - 1} - 4 {x}^{1 - 1} + 0 \\ g'(x) = 2mx - 2m - 4[/tex]

variabel dipangkatkan dengan nol maka hasilnya sama dengan 1

mx²- 2 mx+m = 2x² -3

mx²- 2x²- 2 mx+m + 3 = 0

(m - 2)x² - 2mx + m + 3 = 0

Supaya grafik fungsi f(x) = mx²- 2 mx+m seluruhnya diatas grafik fungsi g(x)=2x² -3

maka nilai diskriminan (D) > 0

(m - 2)x² - 2mx + m + 3 = 0

a = m - 2, b = -2m dan c = m + 3

D > 0

b² - 4ac > 0

(-2m)² - 4(m -2)(m + 3) > 0

4m² - 4(m² + m - 6) > 0

4m² - 4m² - 4m + 24 > 0

-4m + 24 > 0

-4m > - 24

4m < 24

m < 24/4
