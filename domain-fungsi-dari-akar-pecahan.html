1. domain dan range fungsi dari fungsi rasional (pecahan)​


Penjelasan dengan langkah-langkah:

Dalam matematika, fungsi rasional adalah fungsi yang dinyatakan sebagai pecahan polinomial, di mana pembilang dan penyebutnya adalah polinomial. Untuk fungsi rasional f(x), domain dan range-nya dapat dijelaskan sebagai berikut:

1. Domain (daerah asal):

Domain dari fungsi rasional adalah himpunan semua nilai x yang dapat digunakan sebagai input dalam fungsi tersebut tanpa menyebabkan pembagian dengan nol dalam penyebut. Jadi, domain f(x) adalah semua nilai x yang membuat penyebutnya bukan nol.

2. Range (daerah hasil):

Range dari fungsi rasional tergantung pada bentuk fungsi dan batasan-batasan yang mungkin ada. Range adalah himpunan semua nilai y yang dapat dihasilkan oleh fungsi tersebut untuk nilai x dalam domainnya. Range bisa beragam tergantung pada karakteristik fungsi rasional tersebut.

Sebagai contoh, jika kita memiliki fungsi rasional f(x) = (3x + 2) / (x - 1), maka domain f(x) adalah semua nilai x kecuali x = 1 (karena itu akan membuat penyebutnya nol). Range f(x) akan bergantung pada cara grafik fungsi ini berperilaku dan apakah ada batasan lain pada variabel x. Range biasanya dapat ditentukan dengan mengamati grafik fungsi atau dengan menggunakan analisis matematis lebih lanjut.

19. Tentukan domain dan range dari fungsi-fungsi yang diberikan berikut : t(x) = akar x2 + 2x


Penjelasan dengan langkah-langkah:

Untuk menentukan domain dan range dari fungsi t(x) = √(x^2 + 2x), kita perlu memperhatikan batasan-batasan dari akar kuadrat dan ekspresi di dalamnya.

Domain mengacu pada nilai-nilai x yang diperbolehkan dalam fungsi. Dalam hal ini, ekspresi di bawah akar kuadrat harus bernilai non-negatif, yaitu x^2 + 2x ≥ 0. Kita bisa mencari titik-titik di mana ekspresi ini sama dengan nol:

x^2 + 2x = 0

x(x + 2) = 0

Dari sini kita mendapatkan x = 0 dan x = -2 sebagai titik di mana ekspresi x^2 + 2x = 0. Jadi, domain fungsi t(x) adalah semua nilai x yang memenuhi x ≥ -2, karena seiring dengan nilai x yang lebih kecil dari -2 akan menghasilkan hasil negatif di dalam akar kuadrat.

Untuk range, kita perlu mencari nilai minimum dan maksimum dari fungsi t(x). Karena akar kuadrat selalu menghasilkan nilai non-negatif, nilai minimum dari t(x) adalah 0. Untuk mencari nilai maksimum, kita dapat memperhatikan bahwa x^2 + 2x adalah bilangan bulat positif atau nol (dalam domain yang telah ditentukan). Oleh karena itu, akar kuadrat dari bilangan bulat positif atau nol akan menghasilkan bilangan bulat positif atau nol. Jadi, range fungsi t(x) adalah semua nilai t yang diperoleh saat x ≥ -2. Secara formal, range dari fungsi ini adalah [0, ∞).

23. domain dari fungsi f(x) akar 9-x^2


Domain dari fungsi f(x) = √(9 - x²)

Pembahasan :

Daerah asal (domain) dari y = √(f(x)) adalah f(x) ≥ 0

Jadi

f(x) = √(9 - x²) terdefinisi jika

(9 - x²) ≥ 0
(3 + x)(3 - x) ≥ 0
x = -3 atau x = 3

Garis bilangan
.......... (-3) ....... (3) .......

uji titik
• untuk daerah sebelah kiri (-3) kita pilih x = -4
(3 + (-4)) (3 - (-4)) = (-1) (7) = -7 => negatif
• untuk daerah antara (-3) dan (3) kita pilih x = 0
(3 + 0) (3 - 0) = (3) (3) = 9 => positif
• untuk daerah sebelah kanan 3 kita pilih x = 4
(3 + 4) (3 - 4) = (7) (-1) = -7 ==> negatif

Jadi garis bilangannya
------ [-3] ++++ [3] -----
ambil daerah yang positif karena ≥ 0
-3 ≤ x ≤ 3

Sehingga domain dari fungsi f(x) = √(9 - x²) adalah
Df = {x | -3 ≤ x ≤ 3, x ∈ R}
