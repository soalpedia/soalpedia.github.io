1. aTombol yang digunakan untuk menggandakan data adalah ....a Ctrl+ C dan Ctrl + V b. Ctrl+ C dan Ctrl + O c. Ctrl+​


Tombol atau shortcut yang digunakan untuk menggunakan data adalah Ctrl+ C dan Ctrl + V (Jawaban A). Simak pada bagian pembahasan.

Pembahasan

Di dalam menjalankan sebuah aplikasi misalnya kita ambil contoh pada microsoft excel, maka akan terdapat banyak toolbars yang akan kita gunakan untuk membuat sebuah action atau perintah.

Pada umumnya kita menggunakan mouse dan melakukan perintah dengan klik kanan atau pun klik kiri. Namun untuk sebagian orang maka kita dapat menggunakan jalan pintas (Shortcut) untuk beberapa action sebagaimana contoh berikut di bawah ini:

Ctrl + c = Untuk menduplikat data.Ctrl + v = Untuk menempel (paste) data yang tadi kita copy.Ctrl + a = Untuk melakukan seleksi pada semua bagian.Ctrl + s = untuk menyimpan file.Ctrl + o = untuk membuka file.Ctrl + n = untuk membuat document baru.

Selain pada microsoft excel maka shortcut tersebut bisa diaplikasikan pada aplikasi lainnya.

Demikian, semoga membantu!

18. Untuk membuat dokumen dengan orientasi kertas horisontal, maka digunakan...

Di Microsoft Word, bila kita ingin mengubah orientasi kertas maka kita bisa pilih menu Layout > Orientation. Lalu pilih Portrait (Vertikal) atau Landscape (Horisontal).

(https://support.microsoft.com/en-us/office/change-page-orientation-to-landscape-or-portrait-9b5ac1af-9998-4a37-962b-a82b689572a9)

19. Tombol kombinasi keyboard yang digunakan untuk membatalkan perintah adalah...

Untuk membatalkan perintah yang telah kita lakukan sebelumnya, kita bisa klik Undo di Quick Access Toolbar ataupun menggunakan shortcut keyboard Ctrl + Z.

(https://support.microsoft.com/en-us/topic/undo-redo-or-repeat-an-action-84bdb9bc-4e23-4f06-ba78-f7b893eb2d28?ui=en-us&rs=en-us&ad=us#:~:text=To%20undo%20an%20action%20press,want%20to%20undo%20multiple%20steps.)

20. Untuk membuat spasi baris 1,5 dapat digunakan tombol kombinasi keyboard...

Mode spasi baris 1,5 dapat kita lakukan dengan menekan tombol Ctrl + 5 di keyboard kita, ataupun dengan menekan menu Home > Line and Paragraph Spacing > Line Spacing Options > Line Spacing > 1.5 lines

(https://tips.slaw.ca/2012/technology/keyboard-shortcuts-for-easily-changing-line-spacing-in-ms-word/)

(https://support.microsoft.com/en-us/topic/change-the-line-spacing-in-word-04ada056-b8ef-4b84-87dd-5d7c28a85712?ui=en-us&rs=en-us&ad=us)
