39. Apa fungsi dari merge and center,bold,italic,wrap text,underline


Jawaban:

Fungsi di atas dapat dijumpai dalam program seperti Ms. Excel:

Merge and center: berfungsi untuk menggabungkan 2 sel atau lebih sekaligus memposisikan perataan teks ke bagian tengah.

Bold: berfungsi untuk menebalkan teks.

Italic: berfungsi untuk memiringkan teks.

Wrap text: berfungsi untuk mengatur tesk supaya posisinya menyesuaikan lebar kolom.

Underline: berfungsi untuk memberikan garis bawah pada teks.

Penjelasan:

Perhatikan contoh masing-masing ikon pada gambar lampiran:

- Merge and center ditunjukkan tanda panah biru

- Bold ditunjukkan tanda panah merah

- Italic ditunjukkan tanda panah kuning

- Wrap text ditunjukkan tanda panah hitam

- Underline ditunjukkan tanda panah hijau


40. gambar ikon merge and center tidak dapat di gunakan solusi bagaimana?


shortcut : Alt + HMC

detect : coba pilih minimal 2 sel , dengan cara klik sel + shift + left arrow
lalu tekan Alt + H + M + C

problem : jika icon Mc " ( merge and center ) masih belum dapat digunakan ada dua kemungkinan ,
1. berubah fungsi
2. produk office "broken"
3. masa linsensi sudah habis ( biasanya gak ngaruh )

solution
1. klik menu file --> options --> Quick Asscess Toolbar --> all command --> M --> Merge and Center --> Input --> Save --> Ok

2. re-install kembali produk MSOFFICE atau upgrade lisensi

3. cari SN baru !
