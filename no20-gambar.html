Biaya minimum proyek, B min = __?

Jawab :
Step 1
---------
Cari Biaya total = B = ___?
B = (x + 500/x - 40).x
B = (x² + 500 - 40x) Juta 

agar mencapai minimum maka persamaan B diturunkan menjadi 
B' = 2x - 40 = 0
2x = 40
x = 20 hari

Jadi biaya minimumnya untuk 20 hari sebesar
B min = (x² + 500 - 40x) Juta
B min = ((20)² + 500 - 40.20) Juta
B min = (400 + 500 - 800) Juta
B min = 100 Juta

Jawaban : C
