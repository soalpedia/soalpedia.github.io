+ 0 ≤ x ≤ 1

+ 1 ≤ x ≤ 2

+ 2 ≤ x ≤ 3

f(1) = 1 - 3 = -2

Pada interval kedua (1 ≤ x ≤ 2), ambil x = 2, maka :

f(2) = 2 - 3 = -1

Pada interval ketiga (2 ≤ x ≤ 3), ambil x = 3, maka :

f(4) = 4 - 3 = 1

f(5) = 5 - 3 = 2

[tex]\sum\limits^5_{i=1}{(x_i-3).\Delta{x}}=((-2)+(-1)+(0)+(1)+(2))(1)=0[/tex]

[tex]\Delta{x}=\frac{5}{n}[/tex]

Sekarang kita tentukan harga xi nya.

[tex]x_1=0+\frac{5}{n}=\frac{5\times1}{n}\\x_2=0+\frac{10}{n}=\frac{5\times2}{n}\\\vdots\\x_i=\frac{5i}{n}[/tex]

Berarti jumlah Riemannya adalah :

[tex]\begin{aligned}

\lim_{n\to\infty}{\sum\limits^{n}_{i=1}{f(x_i)\Delta{x}}}&=\lim_{n\to\infty}{\sum\limits^{n}_{i=1}{\left(\frac{5i}{n}-3\right)\left(\frac{5}{n}\right)}}\\

&=lim_{n\to\infty}{\sum\limits^{n}_{i=1}{\left(\frac{25i}{n^2}-\frac{15}{n}\right)}}\\

&=\lim_{n\to\infty}{-\left(\frac{5n+25}{2n}\right)}\\

6. Tunjukk bahwa barisan Y = {yn} dengan yn = ( n + (-1)") bukan barisan Cauchy.​


Jawaban:

Untuk membuktikan bahwa barisan Y = {yn} dengan yn = (n + (-1)") bukan barisan Cauchy, kita perlu menunjukkan adanya dua suku dalam barisan tersebut yang memiliki selisih yang tetap, tetapi selisih tersebut tidak mendekati nol saat kedua suku tersebut mendekati tak hingga.

Dalam barisan Y, suku ke-n diberikan oleh yn = (n + (-1)"). Mari kita pilih dua suku, ym dan yn, di mana m dan n adalah bilangan bulat positif, dan misalkan m > n. Dalam hal ini, kita perlu menunjukkan bahwa selisih |ym - yn| tidak mendekati nol saat m dan n mendekati tak hingga.

Selisih antara dua suku dalam barisan ini adalah:

|ym - yn| = |(m + (-1)") - (n + (-1)")|

= |m + (-1)" - n + (-1)"|

= |m - n|

Perhatikan bahwa selisih |ym - yn| adalah |m - n|, yang tidak bergantung pada nilai n atau m. Jadi, tidak peduli seberapa besar n dan m, selisih ini tetap konstan.

Jika kita ambil dua bilangan bulat positif yang cukup besar, misalnya m = 10 dan n = 5, maka selisihnya adalah |10 - 5| = 5. Selisih ini tidak mendekati nol saat m dan n mendekati tak hingga.

Dengan demikian, dapat disimpulkan bahwa barisan Y = {yn} dengan yn = (n + (-1)") bukanlah barisan Cauchy, karena tidak ada dua suku dalam barisan ini yang selisihnya mendekati nol saat suku-suku tersebut mendekati tak hingga.

Contoh bukan barisan Cauchy, yaitu :
<xn>, xn = 1 + [tex] \frac{1}{2} [/tex] + ... + [tex] \frac{1}{n} [/tex]
Barisan di atas divergen dan tidak terbatas.

35. Tentukan jumlah Riemann fungsi g(x) = −2x + 4 pada interval [1, 5] (menggunakan n subinterval dengan lebar sama panjang).


Materi : Integral Tentu

Untuk mencari jumlah Riemann dari fungsi tersebut, mula - mula kita cari dulu [tex]\Delta{x_i}[/tex]

g(x) = 4 - 2x dengan 1 ≤ x ≤ 5

Nilai delta tersebut dapat di cari dengan persamaan :

[tex]\Delta{x_i}=\frac{b-a}{n}[/tex] dengan n sub interval dengan lebar yang sama.

Jadi :

[tex]\Delta{x_i}=\frac{5-1}{n}=\frac{4}{n}[/tex]

Selanjutnya, cari titik - titik wakil dengan cara :

[tex]x_1=1+\frac{4}{n}\\x_2=1+2\cdot\frac{4}{n}=1+\frac{8}{n}\\x_3=1+3\cdot\frac{4}{n}=1+\frac{12}{n}\\\vdots\\x_i=1+i\cdot\frac{4}{n}=1+\frac{4i}{n}[/tex]

Sekarang, cari nilai f(x) nya, diperoleh :

[tex]f(x_i)=4-2\left(1+\frac{4i}{n}\right)=2-\frac{8i}{n}[/tex]

Untuk pencarian jumlah Riemannya dengan n tak terhingga sub interval, gunakan rumus :

[tex]S=\lim_{x\to\infty}{\left(\sum\limits^{n}_{i=1}{f(x_i)\Delta{x_i}}\right)}\\[/tex]

Sehingga :

[tex]S=\lim_{n\to\infty}{\left(\sum\limits^{n}_{i=1}{f(x_i)\Delta{x_i}}\right)}=\lim_{n\to\infty}{\left(\sum\limits^{n}_{i=1}{\left(2-\frac{8i}{n}\right)\left(\frac{4}{n}\right)}\right)}\\S=\lim_{n\to\infty}{\left(\sum\limits^{n}_{i=1}{\left(\frac{8}{n}-\frac{32i}{n^2}\right)}\right)}=\lim_{n\to\infty}{\left(\frac{8}{n}\sum\limits^{n}_{i=1}{1}-\frac{32}{n^2}\sum\limits^{n}_{i=1}{i}\right)}\\[/tex]

Seperti kita ketahui nilai dari :

[tex]\sum\limits^{n}_{i=1}{i}=\frac{1}{2}n(n+1)[/tex] dan [tex]\sum\limits^{n}_{i=1}{1}=n[/tex]

Gunakan ini untuk memperoleh nilai dari sigma i dan 1, sehingga :

[tex]S=\lim_{n\to\infty}{\left(\frac{8}{n}\sum\limits^{n}_{i=1}{1}-\frac{32}{n^2}\sum\limits^{n}_{i=1}{i}\right)}=\lim_{n\to\infty}{\left(8-\frac{32}{n^2}\left(\frac{1}{2}n(n+1)\right)\right)}\\S=\lim_{n\to\infty}{\left(8-\frac{16(n+1)}{n}\right)}=8\lim_{n\to\infty}{\left(1-\frac{2(n+1)}{n}\right)}\\S=8\lim_{n\to\infty}{\left(\frac{-n-2}{n}\right)}=-8[/tex]

Jadi, jumlah Riemannya adalah -8.
