7. apa arti dari oshiete yo


Pembahasan

Arti 教えてよ (oshiete yo) memiliki beberapa arti sebagai berikut :

Ajari aku.Beritahu dong.Tolong jelaskan.Katakanlah.Ceritakanlah.

-------

教えて (oshiete)

Bisa diartikan ajari, ceritakan, jelaskan, beritahu. Kata ini juga termasuk jenis doushi atau disebut kata kerja, selain itu kata 教えて (oshiete) berasal dari kata 教え (oshie) yang berarti pelajaran, ajaran.

Kata 教えて (oshiete) tergolong kata kerja dikarenakan terdapat bentuk て (te). Sehingga jika dibentuk kedalam kata kerja, maka biasa kata ini diungkapkan ketika ingin seseorang diberitahukan atau diajari oleh orang lain.

よ (yo)

Ini termasuk golongan partikel, yang dimana konteksnya biasa digunakan sebagai jenis seruan maupun sebagai penegasan.

9. tatte kudasai, suwatte kudasai, yonde kudasai...Tolong ubah ke bahasa indonesia...


N5 Level A || Bab: 16

Pembahasan:
• kata perintah
( って、て、で、んで + 下さい)

•| R u m u s |•
↪ kata kerjaます ----> kata kerjaって、て、で、んで + 下さい

=== Jawaban ===

↪立って下さい (tatte kudasai): tolong berdiri / diharapkan berdiri.
↪座って下さい (suwatte kudasai): tolong duduk.
↪読んで下さい (yonde kudasai): tolong baca.

❇ Penjelasan:
↪立って (tatte) oerubahasan dari 立ちます (tachimasu) yang mengartikan "Duduk"
↪座って (suwatte) perubahasan dari 座る (suwaru) yang mengartikan "duduk"
↪読んで (yonde) perubahan dari 読みます / 読む (yomimasu / yomu) yang mengartikan "baca"

•| C A T A T A N |•
↪ Kalimat tersebut masuk kedalam kategori kalimat perintah.
Jikalau memakai "kudasai", mengartikan kalimat tersebut sopan dan lebih condong kepada meminta tolong. Karena " kudasai" di sini memiliki makna yamg sama dengan "onegai"
↪ Jika kata "kudasai" dihilangkan masih akan mengartikan arti yang sama. Tetapi bedanya menjadi kalimat non formal.

14. apa arti"教えて"(oshiete kuremasenka)​


oshiete kuremasenka artinya adalah bisakah ajari aku? atau bisakah memberitahuku?

Pembahasan

教えてくれませんか？

Oshiete kuremasenka?

Artinya :

Bisakah ajari aku?Bisakah memberitahuku?Bisakah kamu ceritakan?

----------

Pola kalimat diatas

Memiliki polanya yaitu sebagai berikut :

Kata kerja (bentuk -te) + kuremanseka.

Biasa kata kuremasenka terletak setelah kata kerja bentuk -te.

Pola kalimat tersebut bertujuan untuk membuat kalimat tanya yang digunakan dengan tujuan meminta bantuan atau meminta pertolongan kepada orang lain.

教えて (oshiete) artinya adalah sebagai berikut ini :

Ajari.Beritahu.Ceritakan.Katakan.

Selain itu, kata 教えて (oshiete) merupakan jenis kata kerja dengan bentuk て (te). Kata くれませんか (kuremanseka) itu berarti bisakah?, bisa tolong? atau bolehkah?

16. Yamete kudasai aiyamete kudasai​


Penggunaan kata "Yamete" (やめて) merupakan suatu kata yang memiliki makna "berhenti" atau "hentikan". Untuk kata "Kudasai" (ください) merupakan suatu kata penanda lingual yang memiliki makna sebagai "harap" atau "tolong".

Apabila kedua kata tersebut digabungkan menjadi satu, "Yamete Kudasai" (やめてください) mengandung suatu makna "Hentikanlah segala jenis perbuatan yang kiranya hendak dilaksanakan pada lain kesempatan waktu atau jenis perbuatan yang akan dirimu laksanakan pada saat itu juga".


17. Kalimat perintah di bawah ini benar, kecuali... a. E o mite kudasai b. Sakubun o kaite kudasai c. Ongaku o kiite kudasai d. Mae ni kiite kudasaie. Isu ni suwatte kudasai​


Jawaban:

c. Ongaku o kiite kudasai

Penjelasan:

1. E o mite kudasai

Arti dari E o mite kudasai adalah Silakan lihat gambarnya sedangkan artinya kalimat perintah sendiri kalimat yang berisi permintaan/menyuruh orang lain untuk melakukan sesuatu yang kita kehendaki Silahkan itu termasuk dalam kalimat perintah. Jadi untuk jawaban yang A kurang tepat.

2. Sakubun o kaite kudasai

Arti dari Sakubun o kaite kudasai adalah Silahkan menulis esai Yang bermaksud bahwa si penulis menyuruh kita untuk menuliskan esai dan disini juga terdapat kalimat Silahkan. Jadi, jawaban B juga kurang tepat.

3. Ongaku o kiite kudasai

Arti dari Ongaku o kiite kudasai adalah Tolong Dengarkan Musiknya yang berarti kita sedang memohon kepada seseorang atau bahkan lebih untuk mendengarkan sebuah musik ya guys. Dan Tolong termasuk kalimat Permohonan. Jadi, jawaban yang C adalah jawaban yang paling tepat.

4. Mae ni kiite kudasai

Artinya adalah Silahkan bertanya sebelumnya Yang berarti kita disuruh bertanya sama orang yang menuruh kita ini dan ini termasuk kalimat perintah. Jadi, jawaban yang D kurang tepat.

5. Isu ni suwatte kudasai

Artinya adalah Silahkan duduk di kursi yang berarti orang ini mempersilahkan kita untuk duduk di sebuah kursi dan berarti adalah kalimat perintah. Jadi jawaban yang E sama sekali tidak tepat.

23. perbedaan bentuk "te kudasai" dan "o kkerja kudasai" ?


Perbedaan bentuk 'te kudasai' dan 'o∼kudasai' adalah bentuk 'o∼kudasai' merupakan bentuk permintaan atau permohonan yang lebih sopan daripada bentuk 'te kudasai'. Kedua bentuk tersebut sama-sama sopan, namun bentuk 'o∼kudasai' lebih sopan lagi.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Untuk membuat kalimat permintaan, permohonan, atau perintah secara halus dalam Bahasa Jepang, bisa menggunakan bentuk 'te kudasai' dan 'o∼kudasai' ini. Untuk memahami lebih lanjut mengenai kedua bentuk ini bisa disimak pembahasan di bawah.

てください (Te kudasai)

Untuk membuat kata kerja bentuk 'te kudasai', cara nya dengan mengubah kata kerja nya ke bentuk Te terlebih dahulu, kemudian ditambahkan kata Kudasai yang berarti "Mohon", "Tolong", dsb. Untuk cara mengubah kata kerja ke bentuk Te bisa dilihat di link yang saya berikan di bawah nanti.

Pola : KK(Bentuk Te) + Kudasai

Contoh kalimat :

Suwatte kudasai = Silahkan dudukYamete kudasai = Mohon hentikanYurushιte kudasai = Tolong maafkan

お～ください (O∼kudasai)

Untuk membuat kata kerja bentuk 'o∼kudasai', cara nya hanya dengan meletakkan kata kerja bentuk stem (bentuk masu tanpa menggunakan masu nya) di antara kata O dan Kudasai. Alasan kenapa bentuk ini lebih sopan dari bentuk 'te kudasai' adalah karena menggunakan honorifik (bahasa penghormatan), yaitu O di awal kata.

Pola : O + KK(Bentuk stem) + Kudasai

Contoh kalimat :

Osuwari kudasai = Silahkan dudukOyame kudasai = Mohon hentikanOyurushi kudasai = Tolong maafkan
