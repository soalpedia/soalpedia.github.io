8. Gambarlah grafik fungsi trigonometri dari y= sin x+2​


Jawab:

Buatlah grafik dengan memasukkan titik y, dan maka titik x muncul.

Contoh

jika titik y = 1, maka 1 = sinx+2, sin x = 1-2, sin x = -1, arcsin-1 = -π/2 (-π/2, 1)

jika titik y = 2, maka 2 = sinx+2, sin x = 2-2, sin x = 0, arcsin 0 = 0 (2,0)

jika titik y = 3, maka 3 = sinx+2, sin x = 3-2, sin x = 1, arcsin 1 = π/2 (π/2, 3)

Lanjutkan dengan x

jika titik x = π/2, maka y = sinπ/2 + 2 = 3 (π/2, 3)

jika titik x = 2π/2 atau x = π, maka y = sinπ+2 = 2 (π, 2)

jika titik x = 3π/2, maka y = sin3π/2+2 = (3π/2, 1)

jika titik x = 4π/2 atau x = 2π, maka y = sin2π+2 = (2π, 2)

Terusin aja sampai membentuk punuk unta.
