persamaan logika pada gerbang logika di atas adalah sebagai berikut

Kita perlu melihat gerbang terakhir yang membentuk output Y. Gerbang terakhir adalah OR 3 input, maka persamaan boolean ini akan memiliki 3 suku utama.

Kemudian kita terjemahkan dari inputan OR 3 input yang pertama:

Yaitu A'BC'

Kemudian kita terjemahkan inputan OR 3 input yang kedua yaitu:

A'BC + A'B'C

Kemudian kita terjemahkan inputan OR 3 input yang ketiga yaitu:

AC'

Maka Y = A'BC' + (A'BC + A'B'C) + AC'

Kemudain kita coba sederhanakan tiap suku supaya lebih sederhana

Untuk itu kita bisa lihat teorema aljabar boolean berikut ini

Teorema-teorema Aljabar boolean  

1. X+0 = X  X.1 = X

2. X+1 = 1  X.0 = 0

3. X+X = X  X.X = X

5. X+X’ = 1  X.X’ = 0

Hukum Komutatif

6. X+Y = Y+X  XY=YX

Hukum Asosiatif

7. (X+Y)+Z = X+(Y+Z) = X+Y+Z  (XY)Z = X(YZ) = XYZ

Hukum Distributif

8. X (Y+Z) = XY+XZ  X+YZ = (X+Y)(X+Z)

Hukum Absorbtif

9. XY+XY’ = X  (X+Y)(X+Y’) = X

10. X+XY = X  X(X+Y) = X

11. (X+Y’) = XY  XY’+Y = X+Y

Hukum DeMorgan

12. (X+Y+Z+...)D = XYZ ...  (XYZ...) = X’+Y’+Z’+...

Y = A'BC' + (A'BC + A'B'C) + AC'

kita sederhanakan suku kedua

Pembahasan

Singkat saja ya. Jika sudah mengerti fungsi boolean dalam bentuk minterm dan maxterm, pasti mudah.

ΠM(1,2,5,6) adalah bentuk maxterm dari Σm(0,3,4,7). Mengapa?

Pada fungsi boolean dengan 3 variabel, terdapat 2³ = 8 output, yaitu output 0 (000 biner) hingga output 7 (111 biner). Dari ke-8 output/hasil pada tabel kebenaran, minterm diperoleh dari nilai kebenaran 1 dan maxterm diperoleh dari nilai kebenaran 0.

Maka, dari ΠM(1,2,5,6), yang belum ada dari 0 sampai 7 adalah 0, 3, 4, dan 7, sehingga mintermnya adalah Σm(0,3,4,7).

Kita lanjutkan penyelesaiannya.

F = Σm(0,3,4,7) + ΠM(1,2,5,6)
F = Σm(0,3,4,7) + Σm(0,3,4,7)

Ingat bahwa pada aljabar boolean, p + p = p OR p = true/benar.

Dengan demikian, F = Σm(0,3,4,7) + ΠM(1,2,5,6) = true/benar, dan merupakan tautologi.

34. a. Buktikan dengan 2 cara (Hukum-hukum Aljabar Boolean dan tabel kebenaran) bahwa untuk sembarang elemen a,b, dan c dari aljabar Boolean maka kesamaan berikut adalah benar! a. a + ab = a b. (a+b)(a+c) = a+bc b. Tentukan komplemen fungsi 2 cara (Hukum De Morgan dan Prinsip Dualitas) untuk fungsi berikut: a. f(x,y,z) = xy' (yz'+y) b. f(x,y,z) = x'y (yz'+y')Bantu jawab ​


Untuk membuktikan kesamaan (a+b)(a+c) = a+bc tanpa menggunakan tabel kebenaran, kita dapat menggunakan hukum De Morgan dan prinsip dualitas. Berikut adalah langkah-langkahnya:

1. Buktikan menggunakan Hukum De Morgan:

  Dengan Hukum De Morgan, kita dapat mengubah operasi perkalian menjadi operasi penjumlahan dan sebaliknya. Berikut adalah langkah-langkahnya:

  (a+b)(a+c) = a + ab + ac + bc  (distributif)

             = a(1+b+c) + bc    (distributif)

             = a(1) + bc         (a+a = a dan a+1 = 1)

             = a + bc

  Dengan menggunakan Hukum De Morgan, kita dapat membuktikan bahwa (a+b)(a+c) = a+bc.

2. Buktikan menggunakan Prinsip Dualitas:

  Prinsip Dualitas menyatakan bahwa jika suatu persamaan boolean benar, maka persamaan yang diperoleh dengan menukar operasi penjumlahan dan perkalian serta mengganti 0 dengan 1 dan sebaliknya juga benar.

  Kita dapat menggunakan prinsip dualitas untuk membuktikan kesamaan ini. Berikut adalah langkah-langkahnya:

  Diberikan: (a+b)(a+c) = a+bc

  Dualitas:

  - Ganti operasi penjumlahan dengan perkalian dan sebaliknya.

  - Ganti 0 dengan 1 dan sebaliknya.

  Kita akan mengganti penjumlahan dengan perkalian dan sebaliknya, serta mengganti 0 dengan 1 dan sebaliknya pada persamaan tersebut:

  (a*b)+(a*c) = a+b*c

  Dalam persamaan ini, kita dapat melihat bahwa kedua sisi persamaan adalah sama setelah mengikuti prinsip dualitas. Oleh karena itu, kesamaan (a+b)(a+c) = a+bc benar.

Dengan demikian, kita telah membuktikan kesamaan (a+b)(a+c) = a+bc menggunakan hukum De Morgan dan prinsip dualitas tanpa menggunakan tabel kebenaran.
