19. perbedaan windows 7 dan windows xp​


Jawaban:

Di bawah ini adalah 10 perbedaan utama antara Windows XP dan Windows 7.

1. Tidak Ada e-Mail Client

Outlook Express (OE) telah dihilangkan pada Windows Vista namun digantikan oleh Windows Mail. Anehnya, Windows tidak dilengkapi dengan e-mail client apapun saat pembelian. Pengguna harus membelinya lagi seperti Outlook atau menggunakan servis gratis seperti Windows Live Mail, atau download sebuah e-mail client yang open source seperti Thunderbird.

2. 32-bit dan 64-bit

Memang sebenarnya Windows XP juga memiliki versi 64-bit, namun tidak banyak orang yang mengetahuinya. Saat Anda melakukan upgrade dari Windows XP ke Windows 7, Anda harus memilih versi antara 32-bit atau 64-bit. Anda harus memilihnya berdasarkan hardware, driver, dan software yang Anda miliki.

3. Aero Desktop

Aero Desktop sebenarnya hanya mempercantik tampilan Windows 7 saja, Fitur seperti Aero Snap dapat memudahkan Anda untuk mengatur jendela-jendela yang terbuka dan tranparansi yang ada dapat memudahkan Anda untuk melihat di balik jendela-jendela yang sedang terbuka.

4. Documents and Settings

Folder Documents and Settings yang merupakan tempat dari segala file dan folder personal telah digantikan dengan folder Users.

5. Start Menu

Start Menu pada Windows 7 telah banyak berubah dan mendapat banyak kritik. Start menu tidak lagi menggunakan fly-outs dan scroll-outs untuk memperlihatkan Anda shortcut program dan folder apa saja yang ada di komputer Anda.

Sekarang Anda diharuskan untuk menggunakan scroll bar untuk mengakses shortcut yang tidak terlihat karena telah mencapai batas tempat yang dapat terlihat. Tapi jika Anda lebih senang dengan style Windows XP, Anda dapat mengubahnya sendiri.

6. Ribbon

Pada Office 2007 telah terlihat bagaimana ribbon bekerja seperti menu drop-down dan toolbar. Di Windows 7 Anda juga dapat melihat dan merasakannya pada Microsoft Paint atau Wordpad.

7. Libraries

Libraries pada Windows 7 tidak lebih dari sekedar kumpulan file yang serupa. Kumpulan konten serupa yang terletak di beberapa tempat di komputer Anda akan dibawa ke dalam sistem Library untuk memudahkan pencarian. Tentu saja Anda dapat memilih untuk menggunakan Libraries atau tidak.

8. DirectX 11

Jika Anda seorang gamer, Anda pasti tahu bahwa Anda harus terus mengikuti perkembangan hardware dan software untuk dapat menjalankan game-game yang makin berat. Secara default, Windows XP tidak men-support DirectX di atas 9.0c sehingga bila game Anda membutuhkan versi yang lebih tinggi seperti 10 dan 11, Anda tidak punya pilihan lain selain meng-upgrade Windows.

9. HomeGroup

HomeGroup adalah sebuah perubahan besar dalam home networking yang membuat paradigma sebelumnya tampak menjadi kuno dengan menggabungkan kesederhanaan, pengaturan yang mudah, dan koneksi yang stabil.

10. Touch Support

Windows 7 dahulu merupakan operating system pertama yang men-support touch atau sentuhan sebagai interface

22. berikut file dalam dos yang berfungsi sebagai boot loader adalah


file dalam dos yang berfungsi sebagai boot loader adalah mbldr.exe (CMIIW) File tersebut berfungsi sebagai boot loader membawa OS untuk memasuki boot yang ada di dalam sistem operasi.

MBLDR merupakan kependekan atau singkatan dari Master Boot Loader. Mbldr dibuat dengan base OS Boot Select (OS-BS) by Thomas wolfRAM


Pembahasan

Sebagai intermezzo, Boot loader atau dalam bahasa indonesia pemuat boot. Bootloader merupakan program komputer yang memuat sistem operasi (OS) untuk komputer secara sederhananya digunakan untuk memanggil boot sistem operasi dan memilih akan menggunakan Ssistem operasi yang mana.

Bootloader dapat digunakan untuk mengkonfigurasi  dan sarana untuk memilih darimana memulai kernel. Hal ini memungkinkan untuk melakukan aktivitas boot yang memiliki 2 Sistem operasi seperti contoh untuk mengunakan Linux alternatif, atau Windows. Bootloader adalah komponen penting dari proses booting yang harus dipastikan dalam keadaan baik, memori bootloader tersebut tersimpan dalam memori non-volatile contohnya seperti memori flash.

34. apa yang anda ketahui tentang boot loader pada linux ?


Bootloader atau boot manager merupakan sebuah aplikasi yang terdapat hampir di seluruh sistem operasi termasuk linux. Boot manager ini digunakan untuk multiple boot. Dengan boot manager, kita bisa mengatur proses booting. Bila kita menginginkan OS yang kita gunakan dibaca oleh BIOS tanpa menggunakan Disk Boot maka kita memerlukan Boot Loader program yang terinstall pada MBR ( Master Boot Record ). Boot loader program dimuat di dalam BIOS komputer dan bertugas untuk membaca kernel yang ada di dalam suatu sistem operasi serta memberi kendali terhadap jalannya sistem pada kernel. Kernel akan dapat melakukan inisiasi pada sistem serta mengendalikannya. Sebagai contoh, saat kita ingin menggunakan 2 sistem operasi pada satu komputer, misal ingin menginstall ubuntu dimana kita telah menginstall windows vista di komputer. Apabila ubuntu telah terinstall di hardisk maka secara otomatis ubuntu akan mengeluarkan pilihan booting untuk multiple boot. Secara singkat, boot loader adalah program software pertama yang berjalan ketika komputer dimulai. Hal ini bertanggung jawab untuk loading dan transfer kontrol ke kernel sistem operasi perangkat lunak (seperti Hurd atau Linux). Kernel, pada gilirannya, menginisialisasi sisa dari sistem operasi (misalnya GNU).
