17. turunan pertama dari f(x) cot (x²+x√x)​


Jawaban

[tex]f'(x) =(2x - \frac{3}{2} \sqrt{x} ) \: \cot²(x² + x√x)[/tex]

Pembahasan

f(x) = cot (x² + x√x)

[tex]cot (x² + x√x) = \frac{u}{v} \\ cot (x² + x√x) = \frac{ \cos(x² + x√x) }{ \sin(x² + x√x) } [/tex]

u = cos (x² + x√x)

[tex]u' = - ( \frac{3}{2} \sqrt{x} - 2x)sin( {x}^{2} + x \sqrt{x} )[/tex]

v = sin (x² + x√x)

[tex]v' = ( \frac{3}{2} \sqrt{x} - 2x) \cos( {x}^{2} + x \sqrt{x} )[/tex]

[tex]f'(x) = \frac{u'v - uv'}{ {v}^{2} } [/tex]

untuk lebih singkat, gunakan permisalan

[tex]\frac{3}{2} \sqrt{x} - 2x = a[/tex]

x² + x√x = b

[tex]f'(x) = \frac{ - a \: sin ²(b) -a\cos ²(b)}{ { \sin}^{2} (b)}[/tex]

[tex]f'(x) = \frac{ - a \: sin ²(b) -a(1 - { \sin}^{2}(b) )}{ { \sin}^{2} (b)}[/tex]

[tex]f'(x) = \frac{ - a \: sin ²(b) -a + { \sin}^{2}(b) }{ { \sin}^{2} (b)}[/tex]

[tex]f'(x) = 1- a + \frac{ - a }{ { \sin}^{2} (b)}[/tex]

f'(x) = 1 -a -a csc²(b)

f'(x) = 1 -a (1 + csc²(b))

[tex]f'(x) = 1 - (2x - \frac{3}{2} \sqrt{x} ) \: ( 1 + \csc(x² + x√x) )[/tex]

[tex]f'(x) = 1 - ( 1 + \csc²(x² + x√x) ) \: (2x - \frac{3}{2} \sqrt{x} )[/tex]

[tex]f'(x) = (1 - \csc²(x² + x√x)) \: (2x - \frac{3}{2} \sqrt{x} )[/tex]

[tex]f'(x) =(2x - \frac{3}{2} \sqrt{x} ) \: \cot²(x² + x√x)[/tex]

18. Tentukan turunan f(x) =cot x


Jawab:

-(csc x)^2

Penjelasan dengan langkah-langkah:

f(x) = cot x

= cos x/sin x

Misalkan

u = cos x

u' = -sin x

v = sin x

v' = cos x

Maka,

f'(x) = (u'v - uv')/v^2

= (-sin x . sin x - cos x . cos x)/(sin x)^2

= (-(sin x)^2 - (cos x)^2)/(sin x)^2

= -((sin x)^2 + (cos x)^2)/(sin x)^2

= -1/(sin x)^2

= -(csc x)^2

33. Tentukan turunan fungsi dari [tex]\displaystyle y=\frac{\tan x}{1-\cot x}+\frac{\cot x}{1-\tan x}[/tex] dan nyatakan hasilnya dalam bentuk paling sederhana.


Turunan fungsi dari

[tex]\begin{aligned}\vphantom{\bigg|}y&=\frac{\tan x}{1-\cot x}+\frac{\cot x}{1-\tan x}\\\end{aligned}[/tex]

dalam bentuk yang paling sederhana adalah

[tex]\boxed{\:y'=\sec^2x-\csc^2x\:}[/tex]

atau

[tex]\boxed{\:y'=-4\cot(2x)\csc(2x)\:}[/tex]
 

Penjelasan dengan langkah-langkah:

Sederhanakan dulu, baru turunkan.

[tex]\begin{aligned}\vphantom{\bigg|}y&=\frac{\tan x}{1-\cot x}+\frac{\cot x}{1-\tan x}\\\vphantom{\bigg|}&=\frac{\tan x}{\ 1-\frac{\cos x}{\sin x}\ }+\frac{\cot x}{\ 1-\frac{\sin x}{\cos x}\ }\\\vphantom{\bigg|}&=\frac{\tan x}{\ \frac{\sin x-\cos x}{\sin x}\ }+\frac{\cot x}{\ \frac{\cos x-\sin x}{\cos x}\ }\\\vphantom{\bigg|}&=\frac{\sin x\tan x-\cos x\cot x}{\sin x-\cos x}\\\vphantom{\bigg|}&=\frac{\frac{\sin^2x}{\cos x}-\frac{\cos^2x}{\sin x}}{\sin x-\cos x}\end{aligned}[/tex]
[tex]\begin{aligned}y\vphantom{\bigg|}&=\frac{1}{\sin x\cos x}\cdot\frac{\sin^3x-\cos^3x}{\sin x-\cos x}\\\vphantom{\bigg|}&=\frac{1}{\sin x\cos x}\cdot\left(\sin^2x+\sin x\cos x+\cos^2x\right)\\\vphantom{\bigg|}&=\frac{1}{\sin x\cos x}\cdot\left(1+\sin x\cos x\right)\\\vphantom{\bigg|}&=\frac{1}{\sin x\cos x}+1\\\vphantom{\bigg|}&=\frac{2}{2\sin x\cos x}+1\\\vphantom{\bigg|}&=\frac{2}{\sin2x}+1\\y\vphantom{\Big|}&=2\csc2x+1\end{aligned}[/tex]

Turunannya:

[tex]\begin{aligned}y'&=\frac{d}{dx}(2\csc(2x)+1)\\&=\frac{d}{dx}(2\csc(2x))+\frac{d}{dx}(1)\\&=2\cdot\frac{d}{dx}(\csc(2x))\\&=2\left(-\cot(2x)\right)\cdot\csc(2x)\cdot\frac{d}{dx}(2x)\\&=-2\cot(2x)\csc(2x)\cdot2\\y'&=\boxed{\:-4\cot(2x)\csc(2x)\:}\\&=(-4)\frac{\cos(2x)}{\sin^2(2x)}\\&=(-4)\frac{\cos^2x-\sin^2x}{4\sin^2 x\cos^2 x}\\&=(-1)\frac{\cos^2x-\sin^2x}{\sin^2 x\cos^2 x}\\&=(-1)\left(\frac{1}{\sin^2x}-\frac{1}{\cos^2x}\right)\\y'&=\boxed{\:\sec^2x-\csc^2x\:}\end{aligned}[/tex]
[tex]\blacksquare[/tex]

[tex]f'(x)= \lim_{h \to 0} \frac{f(x+h)-f(x)}{h}~~~~\to~f(x)=cotx\\\\f'(x)=\lim_{h \to 0} \frac{cot(x+h)-cotx}{h}\\\\f'(x)=\lim_{h \to 0} \frac{\frac{1}{tan(x+h)}-\frac{1}{tanx}}{h}\\\\f'(x)=\lim_{h \to 0} \frac{\frac{tanx-tan(x+h)}{tan(x+h).tanx}}{h}\\\\f'(x)=\lim_{h \to 0} \frac{tanx-tan(x+h)}{tan(x+h).tanx.h}~~~~~~...gunakan~tan(a-b)=\frac{tana-tanb}{1+tana.tanb} \\\\f'(x)=\lim_{h \to 0} \frac{tan(x-x-h)(1+tanx.tan(x+h)}{tan(x+h).tanx.h}\\[/tex]

[tex]\\f'(x)=\lim_{h \to 0} \frac{tan(-h)}{h}.\frac{(1+tanx.tan(x+h))}{tan(x+h).tanx}\\\\f'(x)=\lim_{h \to 0} \frac{-tan(h)}{h}. \lim_{h \to 0} \frac{(1+tanx.tan(x+h))}{tan(x+h).tanx}\\\\f'(x)=-1.\frac{(1+tanx.tan(x+0))}{tan(x+0).tanx}\\\\f'(x)=-\frac{1+tan^2x}{tan^2x}\\\\f'(x)=-\frac{sec^2x}{tan^2x}\\\\f'(x)=-\frac{1}{cos^2x}.\frac{cos^2x}{sin^2x}\\\\f'(x)=-cosec^2x\\\\\\maka~turunan~dari~f(x)=cotx~adalah~f'(x)=-cosec^2x[/tex]

40. buktikan rumus dasar turunan fungsi trigonometri berikut dengan f(x)=cot x adalah f'=-csc² x, plis bntuin dong​


Terbukti bahwa turunan dari [tex]f(x)=cotx[/tex] adalah [tex]\boldsymbol{f'(x)=-csc^2x}[/tex].

PEMBAHASAN

Turunan atau Diferensial merupakan pengukuran terhadap bagaimana fungsi berubah seiring perubahan nilai input. Turunan didefinisikan sebagai berikut :

[tex]f'(x)= \lim_{h \to 0} \frac{f(x+h)-f(x)}{h}[/tex]

Selain menggunakan definisi, turunan dari sautu fungsi f(x) dapat kita cari juga dengan menggunakan rumus - rumus berikut ini.

[tex](i)~y=ax^k~~\to~~y'=kax^{k-1}[/tex]

[tex](ii)~y=f(x)\pm g(x)~~\to~~y'=f'(x)\pm g'(x)[/tex]

[tex](iii)~y=f(x)g(x)~~\to~~y'=f'(x)g(x)+f(x)g'(x)[/tex]

[tex](iv)~y=\frac{f(x)}{g(x)}~~\to~~y'=\frac{f'(x)g(x)-f(x)g'(x)}{[g(x)]^2}[/tex]

Rumus turunan untuk fungsi trigonometri :

[tex](i)~y=sinax~~\to~~y'=acosax[/tex]

[tex](ii)~y=cosax~~\to~~y'=-asinax[/tex]

[tex](iii)~y=tanax~~\to~~y'=asec^2ax[/tex]

.

DIKETAHUI

Turunan dari [tex]f(x)=cotx[/tex] adalah [tex]f'(x)=-csc^2x[/tex].

.

DITANYA

Buktikan rumus turunan fungsi tersebut.

.

PENYELESAIAN

> Menggunakan definisi turunan.

[tex]f(x)=cotx[/tex]

[tex]f(x)=\frac{1}{tanx}[/tex]

.

[tex]f'(x)= \lim_{h \to 0} \frac{f(x+h)-f(x)}{h}[/tex]

[tex]f'(x)= \lim_{h \to 0} \frac{\frac{1}{tan(x+h)}-\frac{1}{tanx}}{h}[/tex]

[tex]f'(x)= \lim_{h \to 0} \frac{\frac{tanx-tan(x+h)}{tan(x+h)tanx}}{h}[/tex]

[tex]f'(x)= \lim_{h \to 0} \frac{tanx-\left ( \frac{tanx+tanh}{1-tanx.tanh} \right )}{htan(x+h)tanx}[/tex]

[tex]f'(x)= \lim_{h \to 0} \frac{\frac{tanx(1-tanx.tanh)-tanx-tanh}{1-tanx.tanh}}{h\left ( \frac{tanx+tanh}{1-tanx.tanh} \right )tanx}[/tex]

[tex]f'(x)= \lim_{h \to 0} \frac{tanx-tan^2x.tanh-tanx-tanh}{htanx(tanx+tanh)}[/tex]

[tex]f'(x)= \lim_{h \to 0} \frac{-tanh(tan^2x+1)}{htanx(tanx+tanh)}[/tex]

[tex]f'(x)= \lim_{h \to 0} -\frac{tanh}{h} \times\lim_{h \to 0}\frac{tan^2x+1}{(tanx+tanh)tanx}[/tex]

[tex]f'(x)=-1\times\frac{tan^2x+1}{(tanx+tan0)tanx}[/tex]

[tex]f'(x)=-\frac{tan^2x+1}{(tanx+0)tanx}[/tex]

[tex]f'(x)=-\frac{tan^2x-1}{tan^2x}[/tex]

[tex]f'(x)=-(1+cot^2x)[/tex]

[tex]f'(x)=-csc^2x~(\boldsymbol{terbukti})[/tex]

.

> Menggunakan rumus turunan.

[tex]f(x)=cotx[/tex]

[tex]f(x)=\frac{cosx}{sinx}[/tex]

.

Misal :

[tex]u=cosx~\to~u'=-sinx[/tex]

[tex]v=sinx~\to~v'=cosx[/tex]

.

Maka :

[tex]f'(x)=\frac{u'v-uv'}{v^2}[/tex]

[tex]f'(x)=\frac{-sinx(sinx)-cosx(cosx)}{sin^2x}[/tex]

[tex]f'(x)=\frac{-sin^2x-cos^2x}{sin^2x}[/tex]

[tex]f'(x)=\frac{-(sin^2x+cos^2x)}{sin^2x}[/tex]

[tex]f'(x)=-\frac{1}{sin^2x}[/tex]

[tex]f'(x)=-csc^2x~(\boldsymbol{terbukti})[/tex]

.

KESIMPULAN

Terbukti bahwa turunan dari [tex]f(x)=cotx[/tex] adalah [tex]\boldsymbol{f'(x)=-csc^2x}[/tex].

.
