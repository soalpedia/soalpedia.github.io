1. Apakah fungsi komputer server (server komputer) pada internet?


- Sebagai pusat dari sebuah jaringan 
- Menyediakan akses data bagi client 
- Mengatur lalu lintas data antara server dengan client, sebaliknya dan antar client 
- Mengatur hak akses pada jaringan & client
- Melindungi jaringan dari ancaman security/keamanan seperti malware dan virus

10. Komputer Server juga berfungsi sebagai gerbang komputer yang disebut…. *​


Komputer Server juga berfungsi sebagai gerbang komputer yang disebut proxy server. Proxy Server disebut juga dengan peladen pewali. Proxy Server diletakkan di antara aplikasi server dengan aplikasi client. Proxy Server berperan sebagai penghubung komputer dengan internet.

Penjelasan

Berdasarkan karakteristiknya, proxy server dibedakan menjadi :

Anonymous Proxy adalah jenis proxy yang memberikan informasi mengenai komputer pengguna.High Anonymity Proxy adalah jenis proxy yang tidak menyediakan alamat IP original.Transparent Proxy adalah jenis proxy yang melanjutkan permintaan pengguna ke sumber yang diinginkan tanpa mengungkap informasi komputer pengguna.Reverse Proxy adalah jenis proxy yang digunakan untuk mencegah klien terhubung dengan data.

18. komputer server berfungsi sebagai


komputer server berfungsi sebagai induk dari komputer yang terhubung di sebuah jaringan

1. Aplication Server, .server ini berfungsi sebagai media penyimpanan aplikasi-aplikasi yang nantinya akan di akses dari komputer client. Misalnya aplikasi transaksi jual beli dan aplikasi lainnya yang jalan di intranet atau internet.

2. Proxy Server, biasanya digunakan untuk mengatur lalu lintas di jaringan melalui pengaturan proxy. Orang awam lebih mengenal proxy server untuk mengkoneksikan komputer klien ke Internet.

3.Database Server, berfungsi untuk menyimpan data-data. Ada beberapa orang meletakkan data terpisah dari server aplikasinya, biasanya terinstal semacam Database management system, seperti oracle, MS SQL Server, MySQL dan sebagainya

4. Game Server, server yang berfungsi menyediakan game online, sehingga kita bisa bermain di jaringanPrinter Server, berfungsi mengatur printer yang berjalan pada sebuah jaringan.

5. Web Server, menyediakan konten statis ke web browser dengan memuat file dari disk dan mentransfer melalui jaringan ke web browser pengguna. Dan lain sebagainya.

36. Apakah fungsi komputer server pada jaringan komputer?


1. Bertanggung jawab melayani permintaan komputer client.

2. Menyediakan resource untuk di gunakan bersama baik itu perangkat keras ataupun berupa aplikasi agar dapat di gunakan di semua komputer client di dalam jaringan.

3. Bertanggungjawab mengatur lalu lintas data.

4. Dapat menyimpan file, data untuk di akes bersama menggunakan file sharing.

5. Mampu mengatur hak akses level dalam jaringan, sehingga tidak semua client bisa membuka data yang di simpan di komputer server.

6.  menyediakan database atau aplikasi yang dapat di jalankan di semua komputer.

7. Melindungi komputer client dengan memasang firewall atau anti malware di komputer server.
