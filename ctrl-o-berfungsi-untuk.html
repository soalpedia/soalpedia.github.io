33. a. fungsi dari Ctrl + fb. fungsi dari Ctrl + oc. fungsi dari Ctrl + dd. fungsi dari Ctrl + ve. fungsi dari Ctrl + uf. fungsi dari Ctrl + cnote:✨ mau habisin points ✨jawab dengan benar ✔️❌ ngasal auto lapor mod ❌​

a. Ctrl + F - Temukan Kalimat/Huruf (Find). Ctrl + G - Pergi ke (go to). Ctrl + H - Temukan Kalimat dan Tindih (Find and Replace). Ctrl + I - Membuat cetak miring (Italic)

b. Fungsi ini menggantikan klik kanan > open pada mouse. Ctrl + N = fungsi shortcut untuk membuat berkas baru/New. Ctrl + O = fungsi shortcut untuk membuka berkas sebelumnya/Open yang sudah disimpan

c. Ctrl + T = fungsi shortcut untuk membuka tab baru pada browser. Ctrl + D = fungsi shortcut untuk melakukan bookmark pada halaman yang sedang di buka

d. Ctrl + X = fungsi untuk cut/memotong teks yang dipilih. Ctrl + C = fungsi untuk copy/menduplikat teks yang dipilih. Ctrl + V = fungsi untuk paste/meletakan teks yang dicopy/cut sebelumnya

f. Ctrl + C = fungsi untuk copy/menduplikat teks yang dipilih.

● Fungsi dari Ctrl + F = mencari File dan membuka menu/kotak dialog Find and Replace

● Fungsi dari CTRL + O = membuka menu/kotak dialog open

● Fungsi dari CTRL + V = mentempel/menampilkan hasil teks yang sudah di copy sebelumnya

● Fungsi dari CTRL + U = membuat garis bawah teks

● Fungsi dari CTRL + C = mengcopy/menyalin teks maupun File

36. aTombol yang digunakan untuk menggandakan data adalah ....a Ctrl+ C dan Ctrl + V b. Ctrl+ C dan Ctrl + O c. Ctrl+​


Tombol atau shortcut yang digunakan untuk menggunakan data adalah Ctrl+ C dan Ctrl + V (Jawaban A). Simak pada bagian pembahasan.

Pembahasan

Di dalam menjalankan sebuah aplikasi misalnya kita ambil contoh pada microsoft excel, maka akan terdapat banyak toolbars yang akan kita gunakan untuk membuat sebuah action atau perintah.

Pada umumnya kita menggunakan mouse dan melakukan perintah dengan klik kanan atau pun klik kiri. Namun untuk sebagian orang maka kita dapat menggunakan jalan pintas (Shortcut) untuk beberapa action sebagaimana contoh berikut di bawah ini:

Ctrl + c = Untuk menduplikat data.Ctrl + v = Untuk menempel (paste) data yang tadi kita copy.Ctrl + a = Untuk melakukan seleksi pada semua bagian.Ctrl + s = untuk menyimpan file.Ctrl + o = untuk membuka file.Ctrl + n = untuk membuat document baru.

Selain pada microsoft excel maka shortcut tersebut bisa diaplikasikan pada aplikasi lainnya.

Demikian, semoga membantu!
