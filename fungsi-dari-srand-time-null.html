8. jelaskan cara merumuskan hipotesis null dan hipotesis alternatif


Hipotensi nol(Ho) yaitu hipotesis yg akan diuji. biasanya, hipotesis ini merupakan pernyataan yg menunjukkan bahwa suatu parameter populasi memiliki nilai tertentu. hipotesis nol bisa dinyatakan dengan kata kata "tidak ada perbedaan". Hipotesis alternatif (H1) adalah pernyataan yg sama dengan parameter populasi yg sama dengan yg digunakan dalam hipotesis nol. biasanya hipotesis ini merupakan pernyataan yang menyatakan bahwa parameter populasi tersebut memiliki nilai yg berbeda dari pernyataan yg telah disebutkan dalam hipotesis nol.

34. Pengertian dari CASCADE,RESTRICT,SET NULL dan NO ACTION pada mysql....Tolongbdi bantu ya kakak kakak sekalian terimakasih


RESCRICT adalah jika kita menghapus atau merubah baris data dalam tabel A maka tidak akan diperbolehkan jika pada tabel B masih ditemukan relasi datanya. InnoDB dapat menolak perintah perubahan atau penghapusan tersebut.

CASCADE adalah jika kita menghapus atau merubah baris data dalam tabel A secara otomatis akan menghapus atau merubah baris yang sesuai dalam tabel B.

38. contoh query update agar bisa menampung nilai NULL​


Untuk mengupdate sebuah record di tabel database dengan nilai NULL, kamu dapat menggunakan perintah `UPDATE` seperti ini:

UPDATE tabel SET kolom = NULL WHERE kondisi;

Contoh:

UPDATE users SET email = NULL WHERE id = 1;

Ini akan mengupdate kolom email untuk record dengan id 1 dengan nilai NULL.

Perhatikan bahwa jika kamu menggunakan tanda petik tunggal (') atau tanda kutip (") untuk menyatakan nilai NULL, maka akan dianggap sebagai string, bukan sebagai nilai NULL. Jadi pastikan untuk tidak menggunakan tanda petik apapun saat menyatakan nilai NULL.

Jika kamu menggunakan salah satu dari bahasa pemrograman yang menggunakan tanda petik tunggal atau tanda kutip sebagai tanda string, maka Anda harus menggunakan tanda backslash () untuk mengabaikan tanda petik tersebut, seperti ini:

UPDATE users SET email = '\'NULL\'' WHERE id = 1;

Atau, kamu dapat menggunakan tanda petik ganda (") untuk menyatakan nilai NULL, seperti ini:

UPDATE users SET email = "NULL" WHERE id = 1;

Namun, lebih disarankan untuk tidak menggunakan tanda petik sama sekali dan hanya menggunakan NULL saja.
