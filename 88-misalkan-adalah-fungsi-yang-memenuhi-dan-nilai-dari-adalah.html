1. misalkan nilai dari kategori B adalah 22 dan jumlah nilai dari seluruh kategori adalah 88, maka besarnya persentase dan besarnya derajat dari B itu masing-masing adalah? ​


Presentase dari kategori B terhadap seluruh kategori adalah 25% dan besarnya derajat kategori B terhadap seluruh kategori adalah 90°.

Penjelasan dengan langkah-langkah:

Diketahui :

Jumlah kategori B = 22

Jumlah seluruh kategori = 88

Ditanya : Besarnya presentase dan derajat dari ketegori B

Jawab :

Untuk menentukan besarnya presentase dan derajat dari kategori B terhadap keseluruhan kategori, kita bisa menggunakan rumus berikut

% Kategori B = (Jumlah kategori B / Jumlah seluruh kategori ) x 100%

Derajat kategori B = (Jumlah kategori B / Jumlah seluruh kategori ) x 360°

Menentukan presentase dari kategori B

% Kategori B = (Jumlah kategori B / Jumlah seluruh kategori ) x 100%

% Kategori B = ( 22 / 88 ) x 100%

% Kategori B = 25 %

Menentukan besar derajat dari kategori B

Derajat kategori B = (Jumlah kategori B / Jumlah seluruh kategori ) x 360°

Derajat kategori B = ( 22/ 88 ) x 360°

Derajat kategori B = 90°

Sehingga besarnya presentase dari kategori B terhadap seluruh kategori adalah 25% dan besarnya derajat kategori B terhadap seluruh kategori adalah 90°.

23. Sebuah bak mandi berbentuk balok dengan ukuran panjang = a, lebar = b, dan tinggi = Misalkan a = 2b dan c = 4b. Jika volume bak mandi tersebut 88 liter, berapa nilai b?


a = panjang

b = lebar

c = tinggi

V = p x l x t

88 = 2b x l x 4b

88 = 8b x l

l = 88 : 8b

l = 11b

jadi nilai b = 11b

Diketahui

Bak mandi berbentuk balok dengan ukuran:

panjang= 2 b

tinggi =4 b

Volume = 88 liter

Ditanya = Berapa Nilai b?

Penjelasan

Nilai b adalah nilai lebar karena pada soal yang ditanyakan adalah lebar yang menggunakan Variabel b.

Untuk b dekat angka adalah Variabel atau pengganti satuan dengan huruf . seperti 2 b dan 4 b ini sama halnya dengan 4 cm dan 2 cm akan tetapi satuannya lah yang diubah menjadi huruf sebagai pengganti satuan.

Untuk yang belum tau apa itu Variabel. Variabel adalah huruf pada suatu bilangan tertentu sebagai pengganti angka ataupun bilangan . Biasanya Variabel terdapat pada Aljabar dan Himpunan.

Penyelesaian

Rumus Lebar Balok

[tex]l = \frac{p \times t}{volume} [/tex]

[tex]l = \frac{2b \times 4b}{88} [/tex]

[tex]l = \frac{8b}{88 \: liter} [/tex]

[tex]l = 11b[/tex]

Jadi Lebar Balok tersebut adalah 11 b
