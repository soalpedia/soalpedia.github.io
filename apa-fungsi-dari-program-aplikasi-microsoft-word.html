16. Seperti apakah cara memulai program aplikasi Microsoft Word 2007?


Aplikasi Microsoft Word 2007 merupakan aplikasi yang sangat populer untuk melakukan pengolahan kata. Dengan aplikasi ini, kita bisa melakukan format pengetikan seperti penomoran, pengaturan spasi, margin (jarak pinggir kertas), dan jenis huruf (font). Dengan demikian, dokumen yang di buat menggunakan aplikasi ini lebih rapi, jika terjadi kesalahan dapat dikoreksi kembali, dapat disimpan sehingga dapat diambil ketika diperlukan kembali, dan dapat dicetak di atas kertas.

Untuk bisa memulai program aplikasi Microsoft Word 2007, ada beberapa cara, yaitu:

32. Microsoft word merupakan program aplikasi ...


Microsoft Word merupakan program aplikasi dari microsoft office yang dimana biasanya sering di gunakan dalam pengelolahan teks, pengelolahan dokumen, dan juga dalam sebuah laporan. Yang mana Microsoft Word itu sendiri sangat banyak digunakan untuk mempermudah pengguna dalam menyelesaikan sebuah pekerjaan.

Pembahasan

Adapun terdapat beberapa manfaat dari Microsoft Word, yang mana diantaranya ialah dapat kita lihat sebagai berikut:

Dapat memudahkan dalam hal yang berkaitan dengan berkomunikasi.Dapat digunakan untuk membuat sebuah dokumen.Menjadikan sebuah dokumen tersebut menjadi lebih menarik.Dapat menyisipkan dan menambahkan gambar.Dapat meminimalisir jika terdapat sebuah kesalahan dalam penulisan.Dapat membuat berbagai jenis surat.

Microsoft word itu sendiri memiliki sebuah tampilan yang user friendly yang mana dapat memudahkan para penggunanya dan juga dapat digunakan di beberapa OS, sepert MacOS, Linux, dan Windows. Seperti aplikasi lainnya, Microsoft word juga memiliki beberapa kelebihan,  yang mana diantaranya ialah dapat kita lihat sebagai berikut:

Tampilan yang user friendly.Fitur pengolah kata yang lengkap.Memiliki template yang banyak.Dapat menyimpan dokumen dengan beberapa format.
