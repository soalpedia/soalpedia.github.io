4. Apakah pengertian dari database, server, proxy, ssh, dan ip :)


database, menurut saya adalah sekumpulan data yang telah tersimpan dan diatur, sehingga bisa dicari atau diambil debngan efisien ...
server, sebuah sistem komputer yang menyediakan jenis layanan tertentu dalam sebuah jaringan komputer ( kurang paham )
proxy, suatu mekanisme dan service serta jembatan anda ke facebook.com sekaligus memberikan konten yang sudah/bisa di cache/save di direktory proxy server untuk saat nanti jika anda mengakses ke facebook.com lagi tanpa harus mengambil langsung dari server facebook. 
ssh, (penjelasanya susah, tapi saya sering gunain ssh untuk gratisan)
ip, nama komputer anda dalam nilai angka di jaringan, komputer mengenali komputer lainnya dengan angka misal 192.168.1.100 bukannya nama dalam huruf seperti hostname atau domain seperti facebook.com. 

database = kumpulan informasi yang disimpan di dalam komputer secara sistematik untuk memperoleh informasi dari basis data tersebut.
server = sebuah sistem komputer yang menyediakan jenis layanan tertentu dalam sebuah jaringan komputer.
proxy = suatu server yang menyediakan layanan untuk meneruskan setiap permintaan kita kepada server lain di internet.
ssh = (Secure Shell) protokol jaringan yang memungkinkan pertukaran data melalui saluran aman antara dua perangkat jaringan.
ip = (internet protocol) suatu cara untuk mengetahui asal atau alamat suatu komputer berupa sistem penomoran masing-masing komputer yang bersifat unik atau tidak sama. Sistem penomoran itu sendiri terdiri dari empat bagian yang dipisahkan oleh titik contoh : 202.155.245.2

24. Pentingnya media penyimpanan untuk sebuah sistem database server


1.Efisien
Dapat menghasilkan informasi yang dibutuhkan secara cepat, tergantung dari perancangannya sendiri karna dapat menseleksi informasi yang diinginkan.

2.Kontrol data
Mempermudah pengontrolan data menjadi terpusat dari banyak admin,divisi ketika ingin update cukup dengan satu database saja.

3.Biaya
Dengan memiliki system database terpusat, jadi lebih menghemat biaya perangkat penyimpanan, karna hanya dengan satu server utama, divisi lain tidak memerlukan perangkat penyimpanan.

4.Keamanan data
Mayoritas sistem penyimpanan data saat ini memiliki control pengguna, seperti dapat memberikan grant/izin/hak aksen kepada semua pengguna tergantung kepentingan dan kebutuhan masing-masing pengguna,jadi data akan tetap aman dan terkontrol walaupun banyak penggunanya, Selain itu setiap database yang disimpan diperlukan sebuah password untuk dapat mengaksesnya.

5.Membantu pembuatan aplikasi baru
Ketika perusahaan membutuh kan aplikasi baru dan sistem databasenya sudah tersusun dengan baik maka programmer hanya perlu membuat tampilan interface yang dibutuhkan oleh perusahaan.

jenis topologi

1. Topologi Ring Pada topologi ring setiap komputer di hubungkan dengan komputer lain dan seterusnya sampai kembali lagi ke komputer pertama, dan membentuk lingkaran sehingga disebut ring, topologi ini berkomunikasi menggunakan data token untuk mengontrol hak akses komputer untuk menerima data, misalnya komputer 1 akan mengirim file ke komputer 4, maka data akan melewati komputer 2 dan 3 sampai di terima oleh komputer 4, jadi sebuah komputer akan melanjutkan pengiriman data jika yang dituju bukan IP Address dia.

2. Topologi Bus Topologi jaringan komputer bus tersusun rapi seperti antrian dan  menggunakan cuma satu kabel coaxial dan setiap komputer terhubung ke kabel menggunakan konektor BNC, dan kedua ujung dari kabel coaxial harus diakhiri oleh terminator.

3. Topologi Star Topologi ini membentuk seperti bintang karena semua komputer di hubungkan ke sebuah hub atau switch dengan kabel UTP, sehingga hub/switch lah pusat dari jaringan dan bertugas untuk mengontrol lalu lintas data, jadi jika komputer 1 ingin mengirim data ke komputer 4, data akan dikirim ke switch dan langsung di kirimkan ke komputer tujuan tanpa melewati komputer lain.

4. Topologi Mesh Pada topologi ini setiap komputer akan terhubung dengan komputer lain dalam jaringannya menggunakan kabel tunggal, jadi proses pengiriman data akan langsung mencapai komputer tujuan tanpa melalui komputer lain ataupun switch atau hub.

5. Topologi Tree Topologi jaringan komputer Tree merupakan gabungan dari beberapa topologi star yang dihubungan dengan topologi bus, jadi setiap topologi star akan terhubung ke topologi star lainnya menggunakan topologi bus, biasanya dalam topologi ini terdapat beberapa tingkatan jaringan, dan jaringan yang berada pada tingkat yang lebih tinggi dapat mengontrol jaringan yang berada pada tingkat yang lebih rendah

34. Jelaskan keterkaitan antara dns server, web server dan database server

DNS = Domain Name Server, berfungsi untuk mengubah alamat ip kita menjadi sebuah nama yang kita inginkan dan memudahkan orang lain dalam mengakses website kita.

Web server = Sebuah server perangkat lunak yang berfungsi untuk melayani permintaan(request) klien berupa HTTP atau HTTPS dan akan mengirimkan respon sesuai yang diminta.

Database server = Sebuah program komputer yang menyediakan layanan pengelolaan basis data dan melayani sebuah komputer yang kemudian disalurkan ke komputer lainnya menggunakan klien-server

DNS diperlukan untuk memudahkan klien dalam mengakses website kita. Web server dibutuhkan agar terjadi request dan respon antara klien dan server. Dan database dibutuhkan untuk pengelolaan data di sebuah website kita yang menjadikannya dynamic(berubah-ubah).

Penjelasan:

Persis dengan namanya, software ini adalah server database rilisan microsoft.

Awalnya, software ini dipakai untuk mengolah database berskala kecil dan menengah. Namun seiring berjalannya waktu, microsoft SQL server sudah bisa digunakan untuk mengolah database berskala besar.

Selain itu, bahasa pemrograman yang digunakan software ini adalah transact-SQL berstandar ISO/ANSI yang sudah populer dipakai oleh microsoft dan sybase.
