Pada aplikasi mircosoft word, terdapat bebrapa menu yang mempermudah para penggunanya. Salah satu menu pada aplikasi microsoft word adalah menu file.

Pembahasan

Microsoft word merupakan sebuah aplikasi pengolah kata yang banyak digunakan oleh hampir seluruh masyarakat di dunia. Microsoft word dibuat dengan mengedepankan user friendy, agar pengguna dapat menggunakan Microsoft word dengan mudah.

Dalam microsoft word, terdapat beberapa menu icon yang memiliki fungsi-fungsi sendiri. Diantara adalah yang terdapat dalam menu file. Berikut adalah pejelasan mengenai icon yang terdapat dalam menu file.

New

Menu new digunakan utnuk mebuka dokumen lembar kerja baru di microsoft word.

Open

Menu open digunakan untuk membuka file dokumen yang tersimpan di komputer.

Save as web page

Menu save as digunakan untuk menyimpan file dokumen ke dalam format tertentu didalam komputer.

Print

Print digunakan untuk mencetak file yang sedang kedalam bentuk cetak atau hard file.

Close

Close digunakan utnuk untuk menutup dokumen file yang masih aktif.

14. Sebutkan Fungsi dari masing - masing isi item pada Menu Bar File (Info, New, Open, Save, Save As, Print, Share, Export, Close, Account, Option) ?


1.. Info.

Memunculkan tab informasi dokumen meliputi ukuran, judul, tag, kategori dokumen (Properties), tanggal pembuatan, dan pembaruan (Related Dates), penulis (Related People), dan mengunci memberikan pasword untuk melindungi privasi dokumen (Protect Workbook)

2.New.

Membuat dokumen baru, mencari dan menerapkan template atau tema online ke dalam project.

3.Open.

Membuka dokumen atau berkas yang sudah pernah dibuat, entah itu dalam komputer (This PC), atau One Drive.

4.Save.

Pada menu file microsoft excel, ikon atau perintah save memiliki fungsi untuk menyimpan dokumen ke dalam komputer maupun OneDrive. Shortcutnya: ctrl & S.

5.. Save As.

Perintah save as pada menu file microsoft excel ber-fungsi untuk menyimpan dokumen yang sudah ada dengan nama yang berbeda. Misalkan: Formulir 1 (Sudah ada), menjadi Formulir 2 (Nama lain yang sudah diisi). Sehingga terdapat 2 berkas yang berbeda. hal ini biasanya digunakan untuk sebagai backup antara file awal, dan yang sudah diedit, atau diperbarui.

6.Print.

Group Print pada menu file microsoft excel ber-fungsi untuk mengatur dokumen sebelum dicetak, meliputi jenis printer, halaman, margins, ukuran kertas, dan jumlah salinan. Akan muncul tab preview di samping kanan untuk mengetahui tampilan dokumen jika dicetak.

7.Share.

Share adalah ikon dalam menu File microsoft excel yang memilki fungsi membagikan dokumen ke orang lain melalui email, maupun OneDrive. Terdapat 2 opsi sharing anatar lain:

share pada menu file excel befungsi

Share with People.

Membagikan project yang dibuat ke orang lain yang terhubung dalam lingkup OneDrive.

Email.

Mengirimkan salinan dokumen melalui surat email, dalam bentuk lampiran.

Send a Link.

Menyispkan sebuah link yang berisi dokumen excel, untuk kebutuhan pengerjaan dokumen dengan banyak orang

8.Export.

Fungsi dari Export pada menu file microsoft excel adalah untuk mengubah format/type dokumen ke format lain seperti txt. Ada 2 sub bagian export antara lain:

export pada menu file

Create PDF/XPS Document.

Untuk membuat file menjadi tipe PDF/XPS.

Change File Type.

Mengubah lembar kerja ke dalam tipe dokumen lain seperti excel 97-2003, Spreadsheet, Txt, CVS dll dengan Save maupun Save As.

9.Close.

Fungsi utama Close yang merupakan bagian dari menu file adalah menutup dokumen yang sedang dikerjakan, tanpa mengakhir software microsoft excel.

10.Account.

Tab account yang dapat diakses melalui menu file excel, ber-fungsi untuk mengelola pengaturan akun, tema sistem, login office, dan informasi dari product office, seperti update terakhir, dan informasi software yang digunakan.

11.Options.

Fitur Options adalah sebuah fitur yang terdapat pada menu file excel, yang terdiri atas: General, Formulas, Data, Proofing, Save, Language, Ease of Acces, Advanced, Customize Ribbon, Quick Acces Toolbar, Add-ins, Trust Center, yang memilliki fungsi untuk mengatur toolbar seperti menyembunyikan, dan memunculkan Ribbon.

32. Bagaimana Cara menampilkan menu-menu New, Open, Save, Save As, Print, Prepare, Send, Publish dan Close?


Jawaban:

Yaitu, dengan cara klik file pada bagian pojok kiri atas, dengan mengklik tesebut kamu akan dibawa atau ditampilkan menu-menu tersebut.

Guna dari menu-menu tersebut ialah, salah satunya save as yaitu untuk menyimpan dokumen kita yang sudah selesai maupun belum ke dalam file yang sudah disiapkan dan kemudian akan tersimpan ke file explorer kamu.

Penjelasan:

Microsoft Word adalah sebuah program yang merupakan bagian dari paket instalasi Microsoft Office, berfungsi sebagai perangkat lunak pengolah kata meliputi membuat, mengedit, dan memformat dokumen.

Fungsi menu-menu tersebut ialah:

New yaitu, untuk membuat dokumen baru dan di menu tersebut akan ada dua pilihan yaitu blank document dan single spaced (blank)Open yaitu, untuk membuka file yang sudah kamu simpan di file explore jika kamu ingin mengedit kembali dokumen tersebut.Save yaitu, menyimpan dokumen tersebut, namun menu ini menyimpan jika dokumen tersebut sudah dibuatkan folder di file explorer.Save As yaitu, menu ini digunakan untuk menyimpan dokumen kita yang sudah selsai ke dalam file explorer kita.Print yaitu, menu jika dokumen kita akan dicetak kedalam sebuah kertas maupun bidang lainya.Send/Share yaitu, menu yang digunakan untuk membagikan dokumen kita ke orang lain melalui e-mail.Publish yaitu, menu untuk menerbitkan dokumen yang telah dibuat ke aplikasi Power BI.Close yaitu, menu yang digunakan untuk menutup file atau kembali ke halaman utama tampa diakhiri dengan save ataupun lainya.
